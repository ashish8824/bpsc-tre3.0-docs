## 1. THE BIG PICTURE — WHAT IS ANALOGY?

**Analogy** means **similarity of relationship**. In an analogy question, two words (or numbers or letters) are given in a specific relationship. You must identify THAT SAME RELATIONSHIP and apply it to find the missing word.

> **Golden Rule:** The RELATIONSHIP between the pair matters — NOT the individual words themselves.

---

### Why Analogy Is Important in Exams:

```
Tested in:
→ SSC CGL / CHSL / MTS     → 3–5 questions
→ UPSC CSAT                → 2–3 questions
→ IBPS PO / Clerk          → 2–4 questions
→ RRB NTPC / Group D       → 3–5 questions
→ State PSC Exams          → 4–6 questions
→ Railways                 → 3–5 questions
→ Defence (NDA/CDS)        → 3–4 questions

Time per question: 20–40 seconds (fastest topic when mastered!)
```

---

### The Core Format:

```
FORMAT 1 (Word Pair):
    A : B :: C : ?
    "A is to B as C is to ?"
    Find what D should be such that A:B = C:D

FORMAT 2 (Choose the Pair):
    A : B :: ? : ?
    "A is to B as which pair has the same relationship?"

FORMAT 3 (Odd One Out):
    Which pair does NOT follow the same relationship?

FORMAT 4 (Number/Letter Analogy):
    7 : 49 :: 5 : ?
    AB : CD :: EF : ?
```

---

### How the Relationship Works:

```
Example: Doctor : Hospital :: Teacher : ?

Step 1: Identify relationship in Doctor : Hospital
        → "A doctor WORKS IN a hospital"
        → Relationship: "Person : Place of work"

Step 2: Apply same relationship to Teacher : ?
        → "A teacher WORKS IN a ___"
        → Answer: School

WRONG APPROACH: Thinking "Doctor and Hospital are both related to health"
               and then looking for something health-related for Teacher.

RIGHT APPROACH: Find the PRECISE relationship (works in), then apply.
```

## 2. HOW TO READ AN ANALOGY QUESTION

---

### The Standard Reading Format:

```
"A : B :: C : D"
 ↑   ↑     ↑   ↑
 1st 2nd   3rd 4th
word word  word word

Read as: "A is to B AS C is to D"

The relationship between A and B = The relationship between C and D
```

---

### Key Principles:

```
PRINCIPLE 1 — ORDER MATTERS:
"Doctor : Hospital" ≠ "Hospital : Doctor"
"Doctor works in Hospital" is the relationship.
If given "Hospital : Doctor", the relationship is REVERSED: "where doctors work"

PRINCIPLE 2 — IDENTIFY THE EXACT TYPE OF RELATIONSHIP:
Don't be vague. "Doctor and Hospital are connected" is too vague.
Be precise: "Person → Place of work" or "Worker → Workplace"

PRINCIPLE 3 — TEST YOUR RELATIONSHIP WITH THE THIRD WORD:
Once you identify the relationship, test it with word 3.
If it gives a unique, logical answer = you found the right relationship.

PRINCIPLE 4 — ELIMINATE WRONG OPTIONS:
Use the options to narrow down.
If multiple options seem valid, go back and make your relationship MORE specific.
```

---

### The Three-Step Method:

```
STEP 1: IDENTIFY — What is the relationship between words 1 and 2?
STEP 2: APPLY — Apply that exact relationship starting from word 3.
STEP 3: VERIFY — Check your answer against all options.
```

## 3. CATEGORY 1 — RELATIONSHIP BASED ANALOGIES

> The most common category. The relationship between two words/things is the key.

---

### Sub-Type 1A — Individual to Group (Member : Class):

```
Dog : Canine :: Cat : ?
→ Dog belongs to the Canine family
→ Cat belongs to the Feline family
Answer: Feline

Sparrow : Bird :: Rose : ?
→ Sparrow IS A Bird (member of group)
→ Rose IS A Flower
Answer: Flower

Wheat : Cereal :: Potato : ?
→ Wheat is a type of Cereal
→ Potato is a type of Vegetable
Answer: Vegetable
```

---

### Sub-Type 1B — General to Specific (Class : Member):

```
Metal : Iron :: Fuel : ?
→ Iron is a type of Metal
→ Petrol is a type of Fuel
Answer: Petrol

Furniture : Chair :: Clothing : ?
→ Chair is a type of Furniture
→ Shirt/Dress is a type of Clothing
Answer: Shirt (or Dress, depending on options)
```

---

### Sub-Type 1C — Synonym/Antonym Relationships:

```
SYNONYMS:
Happy : Joyful :: Sad : ?
→ Happy and Joyful mean the same
→ Sad and Miserable mean the same
Answer: Miserable

ANTONYMS:
Hot : Cold :: Day : ?
→ Hot is opposite of Cold
→ Day is opposite of Night
Answer: Night

North : South :: East : ?
→ North is opposite of South
→ East is opposite of West
Answer: West
```

---

### Sub-Type 1D — Symbol / Representation:

```
Dove : Peace :: Red : ?
→ Dove is a symbol/representation of Peace
→ Red is a symbol of Danger/Anger/Stop
Answer: Danger

White Flag : Surrender :: Green Light : ?
→ White Flag symbolizes Surrender
→ Green Light symbolizes Go/Permission
Answer: Go

Skull : Death :: Heart : ?
→ Skull symbolizes Death
→ Heart symbolizes Love
Answer: Love
```

---

### Sub-Type 1E — Gender Relationships:

```
King : Queen :: Man : ?
→ King is male, Queen is female counterpart
→ Man is male, Woman is female counterpart
Answer: Woman

Bull : Cow :: Drake : ?
→ Bull (male) : Cow (female) — both cattle
→ Drake (male duck) : Duck (female duck)
Answer: Duck

Lion : Lioness :: Tiger : ?
→ Lion (male) : Lioness (female)
→ Tiger (male) : Tigress (female)
Answer: Tigress

Ram : Ewe :: Gander : ?
→ Ram (male sheep) : Ewe (female sheep)
→ Gander (male goose) : Goose (female goose)
Answer: Goose
```

## 5. CATEGORY 3 — PART AND WHOLE

---

### The Pattern:

```
Part : Whole :: Part : Whole

Chapter : Book :: Page : ?
→ Chapter is a PART of a Book
→ Page is a PART of a... (what contains pages?) Book!
Wait: Page is part of Chapter, and Chapter is part of Book.
Re-read: Chapter : Book = Page : ?
→ Chapter is a part of Book
→ Page is a part of Book? Or page is part of chapter?
→ More precisely: Page:Chapter (as Chapter:Book shows unit:container)
Answer: Chapter

Leaf : Tree :: Petal : ?
→ Leaf is part of a Tree
→ Petal is part of a Flower
Answer: Flower

Wheel : Car :: Propeller : ?
→ Wheel is a component of Car
→ Propeller is a component of Aeroplane/Ship
Answer: Aeroplane

Sentence : Paragraph :: Paragraph : ?
→ Sentence is part of Paragraph
→ Paragraph is part of Essay/Chapter
Answer: Essay
```

## 6. CATEGORY 4 — CAUSE AND EFFECT

---

### The Pattern:

```
Cause : Effect :: Cause : Effect

Fire : Smoke :: Sun : ?
→ Fire causes Smoke (fire → smoke)
→ Sun causes Light/Heat
Answer: Light or Heat

Virus : Disease :: Flood : ?
→ Virus causes Disease
→ Flood causes Destruction
Answer: Destruction

Study : Knowledge :: Practice : ?
→ Study leads to Knowledge
→ Practice leads to Perfection/Skill
Answer: Perfection

Rain : Flood :: Wind : ?
→ Heavy rain causes Flood
→ Heavy wind causes Storm
Answer: Storm

## 7. CATEGORY 5 — DEGREE / INTENSITY

---

### The Pattern:

```
Two words that have the SAME concept but DIFFERENT intensities.

Warm : Hot :: Cool : ?
→ Hot is a more intense version of Warm
→ Cold is a more intense version of Cool
Answer: Cold

Smile : Laugh :: Annoy : ?
→ Laugh is a more intense version of Smile
→ Anger/Rage is a more intense version of Annoy
Answer: Anger

Breeze : Gale :: Stream : ?
→ Gale is a more intense version of Breeze (wind)
→ River/Torrent is a more intense version of Stream (water flow)
Answer: River

Dislike : Hate :: Like : ?
→ Hate is an extreme form of Dislike
→ Love is an extreme form of Like
Answer: Love

## 15. MASTER TRICKS & SHORTCUTS

---

### 🚀 TRICK 1 — The "Bridge Sentence" Technique (Most Important!)

```
CREATE A SPECIFIC BRIDGE SENTENCE connecting Word 1 and Word 2.
Then use that SAME sentence structure for Word 3 to find Word 4.

Example: Pen : Ink :: Car : ?

VAGUE bridge: "Pen and Ink are related" → doesn't help
SPECIFIC bridge: "Pen uses Ink to function"
                  "Car uses _____ to function" → Petrol/Fuel
Answer: Fuel

Example: Goat : Wool :: Silkworm : ?

Bridge: "Wool is produced FROM a Goat"
        "_____ is produced from a Silkworm" → Silk
Answer: Silk
```

---

### 🚀 TRICK 2 — Direction Matters!

```
"A is to B" is NOT the same as "B is to A"

Library : Book :: Gallery : ?
Bridge: "Library is a place that CONTAINS Books"
→ "Gallery is a place that contains ___" → Paintings
Answer: Paintings ✓

If reversed to: Book : Library :: Painting : ?
Bridge: "Books are KEPT IN Libraries"
→ "Paintings are kept in ___" → Gallery
Answer: Gallery ✓

Same answer different path — but direction must be CONSISTENT!
```

---

### 🚀 TRICK 3 — The Elimination Method

```
When you can't figure out the relationship:
1. Look at the answer options.
2. For each option, test: "Does this create the same relationship as A:B?"
3. Eliminate options that create DIFFERENT relationships.
4. The one that creates IDENTICAL relationship = correct answer.

Example: Bird : Nest :: Man : ?
Options: (A) Den  (B) House  (C) Hole  (D) Stable

Test each:
A→Den: "Bird lives in Nest, Man lives in Den" → Den is for foxes, not men ✗
B→House: "Bird lives in Nest, Man lives in House" ✓
C→Hole: Doesn't apply ✗
D→Stable: For horses ✗
Answer: B (House)
```

---

### 🚀 TRICK 4 — The "Most Specific Relationship" Rule

```
When multiple answers seem possible, choose the MOST SPECIFIC relationship.

Example: Doctor : Hospital :: Pilot : ?
Options: (A) Aeroplane  (B) Airport  (C) Sky  (D) Cockpit

All seem valid, but:
"Doctor WORKS IN a Hospital (as workplace)"
"Pilot works in Aeroplane" (Aeroplane is the tool, not workplace)
"Pilot works AT Airport" ← most similar to Hospital (place of work)

Answer: (B) Airport (place of work = most specific match)

HOWEVER: In some questions:
"Sailor : Ship :: Pilot : ?"
Here the focus is on the VEHICLE, not the workplace.
→ Answer: Aeroplane

Context and options determine which relationship is most specific!
```

---

### 🚀 TRICK 5 — Number Analogy Quick Tests

```
When you see number analogy, try these in ORDER:
1. Is it n² (square)?       4:16, 5:25, 6:36
2. Is it n³ (cube)?         2:8, 3:27, 4:64
3. Is it n×(n+1)?           3:12? No. 3:12 = 3×4 ✓
4. Is it n+(n+k)?            5:8 → +3, 7:10 → +3 ✓ (arithmetic)
5. Is it n²+1 or n²-1?      4:17 → 4²+1=17 ✓
6. Is it reversed digits?   13:31, 24:42 ✓
7. Is it digit sum/product? 23: 2+3=5, 14: 1+4=5

CHECK: Does the same operation give the answer you expect from options?
```

---

### 🚀 TRICK 6 — Letter Analogy Quick Method

```
STEP 1: Write the position numbers of all letters.
STEP 2: Find the pattern between positions.
STEP 3: Apply to find the answer letter positions.
STEP 4: Convert positions back to letters.

Example: BEH : CFI :: KNQ : ?

B(2) E(5) H(8) : C(3) F(6) I(9)
Pattern: Each letter in second group = each in first + 1
K(11) N(14) Q(17) → L(12) O(15) R(18) → LOR
Answer: LOR
```

---

### 🚀 TRICK 7 — "Goes With" vs "Opposite Of" Check

```
Before solving, quickly determine:
Is the relationship "similar/same category"?
OR is it "opposite/contrasting"?

If SIMILAR: "Pen : Pencil :: Chalk : ?"
→ Both writing instruments → Find another writing instrument
→ Crayon, Marker, etc.

If OPPOSITE/CONTRAST: "Day : Night :: Awake : ?"
→ Day is opposite of Night → Awake is opposite of Asleep
→ Answer: Asleep

Confusing these two leads to completely wrong answers!
```

---

### 🚀 TRICK 8 — Group the Options First

```
Before you even look at the question deeply:
SCAN the options and group them by type.

Example Options: (A) Dog  (B) Meow  (C) Bark  (D) Puppy

If the question is Cat : Kitten :: Dog : ?
→ A is an animal name ✗ (not the same as kitten relationship)
→ B is a sound ✗
→ C is a sound ✗
→ D is a young one ✓ (same as kitten)
Answer: D

Grouping options by type eliminates wrong-category options instantly.
```

---

### 🚀 TRICK 9 — The "Consistent Direction" Rule for Letters

```
In letter analogies: ALWAYS check if the pattern is:
(a) Same direction: A→B means go +n letters
(b) Opposite direction: A→Z (reverse pattern)
(c) Mirror: ACE→ZXV (each letter reversed)

BEH : KNQ → B(2)E(5)H(8) : K(11)N(14)Q(17) → each +9
Apply to CFI: C(3)F(6)I(9) → L(12)O(15)R(18) = LOR

Don't mix directions unless the pattern itself alternates!
```

## 16. STEP-BY-STEP ATTACK STRATEGY

---

### The Universal 5-Step Method:

```
STEP 1: IDENTIFY — Read Words 1 and 2 carefully.
         What KIND of relationship do they have?
         (Tool-Use? Part-Whole? Cause-Effect? Synonym? etc.)

STEP 2: BRIDGE — Form a precise "bridge sentence":
         "___ [verb] ___ [specific description]"
         Be as SPECIFIC as possible in the bridge sentence.

STEP 3: APPLY — Take Word 3 and apply the SAME bridge sentence.
         "Word 3 [same verb] ___ [same specific description]"

STEP 4: VERIFY — Check your answer against all options.
         If your answer matches an option → done!
         If not → refine your bridge sentence and try again.

STEP 5: ELIMINATE — If still unsure, eliminate obviously wrong options.
         Pick the best remaining option.
```

---

### Decision Tree for Analogy Type:

```
WHAT IS THE RELATIONSHIP?
         │
         ├─ Are they the same kind of thing? → Synonym / Group membership
         │
         ├─ Are they opposites? → Antonym / Contrast relationship
         │
         ├─ Is one a part of the other? → Part-Whole relationship
         │
         ├─ Does one cause the other? → Cause-Effect relationship
         │
         ├─ Does one USE the other? → Tool-Use / Function relationship
         │
         ├─ Does one MAKE the other? → Worker-Product relationship
         │
         ├─ Is one more intense than the other? → Degree relationship
         │
         ├─ Is one the young of the other? → Animal young relationship
         │
         ├─ Is one the sound of the other? → Animal sound relationship
         │
         ├─ Is one the home of the other? → Animal dwelling relationship
         │
         ├─ Are they numbers? → Mathematical operation relationship
         │
         └─ Are they letters? → Positional/Alphabetical relationship
```

## 17. COMMON MISTAKE ALERTS

---

### ⚠️ MISTAKE 1 — Too Vague Bridge Sentence

```
Wrong: "Pen and Ink are related to writing"
Right: "A Pen uses Ink (to write)" — specific and directional!

Wrong bridge → wrong answer. Always be PRECISE.
```

---

### ⚠️ MISTAKE 2 — Ignoring the Direction of Relationship

```
Wrong: Treating "A : B" the same as "B : A"
Right: "Doctor : Hospital" = Doctor works IN Hospital
       "Hospital : Doctor" = Hospital employs/contains Doctor
These have DIFFERENT directions!

Always maintain the SAME DIRECTION (A→B same as C→D).
```

---

### ⚠️ MISTAKE 3 — Choosing the Most Common (Not Most Relevant) Option

```
Example: Cup : Saucer :: ? : ?
Options: (A) Plate:Food  (B) Book:Pen  (C) Lock:Key  (D) Table:Chair

"Cup and Saucer go together (paired items)" → Plate:Food? No, they're not paired.
Lock:Key → paired items that work together ✓

Don't pick the most FAMILIAR option — pick the most RELEVANT relationship.
```

---

### ⚠️ MISTAKE 4 — Number Analogy Calculation Errors

```
Always double-check your math!
4 : 16 = 4² = 16 ✓ but also 4 × 4 = 16 ✓
6 : 36 = 6² = 36 ✓ 

Test BOTH words with your proposed rule.
If rule works for word 1 but not word 2 → wrong rule.
```

---

### ⚠️ MISTAKE 5 — Confusing "Similar" with "Same"

```
Tiger : Cub is NOT the same relationship as Dog : Puppy.
Actually they ARE the same: both are (Animal : Young one).

But:
Tiger : Cat is NOT the same as Dog : Cat.
Tiger : Cat = predator-prey (or big cat : small cat)
Dog : Cat = common pets (or natural enemies)

The CATEGORY of relationship must match, not just surface similarity.
```

---

### ⚠️ MISTAKE 6 — Letter Analogy Position Errors

```
Wrong: Not checking ALL letters in the group, only the first.
Right: Check EVERY letter in the group against the pattern.

ACI: A(1), C(3), I(9) — gap between A→C = 2, C→I = 6. Not consistent!
Actually: A=1, C=3 (+2), I=9 (+6)? That's increasing gaps.
OR: ACI = positions 1,3,9 = 1, 1×3, 3×3 (multiply by 3)?

Always verify EVERY position before concluding the pattern.
```

---

### ⚠️ MISTAKE 7 — "Young One" vs "Female" Confusion

```
Cow is the FEMALE of Bull (not the young one).
Calf is the YOUNG ONE of Cow.

Bull : Cow = Male : Female (gender pair)
Cow : Calf = Adult : Young one

These are DIFFERENT relationships. Don't mix them up!
```

---

### ⚠️ MISTAKE 8 — Over-Relying on One Type

```
Not all analogies about animals are "Animal : Sound."
"Eagle : Eyrie" = Animal : Dwelling
"Eagle : Eaglet" = Animal : Young one
"Eagle : Sky" = Animal : Habitat

Read carefully which aspect of the animal is the focus.
```

## 🏆 TOP 10 EXAM TIPS

1. **The Bridge Sentence is everything** — form a precise, directional sentence connecting words 1 and 2. "A [does/is/has/causes/uses] B in a specific way." Then apply that exact sentence to word 3. Vague bridges lead to wrong answers every time.

2. **Direction matters absolutely** — "Doctor : Hospital" means doctor works IN hospital. If the pair was "Hospital : Doctor," the relationship is reversed. Always maintain consistent direction from words 1→2 and words 3→4.

3. **For number analogies, try square first** — over 40% of number analogy questions in competitive exams use n² (square). If that doesn't work, try n³, then n(n+1), then arithmetic addition. This order-of-testing saves time.

4. **Memorize EJOTY for letter analogies** — E=5, J=10, O=15, T=20, Y=25. From these anchors you can find any letter's position in under 2 seconds. This is the single most important letter analogy shortcut.

5. **"Pony" is NOT the young of horse** — Pony is a small breed. "Foal" is the young of a horse. This is one of the most tested traps. Similarly: Sydney is NOT Australia's capital (Canberra is). Know your exceptions!

6. **Animal sound/dwelling/young one tables are guaranteed marks** — these 3 tables appear in virtually every exam. Memorize them fully and you'll never lose marks on these sub-types.

7. **When options are confusing, use the elimination method** — test each option as an answer. The one that creates the EXACT SAME relationship as words 1:2 = correct answer. Others will create different (even if related) relationships.

8. **Look for the "odd one out" in paired-relationship questions** — when checking if a relationship matches, check the TYPE (tool:use vs tool:maker) AND the DIRECTION (A causes B vs B causes A). These two criteria together help identify the odd pair quickly.

9. **Build a general knowledge foundation** — author:book pairs, country:language, artist:painting, scientist:invention — these require knowledge, not just logic. Maintain a running list of these associations from every practice session.

10. **Practice "most specific" relationship identification** — when multiple answers seem valid (e.g., Pilot could relate to Airport OR Aeroplane), the most specific and direct relationship wins. Ask: "Is this the PRIMARY, DIRECT relationship or just an associated one?" Primary/direct = correct answer.
