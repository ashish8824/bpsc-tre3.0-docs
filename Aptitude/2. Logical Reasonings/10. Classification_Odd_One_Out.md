## 1. THE BIG PICTURE — WHAT IS CLASSIFICATION?

**Classification** (also called **Odd One Out**) questions give you a group of items and ask you to identify which one does NOT belong to the group.

> **Golden Rule:** Three or more items share a COMMON property. One item does NOT share that property. Find that ONE item.

---

### Why It's Important in Exams:

```
Tested in:
→ SSC CGL / CHSL / MTS    → 3–5 questions
→ RRB NTPC / Group D      → 3–5 questions
→ UPSC CSAT               → 2–3 questions
→ IBPS PO / Clerk         → 2–4 questions
→ State PSC Exams         → 4–6 questions
→ Defence (NDA/CDS)       → 3–5 questions
→ Railways                → 4–6 questions

Average time per question: 15–30 seconds
One of the QUICKEST topics to score marks!
```

---

### The Core Logic:

```
Given: Apple, Mango, Potato, Grape

Apple  = Fruit ✓
Mango  = Fruit ✓
Potato = Vegetable ✗ ← ODD ONE OUT
Grape  = Fruit ✓

Three are fruits. Potato is NOT a fruit.
Answer: Potato
```

---

### The Challenge — Multiple Valid Groupings:

```
⚠️ IMPORTANT: Sometimes more than one grouping is possible.
You must find the MOST OBVIOUS and STRONGEST grouping.

Example: 17, 19, 23, 25, 29

All are odd numbers. But:
17, 19, 23, 29 = PRIME numbers
25 = NOT prime (25 = 5×5)

The MORE SPECIFIC grouping (prime numbers) is the intended answer.
Answer: 25

Rule: Always look for the MOST SPECIFIC common property.
```

## 2. HOW TO THINK ABOUT CLASSIFICATION

---

### The Three-Step Framework:

```
STEP 1: SCAN all items quickly.
         Get a feel for what categories they could belong to.

STEP 2: FIND THE MAJORITY GROUP.
         What do 3 out of 4 (or more) items have in common?

STEP 3: IDENTIFY THE ODD ONE.
         The item that does NOT fit the majority group = answer.

ALWAYS CHECK: Is there a more specific grouping that makes the classification cleaner?
```

---

### Types of Common Properties to Look For:

```
CATEGORY TYPE      │  EXAMPLE
───────────────────┼──────────────────────────────────────────
Semantic/Meaning   │  Same category (fruits, animals, tools...)
Mathematical       │  Odd/Even, Prime, Perfect square, Multiple...
Alphabetical       │  Vowels, Consonants, Same-position pattern...
GK Based           │  Same country, same science branch...
Structural         │  Number of syllables, letters, spelling...
Positional         │  Alphabet positions, number patterns...
Relationship       │  All synonyms, all antonyms, all prefixed...
```

### The Pattern:

```

## 8. TYPE 6 — ANALOGY-BASED CLASSIFICATION

> Given pairs that share a relationship. Find the pair that has a DIFFERENT relationship.

---

### The Pattern:

```
All pairs have the same relationship EXCEPT one.

EXAMPLE 46:
(A) Pen : Write   (B) Knife : Cut
(C) Chair : Sit   (D) Hammer : Nail

→ Pen:Write = Tool:Function (pen is used to write) ✓
→ Knife:Cut = Tool:Function ✓
→ Chair:Sit = Object:Function ✓ (debatable)
→ Hammer:Nail = Tool:Material (hammer drives nails)

Wait: Chair:Sit — Chair is not a tool but a place to sit.
More specifically:
Pen writes ✓, Knife cuts ✓, Hammer nails ✓ — all tools performing active actions.
Chair:Sit — chair doesn't DO the sitting; person sits on it ✗ (passive)
Answer: (C) Chair : Sit
```

---

### Working Through Analogy-Based Classification:

```
EXAMPLE 47:
(A) Doctor : Hospital   (B) Teacher : School
(C) Artist : Museum     (D) Judge : Court

→ Doctor WORKS IN Hospital ✓
→ Teacher WORKS IN School ✓
→ Artist WORKS IN Museum? No! Artists work in studios.
  Museum is where art is DISPLAYED, not where artists work ✗
→ Judge WORKS IN Court ✓
Answer: (C) Artist : Museum

EXAMPLE 48:
(A) Bird : Nest   (B) Dog : Kennel
(C) Bee : Hive    (D) Cat : Meow

→ Bird:Nest = Animal:Dwelling ✓
→ Dog:Kennel = Animal:Dwelling ✓
→ Bee:Hive = Animal:Dwelling ✓
→ Cat:Meow = Animal:Sound ✗
Answer: (D) Cat : Meow
```

## 10. MASTER TRICKS & SHORTCUTS

---

### 🚀 TRICK 1 — The Majority Rule

```
The ODD ONE OUT = the item that does NOT share the property of the MAJORITY.

If 3 out of 4 items are fruits and 1 is a vegetable → vegetable is odd.
If 3 out of 4 are prime and 1 is not → non-prime is odd.
If 3 out of 4 are vowels and 1 is consonant → consonant is odd.

ALWAYS find what 3 (or more) share, then the remaining one is odd.
```

---

### 🚀 TRICK 2 — Try Multiple Groupings for Numbers

```
For number classification, always test multiple properties:
1. Odd/Even
2. Prime/Composite
3. Perfect square/cube
4. Multiple of n
5. Sum of digits
6. n² + constant
7. Fibonacci

The MOST SPECIFIC grouping where exactly one item doesn't fit = answer.

Example: 4, 9, 16, 20, 25
→ Even/Odd: 4(E),9(O),16(E),20(E),25(O) → no majority
→ Perfect squares: 4=2², 9=3², 16=4², 20=not square, 25=5²
→ Perfect square grouping: 4,9,16,25 are squares; 20 is not!
Answer: 20
```

---

### 🚀 TRICK 3 — The Alphabet Position Test for Letters

```
When given letter groups:
1. Convert each letter to its position number (A=1...Z=26)
2. Look for mathematical patterns in the positions

BEH → B(2), E(5), H(8): each +3 ✓
CFI → C(3), F(6), I(9): each +3 ✓
DGJ → D(4), G(7), J(10): each +3 ✓
EHK → E(5), H(8), K(11): each +3 ✓
GJN → G(7), J(10), N(14): 7,10,14 → +3,+4 ✗ (should be GJM: 7,10,13)
Answer: GJN is odd one out
```

---

### 🚀 TRICK 4 — Divisibility Quick Tests

```
DIVISIBILITY RULES (use for number classification):

Divisible by 2: Last digit even (0,2,4,6,8)
Divisible by 3: Sum of digits divisible by 3
Divisible by 4: Last 2 digits divisible by 4
Divisible by 5: Last digit 0 or 5
Divisible by 6: Divisible by both 2 and 3
Divisible by 8: Last 3 digits divisible by 8
Divisible by 9: Sum of digits divisible by 9
Divisible by 11: Alternating digit sum divisible by 11

Example: 121, 132, 143, 154
→ 121: div by 11? 1-2+1=0 ✓
→ 132: div by 11? 1-3+2=0 ✓
→ 143: div by 11? 1-4+3=0 ✓
→ 154: div by 11? 1-5+4=0 ✓
All divisible by 11 — look for another pattern.
121=11², 132=11×12, 143=11×13, 154=11×14 ← all × consecutive
Hmm, all are multiples of 11. Not helpful for odd-one-out.
Try: 121=11², others are not perfect squares.
Answer: 121 (only perfect square in this group — if that's the context)

OR: If context is "multiples of 11 that are perfect squares":
Only 121 is a perfect square among these → 121 is unique → depending on the question's perspective.
```

---

```
When word groups seem similar, check their STRUCTURE:

Prefixes to look for: un-, re-, pre-, anti-, dis-, mis-
Suffixes to look for: -tion, -ing, -ly, -ful, -ness, -er, -ist

Example: unhappy, unkind, unfair, careless
→ unhappy = un- prefix ✓
→ unkind = un- prefix ✓
→ unfair = un- prefix ✓
→ careless = -less suffix ✗ (no un- prefix)
Answer: careless

Example: Teacher, Writer, Painter, Artist
→ Teacher = -er suffix ✓

```

---

### 🚀 TRICK 6 — The "Two Valid Groups" Trap Awareness

```
⚠️ EXAM TRAP: Some questions have TWO seemingly valid groupings.
The INTENDED grouping is usually the MORE SPECIFIC one.

Example: 7, 14, 21, 29, 35
→ Multiples of 7: 7, 14, 21, 35 ✓ → 29 is odd ✗
→ Odd numbers: 7, 21, 29, 35 ✓ → 14 is even ✗

TWO valid groupings! Which is correct?
"Multiple of 7" is MORE SPECIFIC than "odd numbers"
→ Answer: 29 (not a multiple of 7)

RULE: The more specific/mathematical grouping takes precedence.
```

---

### 🚀 TRICK 7 — Check Position from Both Ends

```
For letter pairs and coded classifications:

If forward position doesn't give a clear pattern,
try REVERSE position (A=26, B=25... Z=1):

Or: use the SUM rule (A+Z=27, B+Y=27...)

Example: BX, CW, DV, FT
→ B(2)+X(24)=26, C(3)+W(23)=26, D(4)+V(22)=26, F(6)+T(20)=26
All sum to 26! Actually wait: B(2)+X(24)=26 ✓, C(3)+W(23)=26 ✓, D(4)+V(22)=26 ✓, F(6)+T(20)=26 ✓
All sum to 26! Where's the odd one?
In the question, maybe one pair sums to 27 or 25 — that would be odd.
If given: BX, CW, DV, EU
→ B+X=26, C+W=26, D+V=26, E+U=26 ✓
All 26, all valid! No odd one here.
If given: BX, CW, EV, DU
→ B(2)+X(24)=26 ✓
→ C(3)+W(23)=26 ✓
→ E(5)+V(22)=27 ✗ (sum ≠ 26)
→ D(4)+U(21)=25 ✗ (sum ≠ 26)
Two odd ones here — question would have only one.
```

### 🚀 TRICK 9 — The Number Relation Chain

```
For complex number classification, build a "relation chain":

Given: 6, 15, 21, 33, 35

Check: are all divisible by 3?
6÷3=2 ✓, 15÷3=5 ✓, 21÷3=7 ✓, 33÷3=11 ✓, 35÷3=11.67 ✗
Answer: 35 (not divisible by 3)

Alternative: check if they're products of two primes:
6=2×3 ✓, 15=3×5 ✓, 21=3×7 ✓, 33=3×11 ✓, 35=5×7 ✓
All are products of two primes. No odd one here from this property.
But 35 is not divisible by 3 while all others are → 35 is odd.
Answer: 35
```

### 🚀 TRICK 10 — The "Pure Type vs Mixed Type" Rule

```
When all items are of one TYPE except one:

Example:
(A) 2+3=5    (B) 4+5=9    (C) 6-4=2    (D) 3+4=7

→ A = Addition ✓
→ B = Addition ✓
→ C = Subtraction ✗
→ D = Addition ✓
Answer: C (uses subtraction, others use addition)

Example:
(A) 3²=9    (B) 4²=16    (C) 5²=25    (D) 6²=38

→ A = 3²=9 ✓
→ B = 4²=16 ✓
→ C = 5²=25 ✓
→ D = 6²=36, not 38 ✗
Answer: D (incorrect calculation → odd one out)
```

### 🚀 TRICK 11 — Cross-Category Elimination

```
When items seem to span multiple categories, look at what
MOST items have in common that ONE doesn't:

Example: Iron, Steel, Bronze, Copper, Brass

→ Iron = Metal (pure element) ✓
→ Steel = Alloy (iron + carbon) — NOT pure metal ✗
→ Bronze = Alloy (copper + tin) — NOT pure metal ✗
→ Copper = Metal (pure element) ✓
→ Brass = Alloy (copper + zinc) — NOT pure metal ✗

Three are alloys (Steel, Bronze, Brass), two are pure metals (Iron, Copper).
Majority = ALLOYS → odd ones = Iron or Copper

But which is odd? If question gives 4 options (three alloys + one pure), 
the pure metal is odd. If gives 4 options (three pure + one alloy), 
the alloy is odd. Context of the group matters!
```

### 🚀 TRICK 12 — Word Length Pattern Check

```
When all else fails, check:
1. Number of letters in each word
2. Number of syllables
3. Starting/ending letter

Example: Apple, Grape, Mango, Cherry
→ Apple = 5 letters ✓
→ Grape = 5 letters ✓
→ Mango = 5 letters ✓
→ Cherry = 6 letters ✗
Answer: Cherry (6 letters while others have 5)

Example: Big, Fat, Ugly, Old

→ Fat = 3 letters ✓

## 12. STEP-BY-STEP ATTACK STRATEGY

---

### The Universal Method:

```
STEP 1: READ all items once without analyzing.
         Get an overview of what type of items they are.

STEP 2: IDENTIFY the obvious category.
         What do most items belong to? (animals, fruits, numbers...)

STEP 3: TEST each item against the majority category.
         Which one doesn't fit?

STEP 4: IF STUCK — try alternative groupings:
         → Mathematical property (prime, square, multiple)
         → Structural property (letters, syllables, suffix)
         → Sub-category (wild vs domestic, odd vs even)
         → GK property (same country, same inventor)

STEP 5: VERIFY your answer.
         Does the majority group make sense without the odd one?
         Is there exactly ONE item that doesn't fit?

STEP 6: CHECK for traps.
         Is there a more specific grouping that gives a different answer?
         Which grouping is more specific? → Use that one.
```

---

### Decision Tree for Stuck Situations:

```
STUCK ON A QUESTION?
       │
       ├─ Are they WORDS?
       │      │
       │      ├─ Same category? (animals, tools, fruits)
       │      ├─ Same word structure? (letters, syllables)
       │      ├─ Same part of speech? (noun/verb/adjective)
       │      └─ Same prefix/suffix?
       │
       ├─ Are they NUMBERS?
       │      │
       │      ├─ Odd vs Even?
       │      ├─ Prime?
       │      ├─ Perfect square or cube?
       │      ├─ Multiple of n?
       │      ├─ Sum of digits?
       │      └─ Mathematical pattern? (n²+1, n(n+1), etc.)
       │
       └─ Are they LETTERS or CODES?
              │
              ├─ Vowel vs Consonant?
              ├─ Position in alphabet?
              ├─ Pattern within group (gaps, sums)?
              └─ Coded relationship (pair sums, reversals)?
```

## 13. COMMON MISTAKE ALERTS

---

### ⚠️ MISTAKE 1 — Using First Obvious Grouping Without Checking

```
Wrong: Accept the first grouping that comes to mind.
Right: Always check if there's a MORE SPECIFIC grouping.

Example: 4, 9, 25, 49, 35
First thought: All odd? 4 is even → 4 is odd one.
But MORE SPECIFIC: 4=2², 9=3², 25=5², 49=7² are perfect squares of primes.
35 = 5×7 = not a perfect square.
→ Correct answer: 35 (not 4)
```

---

### ⚠️ MISTAKE 2 — Tomato Trap (Botanical vs Culinary)

```
Botanically: Tomato, Cucumber, Capsicum = Fruits (contain seeds)
Culinarily: Tomato, Cucumber = Vegetables
Exam context: Usually culinary classification.

If given: Mango, Apple, Tomato, Potato
→ Exam usually treats Tomato as vegetable alongside Potato
→ Odd one: Mango or Apple (not Tomato)

BUT if given: Tomato, Cucumber, Capsicum, Carrot
→ First three are botanically fruits
→ Carrot is definitely a vegetable
→ Answer: Carrot

Read the context carefully!
```

---

### ⚠️ MISTAKE 3 — Pony vs Foal (Age vs Breed)

```
Pony = Small breed of horse (not a young horse!)
Foal = Young horse

If: Lion:Cub :: Horse:?
→ Answer should be FOAL (young one), not PONY

In classification: 
Puppy, Kitten, Foal, Pony
→ Puppy = young dog ✓
→ Kitten = young cat ✓
→ Foal = young horse ✓
→ Pony = breed of horse (NOT a young one) ✗
Answer: Pony
```

---

### ⚠️ MISTAKE 4 — "1 is not a prime number" Trap

```
1 is NOT a prime number! (Common exam trap)

If given: 1, 2, 3, 5, 7
→ 1 = NOT prime ✗
→ 2, 3, 5, 7 = Prime ✓
Answer: 1

Many students automatically think 1 is prime — it's NOT!
```

---

### ⚠️ MISTAKE 5 — Confusing Alloys with Pure Metals

```
ALLOYS (not pure metals):
Steel, Bronze, Brass, Solder, Duralumin, Gunmetal, Amalgam

PURE METALS:
Gold, Silver, Platinum, Iron, Copper, Aluminium, Zinc, Tin, Lead

If given: Gold, Silver, Iron, Steel
→ Gold = Pure metal ✓
→ Silver = Pure metal ✓
→ Iron = Pure metal ✓
→ Steel = Alloy ✗
Answer: Steel
```

---

### ⚠️ MISTAKE 6 — Planet vs Star vs Moon Confusion

```
PLANETS (8): Mercury, Venus, Earth, Mars, Jupiter, Saturn, Uranus, Neptune
STARS: Sun, Sirius, Betelgeuse (our Sun is a star!)
MOONS/SATELLITES: Moon (Earth's), Titan (Saturn's), etc.
DWARF PLANETS: Pluto (no longer a planet since 2006!)

If given: Earth, Mars, Moon, Jupiter
→ Earth = Planet ✓
→ Mars = Planet ✓
→ Moon = Natural satellite ✗
→ Jupiter = Planet ✓
Answer: Moon
```

---

### ⚠️ MISTAKE 7 — Confusing "Between" and "Except"

```
The question asks for the ODD ONE OUT (the one that does NOT belong).
Some students mistakenly identify what the majority IS instead of what the odd one IS.

Always re-read: "Find the ODD ONE OUT" = find what DOESN'T belong.
Not "find what belongs to the group."
```

---

### ⚠️ MISTAKE 8 — Missing the Sub-Category Level

```
Wrong: Grouping at too broad a level.

Example: Whale, Dolphin, Fish, Shark
→ All aquatic (broad level) → no odd one!
→ Whale, Dolphin = Mammals (breathe air!) ✓
→ Fish, Shark = Fish (breathe through gills) ✓
→ Now we see: Whale and Dolphin are NOT fish, they're marine mammals.
→ If majority is fish (Fish + Shark = 2, Whale + Dolphin = 2) → ambiguous!
→ But: Whale AND Dolphin are both mammals → Fish is odd (only 1) in that grouping.
Actually: 2 fish (Fish, Shark) and 2 mammals (Whale, Dolphin).
This would be given as one of 4 choices with context making it clear.
```

## 🏆 TOP 10 EXAM TIPS

1. **Always find the MAJORITY GROUP first** — look at what 3 (or more) of the items share before trying to identify the odd one. If you identify the odd one without knowing the majority property, you may be wrong about your reasoning.

2. **Use the "Most Specific Grouping" rule** — when two groupings are possible, the more specific/mathematical one is the intended answer. "Prime numbers" beats "odd numbers"; "perfect squares" beats "even numbers."

3. **Never skip the prime number check** — prime/composite classification is the single most tested numerical property in classification questions. Memorize primes up to 100 and key composites (9, 25, 49, 91, 121).

4. **1 is NOT prime — tattoo this on your brain** — this is the most common exam trap. 1 is neither prime nor composite; it's a unit. Any list with 1 and prime numbers = 1 is the odd one.

5. **Remember the "Pony trap" and "Tomato trap"** — Pony ≠ young horse (Foal is young horse). Tomato = botanical fruit (culinary vegetable). These two appear repeatedly in word classification questions.

6. **Use EJOTY for letter positions** — E=5, J=10, O=15, T=20, Y=25. This lets you find any letter's position in under 2 seconds. Convert all letters to numbers and look for the mathematical pattern.

7. **Check ALL properties before answering** — for number classification, systematically test: odd/even → prime → perfect square → perfect cube → multiple → digit sum. Don't stop at the first property that seems to work without verifying it's the strongest grouping.

8. **For word classification, check SUB-categories** — "all animals" is too broad. Ask: are they all wild? All domestic? All birds? All reptiles? The sub-category reveals the odd one that a broader category hides.

9. **The "two valid groups" trap** — if you find two possible odd ones, the intended answer is usually determined by whichever group of 3 is more specific or mathematically precise. If a number question gives you both an "even/odd" answer and a "prime/composite" answer, choose prime/composite.

10. **In analogy-based classification, check DIRECTION** — the relationship "A works IN B" is different from "B houses/contains A." Artist:Gallery fails because artists work in STUDIOS, not galleries (galleries display work, not create it). Always verify the exact direction and type of relationship.
