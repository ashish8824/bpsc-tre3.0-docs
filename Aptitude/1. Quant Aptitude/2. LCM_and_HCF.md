## 1. UNDERSTANDING FACTORS & MULTIPLES (FOUNDATION)

> Before learning HCF and LCM, you MUST be crystal clear on factors and multiples.

### 1.1 What is a Factor?

A **factor** of a number n is any number that divides n **completely** (no remainder).

```
Factors of 12:
  1 Г— 12 = 12   вң“
  2 Г— 6  = 12   вң“
  3 Г— 4  = 12   вң“
  So factors of 12 = {1, 2, 3, 4, 6, 12}
```

**Key Rules about Factors:**
- **1 is a factor of every number.**
- **Every number is a factor of itself.**
- **A factor is always вүӨ the number.**
- A number **n** always has a **finite** number of factors.

### 1.2 What is a Multiple?

A **multiple** of n is obtained by multiplying n by any natural number.

```
Multiples of 6:
  6Г—1=6,  6Г—2=12,  6Г—3=18,  6Г—4=24,  6Г—5=30 ...
  Multiples of 6 = {6, 12, 18, 24, 30, ...}  вҶ’ INFINITE
```

**Key Rules about Multiples:**
- **Every number is a multiple of itself.**
- **A multiple is always вүҘ the number.**
- Every number has **infinite** multiples.

### 1.3 Factor vs Multiple (Side by Side)

| | Factor | Multiple |
|---|---|---|
| Relation | Divides the number | Is divided by the number |
| Size | Always вүӨ number | Always вүҘ number |
| Count | Finite | Infinite |
| Example (for 12) | 1,2,3,4,6,12 | 12,24,36,48... |

## 2. HCF вҖ” HIGHEST COMMON FACTOR

### 2.1 What is HCF?

> **HCF** (also called **GCD** вҖ” Greatest Common Divisor) of two or more numbers is the **LARGEST number that divides ALL the given numbers exactly** (without remainder).

```
Find HCF of 18 and 24:

  Factors of 18 = {1, 2, 3, 6, 9, 18}
  Factors of 24 = {1, 2, 3, 4, 6, 8, 12, 24}

  Common Factors = {1, 2, 3, 6}

  HIGHEST Common Factor = 6

  вҲҙ HCF(18, 24) = 6
```

### 2.2 Other Names for HCF
- GCD (Greatest Common Divisor)
- GCF (Greatest Common Factor)
- HCF is the term used in Indian exams (SSC, CAT, Bank PO, etc.)

### 2.3 Properties of HCF

| Property | Detail |
|----------|--------|
| HCF вүӨ Smallest number | Always |
| HCF divides each number | Completely |
| HCF divides LCM | Always |
| HCF of co-prime numbers | Always = 1 |
| If one number divides the other | HCF = Smaller number |
| HCF of (n, n) | = n |
| HCF of (1, n) | = 1 |

**Examples:**
- HCF(7, 28) = 7 вҶҗ because 7 divides 28 completely
- HCF(17, 31) = 1 вҶҗ both are primes, no common factor
- HCF(15, 15) = 15 вҶҗ same numbers

## 3. LCM вҖ” LOWEST COMMON MULTIPLE

### 3.1 What is LCM?

> **LCM** of two or more numbers is the **SMALLEST number that is divisible by ALL the given numbers**.

```
Find LCM of 4 and 6:

  Multiples of 4 = {4, 8,  [12], 16, 20, [24], 28, ...}
  Multiples of 6 = {6, [12], 18, [24], 30, ...}

  Common Multiples = {12, 24, 36, ...}

  LOWEST Common Multiple = 12

  вҲҙ LCM(4, 6) = 12
```

### 3.2 Properties of LCM

| Property | Detail |
|----------|--------|
| LCM вүҘ Largest number | Always |
| Each number divides LCM | Completely |
| HCF divides LCM | Always |
| LCM of co-prime numbers | = Product of the numbers |
| If one number divides the other | LCM = Larger number |
| LCM of (n, n) | = n |
| LCM of (1, n) | = n |

**Examples:**
- LCM(7, 28) = 28 вҶҗ because 7 divides 28, so 28 is already the LCM
- LCM(4, 9) = 36 вҶҗ co-primes (HCF=1), so LCM = 4 Г— 9 = 36
- LCM(15, 15) = 15 вҶҗ same numbers

## 4. METHODS TO FIND HCF

### Method 1: Listing Factors (Basic вҖ” for small numbers only)

```
HCF(12, 18):
  Factors of 12 = {1, 2, 3, 4, 6, 12}
  Factors of 18 = {1, 2, 3, 6, 9, 18}
  Common = {1, 2, 3, 6}
  HCF = 6
```

### Method 2: Prime Factorization вӯҗ (Most Reliable)

**Steps:**
1. Write prime factorization of each number.
2. Identify **common** prime factors.
3. Take each common factor with the **LOWEST (minimum) power**.
4. Multiply them.

```
Example 1: HCF(60, 90)
  60 = 2ВІ Г— 3 Г— 5
  90 = 2 Г— 3ВІ Г— 5
  Common primes: 2, 3, 5
  Lowest powers: 2В№, 3В№, 5В№
  HCF = 2 Г— 3 Г— 5 = 30 вң“

Example 2: HCF(48, 72, 108)
  48  = 2вҒҙ Г— 3
  72  = 2Ві Г— 3ВІ
  108 = 2ВІ Г— 3Ві
  Common primes: 2, 3
  Lowest powers: 2ВІ, 3В№
  HCF = 4 Г— 3 = 12 вң“
```

### Method 3: Division Method (Euclidean Algorithm) вӯҗ (Fastest for 2 numbers)

**Steps:**
1. Divide the **larger** number by the **smaller**.
2. Replace the larger number with the **remainder**.
3. Repeat until remainder = **0**.
4. The **last divisor** is the HCF.

```
Example: HCF(144, 60)

  144 Г· 60  = 2, remainder 24
   60 Г· 24  = 2, remainder 12
   24 Г· 12  = 2, remainder  0  вҶҗ STOP

  HCF = 12 вң“
```

```
Example: HCF(1071, 462)

  1071 Г· 462 = 2, remainder 147
   462 Г· 147 = 3, remainder  21
   147 Г·  21 = 7, remainder   0  вҶҗ STOP

  HCF = 21 вң“
```

### Method 4: Successive Division (for 3+ numbers)

**Steps:**
1. Find HCF of first two numbers.
2. Find HCF of that result with the third number.
3. Continue for more numbers.

```
Example: HCF(12, 18, 24)

  Step 1: HCF(12, 18)
    18 Г· 12 = 1, rem 6
    12 Г·  6 = 2, rem 0
    HCF(12,18) = 6

  Step 2: HCF(6, 24)
    24 Г· 6 = 4, rem 0
    HCF(6, 24) = 6

  вҲҙ HCF(12, 18, 24) = 6 вң“
```

## 5. METHODS TO FIND LCM

### Method 1: Listing Multiples (Basic вҖ” only for small numbers)

```
LCM(4, 6):
  Multiples of 4: 4, 8, 12, 16, 20, 24...
  Multiples of 6: 6, 12, 18, 24...
  First common = 12
  LCM = 12
```

### Method 2: Prime Factorization вӯҗ (Most Reliable)

**Steps:**
1. Write prime factorization of each number.
2. Take **ALL** prime factors (even if not common).
3. Take each prime with the **HIGHEST (maximum) power**.
4. Multiply them.

```
Example 1: LCM(60, 90)
  60 = 2ВІ Г— 3 Г— 5
  90 = 2 Г— 3ВІ Г— 5
  All primes: 2, 3, 5
  Highest powers: 2ВІ, 3ВІ, 5В№
  LCM = 4 Г— 9 Г— 5 = 180 вң“

Example 2: LCM(12, 18, 24)
  12 = 2ВІ Г— 3
  18 = 2  Г— 3ВІ
  24 = 2Ві Г— 3
  All primes: 2, 3
  Highest powers: 2Ві, 3ВІ
  LCM = 8 Г— 9 = 72 вң“
```

### Method 3: Division Method (Ladder Method) вӯҗ (Fastest for multiple numbers)

**Steps:**
1. Arrange all numbers in a row.
2. Divide by the **smallest prime** that divides **at least one** number.
3. Bring down numbers not divisible by that prime.
4. Repeat until all become 1.
5. LCM = Product of all divisors used.

```
Example: LCM(12, 18, 24, 36)

  2 | 12  18  24  36
  2 |  6   9  12  18
  2 |  3   9   6   9
  3 |  3   9   3   9
  3 |  1   3   1   3
  3 |  1   1   1   1

  LCM = 2 Г— 2 Г— 2 Г— 3 Г— 3 Г— 3 = 216

  Wait вҖ” let me recheck:
  12 = 2ВІГ—3, 18=2Г—3ВІ, 24=2ВіГ—3, 36=2ВІГ—3ВІ
  Highest: 2Ві Г— 3ВІ = 8 Г— 9 = 72 вң“
```

```
Example: LCM(6, 8, 15)

  2 | 6   8   15
  2 | 3   4   15
  2 | 3   2   15
  3 | 3   1   15
  5 | 1   1    5
      1   1    1

  LCM = 2 Г— 2 Г— 2 Г— 3 Г— 5 = 120 вң“
```

## 6. THE GOLDEN RELATIONSHIP: HCF Г— LCM = PRODUCT

### 6.1 The Formula

> ### HCF(a, b) Г— LCM(a, b) = a Г— b
> вҡ пёҸ **This works ONLY for exactly TWO numbers.**

```
Verification:
  a = 12, b = 18
  HCF(12,18) = 6
  LCM(12,18) = 36
  HCF Г— LCM = 6 Г— 36 = 216
  a Г— b = 12 Г— 18 = 216 вң“
```

### 6.2 Using the Formula (Finding Unknown)

**Type 1: Find LCM when HCF and both numbers are known**
```
  LCM = (a Г— b) / HCF
  Example: a=15, b=20, HCF=5
  LCM = (15 Г— 20) / 5 = 300/5 = 60 вң“
```

**Type 2: Find the other number when HCF, LCM, and one number are known**
```
  Other number = (HCF Г— LCM) / Known number
  Example: HCF=8, LCM=96, one number=32
  Other = (8 Г— 96) / 32 = 768/32 = 24 вң“
```

**Type 3: Find HCF when LCM and both numbers are known**
```
  HCF = (a Г— b) / LCM
  Example: a=15, b=20, LCM=60
  HCF = (15 Г— 20) / 60 = 300/60 = 5 вң“
```

### 6.3 Why This Formula Fails for 3+ Numbers

```
  a=6, b=10, c=15
  Product = 6Г—10Г—15 = 900
  HCF(6,10,15) = 1
  LCM(6,10,15) = 30
  HCF Г— LCM = 1 Г— 30 = 30  вү   900  вң—

  The formula does NOT apply for 3 or more numbers.
```

### 6.4 Extended Relationship for 3 Numbers

```
  LCM(a,b,c) = (a Г— b Г— c Г— HCF(a,b,c)) / (HCF(a,b) Г— HCF(b,c) Г— HCF(a,c))
```
(This is complex вҖ” in exams, just use prime factorization for 3+ numbers.)

## 7. HCF & LCM OF FRACTIONS

### 7.1 The Formulas

> ### HCF of Fractions = HCF of Numerators / LCM of Denominators
> ### LCM of Fractions = LCM of Numerators / HCF of Denominators

### 7.2 Step-by-Step Examples

```
Example 1: HCF of (2/3, 4/6, 8/9)

  Step 1: All fractions already in form p/q
  Step 2: HCF of numerators (2, 4, 8) = 2
  Step 3: LCM of denominators (3, 6, 9) = 18
  Step 4: HCF = 2/18 = 1/9 вң“

  Verification: 2/3 Г· (1/9) = 6 вң“ | 4/6 Г· (1/9) = 6 вң“ | 8/9 Г· (1/9) = 8 вң“
  (All are whole numbers вҶ’ correct!)
```

```
Example 2: LCM of (3/4, 5/6, 7/8)

  LCM of numerators (3, 5, 7) = 105
  HCF of denominators (4, 6, 8) = 2
  LCM = 105/2 вң“
```

```
Example 3: HCF and LCM of (1/2, 2/3, 3/4)

  HCF = HCF(1,2,3) / LCM(2,3,4) = 1/12
  LCM = LCM(1,2,3) / HCF(2,3,4) = 6/1 = 6
```

### 7.3 Key Verification Rules

- When you divide each fraction by the HCF of fractions вҶ’ result should be a **whole number**.
- Each fraction should divide the LCM of fractions вҶ’ result should be a **whole number**.

## 8. HCF & LCM OF DECIMALS

### The Method: Convert to Same Decimal Places First

**Steps:**
1. Make all numbers have the **same number of decimal places** (add trailing zeros if needed).
2. Remove decimal points вҶ’ treat as whole numbers.
3. Find HCF or LCM of those whole numbers.
4. Put the **decimal point back** at the same position.

```
Example 1: HCF(0.6, 0.9, 1.2)

  Step 1: All have 1 decimal place already
  Step 2: вҶ’ HCF(6, 9, 12)
           6  = 2 Г— 3
           9  = 3ВІ
           12 = 2ВІ Г— 3
           HCF = 3
  Step 3: Put decimal back вҶ’ 0.3

  вҲҙ HCF = 0.3 вң“
```

```
Example 2: LCM(0.4, 0.8, 1.6)

  All have 1 decimal place вҶ’ LCM(4, 8, 16)
  LCM = 16
  Put decimal back вҶ’ 1.6

  вҲҙ LCM = 1.6 вң“
```

```
Example 3: HCF(1.2, 0.08, 2.4)

  Step 1: Max decimal places = 2
          1.2 вҶ’ 1.20 | 0.08 вҶ’ 0.08 | 2.4 вҶ’ 2.40
  Step 2: вҶ’ HCF(120, 8, 240)
           120 = 2Ві Г— 3 Г— 5
           8   = 2Ві
           240 = 2вҒҙ Г— 3 Г— 5
           HCF = 2Ві = 8
  Step 3: 8 вҶ’ put 2 decimal places back вҶ’ 0.08

  вҲҙ HCF = 0.08 вң“
```

### TYPE 1: Bells / Events Ringing Together (LCM)

**When:** Things happen at different intervals вҶ’ find WHEN they happen together.

> **Answer = LCM of all intervals**

```
Q: Three bells ring at intervals of 6, 8, and 12 minutes.
   If they all ring at 12:00 noon, when will they next ring together?

   LCM(6, 8, 12):
     6 = 2 Г— 3
     8 = 2Ві
    12 = 2ВІ Г— 3
   LCM = 2Ві Г— 3 = 24 minutes

   Answer: 12:00 + 24 min = 12:24 PM вң“
```

### TYPE 2: Tiles / Square / Minimum Area (LCM)

**When:** Find the minimum square/area that can be tiled exactly.

> **Answer = LCM of the tile dimensions**

```
Q: A room floor is to be covered with square tiles of maximum possible size.
   The room is 15m Г— 20m. Find the tile size, number of tiles needed.

   Maximum tile size = HCF(15, 20) = 5 m
   Number of tiles = (15 Г— 20) / (5 Г— 5) = 300/25 = 12 tiles вң“
```

### TYPE 3: Largest Number that Divides All Giving Same Remainder (HCF)

**Pattern:** "Find the LARGEST number that divides a, b, c leaving the SAME remainder r."

**Formula:** HCF of (aвҲ’r, bвҲ’r, cвҲ’r) if r is given.
OR HCF of all **differences** (aвҲ’b, bвҲ’c, aвҲ’c) if r is NOT given.

```
Q: Find the largest number that divides 109, 149, and 229 leaving
   the same remainder in each case.

   Differences:
   149 вҲ’ 109 = 40
   229 вҲ’ 149 = 80
   229 вҲ’ 109 = 120

   HCF(40, 80, 120):
     40 = 2Ві Г— 5
     80 = 2вҒҙ Г— 5
    120 = 2Ві Г— 3 Г— 5
   HCF = 2Ві Г— 5 = 40

   Answer: 40 вң“
   Verification: 109Г·40=2 rem 29 | 149Г·40=3 rem 29 | 229Г·40=5 rem 29 вң“
```

```
Q: Find the largest number dividing 26, 54 leaving remainders 2, 3.

   Subtract remainders: 26вҲ’2 = 24,  54вҲ’3 = 51
   HCF(24, 51):
     24 = 2Ві Г— 3
     51 = 3 Г— 17
   HCF = 3

   Answer: 3 вң“
   Verification: 26Г·3 = 8 rem 2 вң“ | 54Г·3 = 18 rem 0... 
   (Check: 51Г·3=17 rem 0, 24Г·3=8 rem 0 вң“)
```

### TYPE 4: Smallest Number Divisible by All (LCM)

**Pattern:** "Find the smallest number divisible by a, b, c."

> **Answer = LCM(a, b, c)**

```

   12 = 2ВІ Г— 3
   15 = 3 Г— 5
   20 = 2ВІ Г— 5
   27 = 3Ві

### TYPE 5: Smallest Number that Leaves Remainder r When Divided by All (LCM + Add)

**Pattern:** "Find smallest number which when divided by a, b, c leaves remainder r."

> **Answer = LCM(a, b, c) + r**

### TYPE 6: Smallest Number that Leaves (Divisor вҲ’ Remainder) Same (LCM вҲ’ Subtract)

**Pattern:** "When divided by a, b, c the remainder is r less than the divisor each time."

> **Answer = LCM(a, b, c) вҲ’ deficit**

   LCM(8, 9, 12) = 72

   Answer = 72 вҲ’ 1 = 71 вң“
   Verification: 71Г·8=8 rem 7 вң“ | 71Г·9=7 rem 8 вң“ | 71Г·12=5 rem 11 вң“
```

### TYPE 7: n-Digit Largest/Smallest Number Divisible by Given Numbers

**Pattern A:** "Find the LARGEST n-digit number divisible by a, b, c."
```
Steps:
  1. Find LCM(a, b, c) = L
  2. Divide largest n-digit number (e.g., 9999 for 4-digit) by L
  3. Subtract remainder from 9999
  Answer = 9999 вҲ’ remainder
```

**Pattern B:** "Find the SMALLEST n-digit number divisible by a, b, c."
```
Steps:
  1. Find LCM(a, b, c) = L
  2. Divide smallest n-digit number (e.g., 1000 for 4-digit) by L
  3. If remainder = 0, it IS the answer
  4. Else: Answer = 1000 + (L вҲ’ remainder)
```

```
Example A: Largest 4-digit number divisible by 12, 15, 20.
  LCM(12,15,20) = 60
  9999 Г· 60 = 166 remainder 39
  Answer = 9999 вҲ’ 39 = 9960 вң“
  Check: 9960 Г· 60 = 166 вң“

Example B: Smallest 4-digit number divisible by 12, 15, 20.
  LCM = 60
  1000 Г· 60 = 16 remainder 40
  Answer = 1000 + (60 вҲ’ 40) = 1000 + 20 = 1020 вң“
  Check: 1020 Г· 60 = 17 вң“
```

## 10. ADVANCED CONCEPTS

### 10.1 HCF of Powers

```
HCF(a^m, a^n) = a^(min of m,n)
LCM(a^m, a^n) = a^(max of m,n)

Example:
  HCF(2^4, 2^6) = 2^4 = 16
  LCM(2^4, 2^6) = 2^6 = 64
```

### 10.2 Co-Prime Numbers and LCM

> If HCF(a, b) = 1 (co-prime), then:
> **LCM(a, b) = a Г— b**

```
Example: LCM(8, 9) = 8 Г— 9 = 72
  (Since HCF(8,9) = 1, they are co-prime)
```

### 10.3 Number of Pairs with Given HCF

```
If HCF = h, then the two numbers are of the form hВ·a and hВ·b
where HCF(a, b) = 1 (a and b are co-prime).

Also, a Г— b = LCM/HCF.

To find all pairs: find all co-prime factor pairs of (LCM/HCF).
```

```
Example: HCF = 12, LCM = 144. Find all pairs.

  LCM/HCF = 144/12 = 12
  Co-prime factor pairs of 12: (1,12), (3,4)
  [Note: (2,6) is NOT co-prime since HCF(2,6)=2вү 1]

  Pairs of numbers:
  (12Г—1, 12Г—12) = (12, 144)
  (12Г—3, 12Г—4)  = (36, 48)

  Answer: 2 pairs вҶ’ (12,144) and (36,48) вң“
```

### 10.4 LCM and HCF with Algebraic Expressions

```
Find HCF and LCM of (xВІ-4) and (xВІ+4x+4):

  xВІ-4     = (x+2)(x-2)
  xВІ+4x+4 = (x+2)ВІ

  HCF = (x+2)          вҶҗ common factor with lowest power
  LCM = (x+2)ВІ(x-2)    вҶҗ all factors with highest power вң“
```

### 10.5 Consecutive Numbers and HCF

> **HCF of any two consecutive integers = 1**
> **HCF of any two consecutive even numbers = 2**
> **HCF of any two consecutive odd numbers = 1**

```
Examples:
  HCF(n, n+1) = 1 always
  HCF(2k, 2k+2) = 2 always
  HCF(100, 101) = 1 вң“
```

### 10.6 HCF of (a^n вҲ’ 1) Type

> HCF(a^m вҲ’ 1, a^n вҲ’ 1) = a^(HCF(m,n)) вҲ’ 1

```
Example: HCF(2^12 вҲ’ 1, 2^8 вҲ’ 1)
  HCF(12, 8) = 4
  Answer = 2^4 вҲ’ 1 = 15 вң“
```

### 10.7 When Numbers are Expressed as Products

```
HCF and LCM of (12 Г— 24) and (16 Г— 18):

  Method: Factorize each product together.
  12 Г— 24 = 288 = 2вҒө Г— 3ВІ
  16 Г— 18 = 288 = 2вҒө Г— 3ВІ
  HCF = LCM = 288 (same number!)
```

## 11. SHORT TRICKS & SPEED METHODS

---

### вҡЎ Trick 1: Quick HCF Using Subtraction (for close numbers)

For two numbers close to each other:
**HCF divides their difference.**

So find difference first, then check its factors.

```
HCF(91, 119):
  Difference = 119 - 91 = 28
  Factors of 28: 1, 2, 4, 7, 14, 28
  Check which divides both:
  28: 91Г·28=3.25вң—  |  14: 91Г·14=6.5вң—  |  7: 91Г·7=13вң“, 119Г·7=17вң“
  HCF = 7  вң“  (Done in seconds!)
```

---

### вҡЎ Trick 2: Co-Prime Shortcut for LCM

If HCF(a,b) = 1:  **LCM = a Г— b**

```
LCM(17, 19) = 17 Г— 19 = 323  (both prime вҶ’ co-prime)
LCM(8, 9)   = 8 Г— 9   = 72   (HCF=1)
LCM(4, 25)  = 4 Г— 25  = 100  (HCF=1)
```

---

### вҡЎ Trick 3: LCM from HCF (2 numbers only)

```
LCM = (a Г— b) / HCF

Faster form: LCM = (a/HCF) Г— b  OR  (b/HCF) Г— a
```

---

### вҡЎ Trick 4: Check if LCM & HCF are Consistent

> **HCF must always divide LCM.**
> If HCF does NOT divide LCM вҶ’ the given values are IMPOSSIBLE.

```
HCF = 8, LCM = 60?
  60 Г· 8 = 7.5  вҶ’ NOT a whole number
  вҶ’ IMPOSSIBLE combination вң—

HCF = 6, LCM = 60?
  60 Г· 6 = 10  вң“
  вҶ’ Valid combination вң“
```

---

### вҡЎ Trick 5: Numbers in Form of HCF

> If HCF(a, b) = h, then **a = hВ·m** and **b = hВ·n** where HCF(m, n) = 1.

```
HCF = 15, numbers are in ratio 2:5
  Numbers = 15Г—2, 15Г—5 = 30, 75
  LCM = 15 Г— 2 Г— 5 = 150 вң“
```

---

### вҡЎ Trick 6: LCM of (1 to n) Shortcuts

Memorize these for speed:
```
  LCM(1 to 5)  = 60
  LCM(1 to 6)  = 60
  LCM(1 to 7)  = 420
  LCM(1 to 8)  = 840
  LCM(1 to 9)  = 2520
  LCM(1 to 10) = 2520
```

---

### вҡЎ Trick 7: Pattern вҖ” Deficit Type Problems

When each divisor exceeds its remainder by the **same fixed number k**:

> **Answer = LCM вҲ’ k**

```
Divided by 5 вҶ’ remainder 3 (deficit = 2)
Divided by 6 вҶ’ remainder 4 (deficit = 2)
Divided by 7 вҶ’ remainder 5 (deficit = 2)
вҶ’ Same deficit = 2
вҶ’ Answer = LCM(5,6,7) вҲ’ 2 = 210 вҲ’ 2 = 208 вң“
```

---

### вҡЎ Trick 8: Sum/Difference Pattern

For TYPE 3 problems (same remainder, divisor not given):
- **Always take pairwise differences** of all given numbers.
- Find HCF of those differences.

```
Numbers: 34, 58, 94
  Differences: 58вҲ’34=24, 94вҲ’58=36, 94вҲ’34=60
  HCF(24, 36, 60) = 12
  Answer = 12 вң“
```

---

### вҡЎ Trick 9: Quick LCM for Consecutive Numbers

```
LCM(n, n+1) = n(n+1)  вҶҗ always, since consecutive integers are co-prime

LCM(5,6) = 30  вң“
LCM(9,10) = 90 вң“
```

---

### вҡЎ Trick 10: Multiple of HCF Constraint

> Any common factor of a and b **must be a factor of HCF(a,b)**.

So if asked "Is 8 a common factor of 24 and 36?":
- HCF(24,36) = 12
- Does 8 divide 12? 12Г·8 = 1.5 вң—
- **No, 8 is NOT a common factor.** вң“

---

### вҡЎ Trick 11: Shortcut Divisibility Check via LCM

> "Is N divisible by all of a, b, c?" вҶ’ Check if LCM(a,b,c) divides N.

```
Is 360 divisible by 4, 6, 9?
  LCM(4,6,9) = 36
  360 Г· 36 = 10 вң“
  Yes! вң“
```

---

### вҡЎ Trick 12: Fraction HCF/LCM Memory Hook

```
HCF of fractions вҶ’ think "H for Humble вҶ’ small numerator, big denominator"
              HCF = HCF(top) / LCM(bottom)

LCM of fractions вҶ’ think "L for Large вҶ’ big numerator, small denominator"
              LCM = LCM(top) / HCF(bottom)
```

## 12. COMMON MISTAKES TO AVOID

| вқҢ Mistake | вң… Correct Approach |
|-----------|---------------------|
| Using HCFГ—LCM = product for 3 numbers | Only valid for exactly 2 numbers |
| Forgetting HCF must divide LCM | Always verify this |
| Not reducing fractions before HCF/LCM | Simplify first if needed |
| Confusing "same remainder" type with "different remainder" type | Identify type carefully |
| HCF of fractions: dividing numerators by HCF of denominators | Formula is HCF(num)/LCM(den) |
| Taking LCM when HCF is needed in cutting/packing problems | LARGEST piece вҶ’ HCF; SMALLEST time together вҶ’ LCM |
| In TYPE 6: not checking if deficits are equal | Must verify before applying trick |
| Ignoring the word "SAME" in remainder problems | "Same remainder" вҶ’ use differences |
