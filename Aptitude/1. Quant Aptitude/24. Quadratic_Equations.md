## 1. THE BIG PICTURE — WHAT IS A QUADRATIC EQUATION?

---

### 🔑 Definition:

> A **Quadratic Equation** is a polynomial equation of **degree 2** (highest power of the variable is 2).

$$\boxed{ax^2 + bx + c = 0 \quad \text{where } a \neq 0}$$

---

### Why "Quadratic"?

- "Quad" = Square (Latin)
- The highest term has degree **2** (a square)
- Always has **exactly 2 roots** (counting multiplicity, in complex numbers)

---

### Examples vs Non-Examples:

| Equation | Quadratic? | Reason |
|---------|-----------|--------|
| x² − 5x + 6 = 0 | ✅ Yes | Degree = 2 |
| 2x² + 3x = 0 | ✅ Yes | c = 0, still quadratic |
| x² = 0 | ✅ Yes | b = c = 0 |
| x + 5 = 0 | ❌ No | Degree = 1 (linear) |
| x³ − x = 0 | ❌ No | Degree = 3 (cubic) |
| 0x² + x + 1 = 0 | ❌ No | a = 0 means not quadratic |

---

### The Big Picture — Concept Map:

```
           QUADRATIC EQUATION: ax² + bx + c = 0
                              │
              ┌───────────────┼───────────────┐
              │               │               │
        SOLVING          DISCRIMINANT     ROOTS
        METHODS          (D = b²-4ac)   RELATIONSHIP
              │               │               │
     ┌────────┼────────┐   D>0: 2 real   Sum = -b/a
     │        │        │   D=0: 1 real   Product = c/a
 Factoring Quadratic  Completing  D<0: 2 complex
           Formula    the Square
```

## 2. STANDARD FORM & KEY TERMINOLOGY

---

### Standard Form:

$$ax^2 + bx + c = 0 \quad (a \neq 0)$$

| Symbol | Name | Role |
|--------|------|------|
| a | Leading coefficient | Coefficient of x² |
| b | Middle coefficient | Coefficient of x |
| c | Constant term | Independent term |
| α, β | Roots | Values of x that satisfy the equation |

---

### Monic Quadratic:

When a = 1: $x^2 + bx + c = 0$ (called **monic** form)

The roots satisfy: $x^2 − (\alpha+\beta)x + \alpha\beta = 0$

---

### Reduced/Depressed Form:

If one coefficient is 0:

| Form | Example |
|------|---------|
| b = 0: Pure quadratic | $ax^2 + c = 0$ → $x = \pm\sqrt{-c/a}$ |
| c = 0: Incomplete quadratic | $ax^2 + bx = 0$ → $x(ax+b)=0$ |

---

## 3. METHODS OF SOLVING QUADRATIC EQUATIONS

---

### METHOD 1: Factoring (Factorisation)

> Split the middle term so that the product of two factors equals the quadratic.

**The Split-Middle-Term Technique:**

For $ax^2 + bx + c = 0$:

1. Find two numbers whose **product = ac** and **sum = b**
2. Split bx into two terms using these numbers
3. Factor by grouping

---

#### Example 1: Simple Factoring

> Solve: $x^2 - 5x + 6 = 0$

Product = 6, Sum = -5 → Numbers: -2 and -3.

$$x^2 - 2x - 3x + 6 = 0$$
$$x(x-2) - 3(x-2) = 0$$
$$(x-2)(x-3) = 0$$
$$\boxed{x = 2 \text{ or } x = 3}$$

---

#### Example 2: With Leading Coefficient

> Solve: $6x^2 - 13x + 6 = 0$

Product = 6×6 = 36, Sum = -13 → Numbers: -9 and -4.

$$6x^2 - 9x - 4x + 6 = 0$$
$$3x(2x-3) - 2(2x-3) = 0$$
$$(2x-3)(3x-2) = 0$$
$$\boxed{x = \frac{3}{2} \text{ or } x = \frac{2}{3}}$$

---

#### Example 3: c = 0 Form

> Solve: $3x^2 - 7x = 0$

$$x(3x-7) = 0$$
$$\boxed{x = 0 \text{ or } x = \frac{7}{3}}$$

---

#### Example 4: b = 0 Form

> Solve: $x^2 - 9 = 0$

$$x^2 = 9 \Rightarrow x = \pm 3$$
$$\boxed{x = 3 \text{ or } x = -3}$$

---

### METHOD 2: Quadratic Formula

> The universal method. Works for ALL quadratic equations.

$$\boxed{x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}}$$

**Memory Tip:** "Negative b, plus or minus square root of b squared minus 4ac, all over 2a."

---

#### Example 5: Quadratic Formula — Standard

> Solve: $2x^2 - 3x - 2 = 0$

a=2, b=-3, c=-2.

$$x = \frac{3 \pm \sqrt{9 + 16}}{4} = \frac{3 \pm \sqrt{25}}{4} = \frac{3 \pm 5}{4}$$

$$x = \frac{3+5}{4} = 2 \quad \text{or} \quad x = \frac{3-5}{4} = -\frac{1}{2}$$

$$\boxed{x = 2 \text{ or } x = -\frac{1}{2}}$$

---

#### Example 6: Irrational Roots

> Solve: $x^2 - 4x + 1 = 0$

a=1, b=-4, c=1.

$$x = \frac{4 \pm \sqrt{16-4}}{2} = \frac{4 \pm \sqrt{12}}{2} = \frac{4 \pm 2\sqrt{3}}{2} = 2 \pm \sqrt{3}$$

$$\boxed{x = 2+\sqrt{3} \text{ or } x = 2-\sqrt{3}}$$

---

### METHOD 3: Completing the Square

> Convert $ax^2 + bx + c = 0$ into the form $(x+h)^2 = k$.

**Steps:**

1. Divide by a: $x^2 + \frac{b}{a}x + \frac{c}{a} = 0$
2. Move constant: $x^2 + \frac{b}{a}x = -\frac{c}{a}$
3. Add $\left(\frac{b}{2a}\right)^2$ to both sides
4. Factor left as perfect square
5. Take square root

---

#### Example 7: Completing the Square

> Solve: $x^2 + 6x - 7 = 0$

$$x^2 + 6x = 7$$
$$x^2 + 6x + 9 = 7 + 9 = 16$$
$$(x+3)^2 = 16$$
$$x+3 = \pm 4$$
$$\boxed{x = 1 \text{ or } x = -7}$$

---

#### Example 8: Completing the Square — with Coefficient

> Solve: $2x^2 + 8x - 10 = 0$

Divide by 2: $x^2 + 4x - 5 = 0$

$$x^2 + 4x = 5$$
$$x^2 + 4x + 4 = 9$$
$$(x+2)^2 = 9 \Rightarrow x+2 = \pm 3$$
$$\boxed{x = 1 \text{ or } x = -5}$$

---

### METHOD 4: Graphical Method

> Plot $y = ax^2 + bx + c$. Where the parabola crosses the x-axis = roots.

- 2 crossing points → 2 real roots
- 1 touching point → 1 repeated root
- No crossing → complex roots

---

### Choosing the Right Method:

| Situation | Best Method |
|-----------|------------|
| Factors are obvious integers | Factoring |
| General/irrational roots | Quadratic Formula |
| Vertex form needed | Completing the Square |
| Perfect square pattern | Direct square root |
| Always works | Quadratic Formula |

## 4. THE DISCRIMINANT — NATURE OF ROOTS

---

### Definition:

$$\boxed{D = b^2 - 4ac}$$

The discriminant **D** tells us the **nature** (type) of roots **without actually solving** the equation.

---

### Nature of Roots Table:

| Discriminant | Roots | Nature |
|-------------|-------|--------|
| **D > 0** | Real and Distinct | Two different real roots |
| **D = 0** | Real and Equal | One repeated real root |
| **D < 0** | Complex/Imaginary | No real roots |
| **D ≥ 0** | Real | Both real roots exist |
| **D is perfect square** | Rational and Distinct | (when a, b, c are integers) |

---

### Visualising with Parabola:

```
D > 0               D = 0               D < 0
 (2 real roots)     (1 repeated root)   (no real roots)

    ∩                   ∩                   ∩
   / \               ___|___               / \
  /   \             |       |             /   \
─x───x─              ──x──               ─────
  Cuts x-axis        Touches x-axis      Above x-axis
  at 2 points        at 1 point          (no crossing)
```

---

### Examples — Discriminant:

#### Example 9: Find Nature Without Solving

> Find the nature of roots of (a) $x^2 - 5x + 3 = 0$ (b) $x^2 - 6x + 9 = 0$ (c) $x^2 + x + 1 = 0$

(a) D = 25 - 12 = **13 > 0** → Two distinct real roots

(b) D = 36 - 36 = **0** → Equal real roots (x = 3, 3)

(c) D = 1 - 4 = **-3 < 0** → No real roots (complex roots)

---

#### Example 10: Find k for Equal Roots

> Find k if $2x^2 + kx + 8 = 0$ has equal roots.

For equal roots: D = 0.
$$k^2 - 4(2)(8) = 0 \Rightarrow k^2 = 64 \Rightarrow k = \pm 8$$

---

#### Example 11: Find Range for Real Roots

> Find values of k for which $x^2 - 2kx + k + 2 = 0$ has real roots.

For real roots: D ≥ 0.
$$(−2k)^2 - 4(1)(k+2) \geq 0$$
$$4k^2 - 4k - 8 \geq 0$$
$$k^2 - k - 2 \geq 0$$
$$(k-2)(k+1) \geq 0$$
$$\boxed{k \leq -1 \text{ or } k \geq 2}$$

## 5. RELATIONSHIP BETWEEN ROOTS AND COEFFICIENTS (VIETA'S FORMULAS)

---

### The Core Formulas:

For $ax^2 + bx + c = 0$ with roots α and β:

$$\boxed{\alpha + \beta = -\frac{b}{a}} \quad \text{(Sum of roots)}$$

$$\boxed{\alpha \cdot \beta = \frac{c}{a}} \quad \text{(Product of roots)}$$

---

### Memory Aid:

$$\text{Sum} = -\frac{b}{a} = \frac{-\text{coefficient of x}}{\text{coefficient of x}^2}$$

$$\text{Product} = \frac{c}{a} = \frac{\text{constant term}}{\text{coefficient of x}^2}$$

---

### Quick Derivation (Know Why It Works):

If α and β are roots:
$$ax^2 + bx + c = a(x-\alpha)(x-\beta)$$
$$= a[x^2 - (\alpha+\beta)x + \alpha\beta]$$
$$= ax^2 - a(\alpha+\beta)x + a\alpha\beta$$

Comparing: $b = -a(\alpha+\beta)$ → $\alpha+\beta = -b/a$
And: $c = a\alpha\beta$ → $\alpha\beta = c/a$

---

### Examples — Vieta's Formulas:

#### Example 12: Find Sum and Product

> Find sum and product of roots of $3x^2 - 7x + 2 = 0$ **without solving**.

Sum = $\frac{7}{3}$, Product = $\frac{2}{3}$

---

#### Example 13: Find a, b, c from Root Properties

> A quadratic equation has roots 4 and −3. Write the equation.

Sum = 4 + (−3) = 1, Product = 4 × (−3) = −12.

Equation: $x^2 - (\text{sum})x + \text{product} = 0$

$$\boxed{x^2 - x - 12 = 0}$$

---

#### Example 14: Find Expression Without Solving

> If α, β are roots of $x^2 - 3x + 1 = 0$, find $\alpha^2 + \beta^2$.

Sum α+β = 3, Product αβ = 1.

$$\alpha^2 + \beta^2 = (\alpha+\beta)^2 - 2\alpha\beta = 9 - 2 = \textbf{7}$$

---

#### Example 15: Advanced Symmetric Function

> If α, β are roots of $x^2 - 5x + 3 = 0$, find $\alpha^3 + \beta^3$.

α+β = 5, αβ = 3.

$$\alpha^3 + \beta^3 = (\alpha+\beta)^3 - 3\alpha\beta(\alpha+\beta) = 125 - 45 = \textbf{80}$$

---

#### Example 16: Reciprocal Roots

> If α, β are roots of $2x^2 - 5x + 3 = 0$, find the equation with roots $1/\alpha$ and $1/\beta$.

$\frac{1}{\alpha} + \frac{1}{\beta} = \frac{\alpha+\beta}{\alpha\beta} = \frac{5/2}{3/2} = \frac{5}{3}$

$\frac{1}{\alpha} \cdot \frac{1}{\beta} = \frac{1}{\alpha\beta} = \frac{1}{3/2} = \frac{2}{3}$

Equation: $x^2 - \frac{5}{3}x + \frac{2}{3} = 0$ → Multiply by 3: $3x^2 - 5x + 2 = 0$

---

## 6. FORMATION OF QUADRATIC EQUATION FROM ROOTS

---

### The Master Formula:

If roots are α and β:

$$\boxed{x^2 - (\alpha + \beta)x + \alpha\beta = 0}$$

Or equivalently:

$$x^2 - (\text{Sum})x + (\text{Product}) = 0$$

---

### Examples:

#### Example 17: From Given Roots

> Form quadratic with roots $3+\sqrt{2}$ and $3-\sqrt{2}$.

Sum = $(3+\sqrt{2}) + (3-\sqrt{2}) = 6$

Product = $(3+\sqrt{2})(3-\sqrt{2}) = 9-2 = 7$

$$\boxed{x^2 - 6x + 7 = 0}$$

---

#### Example 18: Double Root

> Form equation with equal roots 5, 5.

Sum = 10, Product = 25.
$$\boxed{x^2 - 10x + 25 = 0} = (x-5)^2 = 0$$

---

#### Example 19: Roots of New Equation from Old

> Roots of $x^2 - 3x + 2 = 0$ are α, β. Form equation with roots $\alpha^2$ and $\beta^2$.

From x² - 3x + 2 = 0: α+β = 3, αβ = 2.

New sum = $\alpha^2+\beta^2 = (\alpha+\beta)^2 - 2\alpha\beta = 9-4 = 5$

New product = $\alpha^2\beta^2 = (\alpha\beta)^2 = 4$

$$\boxed{x^2 - 5x + 4 = 0}$$

---

#### Example 20: Roots Increased by k

> If α, β are roots of $x^2 - 2x + 3 = 0$, form equation with roots α+2 and β+2.

Substitute x → x−2 in original equation (shift roots by 2):

$(x-2)^2 - 2(x-2) + 3 = 0$

$x^2 - 4x + 4 - 2x + 4 + 3 = 0$

$$\boxed{x^2 - 6x + 11 = 0}$$

## 7. MAXIMUM AND MINIMUM OF QUADRATIC EXPRESSION

---

### The Vertex Form:

$$y = ax^2 + bx + c = a\left(x + \frac{b}{2a}\right)^2 + \left(c - \frac{b^2}{4a}\right)$$

---

### Maximum/Minimum Value:

| Condition | Shape | Extreme Value | At x = |
|-----------|-------|--------------|--------|
| **a > 0** | Opens UP ∪ | **Minimum** = $c - \frac{b^2}{4a}$ | $x = -\frac{b}{2a}$ |
| **a < 0** | Opens DOWN ∩ | **Maximum** = $c - \frac{b^2}{4a}$ | $x = -\frac{b}{2a}$ |

$$\boxed{\text{Extreme value} = \frac{4ac - b^2}{4a} = -\frac{D}{4a}}$$

---

### Examples:

#### Example 21: Minimum Value

> Find the minimum value of $2x^2 - 8x + 5$.

a = 2 > 0 → Minimum exists.

$$x_{min} = -\frac{-8}{2 \times 2} = \frac{8}{4} = 2$$

$$\text{Min value} = 2(4) - 8(2) + 5 = 8 - 16 + 5 = \textbf{-3}$$

OR using formula: $\frac{4(2)(5) - 64}{4 \times 2} = \frac{40-64}{8} = \frac{-24}{8} = -3$ ✓

---

#### Example 22: Maximum Value

> Find the maximum value of $-3x^2 + 12x - 5$.

a = −3 < 0 → Maximum exists.

$$x_{max} = -\frac{12}{2 \times (-3)} = 2$$

$$\text{Max} = -3(4) + 12(2) - 5 = -12+24-5 = \textbf{7}$$

## 8. SIGN OF QUADRATIC EXPRESSION

---

### When is ax² + bx + c Positive/Negative?

Let f(x) = ax² + bx + c with roots α, β (α < β when D > 0).

| Case | a > 0 | a < 0 |
|------|-------|-------|
| **D > 0** | f(x) > 0 when x < α or x > β; f(x) < 0 when α < x < β | f(x) < 0 when x < α or x > β; f(x) > 0 when α < x < β |
| **D = 0** | f(x) ≥ 0 for all x (= 0 only at x = α) | f(x) ≤ 0 for all x |
| **D < 0** | f(x) > 0 for ALL x | f(x) < 0 for ALL x |

---

### Visual Summary:

```
f(x) = ax² + bx + c

a > 0, D > 0:           a > 0, D < 0:
++++│---│++++            ++++++++++++
    α   β                (always +ve)

a < 0, D > 0:           a < 0, D < 0:
----│+++│----            -----------
    α   β                (always -ve)
```

---

#### Example 23: Sign Analysis

> For what values of x is $x^2 - 5x + 6 > 0$?

Roots: x = 2 and x = 3. a = 1 > 0.

$x^2 - 5x + 6 > 0$ when $x < 2$ or $x > 3$.

$$\boxed{x \in (-\infty, 2) \cup (3, +\infty)}$$

## 9. COMMON ROOTS BETWEEN TWO EQUATIONS

---

### Case 1: One Common Root

If α is common root of $a_1x^2 + b_1x + c_1 = 0$ and $a_2x^2 + b_2x + c_2 = 0$:

Eliminate α² (subtract multiples):

$$\alpha = \frac{b_1c_2 - b_2c_1}{a_1b_2 - a_2b_1}$$

---

### Case 2: Both Roots Common

If both roots are common:

$$\frac{a_1}{a_2} = \frac{b_1}{b_2} = \frac{c_1}{c_2}$$

---

#### Example 24: Find Common Root

> Find the common root of $x^2 - 5x + 6 = 0$ and $x^2 - 3x + 2 = 0$.

Roots of first: 2, 3. Roots of second: 1, 2.

**Common root = 2**

Algebraically: Subtract equations:
$-2x + 4 = 0 \Rightarrow x = 2$ ✓

## 10. SYMMETRIC FUNCTIONS OF ROOTS

---

### Key Identities (α+β = S, αβ = P):

| Expression | In terms of S and P |
|-----------|---------------------|
| $\alpha^2 + \beta^2$ | $S^2 - 2P$ |
| $\alpha^2 - \beta^2$ | $(\alpha-\beta)(\alpha+\beta) = (\sqrt{S^2-4P}) \cdot S$ |
| $\alpha^3 + \beta^3$ | $S^3 - 3PS$ |
| $\alpha^3 - \beta^3$ | $(\alpha-\beta)(S^2 - P)$ |
| $\alpha^4 + \beta^4$ | $(S^2-2P)^2 - 2P^2$ |
| $\frac{1}{\alpha} + \frac{1}{\beta}$ | $\frac{S}{P}$ |
| $\frac{1}{\alpha^2} + \frac{1}{\beta^2}$ | $\frac{S^2-2P}{P^2}$ |
| $\alpha^2\beta + \alpha\beta^2$ | $SP$ |
| $(\alpha-\beta)^2$ | $S^2 - 4P$ |
| $|\alpha - \beta|$ | $\sqrt{S^2 - 4P} = \frac{\sqrt{D}}{|a|}$ |

---

### Examples:

#### Example 25: Multiple Symmetric Functions

> If α, β are roots of $x^2 - 4x + 1 = 0$, find: (a) $\alpha^2+\beta^2$ (b) $\frac{1}{\alpha}+\frac{1}{\beta}$ (c) $\alpha^3+\beta^3$ (d) $(\alpha-\beta)^2$

S = α+β = 4, P = αβ = 1.

(a) $S^2 - 2P = 16 - 2 = \textbf{14}$

(b) $\frac{S}{P} = \frac{4}{1} = \textbf{4}$

(c) $S^3 - 3PS = 64 - 12 = \textbf{52}$

(d) $S^2 - 4P = 16 - 4 = \textbf{12}$ → $|\alpha-\beta| = \sqrt{12} = 2\sqrt{3}$

## 11. QUADRATIC IN DISGUISE

---

### Types of Hidden Quadratics:

| Substitution | Resulting Equation | Apply When |
|-------------|-------------------|------------|
| $t = x^2$ | $at^2 + bt + c = 0$ | Biquadratic equations |
| $t = x + 1/x$ | $t^2 \pm 2 = ...$ | Reciprocal equations |
| $t = \sqrt{x}$ | $at^2 + bt + c = 0$ | Equations with √x |
| $t = e^x$ or $t = a^x$ | Exponential to quadratic | Exp equations |
| $t = \log x$ | Log equations | Log quadratics |

---

### Examples:

#### Example 26: Biquadratic (t = x²)

> Solve: $x^4 - 10x^2 + 9 = 0$

Let $t = x^2$:

$$t^2 - 10t + 9 = 0 \Rightarrow (t-1)(t-9) = 0$$

$t = 1$ or $t = 9$

$x^2 = 1 \Rightarrow x = \pm 1$ or $x^2 = 9 \Rightarrow x = \pm 3$

$$\boxed{x = \pm 1, \pm 3}$$

---

#### Example 27: Reciprocal Type

> Solve: $x + \frac{1}{x} = \frac{5}{2}$

Multiply by 2x: $2x^2 - 5x + 2 = 0$

$(2x-1)(x-2) = 0 \Rightarrow x = \frac{1}{2}$ or $x = 2$

---

#### Example 28: Complex Reciprocal Equation

> Solve: $x^2 + \frac{1}{x^2} - 5\left(x + \frac{1}{x}\right) + 6 = 0$

Let $t = x + \frac{1}{x}$. Then $t^2 = x^2 + 2 + \frac{1}{x^2}$ → $x^2 + \frac{1}{x^2} = t^2 - 2$.

$$t^2 - 2 - 5t + 6 = 0 \Rightarrow t^2 - 5t + 4 = 0 \Rightarrow (t-1)(t-4) = 0$$

$t = 1$: $x + 1/x = 1$ → $x^2 - x + 1 = 0$ → D = 1-4 < 0 → No real roots.

$t = 4$: $x + 1/x = 4$ → $x^2 - 4x + 1 = 0$ → $x = 2 \pm \sqrt{3}$

$$\boxed{x = 2+\sqrt{3} \text{ or } x = 2-\sqrt{3}}$$

---

#### Example 29: Exponential Quadratic

> Solve: $4^x - 3 \cdot 2^x + 2 = 0$

$4^x = (2^x)^2$. Let $t = 2^x$:

$$t^2 - 3t + 2 = 0 \Rightarrow (t-1)(t-2) = 0$$

$t = 1 \Rightarrow 2^x = 1 \Rightarrow x = 0$

$t = 2 \Rightarrow 2^x = 2 \Rightarrow x = 1$

$$\boxed{x = 0 \text{ or } x = 1}$$

---

#### Example 30: Word Problem — Quadratic

> The product of two consecutive positive integers is 56. Find them.

Let integers = n and n+1.
$n(n+1) = 56 \Rightarrow n^2 + n - 56 = 0$

$(n+8)(n-7) = 0 \Rightarrow n = 7$ (positive)

**Integers: 7 and 8**

## 12. GRAPH OF QUADRATIC EQUATION (PARABOLA)

---

### Key Features of Parabola y = ax² + bx + c:

| Feature | Formula | Description |
|---------|---------|-------------|
| **Vertex** | $\left(-\frac{b}{2a}, \frac{4ac-b^2}{4a}\right)$ | Highest/lowest point |

| **x-intercepts** | Roots of equation | Where graph crosses x-axis |
| **y-intercept** | $(0, c)$ | Where graph crosses y-axis |
| **Direction** | a > 0: upward ∪; a < 0: downward ∩ | |

### Shape Summary:

```
a > 0 (Upward)          a < 0 (Downward)

     |                       ___
     |  /\                  /   \
  ___| /  \           _____|     |_____
 /   |/    \               |     |
─────+──────              ─+─────+─
```

## 13. QUICK TRICKS & SHORTCUTS

---

### 🚀 TRICK 1: Quick Factoring — Product-Sum Method

For $x^2 + bx + c = 0$: Find two numbers with product c and sum b.

Example: $x^2 - 7x + 12 = 0$ → product 12, sum -7 → -3, -4 → $(x-3)(x-4) = 0$

---

### 🚀 TRICK 2: Vieta's — Instant Sum and Product

Don't solve! For $ax^2+bx+c=0$:
$$S = \frac{-b}{a}, \quad P = \frac{c}{a}$$

---

### 🚀 TRICK 3: α² + β² in One Step

$$\alpha^2 + \beta^2 = (\alpha+\beta)^2 - 2\alpha\beta = S^2 - 2P$$

---

### 🚀 TRICK 4: α³ + β³ Formula

$$\alpha^3 + \beta^3 = (\alpha+\beta)(\alpha^2 - \alpha\beta + \beta^2) = S(S^2 - 3P)$$

---

### 🚀 TRICK 5: |α − β| Without Finding Roots

$$|\alpha - \beta| = \frac{\sqrt{D}}{|a|}$$

---

### 🚀 TRICK 6: Equation with Roots k×α and k×β (Scale Roots)

Replace x by x/k in original equation.

---

### 🚀 TRICK 7: Equation with Roots (α+k) and (β+k) (Shift Roots)

Replace x by (x−k) in original equation.

---

### 🚀 TRICK 8: Equation with Roots 1/α and 1/β

Reverse coefficients: if $ax^2+bx+c=0$, new equation is $cx^2+bx+a=0$.

---

### 🚀 TRICK 9: Equation with Roots −α and −β

Replace x by −x: if $ax^2+bx+c=0$, new is $ax^2-bx+c=0$ (flip sign of b).

---

### 🚀 TRICK 10: Equal Roots Condition

D = 0 → b² = 4ac. Use this to find unknown parameter.

---

### 🚀 TRICK 11: Both Roots Positive

Both roots positive iff: $S > 0$, $P > 0$, and $D \geq 0$.

---

### 🚀 TRICK 12: Both Roots Negative

Both roots negative iff: $S < 0$, $P > 0$, and $D \geq 0$.

---

### 🚀 TRICK 13: Roots of Opposite Signs

Product P < 0 (one root positive, one negative, regardless of D).

---

### 🚀 TRICK 14: One Root is Zero

c = 0 in $ax^2+bx+c=0$.

---

### 🚀 TRICK 15: Both Roots Zero

b = 0 and c = 0, so equation is $ax^2 = 0$.

---

### 🚀 TRICK 16: Biquadratic — Always Use t = x²

$ax^4 + bx^2 + c = 0$ → let $t = x^2$ → solve quadratic in t.

---

### 🚀 TRICK 17: Roots Reciprocal of Each Other

Product αβ = 1 → c = a. The equation $ax^2+bx+c=0$ with $a=c$.

---

### 🚀 TRICK 18: D is Perfect Square → Rational Roots (if a,b,c rational)

If D = k² (perfect square) and a,b,c integers → roots are rational.

---

### 🚀 TRICK 19: Nature from Coefficients Pattern

For $x^2 - (α+β)x + αβ = 0$:
- $αβ < 0$ → Roots of opposite sign (no need to check D)
- $α+β > 0$ and $αβ > 0$ → Both roots positive
- $α+β < 0$ and $αβ > 0$ → Both roots negative

---

### 🚀 TRICK 20: Sum and Product Without Solving — Use These Always

For any quadratic problem, ALWAYS write S = −b/a and P = c/a first. Then express whatever is asked in terms of S and P.

## 14. COMMON MISTAKE ALERTS

---

### ⚠️ MISTAKE 1: Forgetting ± in Square Root

$x^2 = 9 \Rightarrow x = \pm 3$ (**not** just x = 3)

The square root always yields **two** solutions: positive and negative.

---

### ⚠️ MISTAKE 2: Dividing by Variable (Losing a Root)

**Wrong:** $x^2 = 5x$ → divide by x → $x = 5$ (loses x = 0!)
**Right:** $x^2 - 5x = 0$ → $x(x-5) = 0$ → x = 0 or x = 5

Never divide both sides by a variable expression — you might be dividing by 0.

---

### ⚠️ MISTAKE 3: Sum of Roots Sign Error

For $ax^2 + bx + c = 0$:

**Wrong:** Sum = b/a
**Right:** Sum = **-b/a** (there's a negative sign!)

The formula is $\alpha + \beta = \frac{-b}{a}$, not $\frac{b}{a}$.

---

### ⚠️ MISTAKE 4: Discriminant Formula Error

**Wrong:** D = b² + 4ac
**Right:** D = b² **−** 4ac

It's **minus** 4ac, not plus!

---

### ⚠️ MISTAKE 5: Not Checking for Extraneous Solutions

After solving, always substitute back to verify, especially when:
- Square roots are taken (both sides)
- Equation had fractions (could make denominator 0)
- Absolute values were involved

---

### ⚠️ MISTAKE 6: Forming Equation — Wrong Sign for Sum

Quadratic from roots α, β:

**Wrong:** $x^2 + (\alpha+\beta)x + \alpha\beta = 0$
**Right:** $x^2 - (\alpha+\beta)x + \alpha\beta = 0$ (subtract sum, add product)

---

### ⚠️ MISTAKE 7: (α − β)² ≠ (α+β)² − αβ

**Wrong:** $(\alpha-\beta)^2 = (\alpha+\beta)^2 - \alpha\beta$
**Right:** $(\alpha-\beta)^2 = (\alpha+\beta)^2 - 4\alpha\beta = S^2 - 4P$

---

### ⚠️ MISTAKE 8: Perfect Square Factoring Error

**Wrong:** $(x+3)^2 = x^2 + 9$
**Right:** $(x+3)^2 = x^2 + 6x + 9$ (don't forget 2ab middle term!)

---

### ⚠️ MISTAKE 9: α³+β³ Using Wrong Formula

**Wrong:** $\alpha^3+\beta^3 = (\alpha+\beta)^3$
**Right:** $\alpha^3+\beta^3 = (\alpha+\beta)^3 - 3\alpha\beta(\alpha+\beta) = S^3 - 3PS$

---

### ⚠️ MISTAKE 10: Using Factoring When D is Not a Perfect Square

If D is not a perfect square (like D = 5, 7, 11), factoring won't give clean rational roots. Use the Quadratic Formula instead.

### Problem Type → Best Approach:

| Problem | Approach |
|---------|---------|
| Find roots | Factor or Quadratic Formula |
| Nature of roots | Compute D = b²-4ac |
| Sum/Product of roots | S = -b/a, P = c/a (don't solve!) |
| α²+β², α³+β³ | Express in terms of S and P |
| Form equation from roots | x² - Sx + P = 0 |
| Equal roots condition | Set D = 0, solve for parameter |
| Real roots condition | Set D ≥ 0, solve inequality |
| Max/min value | x = -b/2a, evaluate at vertex |
| Biquadratic | Substitute t = x² |
| Reciprocal equation | Substitute t = x + 1/x |
| Exponential quadratic | Substitute t = aˣ |

## 🏆 TOP EXAM TIPS

1. **Sum = -b/a (MINUS b over a)** — The minus sign is the most forgotten part. Always write it first.
2. **D decides everything about roots** — Compute D before doing anything else in any quadratic problem.
3. **Never divide by variable** — $x^2 = 5x$ → factor, don't divide by x (you'll lose x=0).
4. **Always verify in equations with square roots** — Check for extraneous solutions after squaring.
5. **Form equation: x² − Sx + P = 0** — Minus the sum, plus the product.
6. **Symmetric functions in S and P** — Express everything in terms of S=α+β and P=αβ.
7. **Equal roots → D = 0** — The most common "find k" exam question.
8. **Biquadratic: always use t = x²** — Then solve quadratic, then square root.
9. **Reciprocal equations: use t = x + 1/x** — Then t² = x² + 2 + 1/x², so x² + 1/x² = t² - 2.
10. **Both roots positive** — Need ALL THREE: S>0 AND P>0 AND D≥0.
