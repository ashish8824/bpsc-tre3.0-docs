## 1. WHAT IS AVERAGE? — FOUNDATION

### 1.1 The Concept

> **Average** (also called **Arithmetic Mean**) is a single number that represents or summarises a group of numbers. It is the value each number would have if they were all made equal.

```
Imagine 5 people have different amounts of money:
  Person A: ₹10
  Person B: ₹20
  Person C: ₹30
  Person D: ₹40
  Person E: ₹50

Total = ₹150. If we redistribute equally:
  Each gets 150/5 = ₹30

  ₹30 is the AVERAGE — the "fair share" if all were equal.
```

### 1.2 Real-Life Meaning

```
Average tells us the CENTRAL TENDENCY — where values cluster.

  • Average marks → how well the class performed overall
  • Average speed → representative speed for entire journey
  • Average age → typical age of a group
  • Average income → typical earning in a population
  • Batting average → typical runs per innings
```

### 1.3 Key Intuitive Facts

```
FACT 1: Average always lies BETWEEN the minimum and maximum values.
  If values are 10, 20, 30, 40, 50 → Average=30, lies between 10 and 50 ✓

FACT 2: If all numbers are equal, average = that number.
  5, 5, 5, 5, 5 → Average = 5 ✓

FACT 3: Average can be a DECIMAL even if all numbers are integers.
  1, 2, 4 → Average = 7/3 = 2.33... ✓

FACT 4: Sum = Average × Count (ALWAYS — this is your most used tool)

FACT 5: Average is affected by EVERY value in the set.
  (Unlike median or mode which may not be)
```

## 2. CORE FORMULA & VARIATIONS

### 2.1 The Master Formula

```
╔══════════════════════════════════════════════════════════════╗
║                                                              ║
║         Average = Sum of all values                         ║
║                   ─────────────────                         ║
║                   Number of values                          ║
║                                                              ║
║              A  =  S / n                                     ║
║                                                              ║
║   Rearranged:                                               ║
║     Sum    S  =  A × n          ← USE THIS MOST             ║
║     Count  n  =  S / A                                       ║
║                                                              ║
╚══════════════════════════════════════════════════════════════╝
```

### 2.2 Step-by-Step Basic Examples

```
Example 1: Find average of 15, 25, 35, 45, 55.
  Sum = 15+25+35+45+55 = 175
  Count = 5
  Average = 175/5 = 35 ✓

Example 2: Average of 7 numbers is 24. Find their sum.
  Sum = Average × Count = 24 × 7 = 168 ✓

Example 3: Sum of 9 observations is 315. Find average.
  Average = 315/9 = 35 ✓
```

### 2.3 Finding Missing Number Using Average

```
Example: Average of 5 numbers is 40. Four of them are
         30, 42, 38, 46. Find the fifth number.

  Sum of all 5 = 40 × 5 = 200
  Sum of 4 known = 30+42+38+46 = 156
  Fifth number = 200 − 156 = 44 ✓

TEMPLATE:
  Missing number = (Average × Total count) − (Sum of known numbers)
```

### 2.4 Finding New Average When One Changes

```
Example: Average of 6 numbers is 30. If one number 24 is
         replaced by 42, find new average.

  Old sum = 30 × 6 = 180
  New sum = 180 − 24 + 42 = 198
  New average = 198/6 = 33 ✓

SHORTCUT: Change in average = Change in number / Total count
  Change in number = 42−24 = 18
  Change in average = 18/6 = 3
  New average = 30 + 3 = 33 ✓
```

## 3. AVERAGE OF SPECIAL NUMBER SERIES

> These are HIGH-FREQUENCY exam shortcuts. Memorise all of them.

### 3.1 Average of Consecutive Natural Numbers

```
╔══════════════════════════════════════════════════════════════╗
║   Average of consecutive natural numbers from 1 to n:       ║
║                                                              ║
║              Average = (n + 1) / 2                          ║
║                                                              ║
║   Examples:                                                  ║
║     1 to 10:  (10+1)/2 = 5.5   ✓                           ║
║     1 to 20:  (20+1)/2 = 10.5  ✓                           ║
║     1 to 99:  (99+1)/2 = 50    ✓                           ║
║     1 to 100: (100+1)/2 = 50.5 ✓                           ║
╚══════════════════════════════════════════════════════════════╝
```

### 3.2 Average of Consecutive Numbers (Any Range)

```
╔══════════════════════════════════════════════════════════════╗
║   Average of consecutive integers from a to b:               ║
║                                                              ║
║              Average = (a + b) / 2                          ║
║                                                              ║
║   Examples:                                                  ║
║     10 to 30:  (10+30)/2 = 20  ✓                           ║
║     15 to 75:  (15+75)/2 = 45  ✓                           ║
║     31 to 50:  (31+50)/2 = 40.5 ✓                          ║
╚══════════════════════════════════════════════════════════════╝

WHY? Consecutive numbers are symmetric around their middle.
  10, 11, 12, 13, 14 → Middle = 12, Average = (10+14)/2 = 12 ✓
```

### 3.3 Average of Odd Numbers from 1 to n

```
╔══════════════════════════════════════════════════════════════╗
║   Average of first n odd numbers (1,3,5,...,2n−1):          ║
║                                                              ║
║              Average = n                                     ║
║                                                              ║
║   Examples:                                                  ║
║     First 5 odd (1,3,5,7,9):    Avg = 5    ✓               ║
║     First 10 odd (1,3,...,19):  Avg = 10   ✓               ║
║     First 50 odd:               Avg = 50   ✓               ║
╚══════════════════════════════════════════════════════════════╝
```

### 3.4 Average of Even Numbers from 2 to n

```
╔══════════════════════════════════════════════════════════════╗
║   Average of first n even numbers (2,4,6,...,2n):           ║
║                                                              ║
║              Average = n + 1                                 ║
║                                                              ║
║   Examples:                                                  ║
║     First 5 even (2,4,6,8,10):   Avg = 6   ✓               ║
║     First 10 even (2,4,...,20):  Avg = 11  ✓               ║
║     First n even numbers:        Avg = n+1 ✓                ║
╚══════════════════════════════════════════════════════════════╝
```

### 3.5 Average of Squares of First n Natural Numbers

```
   Average of (1² + 2² + 3² + ... + n²) = (n+1)(2n+1) / 6

   Example: Average of squares of 1 to 5:
     = (5+1)(2×5+1)/6 = 6×11/6 = 11 ✓
     Verify: (1+4+9+16+25)/5 = 55/5 = 11 ✓
```

### 3.6 Average of Cubes of First n Natural Numbers

```
   Average of (1³ + 2³ + ... + n³) = n(n+1)² / 4

   Example: Average of cubes of 1 to 4:
     = 4×25/4 = 25 ✓
     Verify: (1+8+27+64)/4 = 100/4 = 25 ✓
```

### 3.7 Average of Arithmetic Progression (AP)

```
Any AP: a, a+d, a+2d, ..., l  (first term a, last term l)

   Average of AP = (First term + Last term) / 2
                 = (a + l) / 2

   Also = Middle term (if odd number of terms)

   Example: 5, 10, 15, 20, 25, 30, 35
     Average = (5+35)/2 = 20 ✓
     Middle term = 20 ✓  (4th of 7 terms)

   Example: 4, 8, 12, 16, 20
     Average = (4+20)/2 = 12 ✓
```

## 4. EFFECT OF ADDING/REMOVING A NUMBER

### 4.1 When a New Number is ADDED to the Group

```
╔══════════════════════════════════════════════════════════════╗
║   If average of n numbers = A                                ║
║   A new number x is added                                    ║
║                                                              ║
║   New sum     = A×n + x                                      ║
║   New count   = n + 1                                        ║
║   New average = (A×n + x) / (n+1)                           ║
║                                                              ║
║   Change in average = (x − A) / (n+1)                       ║
║     If x > A: average INCREASES                              ║
║     If x < A: average DECREASES                              ║
║     If x = A: average UNCHANGED                              ║
╚══════════════════════════════════════════════════════════════╝
```

```
Example: Average of 8 numbers = 25.
         A 9th number = 35 is added. Find new average.

Method 1 (Formula):
  New average = (25×8 + 35)/9 = (200+35)/9 = 235/9 = 26.11

Method 2 (Change method):
  x=35 > A=25 → average increases
  Change = (35−25)/(8+1) = 10/9 ≈ 1.11
  New average = 25 + 10/9 ≈ 26.11 ✓

Example: Average of 10 numbers = 50.
         A number 20 is added. New average?
  Change = (20−50)/11 = −30/11 ≈ −2.73
  New average = 50 − 30/11 ≈ 47.27 ✓
```

### 4.2 When a Number is REMOVED from the Group

```
   If average of n numbers = A
   Number x is removed

   New sum     = A×n − x
   New count   = n − 1
   New average = (A×n − x) / (n−1)

   Change in average = (A − x) / (n−1)
     If x < A: average INCREASES (removing below-average number)
     If x > A: average DECREASES (removing above-average number)
```

```
Example: Average of 6 numbers = 30.
         Number 18 is removed. New average?

  New average = (30×6 − 18)/(6−1) = (180−18)/5 = 162/5 = 32.4

  Intuition: 18 < 30 → removing it should INCREASE average ✓
```

## 5. EFFECT OF REPLACING A NUMBER

```
╔══════════════════════════════════════════════════════════════╗
║   If old number x is replaced by new number y:              ║
║                                                              ║
║   Change in sum     = y − x                                  ║
║   Change in average = (y − x) / n                           ║
║                                                              ║
║   New average = Old average + (y−x)/n                       ║
╚══════════════════════════════════════════════════════════════╝
```

### 5.1 Finding the Replaced Number

```
Example: Average of 8 numbers = 40. If one number 56 is
         replaced by another number, new average = 38.
         Find the new number.

  Change in average = 38 − 40 = −2
  Change in sum = −2 × 8 = −16
  New number = Old number + Change in sum
             = 56 + (−16) = 40 ✓

  Verify: Change = (40−56)/8 = −16/8 = −2 ✓
```

```
Example: Average of 5 students' weight = 50 kg.
         When one student weighing 60 kg leaves and another joins,
         average becomes 52 kg. Find weight of new student.

  Change in average = 52 − 50 = +2
  Change in sum = 2 × 5 = +10
  New student's weight = 60 + 10 = 70 kg ✓

  Or: New student = Old out + (Change in avg × n)
                  = 60 + (2 × 5) = 70 kg ✓
```

### 5.2 The Universal Replacement Formula

```
╔══════════════════════════════════════════════════════════════╗
║                                                              ║
║  New number = Out number + (New avg − Old avg) × n          ║
║                                                              ║
║  Out number = New number − (New avg − Old avg) × n          ║
║                                                              ║
╚══════════════════════════════════════════════════════════════╝

Examples:
  New avg > Old avg → New number > Out number
  New avg < Old avg → New number < Out number
```

## 6. WEIGHTED AVERAGE

### 6.1 What is Weighted Average?

> When different groups have **different sizes** (weights), we can't just average the averages. We must use Weighted Average.

```
╔══════════════════════════════════════════════════════════════╗
║                                                              ║
║   Weighted Average = (w₁×a₁ + w₂×a₂ + ... + wₙ×aₙ)        ║
║                      ─────────────────────────────           ║
║                           w₁ + w₂ + ... + wₙ               ║
║                                                              ║
║   where wᵢ = weight (size/count) of each group              ║
║         aᵢ = average (value) of each group                  ║
╚══════════════════════════════════════════════════════════════╝
```

### 6.2 Why Simple Average Fails

```
Class A: 30 students, average marks = 70
Class B: 20 students, average marks = 80

WRONG: Simple average = (70+80)/2 = 75
RIGHT: Weighted average = (30×70 + 20×80)/(30+20)
                        = (2100+1600)/50
                        = 3700/50 = 74 ✓

The class with MORE students (A) pulls average CLOSER to 70.
```

### 6.3 Weighted Average Examples

```
Example: A person travels 3 hours at 40 km/h, 2 hours at 60 km/h,
         and 1 hour at 90 km/h. Find average speed.

  (Since TIMES are equal weights here, not distances)
  Weighted average speed = (3×40 + 2×60 + 1×90) / (3+2+1)
                         = (120+120+90)/6
                         = 330/6 = 55 km/h ✓
```

```
Example: Mixture: 5 kg of rice at ₹20/kg and 3 kg at ₹30/kg.
         Find average price of mixture.

  Average price = (5×20 + 3×30)/(5+3)
               = (100+90)/8
               = 190/8 = ₹23.75/kg ✓
```

### 6.4 Weighted Average Using Alligation (Fast Method)

```
When combining ONLY two groups:
  Use ratio method → no need to multiply large numbers.

Class A: n₁=30, avg=70; Class B: n₂=20, avg=80; Combined avg=?
  By alligation: difference from 70 = (n₂/n₁+n₂)×(80−70)
  Or: Avg = 70 + [20/(30+20)]×(80−70) = 70 + 4 = 74 ✓
```

## 7. AVERAGE SPEED

> Average Speed is the MOST MISUNDERSTOOD concept. Read carefully.

### 7.1 The Critical Rule

```
╔══════════════════════════════════════════════════════════════╗
║                                                              ║
║   Average Speed ≠ Average of Speeds (almost always)         ║
║                                                              ║
║   Average Speed = Total Distance / Total Time               ║
║                                                              ║
╚══════════════════════════════════════════════════════════════╝
```

### 7.2 Case 1: Same Distance at Different Speeds

```
Person travels from A to B at speed u and returns at speed v.
(SAME distance, different speeds)

   Average Speed = 2uv / (u + v)   ← Harmonic Mean formula

Example: Goes at 60 km/h, returns at 40 km/h.
  Average speed = 2×60×40/(60+40) = 4800/100 = 48 km/h ✓

IMPORTANT: Average speed (48) < Arithmetic mean of speeds (50)
  (Always less when distances are equal — harmonic mean < arithmetic mean)
```

### 7.3 Case 2: Same Time at Different Speeds

```
Person travels for equal TIME at speeds u and v.

   Average Speed = (u + v) / 2   ← Simple Arithmetic Mean

Example: Travels 2 hours at 60 km/h and 2 hours at 40 km/h.
  Average speed = (60+40)/2 = 50 km/h ✓

  Verify: Total distance = 2×60+2×40=200; Total time=4; Avg=50 ✓
```

### 7.4 Case 3: Three Segments (General)

```
   Average Speed = Total Distance / Total Time
                 = (d₁+d₂+d₃) / (d₁/s₁ + d₂/s₂ + d₃/s₃)

Example: Travels 60 km at 30 km/h, 90 km at 45 km/h, 120 km at 60 km/h.
  Total distance = 270 km
  Total time = 60/30 + 90/45 + 120/60 = 2+2+2 = 6 hours
  Average speed = 270/6 = 45 km/h ✓
```

### 7.5 Special Case: Three Equal Parts of Distance

```
If a journey is divided into THREE equal parts of distance
at speeds a, b, c:

   Average speed = 3abc / (ab+bc+ca)

Example: Speeds 30, 40, 60 for equal distances.
  = 3×30×40×60 / (30×40+40×60+60×30)
  = 216000 / (1200+2400+1800)
  = 216000/5400 = 40 km/h ✓
```

## 8. AVERAGE IN AGES PROBLEMS

### 8.1 Basic Age Average

```
Example: Average age of 5 family members = 32 years.
         Youngest member's age = 8 years.
         Find average age at time of youngest's birth.

  Current sum of ages = 32 × 5 = 160
  8 years ago, each person was 8 years younger:
  Sum 8 years ago = 160 − 8×5 = 160 − 40 = 120
  Average 8 years ago = 120/5 = 24 years ✓
```

### 8.2 New Member Joins/Leaves

```
Example: Average age of 6 members = 40 years.
         A new baby is born. New average = ?
         (Baby's age = 0)

  New average = (40×6 + 0)/7 = 240/7 ≈ 34.3 years ✓

  Change = (0−40)/7 = −40/7 ≈ −5.7 years
  New average = 40 − 5.7 = 34.3 years ✓
```

### 8.3 Replacing One Member

```
Example: Average age of 10 employees = 35 years.
         One employee aged 55 retires, replaced by a new employee.
         New average = 32 years. Find new employee's age.

  New employee's age = 55 + (32−35)×10
                     = 55 + (−30)
                     = 25 years ✓

  Verify: Sum changes by 32×10 − 35×10 = 320−350 = −30
          New employee = 55−30 = 25 ✓
```

### 8.4 Age Problems with Time Passage

```
GOLDEN RULE: After k years, average age of a group of n people
             INCREASES by exactly k years (if no one leaves/joins).

Example: Average age now = 30. Average age after 5 years?
  = 30 + 5 = 35 ✓ (Each person gets 5 years older)

Example: Average age 5 years ago = 25. Average age now?
  = 25 + 5 = 30 ✓
```

### 8.5 Tricky Age Problem

```
Example: Average of ages of a father and his 2 children = 27.
         Mother's age is 3 years more than father's.
         Average age of all 4 = 28. Find mother's age.

  Sum of father + 2 children = 27×3 = 81
  Sum of all 4 = 28×4 = 112
  Mother's age = 112−81 = 31 years ✓

  Father's age = 31−3 = 28 years
  Check: sum of children = 81−28 = 53 (valid, each ≈ 26.5)
```

## 9. AVERAGE IN SCORES / MARKS PROBLEMS

### 9.1 Score Average with Error Correction

```
Example: A student's average for 10 subjects = 75.
         He found marks of one subject were read as 56 instead of 65.
         Find correct average.

  Wrong sum = 75×10 = 750
  Correct sum = 750 − 56 + 65 = 759
  Correct average = 759/10 = 75.9 ✓

SHORTCUT:
  Change in average = (65−56)/10 = 9/10 = 0.9
  Correct average = 75 + 0.9 = 75.9 ✓
```

### 9.2 Average Marks Across Subjects

```
Example: In 5 subjects, a student scored:
  Maths=90, Science=85, English=70, History=65, Hindi=75.
  Average marks = (90+85+70+65+75)/5 = 385/5 = 77 ✓
```

### 9.3 Finding How Many Subjects to Achieve Target Average

```
Example: Average of 5 subjects = 70. What should be the
         score in 6th subject to make average 75?

  Required sum of 6 = 75×6 = 450
  Current sum of 5 = 70×5 = 350
  Score needed in 6th = 450−350 = 100

  But maximum marks = 100, so just achievable. ✓
```

## 10. AVERAGE IN CRICKET / SPORTS PROBLEMS

### 10.1 Batting Average

```
Batting Average = Total runs / Number of innings (not out handled separately)

If not-outs: Batting Average = Total runs / Innings where batsman got out
```

### 10.2 Classic Cricket Average Problem

```
Example: A cricketer's average after 20 innings = 35.
         He scores 87 in the 21st innings.
         Find new average.

  Old sum = 35×20 = 700
  New sum = 700+87 = 787
  New average = 787/21 = 37.48 ✓

SHORTCUT:
  Change = 87−35 = 52 (extra runs above average)
  Increase in average = 52/21 ≈ 2.48
  New average = 35 + 52/21 ≈ 37.48 ✓
```

### 10.3 Average to be Maintained

```
Example: After 15 innings, average = 42. To achieve overall
         average of 44 after 20 innings, what average is needed
         in last 5 innings?

  Required total = 44×20 = 880
  Current total  = 42×15 = 630
  Runs needed in 5 innings = 880−630 = 250
  Average needed = 250/5 = 50 ✓
```

### 10.4 Average Rises/Falls by Certain Runs

```
Example: After n innings, average = A. After one more innings
         with score S, average rises by k.

  New average = A + k = (A×n + S)/(n+1)
  A(n+1) + k(n+1) = An + S
  An + A + kn + k = An + S
  S = A + k(n+1) = A + kn + k

FORMULA: Score in new innings = Old average + k×(n+1)
         where k = rise in average, n = old number of innings

Example: Average after 16 innings = 30. Average rises by 2.
         Score in 17th innings?
  S = 30 + 2×17 = 30+34 = 64 ✓
```

## 11. AVERAGE OF GROUPS COMBINED

### 11.1 Two Groups Combined

```
╔══════════════════════════════════════════════════════════════╗
║   Group 1: n₁ members, average A₁                           ║
║   Group 2: n₂ members, average A₂                           ║
║                                                              ║
║   Combined average = (n₁A₁ + n₂A₂) / (n₁ + n₂)            ║
╚══════════════════════════════════════════════════════════════╝
```

```
Example: Class A: 25 students, avg marks = 72
         Class B: 35 students, avg marks = 80
         Combined average?

  = (25×72 + 35×80)/(25+35)
  = (1800+2800)/60
  = 4600/60 = 76.67 ✓
```

### 11.2 Three Groups Combined

```
Combined average = (n₁A₁ + n₂A₂ + n₃A₃) / (n₁+n₂+n₃)

Example: 3 sections: 30 at avg 60, 40 at avg 70, 30 at avg 80.
  = (30×60 + 40×70 + 30×80)/100
  = (1800+2800+2400)/100
  = 7000/100 = 70 ✓
```

### 11.3 Finding One Group's Average When Combined is Known

```
Example: Two groups: n₁=20, A₁=?, n₂=30, A₂=60.
         Combined average = 54. Find A₁.

  20×A₁ + 30×60 = 54×50
  20A₁ + 1800 = 2700
  20A₁ = 900
  A₁ = 45 ✓
```

## 12. DEVIATION METHOD (ASSUMED MEAN)

> This is the FASTEST method for calculating averages when numbers are close together.

### 12.1 The Method

```
Step 1: Choose any convenient number as ASSUMED MEAN (A).
        (Best choice: middle value or any round number)
Step 2: Find deviation of each number from A:
        dᵢ = xᵢ − A
Step 3: Find average of deviations: d̄ = Σdᵢ / n
Step 4: Actual Average = A + d̄
```

### 12.2 Worked Example

```
Find average of: 47, 53, 51, 49, 55, 52, 48

Assumed Mean A = 50 (convenient round number)

Deviations:
  47−50 = −3
  53−50 = +3
  51−50 = +1
  49−50 = −1
  55−50 = +5
  52−50 = +2
  48−50 = −2
  ─────────
  Sum = −3+3+1−1+5+2−2 = +5

Average of deviations = 5/7 ≈ 0.71

Actual Average = 50 + 5/7 = 50.71 ✓

Verify: Sum=355, 355/7=50.71 ✓
```

### 12.3 Why This is Faster

```
Instead of adding large numbers (47+53+51+49+55+52+48=355),
you add small deviations (−3+3+1−1+5+2−2=5).

Much less chance of arithmetic error!
```

## 13. ALL WORD PROBLEM PATTERNS

---

### Pattern 1: Average After Adding/Removing

```
Q: Average of 10 numbers = 15. If 20 is added, new average?
  New avg = (15×10+20)/11 = 170/11 ≈ 15.45

Q: Average of 6 = 24. Remove 36. New average?
  New avg = (24×6−36)/5 = (144−36)/5 = 108/5 = 21.6
```

### Pattern 2: Student Passes/Fails Marks

```
Q: Average marks of 30 students = 60. If 5 students failed
   (each scored 20% of average), find average of passed students.

  Total marks = 1800
  Failed students' marks = 5 × (20% of 60) = 5×12 = 60
  Passed students' total = 1800−60 = 1740
  Average of passed = 1740/25 = 69.6 ✓
```

### Pattern 3: Consecutive/Specific Number Problems

```
Q: Average of 5 consecutive odd numbers = 31. Find largest.
  For consecutive odd: average = middle number = 31
  Numbers: 27, 29, 31, 33, 35
  Largest = 35 ✓

Q: Average of 4 consecutive even numbers = 27.
  But 4 consecutive evens can't average to 27 (would be 27.5 or 28)
  If avg=28: middle would be 27.5 → numbers 26,28,30,32 → avg=29 ✗
  Numbers: 25,27,29,31 (consecutive odd avg 28 works)
```

### Pattern 4: Two Groups, One Unknown

```
Q: Average of boys = 70, girls = 65, class = 68.
   Ratio of boys to girls?

  Let boys = b, girls = g
  70b + 65g = 68(b+g)
  70b + 65g = 68b + 68g
  2b = 3g
  b:g = 3:2 ✓
```

### Pattern 5: Error Correction

```
Q: Average of 20 readings = 56. One reading misread as 73
   instead of 37. Correct average?

  Correct sum = 56×20 − 73 + 37 = 1120 − 36 = 1084
  Correct average = 1084/20 = 54.2 ✓

SHORTCUT: Change = (37−73)/20 = −36/20 = −1.8
  Correct avg = 56 − 1.8 = 54.2 ✓
```

### Pattern 6: Average of Averages (Trap!)

```
Q: A scores average 70 in 5 tests, B scores average 80 in 3 tests.
   Overall average of all 8 tests?

  RIGHT: (70×5 + 80×3)/8 = (350+240)/8 = 590/8 = 73.75 ✓
```

```
Q: Average of 15 numbers = 40. When one number is removed,
```

## 14. SHORT TRICKS & SPEED METHODS

---

### ⚡ Trick 1: The "Sum = Average × Count" Power Tool

```
ALWAYS convert average problems to SUM problems first.
Sum is concrete, average is abstract.

  "Average = 40, count = 7"  →  Sum = 280  (now you can work!)
  "New member joins at value 60, sum becomes 340, count = 8"
  → New average = 340/8 = 42.5  ✓
```

---

### ⚡ Trick 2: Change Method for Additions/Removals

```
Instead of computing new sum, just track the CHANGE:

  Average = A, adding number x to group of n:
  New avg = A + (x − A)/(n+1)

  Average = A, removing number x from group of n:
  New avg = A + (A − x)/(n−1)   [or: A − (x−A)/(n−1)]

  Average = A, replacing x with y in group of n:
  New avg = A + (y − x)/n
```

---

### ⚡ Trick 3: Consecutive Numbers → Average = Middle Value

```
Any SET of consecutive numbers (or any AP):
  Average = (First + Last)/2 = Middle value

  11, 12, 13, 14, 15 → Average = 13 (middle) ✓
  10, 14, 18, 22, 26 → Average = (10+26)/2 = 18 (middle) ✓

  For EVEN count of AP:
  Average = (two middle values)/2
  12, 14, 16, 18 → Average = (14+16)/2 = 15 ✓
```

---

### ⚡ Trick 4: Age Average Increases Exactly by Time Elapsed

```
If NO ONE joins or leaves:
  Average age AFTER k years = Average age NOW + k

  No calculation needed!
  Average now = 35 → Average in 10 years = 45  ✓
```

---

### ⚡ Trick 5: Same Distance, Different Speeds → Harmonic Mean

```
MEMORIZE: Equal distances at speeds u and v:
  Average speed = 2uv/(u+v)

  40 km/h and 60 km/h: 2×40×60/100 = 4800/100 = 48  ✓
  30 km/h and 60 km/h: 2×30×60/90  = 3600/90  = 40  ✓
  u and 2u:            2×u×2u/3u   = 4u/3           ✓
```

---

### ⚡ Trick 6: New Score in Cricket to Raise Average by k

```
  Score needed = Old average + k × (n+1)
  where n = current number of innings

  Average after 20 innings = 40, want to raise by 3:
  Score needed = 40 + 3×21 = 40+63 = 103  ✓
```

---

### ⚡ Trick 7: Comparing Two Averages to Find Ratio

```
  Group 1: avg=A₁, Group 2: avg=A₂, Combined avg=A
  Ratio n₁:n₂ = (A₂−A):(A−A₁)   ← Alligation method

  A₁=60, A₂=80, A=68:
  Ratio = (80−68):(68−60) = 12:8 = 3:2  ✓
  (Group with avg 60 has 3 parts, group with avg 80 has 2 parts)
```

---

### ⚡ Trick 8: Assumed Mean for Fast Calculation

```
Pick a round number near the values. Add/subtract small deviations.

  Find avg of 98, 102, 97, 103, 100:
  Assume mean = 100
  Deviations: −2, +2, −3, +3, 0 → sum = 0
  Average = 100 + 0/5 = 100  ✓  (Done in seconds!)

  Find avg of 197, 201, 203, 199, 195:
  Assume 200. Deviations: −3,+1,+3,−1,−5 → sum=−5
  Average = 200 + (−5)/5 = 200−1 = 199  ✓
```

---

### ⚡ Trick 9: Total Runs / Total Amount from Average + Count

```
If average of first 5 = 60 and average of all 8 = 70:
  First 5 sum = 300
  All 8 sum = 560
  Last 3 sum = 260
  Average of last 3 = 260/3 ≈ 86.67  ✓
```

---

### ⚡ Trick 10: Error in Reading — Correct via Difference

```
If one value x was wrongly read as y:
  Correct average = Wrong average + (x−y)/n
                  = Wrong average − (y−x)/n

  x=actual, y=wrong reading, n=count

  Wrong avg=50, n=10, actual value 37, read as 73:
  Correct avg = 50 + (37−73)/10 = 50−3.6 = 46.4  ✓
```

---

### ⚡ Trick 11: Weighted Average Shortcut (Two Groups)

```
  Combined avg = A₁ + [n₂/(n₁+n₂)] × (A₂−A₁)
  (Pulls the combined average from A₁ toward A₂ proportionally)

  n₁=30, A₁=70; n₂=20, A₂=80:
  Combined = 70 + [20/50]×(80−70) = 70 + 4 = 74  ✓
```

---

### ⚡ Trick 12: Sum of Series Formulas for Average

```
1 to n:         Sum = n(n+1)/2    → Avg = (n+1)/2
Odd 1 to (2n−1): Sum = n²         → Avg = n
Even 2 to 2n:   Sum = n(n+1)      → Avg = n+1
Squares 1² to n²: Sum=n(n+1)(2n+1)/6 → Avg=(n+1)(2n+1)/6
Cubes 1³ to n³:  Sum=[n(n+1)/2]²  → Avg=n(n+1)²/4
```

---

### ⚡ Trick 13: Average of Remaining After Removing Best/Worst

```
Q: Average of n = A. If highest (H) is removed, avg of rest?
  Rest avg = (A×n − H)/(n−1)

Q: If both highest H and lowest L are removed, avg of rest?
  Rest avg = (A×n − H − L)/(n−2)
```

---

### ⚡ Trick 14: Average Profit/Loss Per Day/Month

```
Q: Shop's average daily sale for 30 days = ₹500.
   For first 10 days avg = ₹450, last 10 days avg = ₹525.
   Find average for middle 10 days.

  Total = 500×30 = 15000
  First 10 = 450×10 = 4500
  Last 10  = 525×10 = 5250
  Middle 10 = 15000−4500−5250 = 5250
  Middle avg = 5250/10 = ₹525  ✓
```

---

### ⚡ Trick 15: Minimum/Maximum Value Constraints

```
Q: Average of 7 numbers = 50. One number = 90. Maximum possible
   value of another number if all others are at least 30?

  Sum = 350. One number = 90. Remaining 6 sum = 260.
  To MAXIMISE one, MINIMISE the other 5:
  5 numbers at 30 → their sum = 150
  Max value = 260−150 = 110  ✓
```

## 15. COMMON MISTAKES TO AVOID

| ❌ Mistake | ✅ Correct Approach | Example |
|-----------|---------------------|---------|
| Average of averages = simple mean | Use weighted average (multiply by counts) | Class A (30, avg 70) and B (20, avg 80) → not (70+80)/2=75 |
| Average speed = (s₁+s₂)/2 always | Only if time is equal; use 2s₁s₂/(s₁+s₂) for equal distance | 40 and 60 same dist → avg=48 not 50 |
| Adding a number doesn't change count | Count increases by 1 | Avg of 5 → becomes avg of 6 |
| Age average after years doesn't change | Increases by exact number of years | Avg 30 today → avg 35 in 5 years |
| New member replaces: same sum | Sum changes by (new − old) value | Remove 60, add 80: sum +20, avg +20/n |
| Consecutive numbers: avg=first+last | Avg=(first+last)/2 NOT first+last | 10 to 20: avg=15, not 30 |
| n numbers, remove 1: still n | Count reduces to n−1 | Be careful with denominator |
| Error correction: add wrong difference | Correct avg = wrong avg + (actual−wrong)/n | Read 73 for 37: diff=37−73=−36, divide by n |
| Cricket: average per innings includes not-outs | Not-out innings not counted in denominator | Batting avg handles this specially |
| Weighted avg: weight is count, not value | Weight = number of items/people | 5 kg at ₹20: weight=5, value=20 |
