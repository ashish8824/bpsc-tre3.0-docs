## 1. THE BIG PICTURE — INDICES & SURDS OVERVIEW

---

### 🔑 What are Indices?

> **Indices** (singular: Index) tell us **how many times a number (base) is multiplied by itself**.

$$a^n = \underbrace{a \times a \times a \times \ldots \times a}_{n \text{ times}}$$

- **a** = base
- **n** = index (exponent, power)

---

### 🔑 What are Surds?

> **Surds** are **irrational roots** of rational numbers that cannot be simplified into a rational number.

$$\sqrt{2}, \quad \sqrt{3}, \quad \sqrt[3]{5}, \quad 2\sqrt{7}, \quad \sqrt[4]{11}$$

**Not surds:**
- √4 = 2 (rational) ❌
- √9 = 3 (rational) ❌
- ∛8 = 2 (rational) ❌

---

### The Connection Between Surds and Indices:

$$\sqrt[n]{a} = a^{1/n}$$

$$\sqrt[n]{a^m} = a^{m/n}$$

> Every surd can be written as an index with a fractional exponent!

---

### Overview Diagram:

```
NUMBER TYPES
│
├── RATIONAL (p/q form)
│   ├── Integers: ..., -2, -1, 0, 1, 2, ...
│   └── Fractions: 1/2, 3/4, 7/5, ...
│
└── IRRATIONAL
    ├── Surds: √2, √3, √5, ∛2, ...   ← Today's topic
    └── Non-surds: π, e, ...

INDICES connect these:
    Rational index → may be rational or surd
    Fractional index = surd notation
```

| 3⁴ | 3 × 3 × 3 × 3 | 81 |

---

### Special Index Values — MEMORISE ALL:

#### Zero Exponent:
$$\boxed{a^0 = 1} \quad \text{for any } a \neq 0$$

> Why? a^n / a^n = a^(n-n) = a^0. But a^n/a^n = 1. So a^0 = 1.

#### Negative Exponent:
$$\boxed{a^{-n} = \frac{1}{a^n}}$$

> Example: 2⁻³ = 1/2³ = 1/8

#### Fractional Exponent:
$$\boxed{a^{1/n} = \sqrt[n]{a}}$$

$$\boxed{a^{m/n} = \sqrt[n]{a^m} = \left(\sqrt[n]{a}\right)^m}$$

> Example: 8^(2/3) = (∛8)² = 2² = 4

#### Unit Exponent:
$$a^1 = a$$

## 3. LAWS OF INDICES — ALL RULES

---

### THE COMPLETE SET OF LAWS:

| Law | Rule | Example |
|-----|------|---------|
| **Product Rule** | $a^m \times a^n = a^{m+n}$ | $2^3 \times 2^4 = 2^7$ |
| **Quotient Rule** | $a^m \div a^n = a^{m-n}$ | $3^5 \div 3^2 = 3^3$ |
| **Power of Power** | $(a^m)^n = a^{mn}$ | $(2^3)^4 = 2^{12}$ |
| **Power of Product** | $(ab)^n = a^n b^n$ | $(2\times3)^4 = 2^4 \times 3^4$ |
| **Power of Quotient** | $(a/b)^n = a^n/b^n$ | $(2/3)^3 = 8/27$ |
| **Zero Power** | $a^0 = 1$ | $100^0 = 1$ |
| **Negative Power** | $a^{-n} = 1/a^n$ | $5^{-2} = 1/25$ |
| **Fractional Power** | $a^{m/n} = \sqrt[n]{a^m}$ | $8^{2/3} = 4$ |
| **Root as Power** | $\sqrt[n]{a} = a^{1/n}$ | $\sqrt[4]{16} = 16^{1/4} = 2$ |

---

### Extended Laws:

$$a^m \times b^m = (ab)^m$$

$$\frac{a^m}{b^m} = \left(\frac{a}{b}\right)^m$$

$$a^{-m} = \left(\frac{1}{a}\right)^m = \frac{1}{a^m}$$

$$\left(\frac{a}{b}\right)^{-n} = \left(\frac{b}{a}\right)^n = \frac{b^n}{a^n}$$

---

### The Most Important Identity for Exams:

$$\boxed{a^x = a^y \Rightarrow x = y \quad \text{(when } a \neq 0, 1, -1\text{)}}$$

$$\boxed{a^x = b^x \Rightarrow a = b \quad \text{(when } x \neq 0\text{)}}$$

## 4. INDICES — BASIC TO ADVANCED EXAMPLES

---

### ─── BASIC LEVEL ───

#### Example 1: Simplify Using Product Rule

> Simplify: $2^5 \times 2^3$

$$= 2^{5+3} = 2^8 = \textbf{256}$$

---

#### Example 2: Simplify Using Quotient Rule

> Simplify: $\dfrac{5^7}{5^3}$

$$= 5^{7-3} = 5^4 = \textbf{625}$$

---

#### Example 3: Power of Power

> Simplify: $(3^2)^4$

$$= 3^{2 \times 4} = 3^8 = \textbf{6561}$$

---

#### Example 4: Negative Exponent

> Simplify: $4^{-3}$

$$= \frac{1}{4^3} = \frac{1}{64}$$

---

#### Example 5: Fractional Exponent

> Evaluate: $27^{2/3}$

$$27^{2/3} = (\sqrt[3]{27})^2 = 3^2 = \textbf{9}$$

---

#### Example 6: Mixed

> Evaluate: $\left(\frac{4}{9}\right)^{-3/2}$

$$= \left(\frac{9}{4}\right)^{3/2} = \frac{9^{3/2}}{4^{3/2}} = \frac{(\sqrt{9})^3}{(\sqrt{4})^3} = \frac{3^3}{2^3} = \frac{27}{8}$$

---

### ─── INTERMEDIATE LEVEL ───

#### Example 7: Comparing Exponents

> If $2^x = 8$, find x.

$$2^x = 8 = 2^3 \Rightarrow x = \textbf{3}$$

---

#### Example 8: Solving Equations with Indices

> If $5^{2x-1} = 25^{x-3}$, find x.

$$5^{2x-1} = (5^2)^{x-3} = 5^{2x-6}$$

$$2x-1 = 2x-6 \Rightarrow -1 = -6$$

This gives a contradiction → **No solution** (Inconsistent).

**Corrected problem:** $5^{2x+1} = 25^{x-3}$:
$5^{2x+1} = 5^{2x-6}$ → $2x+1 = 2x-6$ → Still no solution!

Better: $5^{3x-1} = 25^{x-2}$:
$5^{3x-1} = 5^{2(x-2)} = 5^{2x-4}$
$3x-1 = 2x-4$ → **x = -3**

---

#### Example 9: Simplify Mixed Expression

> Simplify: $\dfrac{x^3 y^{-2} z}{x^{-1} y^4 z^{-2}}$

$$= x^{3-(-1)} \cdot y^{-2-4} \cdot z^{1-(-2)} = x^4 \cdot y^{-6} \cdot z^3 = \frac{x^4 z^3}{y^6}$$

---

#### Example 10: Simplify Powers of Same Base

> Simplify: $\dfrac{(2^3 \times 3^2)^4}{(6^2)^5}$

$$= \frac{2^{12} \times 3^8}{(2 \times 3)^{10}} = \frac{2^{12} \times 3^8}{2^{10} \times 3^{10}} = 2^{12-10} \times 3^{8-10} = 2^2 \times 3^{-2} = \frac{4}{9}$$

---

#### Example 11: If aˣ = b, bʸ = c, cᶻ = a — Find xyz

> If $a^x = b$, $b^y = c$, $c^z = a$, prove xyz = 1.

From $a^x = b$: $b = a^x$.
From $b^y = c$: $c = b^y = (a^x)^y = a^{xy}$.
From $c^z = a$: $a = c^z = (a^{xy})^z = a^{xyz}$.

Since $a^{xyz} = a^1 \Rightarrow xyz = \textbf{1}$

---

#### Example 12: Solving for x

> Find x if $\left(\dfrac{81}{16}\right)^x \times \left(\dfrac{4}{9}\right)^3 = \left(\dfrac{3}{2}\right)^{10}$

$$\left(\frac{3^4}{2^4}\right)^x \times \frac{2^6}{3^6} = \frac{3^{10}}{2^{10}}$$

$$\frac{3^{4x}}{2^{4x}} \times \frac{2^6}{3^6} = \frac{3^{10}}{2^{10}}$$

$$3^{4x-6} \times 2^{6-4x} = 3^{10} \times 2^{-10}$$

Equating powers of 3: 4x - 6 = 10 → **x = 4**
Verify powers of 2: 6 - 4x = -10 → 4x = 16 → **x = 4** ✓

---

### ─── ADVANCED LEVEL ───

#### Example 13: Complex Simplification

> Simplify: $\dfrac{(a^2 - b^2)^{1/2} \cdot (a+b)^{-3/2}}{(a-b)^{-1/2}}$

$$= (a^2-b^2)^{1/2} \cdot (a+b)^{-3/2} \cdot (a-b)^{1/2}$$

$$= [(a+b)(a-b)]^{1/2} \cdot (a+b)^{-3/2} \cdot (a-b)^{1/2}$$

$$= (a+b)^{1/2}(a-b)^{1/2} \cdot (a+b)^{-3/2} \cdot (a-b)^{1/2}$$

$$= (a+b)^{1/2-3/2} \cdot (a-b)^{1/2+1/2}$$

$$= (a+b)^{-1} \cdot (a-b)^1 = \frac{a-b}{a+b}$$

---

#### Example 14: Prove the Identity

> If $x = 3^{1/3} + 3^{-1/3}$, prove that $3x^3 - 9x = 10$.

$$x = 3^{1/3} + 3^{-1/3}$$

$$x^3 = (3^{1/3})^3 + (3^{-1/3})^3 + 3(3^{1/3})(3^{-1/3})(3^{1/3}+3^{-1/3})$$

$$= 3 + \frac{1}{3} + 3 \cdot 1 \cdot x = \frac{10}{3} + 3x$$

$$3x^3 = 10 + 9x \Rightarrow 3x^3 - 9x = 10 \quad \checkmark$$

## 5. SURDS — CORE CONCEPTS

---

### 🔑 Definition:

> A **surd** is an **nth root** of a positive rational number that is **irrational** (cannot be expressed as p/q).

$$\sqrt[n]{a} \text{ is a surd when: } a > 0, \quad a \text{ is rational, and } \sqrt[n]{a} \text{ is irrational}$$

---

### Types of Surds:

#### By Order (the 'n' in ⁿ√a):

| Type | Name | Example |
|------|------|---------|
| ²√a | **Quadratic surd** (square root) | √2, √7 |
| ³√a | **Cubic surd** (cube root) | ∛3, ∛5 |
| ⁴√a | **Biquadratic surd** | ⁴√5 |
| ⁿ√a | **n-th order surd** | General |

---

#### By Form:

| Type | Definition | Example |
|------|-----------|---------|
| **Simple surd** | Single surd term | √3, ∛7 |
| **Compound surd** | Sum/difference of two surds | √3 + √5 |
| **Binomial surd** | Sum of rational and surd | 2 + √3 |
| **Pure surd** | Rational number is 1 | √5 (as 1·√5) |
| **Mixed surd** | Rational number ≠ 1 | 3√5, 2∛7 |
| **Similar surds** | Same surd part | 2√3 and 5√3 |
| **Unlike surds** | Different surd parts | √3 and √5 |

---

#### Like/Similar vs Unlike Surds:

| Similar (Can add/subtract) | Unlike (Cannot directly combine) |
|---------------------------|----------------------------------|
| 3√2 + 5√2 = 8√2 | √2 + √3 (stays as is) |
| 4√5 − √5 = 3√5 | ∛2 + √2 (different orders) |

---

## 6. LAWS OF SURDS

---

### The Core Surd Laws:

| Law | Rule | Example |
|-----|------|---------|
| **Product** | $\sqrt{a} \times \sqrt{b} = \sqrt{ab}$ | $\sqrt{2}\times\sqrt{8} = \sqrt{16} = 4$ |
| **Quotient** | $\dfrac{\sqrt{a}}{\sqrt{b}} = \sqrt{\dfrac{a}{b}}$ | $\dfrac{\sqrt{50}}{\sqrt{2}} = \sqrt{25} = 5$ |
| **Power** | $(\sqrt[n]{a})^m = \sqrt[n]{a^m} = a^{m/n}$ | $(\sqrt{3})^4 = 9$ |
| **Root of Root** | $\sqrt[m]{\sqrt[n]{a}} = \sqrt[mn]{a} = a^{1/mn}$ | $\sqrt{\sqrt{3}} = 3^{1/4}$ |
| **n-th Power** | $(\sqrt[n]{a})^n = a$ | $(\sqrt{5})^2 = 5$ |

---

### Important: Order of a Surd

$$\sqrt[mn]{a^m} = \sqrt[n]{a}$$

> The order can be reduced when there's a common factor.

Example: $\sqrt[4]{9} = \sqrt[4]{3^2} = 3^{2/4} = 3^{1/2} = \sqrt{3}$

---

## 7. RATIONALISATION OF SURDS

---

### 🔑 Definition:

> **Rationalisation** = Making the **denominator rational** by multiplying by the appropriate conjugate.

---

### Three Cases:

#### Case 1: Single Surd in Denominator

$$\frac{1}{\sqrt{a}} \times \frac{\sqrt{a}}{\sqrt{a}} = \frac{\sqrt{a}}{a}$$

**Example:**
$$\frac{3}{\sqrt{5}} = \frac{3}{\sqrt{5}} \times \frac{\sqrt{5}}{\sqrt{5}} = \frac{3\sqrt{5}}{5}$$

---

#### Case 2: Binomial Denominator (a + √b) — Multiply by Conjugate (a − √b)

$$\frac{1}{a + \sqrt{b}} \times \frac{a - \sqrt{b}}{a - \sqrt{b}} = \frac{a - \sqrt{b}}{a^2 - b}$$

**Key formula:** $(a + \sqrt{b})(a - \sqrt{b}) = a^2 - b$

**Example:**
$$\frac{1}{2 + \sqrt{3}} = \frac{1}{2+\sqrt{3}} \times \frac{2-\sqrt{3}}{2-\sqrt{3}} = \frac{2-\sqrt{3}}{4-3} = 2-\sqrt{3}$$

---

#### Case 3: Two Surd Denominator (√a + √b) — Multiply by (√a − √b)

$$\frac{1}{\sqrt{a} + \sqrt{b}} \times \frac{\sqrt{a} - \sqrt{b}}{\sqrt{a} - \sqrt{b}} = \frac{\sqrt{a} - \sqrt{b}}{a - b}$$

**Example:**
$$\frac{1}{\sqrt{5} + \sqrt{3}} = \frac{\sqrt{5} - \sqrt{3}}{5-3} = \frac{\sqrt{5} - \sqrt{3}}{2}$$

---

### The Conjugate Rule:

| Expression | Conjugate | Product |
|-----------|-----------|---------|
| a + b | a − b | a² − b² |
| √a + √b | √a − √b | a − b |
| a + √b | a − √b | a² − b |
| √a + b | √a − b | a − b² |

## 8. OPERATIONS ON SURDS

---

### Addition and Subtraction:

**Only LIKE surds** (same order AND same radicand) can be combined.

$$3\sqrt{2} + 5\sqrt{2} = 8\sqrt{2}$$

$$4\sqrt{3} - \sqrt{3} = 3\sqrt{3}$$

$$\sqrt{12} + \sqrt{27} = 2\sqrt{3} + 3\sqrt{3} = 5\sqrt{3}$$

> Always simplify surds first before adding/subtracting!

---

### Multiplication of Surds:

$$\sqrt[m]{a} \times \sqrt[n]{b}$$

To multiply surds of **different orders**, convert to same order first:

$$\sqrt{a} \times \sqrt[3]{b} = a^{1/2} \times b^{1/3} = a^{3/6} \times b^{2/6} = \sqrt[6]{a^3} \times \sqrt[6]{b^2} = \sqrt[6]{a^3 b^2}$$

---

### Comparing Surds of Different Orders:

Convert all to the **same order** (LCM of orders):

$$\sqrt{2} \text{ vs } \sqrt[3]{3}$$

$$= 2^{1/2} \text{ vs } 3^{1/3} = 2^{3/6} \text{ vs } 3^{2/6} = \sqrt[6]{8} \text{ vs } \sqrt[6]{9}$$

Since 9 > 8: **∛3 > √2**

## 9. SURDS — BASIC TO ADVANCED EXAMPLES

---

### ─── BASIC LEVEL ───

#### Example 15: Simplify Surds

> Simplify: (a) √48, (b) √72, (c) √108

(a) $\sqrt{48} = \sqrt{16 \times 3} = 4\sqrt{3}$

(b) $\sqrt{72} = \sqrt{36 \times 2} = 6\sqrt{2}$

(c) $\sqrt{108} = \sqrt{36 \times 3} = 6\sqrt{3}$

---

#### Example 16: Addition of Surds

> Simplify: $3\sqrt{5} + 2\sqrt{5} - \sqrt{5}$

$$= (3+2-1)\sqrt{5} = 4\sqrt{5}$$

---

#### Example 17: Simplify then Add

> Simplify: $\sqrt{50} + \sqrt{18} - \sqrt{8}$

$$= 5\sqrt{2} + 3\sqrt{2} - 2\sqrt{2} = 6\sqrt{2}$$

---

#### Example 18: Multiplication

> Multiply: $\sqrt{6} \times \sqrt{8}$

$$= \sqrt{6 \times 8} = \sqrt{48} = 4\sqrt{3}$$

---

#### Example 19: Rationalise

> Rationalise: $\dfrac{4}{\sqrt{3}}$

$$= \frac{4\sqrt{3}}{3}$$

---

#### Example 20: Rationalise Binomial

> Rationalise: $\dfrac{5}{3 + \sqrt{2}}$

$$= \frac{5(3-\sqrt{2})}{(3)^2 - (\sqrt{2})^2} = \frac{5(3-\sqrt{2})}{9-2} = \frac{5(3-\sqrt{2})}{7} = \frac{15 - 5\sqrt{2}}{7}$$

---

### ─── INTERMEDIATE LEVEL ───

#### Example 21: Rationalise Two-Surd Denominator

> Rationalise: $\dfrac{3}{\sqrt{7} - \sqrt{5}}$

$$= \frac{3(\sqrt{7}+\sqrt{5})}{7-5} = \frac{3(\sqrt{7}+\sqrt{5})}{2}$$

---

#### Example 22: Combined Rationalisation

> Simplify: $\dfrac{1}{1+\sqrt{2}} + \dfrac{1}{\sqrt{2}+\sqrt{3}} + \dfrac{1}{\sqrt{3}+\sqrt{4}}$

$$= \frac{1-\sqrt{2}}{1-2} + \frac{\sqrt{2}-\sqrt{3}}{2-3} + \frac{\sqrt{3}-\sqrt{4}}{3-4}$$

$$= \frac{1-\sqrt{2}}{-1} + \frac{\sqrt{2}-\sqrt{3}}{-1} + \frac{\sqrt{3}-2}{-1}$$

$$= (\sqrt{2}-1) + (\sqrt{3}-\sqrt{2}) + (2-\sqrt{3}) = 2-1 = \textbf{1}$$

---

#### Example 23: Find Value if x = 2 + √3

> If $x = 2 + \sqrt{3}$, find $x + \dfrac{1}{x}$ and $x^2 + \dfrac{1}{x^2}$.

$$\frac{1}{x} = \frac{1}{2+\sqrt{3}} = \frac{2-\sqrt{3}}{(4-3)} = 2-\sqrt{3}$$

$$x + \frac{1}{x} = (2+\sqrt{3}) + (2-\sqrt{3}) = \textbf{4}$$

$$x^2 + \frac{1}{x^2} = \left(x + \frac{1}{x}\right)^2 - 2 = 16 - 2 = \textbf{14}$$

---

#### Example 24: Multiplication with Conjugate

> Multiply: $(3 + \sqrt{5})(3 - \sqrt{5})$

$$= 3^2 - (\sqrt{5})^2 = 9 - 5 = \textbf{4}$$

---

#### Example 25: Squaring a Surd Expression

> Expand: $(\sqrt{3} + \sqrt{2})^2$

$$= 3 + 2\sqrt{6} + 2 = 5 + 2\sqrt{6}$$

---

#### Example 26: Value of Expression

> Simplify: $\dfrac{\sqrt{6}+\sqrt{5}}{\sqrt{6}-\sqrt{5}}$

Multiply by conjugate:
$$= \frac{(\sqrt{6}+\sqrt{5})^2}{6-5} = (\sqrt{6}+\sqrt{5})^2 = 6 + 2\sqrt{30} + 5 = 11 + 2\sqrt{30}$$

---

### ─── ADVANCED LEVEL ───

#### Example 27: Complex Surd Identity

> If $x = \dfrac{\sqrt{5}+1}{\sqrt{5}-1}$ and $y = \dfrac{\sqrt{5}-1}{\sqrt{5}+1}$, find x² + xy + y².

First: $xy = \frac{(\sqrt{5}+1)(\sqrt{5}-1)}{(\sqrt{5}-1)(\sqrt{5}+1)} = \frac{5-1}{5-1} = 1$

$x + y = \frac{(\sqrt{5}+1)^2 + (\sqrt{5}-1)^2}{(\sqrt{5})^2 - 1^2} = \frac{(5+2\sqrt{5}+1)+(5-2\sqrt{5}+1)}{4} = \frac{12}{4} = 3$

$x^2 + y^2 = (x+y)^2 - 2xy = 9 - 2 = 7$

$x^2 + xy + y^2 = 7 + 1 = \textbf{8}$

---

#### Example 28: Nested/Continued Surd

> Simplify: $\sqrt{6 + \sqrt{6 + \sqrt{6 + \ldots}}}$

Let $x = \sqrt{6 + \sqrt{6 + \sqrt{6 + \ldots}}}$

Then $x = \sqrt{6 + x}$

$x^2 = 6 + x \Rightarrow x^2 - x - 6 = 0$

$(x-3)(x+2) = 0 \Rightarrow x = 3$ (taking positive value)

**Answer: 3**

---

#### Example 29: Mixed Surd Product

> Evaluate: $\sqrt[3]{2} \times \sqrt[4]{2}$

$$= 2^{1/3} \times 2^{1/4} = 2^{1/3+1/4} = 2^{7/12} = \sqrt[12]{2^7} = \sqrt[12]{128}$$

---

#### Example 30: Find a + b If Surd Expression Equals a + b√c

> If $\dfrac{3+\sqrt{2}}{3-\sqrt{2}} = a + b\sqrt{2}$, find a and b.

$$\frac{3+\sqrt{2}}{3-\sqrt{2}} = \frac{(3+\sqrt{2})^2}{(3-\sqrt{2})(3+\sqrt{2})} = \frac{9+6\sqrt{2}+2}{9-2} = \frac{11+6\sqrt{2}}{7} = \frac{11}{7} + \frac{6}{7}\sqrt{2}$$

**a = 11/7, b = 6/7**

## 10. COMPARISON OF SURDS

---

### Method: Convert to Same Order Then Compare

#### Step 1: Find LCM of orders (the 'n' in ⁿ√a).
#### Step 2: Convert all surds to this LCM order.
#### Step 3: Compare the radicands.

---

#### Example 31: Compare √2, ∛3, ⁶√7

Orders: 2, 3, 6. LCM = 6.

$$\sqrt{2} = 2^{1/2} = 2^{3/6} = \sqrt[6]{8}$$

$$\sqrt[3]{3} = 3^{1/3} = 3^{2/6} = \sqrt[6]{9}$$

$$\sqrt[6]{7} = 7^{1/6} = \sqrt[6]{7}$$

Comparing radicands: 9 > 8 > 7

$$\therefore \sqrt[3]{3} > \sqrt{2} > \sqrt[6]{7}$$

---

#### Example 32: Which is Greater — √3 or ∛4?

LCM of 2 and 3 = 6.

$$\sqrt{3} = 3^{1/2} = 3^{3/6} = \sqrt[6]{27}$$

$$\sqrt[3]{4} = 4^{1/3} = 4^{2/6} = \sqrt[6]{16}$$

Since 27 > 16: **√3 > ∛4**

## 11. MIXED PROBLEMS — SURDS & INDICES TOGETHER

---

#### Example 33: Convert Surd to Index and Simplify

> Simplify: $\dfrac{\sqrt[3]{x^2} \times \sqrt{x}}{\sqrt[6]{x^5}}$

$$= \frac{x^{2/3} \times x^{1/2}}{x^{5/6}} = x^{2/3 + 1/2 - 5/6}$$

$$= x^{4/6 + 3/6 - 5/6} = x^{2/6} = x^{1/3} = \sqrt[3]{x}$$

---

#### Example 34: Surd Expression to Index Form

> Write $\sqrt{a\sqrt{a\sqrt{a}}}$ as a power of a.

$$= \sqrt{a \cdot \sqrt{a \cdot a^{1/2}}} = \sqrt{a \cdot \sqrt{a^{3/2}}} = \sqrt{a \cdot a^{3/4}} = \sqrt{a^{7/4}} = a^{7/8}$$

---

#### Example 35: Evaluate Mixed Expression

## 12. QUICK TRICKS & SHORTCUTS

---

### 🚀 TRICK 1: Same Base Comparison

If bases are same, larger exponent = larger number (for base > 1).
If bases are same, larger exponent = smaller number (for 0 < base < 1).

---

### 🚀 TRICK 2: Convert to Same Base FIRST

Before applying any law, ensure **same base**. 
8 = 2³, 16 = 2⁴, 27 = 3³, 81 = 3⁴, 125 = 5³, 243 = 3⁵, 1024 = 2¹⁰, 4096 = 2¹².

---

### 🚀 TRICK 3: Three Key Surd Values to Memorise

$$\sqrt{2} \approx 1.414, \quad \sqrt{3} \approx 1.732, \quad \sqrt{5} \approx 2.236$$

These help estimate answers quickly in MCQ exams.

---

### 🚀 TRICK 4: If a + b = a + b form, equate rational and irrational parts

If $p + q\sqrt{n} = a + b\sqrt{n}$ (where √n is irrational):

Then $p = a$ AND $q = b$ separately.

---

### 🚀 TRICK 5: Nested Surd — Set Equal to x, Square Both Sides

For $\sqrt{a + \sqrt{a + \sqrt{a + \ldots}}} = x$:
$x^2 = a + x \Rightarrow x^2 - x - a = 0$

---

### 🚀 TRICK 6: x + 1/x from Conjugate

If $x = a + \sqrt{b}$, then $\frac{1}{x} = a - \sqrt{b}$ (when $a^2 - b = 1$).

This gives $x + \frac{1}{x} = 2a$ instantly!

More generally: rationalise $\frac{1}{x}$, then add.

---

### 🚀 TRICK 7: x² + 1/x² from x + 1/x

$$x^2 + \frac{1}{x^2} = \left(x + \frac{1}{x}\right)^2 - 2$$

$$x^2 - \frac{1}{x^2} = \left(x + \frac{1}{x}\right)\left(x - \frac{1}{x}\right)$$

---

### 🚀 TRICK 8: Rationalise — Always Multiply Top and Bottom

Never rationalise only the denominator — multiply BOTH numerator and denominator by the conjugate.

---

### 🚀 TRICK 9: Telescoping Surd Sum

$$\frac{1}{\sqrt{n}+\sqrt{n+1}} = \sqrt{n+1} - \sqrt{n}$$

So: $\sum \frac{1}{\sqrt{n}+\sqrt{n+1}} = \sqrt{k+1} - 1$ (telescopes beautifully).

---

### 🚀 TRICK 10: Product of Conjugate Always Rational

$(\sqrt{a} + \sqrt{b})(\sqrt{a} - \sqrt{b}) = a - b$ (rational!)

This is WHY we use conjugates for rationalisation.

---

### 🚀 TRICK 11: Simplify ⁿ√(aᵐ) — Check if m/n Simplifies

$$\sqrt[6]{2^4} = 2^{4/6} = 2^{2/3} = \sqrt[3]{4}$$

Always reduce the fraction m/n first.

---

### 🚀 TRICK 12: aˣ = 1 Has Two Cases

Case 1: x = 0 (anything⁰ = 1).
Case 2: a = 1 (1^anything = 1).
Case 3: a = -1 and x is even (-1^even = 1).

Don't miss these!

---

### 🚀 TRICK 13: If aˣ = aʸ But a = ±1

General rule: aˣ = aʸ → x = y **only if a ≠ 0, ±1**.

If a = 1: all x work.
If a = -1: x = y only if both even or both odd.
If a = 0: all x ≠ 0 work.

---

### 🚀 TRICK 14: Square Root Simplification

To simplify √n: find the largest perfect square factor.

$$\sqrt{720} = \sqrt{144 \times 5} = 12\sqrt{5}$$

---

### 🚀 TRICK 15: Comparing Three Surds Quickly

Convert all to 6th power form:

$2^1 = 2, \quad 2^{1/2} = \sqrt{2} \approx 1.414, \quad 2^{1/3} \approx 1.26$

So: $2 > \sqrt{2} > \sqrt[3]{2}$ (as exponent decreases below 1, value decreases for base > 1).

---

### 🚀 TRICK 16: (a+b)² and (a-b)² for Surd Expressions

$$(\sqrt{p} + \sqrt{q})^2 = p + q + 2\sqrt{pq}$$

$$(\sqrt{p} - \sqrt{q})^2 = p + q - 2\sqrt{pq}$$

Use these to recognise surd expressions and simplify $\sqrt{p+q \pm 2\sqrt{pq}} = \sqrt{p} \pm \sqrt{q}$.

---

### 🚀 TRICK 17: Square Root of Binomial Surd

To find $\sqrt{a + b + 2\sqrt{ab}}$:

$$= \sqrt{(\sqrt{a} + \sqrt{b})^2} = \sqrt{a} + \sqrt{b}$$

**Example:** $\sqrt{5 + 2\sqrt{6}} = \sqrt{3 + 2 + 2\sqrt{6}} = \sqrt{3} + \sqrt{2}$

---

### 🚀 TRICK 18: Power Chains — aˣ = b^y

If aˣ = bʸ = c (say), then:

$$c^{1/x} = a, \quad c^{1/y} = b$$

$$c^{1/x + 1/y} = ab \quad \Rightarrow \quad c^{xy/(x+y)} = \ldots$$

## 13. COMMON MISTAKE ALERTS

---

### ⚠️ MISTAKE 1: √a × √b ≠ √(a×b) When a,b < 0

For real surds, √a × √b = √(ab) **ONLY when a, b ≥ 0**.

---

### ⚠️ MISTAKE 2: (a + b)² ≠ a² + b²

$$(\sqrt{2} + \sqrt{3})^2 = 2 + 3 + 2\sqrt{6} = 5 + 2\sqrt{6} \neq 5$$

**Wrong:** $(\sqrt{2}+\sqrt{3})^2 = \sqrt{4}+\sqrt{9} = 2+3 = 5$

Never forget the middle term 2ab!

---

### ⚠️ MISTAKE 3: √(a² + b²) ≠ a + b

$$\sqrt{9 + 16} = \sqrt{25} = 5 \neq 3+4 = 7$$

There is NO rule to split √(a+b) into √a + √b.

---

### ⚠️ MISTAKE 4: a^(m+n) ≠ a^m + a^n

Product rule: a^m × a^n = a^(m+n).
**WRONG:** a^m + a^n = a^(m+n). This is NEVER true!

$2^3 + 2^4 = 8 + 16 = 24 \neq 2^7 = 128$

---

### ⚠️ MISTAKE 5: (a^m)^n ≠ a^(m+n) — Must MULTIPLY

**(a^m)^n = a^(mn)** — multiply, don't add!

**Wrong:** $(2^3)^4 = 2^{3+4} = 2^7$
**Right:** $(2^3)^4 = 2^{3 \times 4} = 2^{12}$

---

### ⚠️ MISTAKE 6: Rationalising — Don't Forget Numerator

When rationalising, multiply **BOTH** numerator and denominator by the conjugate.

**Wrong:** $\frac{3}{2+\sqrt{5}}$. "Denominator becomes $4-5=-1$" and stop.

**Right:** $\frac{3(2-\sqrt{5})}{(2+\sqrt{5})(2-\sqrt{5})} = \frac{3(2-\sqrt{5})}{-1} = -3(2-\sqrt{5}) = -6+3\sqrt{5}$

---

### ⚠️ MISTAKE 7: Adding Unlike Surds

**Wrong:** $\sqrt{2} + \sqrt{3} = \sqrt{5}$
**Right:** $\sqrt{2} + \sqrt{3}$ **cannot be simplified** — they are unlike surds.

---

### ⚠️ MISTAKE 8: Forgetting a⁰ = 1 Doesn't Apply to a = 0

$0^0$ is **indeterminate/undefined** in most contexts.
$a^0 = 1$ only when **a ≠ 0**.

---

### ⚠️ MISTAKE 9: Root of Root — Multiplying Instead of Multiplying Orders

$\sqrt{\sqrt[3]{a}} = \sqrt[2 \times 3]{a} = \sqrt[6]{a}$

**Wrong:** $\sqrt{\sqrt[3]{a}} = \sqrt[2+3]{a} = \sqrt[5]{a}$

Orders are **multiplied**, not added.

---

### ⚠️ MISTAKE 10: Comparing Surds by Direct Inspection

Never compare $\sqrt{3}$ and $\sqrt[3]{5}$ just by "looking" — always convert to same order.

$\sqrt{3} = 3^{1/2} = 3^{3/6} = \sqrt[6]{27}$ and $\sqrt[3]{5} = 5^{1/3} = 5^{2/6} = \sqrt[6]{25}$.

Since 27 > 25: **√3 > ∛5**.

### Problem Type → Approach Guide:

| Problem Type | Key Approach |
|-------------|-------------|
| Same base equation | Equate exponents |
| Different base → same | Convert to prime base |
| x + 1/x (with surd x) | Rationalise 1/x, then add |
| x² + 1/x² | Use (x+1/x)² − 2 |
| Nested surd | Set = x, square both sides |
| Telescoping surd sum | Rationalise each term, cancel middle terms |
| √(a+b+2√ab) | Recognise as (√a+√b)² |
| Comparing surds | LCM of orders → compare radicands |
| Proof: aˣ=bʸ=cᶻ | Set equal to k, express a,b,c in terms of k |
| Surd × Surd (diff orders) | Convert to indices, add powers |

## 🏆 TOP EXAM TIPS

1. **Memorise the laws** — All 9 laws of indices must be instant recall. Practice writing them from memory.
2. **Convert to prime base first** — 8=2³, 27=3³, 25=5². This solves 80% of index equations.
3. **Rationalise with conjugate always** — For (a+√b) denominator, conjugate is (a−√b). For (√a+√b), conjugate is (√a−√b).
4. **Like surds only** — Can add/subtract 3√2 and 5√2, but NOT √2 and √3.
5. **√(a+b) ≠ √a + √b** — This is the most common error in surds. Never split a sum under a root.
6. **(a+b)² always has middle term** — (√2+√3)² = 5 + 2√6, NOT 5. The 2ab term is critical.
7. **Nested surds** — Set equal to x, square, solve quadratic. Take positive root.
8. **Telescoping sums** — Rationalise each term and watch the middle terms vanish.
9. **x + 1/x problems** — First rationalise 1/x (it becomes the conjugate of x), then use algebraic identities.
10. **aˣ = aʸ rule** — Only valid when a ≠ 0, ±1. Remember the special cases!
