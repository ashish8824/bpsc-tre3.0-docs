## 1. THE BIG PICTURE — WHAT IS A LOGARITHM?

---

### 🔑 The Core Idea:

> A **logarithm** answers the question:
>
> **"To what power must I raise the base to get this number?"**

---

### The Intuition:

| Exponential | Question | Answer (Logarithm) |
|------------|---------|-------------------|
| 2³ = 8 | "2 to what power gives 8?" | log₂8 = 3 |
| 10² = 100 | "10 to what power gives 100?" | log₁₀100 = 2 |
| 5¹ = 5 | "5 to what power gives 5?" | log₅5 = 1 |
| 3⁰ = 1 | "3 to what power gives 1?" | log₃1 = 0 |

---

### The Big Picture — Relationship:

```
EXPONENTIAL FORM          LOGARITHMIC FORM
                                                    
   a^x  =  N    ←──→    log_a(N) = x

   ↑         ↑             ↑         ↑
  BASE   RESULT        BASE      POWER

"Base raised to        "Logarithm of N
 power x = N"          to base a = x"
```

---

### Why Logarithms?

Before calculators, log tables were used to:
- Turn **multiplication** into **addition** (log(ab) = log a + log b)
- Turn **division** into **subtraction** (log(a/b) = log a − log b)
- Turn **powers** into **multiplication** (log(aⁿ) = n·log a)

These properties still make logs essential in competitive exams!

## 2. CORE DEFINITION & THE EXPONENTIAL-LOG CONNECTION

---

### The Formal Definition:

$$\boxed{\log_a N = x \iff a^x = N}$$

**Conditions:**
- Base $a > 0$, $a \neq 1$
- Argument $N > 0$ (you can ONLY take log of positive numbers)
- $x$ can be any real number

---

### Reading Logarithms:

| Notation | Read As |
|---------|---------|
| $\log_2 8$ | "log of 8 to the base 2" |
| $\log_{10} 100$ | "log of 100 to the base 10" |
| $\ln 5$ | "natural log of 5" (base e) |
| $\log 1000$ | "log of 1000" (base 10, when base not written) |

---

### The Two-Way Conversion (MASTER THIS):

| Exponential → Log | Log → Exponential |
|------------------|------------------|
| $2^5 = 32$ → $\log_2 32 = 5$ | $\log_2 32 = 5$ → $2^5 = 32$ |
| $10^3 = 1000$ → $\log_{10} 1000 = 3$ | $\log_{10} 1000 = 3$ → $10^3 = 1000$ |
| $3^{-2} = 1/9$ → $\log_3(1/9) = -2$ | $\log_3(1/9) = -2$ → $3^{-2} = 1/9$ |
| $a^0 = 1$ → $\log_a 1 = 0$ | $\log_a 1 = 0$ → $a^0 = 1$ |

---

### Examples — Basic Conversion:

#### Example 1:

> Convert to logarithmic form: (a) $5^3 = 125$ (b) $7^0 = 1$ (c) $4^{-2} = 1/16$

(a) $\log_5 125 = 3$
(b) $\log_7 1 = 0$
(c) $\log_4 (1/16) = -2$

---

#### Example 2:

> Convert to exponential form: (a) $\log_3 81 = 4$ (b) $\log_{10} 0.01 = -2$

(a) $3^4 = 81$
(b) $10^{-2} = 0.01$

## 3. TYPES OF LOGARITHMS

---

### Type 1: Common Logarithm (Base 10)

$$\log_{10} N = \log N$$

> When no base is written, it is **assumed to be 10** (in most exam contexts).

Used in: engineering, science, pH scale, Richter scale, decibels.

---

### Type 2: Natural Logarithm (Base e)

$$\log_e N = \ln N$$

Where $e = 2.71828...$ (Euler's number, irrational like π).

Used in: calculus, growth/decay problems, statistics.

---

### Type 3: Binary Logarithm (Base 2)

$$\log_2 N$$

Used in: computer science, information theory.

---

### Comparison:

| Type | Notation | Base | When Used |
|------|---------|------|----------|
| Common log | log N | 10 | Standard exams |
| Natural log | ln N | e ≈ 2.718 | Advanced math |
| Binary log | log₂ N | 2 | CS problems |

## 4. LAWS / PROPERTIES OF LOGARITHMS

---

### THE SEVEN FUNDAMENTAL LAWS:

---

#### Law 1: Product Rule (Log of Product = Sum of Logs)

$$\boxed{\log_a(M \times N) = \log_a M + \log_a N}$$

> "Logarithm turns multiplication into addition."

Example: $\log_2 (8 \times 4) = \log_2 8 + \log_2 4 = 3 + 2 = 5$ ✓ (since 2⁵ = 32 = 8×4)

---

#### Law 2: Quotient Rule (Log of Quotient = Difference of Logs)

$$\boxed{\log_a\left(\frac{M}{N}\right) = \log_a M - \log_a N}$$

> "Logarithm turns division into subtraction."

Example: $\log_2\left(\frac{32}{4}\right) = \log_2 32 - \log_2 4 = 5 - 2 = 3$ ✓ (since 32/4 = 8 = 2³)

---

#### Law 3: Power Rule (Log of Power = Power × Log)

$$\boxed{\log_a(M^n) = n \cdot \log_a M}$$

> "Logarithm brings the exponent down as a multiplier."

Example: $\log_2(8^3) = 3 \cdot \log_2 8 = 3 \times 3 = 9$ ✓ (since 8³ = 512 = 2⁹)

---

#### Law 4: Log Base = 1 (Log of Base Equals 1)

$$\boxed{\log_a a = 1}$$

> "Any number to the first power is itself."

Examples: $\log_2 2 = 1$, $\log_{10} 10 = 1$, $\log_5 5 = 1$

---

#### Law 5: Log 1 = 0 (Log of 1 is Always 0)

$$\boxed{\log_a 1 = 0}$$

> "Any base to the power 0 = 1."

Examples: $\log_2 1 = 0$, $\log_{10} 1 = 0$, $\log_e 1 = 0$

---

#### Law 6: Log of Reciprocal = Negative Log

$$\boxed{\log_a\left(\frac{1}{N}\right) = -\log_a N}$$

Example: $\log_3 \frac{1}{9} = -\log_3 9 = -2$

---

#### Law 7: Change of Base

$$\boxed{\log_a b = \frac{\log_c b}{\log_c a} = \frac{\ln b}{\ln a} = \frac{\log b}{\log a}}$$

---

### Extended Properties:

$$\log_a(a^x) = x \quad \text{(log and exponential cancel)}$$

$$a^{\log_a x} = x \quad \text{(exponential and log cancel)}$$

$$\log_a b = \frac{1}{\log_b a} \quad \text{(reciprocal base swap)}$$

$$\log_{a^m} b = \frac{1}{m} \log_a b$$

$$\log_{a^m} b^n = \frac{n}{m} \log_a b$$

---

### All Laws at a Glance:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
OPERATION        LOGARITHM DOES...         FORMULA
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Multiply M×N     → Addition              log(MN) = logM + logN
Divide M÷N       → Subtraction           log(M/N) = logM - logN
Power Mⁿ         → Multiplication        log(Mⁿ) = n·logM
Root ⁿ√M         → Division              log(ⁿ√M) = (1/n)logM
Base a           → 1                     log_a(a) = 1
One (1)          → 0                     log_a(1) = 0
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 6. CHANGE OF BASE FORMULA

---

### The Formula:

$$\boxed{\log_a b = \frac{\log_c b}{\log_c a}}$$

Most commonly used as:

$$\log_a b = \frac{\log b}{\log a} = \frac{\ln b}{\ln a}$$

---

### Key Consequence:

$$\log_a b \times \log_b a = 1$$

$$\log_a b = \frac{1}{\log_b a}$$

---

### Chain Rule of Logarithms:

$$\log_a b \times \log_b c \times \log_c d = \log_a d$$

> Intermediate bases cancel! This is extremely useful in exams.

---

### Examples — Change of Base:

#### Example 3:

> Find $\log_4 32$ using change of base.

$$\log_4 32 = \frac{\log 32}{\log 4} = \frac{\log 2^5}{\log 2^2} = \frac{5\log 2}{2\log 2} = \frac{5}{2} = \textbf{2.5}$$

---

#### Example 4:

> Find $\log_8 2$

$$\log_8 2 = \frac{\log 2}{\log 8} = \frac{\log 2}{3\log 2} = \frac{1}{3}$$

---

#### Example 5: Chain Rule

> Simplify: $\log_2 3 \times \log_3 4 \times \log_4 8$

$$= \frac{\log 3}{\log 2} \times \frac{\log 4}{\log 3} \times \frac{\log 8}{\log 4} = \frac{\log 8}{\log 2} = \frac{3\log 2}{\log 2} = \textbf{3}$$

## 7. BASIC TO ADVANCED EXAMPLES

---

### ─── BASIC LEVEL ───

#### Example 6: Evaluate Logarithms

> Find: (a) $\log_2 64$ (b) $\log_5 125$ (c) $\log_3 \frac{1}{27}$

(a) $\log_2 64 = \log_2 2^6 = \textbf{6}$

(b) $\log_5 125 = \log_5 5^3 = \textbf{3}$

(c) $\log_3 \frac{1}{27} = \log_3 3^{-3} = \textbf{-3}$

---

#### Example 7: Using Product Rule

> If $\log 2 = 0.3010$ and $\log 3 = 0.4771$, find $\log 6$.

$$\log 6 = \log(2 \times 3) = \log 2 + \log 3 = 0.3010 + 0.4771 = \textbf{0.7781}$$

---

#### Example 8: Using Power Rule

> Given $\log 2 = 0.3010$, find $\log 32$.

$$\log 32 = \log 2^5 = 5 \times 0.3010 = \textbf{1.5050}$$

---

#### Example 9: Find log of fractions

> Given $\log 2 = 0.3010$, find $\log 0.25$.

$$\log 0.25 = \log \frac{1}{4} = -\log 4 = -2\log 2 = -2 \times 0.3010 = \textbf{-0.6020}$$

---

#### Example 10: Expand Using All Laws

> Expand: $\log\left(\dfrac{x^2 \sqrt{y}}{z^3}\right)$

$$= \log x^2 + \log \sqrt{y} - \log z^3$$

$$= 2\log x + \frac{1}{2}\log y - 3\log z$$

---

#### Example 11: Condense Logs

> Condense: $3\log x - \frac{1}{2}\log y + 2\log z$

$$= \log x^3 - \log y^{1/2} + \log z^2 = \log\left(\frac{x^3 z^2}{\sqrt{y}}\right)$$

---

### ─── INTERMEDIATE LEVEL ───

#### Example 12: Prove Identities

> Show: $\log_a b \times \log_b c = \log_a c$

$$\log_a b \times \log_b c = \frac{\log b}{\log a} \times \frac{\log c}{\log b} = \frac{\log c}{\log a} = \log_a c \quad \checkmark$$

---

#### Example 13: Simplify Complex Expression

> Simplify: $\log_2 \log_2 \log_2 16$

$$\log_2 16 = \log_2 2^4 = 4$$

$$\log_2 \log_2 16 = \log_2 4 = \log_2 2^2 = 2$$

$$\log_2 \log_2 \log_2 16 = \log_2 2 = \textbf{1}$$

---

#### Example 14: Value of Expression

> If $\log_{10} 2 = 0.3010$, find the number of digits in $2^{50}$.

$$\log(2^{50}) = 50 \times 0.3010 = 15.050$$

Characteristic = 15 → Number has **15 + 1 = 16 digits**.

---

#### Example 15: Simplify Sum

> Simplify: $\log_3 4 \times \log_4 5 \times \log_5 6 \times \log_6 27$

$$= \frac{\log 4}{\log 3} \times \frac{\log 5}{\log 4} \times \frac{\log 6}{\log 5} \times \frac{\log 27}{\log 6}$$

$$= \frac{\log 27}{\log 3} = \frac{3\log 3}{\log 3} = \textbf{3}$$

---

#### Example 16: Find Value

> If $\log_{10} 5 = a$ and $\log_{10} 3 = b$, find $\log_{10} 45$ in terms of a and b.

$$\log 45 = \log(9 \times 5) = \log 9 + \log 5 = 2\log 3 + \log 5 = 2b + a$$

---

#### Example 17: Complicated Simplification

> Simplify: $\dfrac{\log \sqrt{8}}{\log 2}$

$$= \frac{\frac{1}{2}\log 8}{\log 2} = \frac{\frac{1}{2} \times 3\log 2}{\log 2} = \frac{3}{2} = \textbf{1.5}$$

---

#### Example 18: Prove

> If $a, b, c$ are in GP, prove that $\log a, \log b, \log c$ are in AP.

$a, b, c$ in GP means $b^2 = ac$.

Taking log: $2\log b = \log a + \log c$

This means $\log a, \log b, \log c$ are in AP (2× middle = sum of first and last). ✓

---

#### Example 19: Value from Given Information

> If $\log 2 = 0.3010$ and $\log 3 = 0.4771$, find $\log 1.5$.

$$\log 1.5 = \log \frac{3}{2} = \log 3 - \log 2 = 0.4771 - 0.3010 = \textbf{0.1761}$$

---

### ─── ADVANCED LEVEL ───

#### Example 20: Multi-Variable Log Problem

> If $\log\left(\dfrac{a+b}{3}\right) = \dfrac{\log a + \log b}{2}$, show that $\dfrac{a}{b} + \dfrac{b}{a} = 7$.

$$\log\frac{a+b}{3} = \frac{1}{2}\log(ab) = \log\sqrt{ab}$$

$$\frac{a+b}{3} = \sqrt{ab}$$

$$(a+b) = 3\sqrt{ab}$$

Squaring: $(a+b)^2 = 9ab$

$$a^2 + 2ab + b^2 = 9ab \Rightarrow a^2 - 7ab + b^2 = 0$$

Dividing by $ab$: $\dfrac{a}{b} - 7 + \dfrac{b}{a} = 0 \Rightarrow \dfrac{a}{b} + \dfrac{b}{a} = 7$ ✓

---

#### Example 21: Logarithmic Identity

> Prove: $\dfrac{1}{\log_2 N} + \dfrac{1}{\log_3 N} + \dfrac{1}{\log_4 N} + \ldots + \dfrac{1}{\log_n N} = \dfrac{1}{\log_{n!} N}$

Using reciprocal property: $\dfrac{1}{\log_k N} = \log_N k$

LHS = $\log_N 2 + \log_N 3 + \log_N 4 + \ldots + \log_N n$

$= \log_N (2 \times 3 \times 4 \times \ldots \times n) = \log_N (n!)$

$= \dfrac{1}{\log_{n!} N}$ = RHS ✓

---

#### Example 22: Find Unknowns

> If $\log_3 x + \log_9 x = 3$, find x.

$$\log_3 x + \log_9 x = \log_3 x + \frac{\log_3 x}{\log_3 9} = \log_3 x + \frac{\log_3 x}{2} = \frac{3}{2}\log_3 x$$

$$\frac{3}{2}\log_3 x = 3 \Rightarrow \log_3 x = 2 \Rightarrow x = 3^2 = \textbf{9}$$

---

## 8. LOGARITHMIC EQUATIONS

---

### Strategy to Solve Log Equations:

```
STEP 1: Identify the type
STEP 2: Use laws to simplify
STEP 3: Convert log = k to exponential: a^k = argument
STEP 4: Solve for the variable
STEP 5: CHECK if solution makes the argument POSITIVE
         (Reject any solution that makes argument ≤ 0)
```

---

### Type 1: Direct — log_a(x) = k

$$\log_a(x) = k \Rightarrow x = a^k$$

#### Example 23:

> Solve: $\log_5 x = 3$

$$x = 5^3 = \textbf{125}$$

---

### Type 2: Two Logs Equal — log f(x) = log g(x)

$$\log_a f(x) = \log_a g(x) \Rightarrow f(x) = g(x) \quad \text{(same base)}$$

#### Example 24:

> Solve: $\log(2x+1) = \log(x+4)$

$$2x+1 = x+4 \Rightarrow x = \textbf{3}$$

Check: 2(3)+1 = 7 > 0 ✓, 3+4 = 7 > 0 ✓

---

### Type 3: Quadratic in log — let log x = t

#### Example 25:

> Solve: $(\log x)^2 - 5\log x + 6 = 0$

Let $t = \log x$:

$$t^2 - 5t + 6 = 0 \Rightarrow (t-2)(t-3) = 0$$

$$t = 2 \text{ or } t = 3$$

$$\log x = 2 \Rightarrow x = 100 \quad \text{or} \quad \log x = 3 \Rightarrow x = 1000$$

**x = 100 or x = 1000**

---

### Type 4: Using All Laws to Solve

#### Example 26:

> Solve: $\log_2 x + \log_2(x-2) = 3$

$$\log_2 x(x-2) = 3 \Rightarrow x(x-2) = 2^3 = 8$$

$$x^2 - 2x - 8 = 0 \Rightarrow (x-4)(x+2) = 0$$

$x = 4$ or $x = -2$

Check: $x = -2$ makes argument $\log(-2)$ and $\log(-4)$ — both invalid (negative).

**x = 4** (only solution)

---

#### Example 27:

> Solve: $\log(x+1) + \log(x-1) = \log 3$

$$\log[(x+1)(x-1)] = \log 3$$

$$(x+1)(x-1) = 3 \Rightarrow x^2 - 1 = 3 \Rightarrow x^2 = 4 \Rightarrow x = \pm 2$$

Check: $x = -2$ gives $\log(-1)$ (invalid). **x = 2** ✓

---

#### Example 28: Complex Equation

> Solve: $\log_2(x+3) = 2 + \log_2(x-1)$

$$\log_2(x+3) - \log_2(x-1) = 2$$

$$\log_2\frac{x+3}{x-1} = 2 \Rightarrow \frac{x+3}{x-1} = 4$$

$$x+3 = 4x-4 \Rightarrow 3x = 7 \Rightarrow x = \frac{7}{3}$$

Check: $x = 7/3$: $x+3 = 16/3 > 0$ ✓, $x-1 = 4/3 > 0$ ✓. **x = 7/3**

---

## 9. LOGARITHMIC INEQUALITIES

---

### Key Rules for Log Inequalities:

$$\text{If } a > 1: \quad \log_a M > \log_a N \iff M > N$$

$$\text{If } 0 < a < 1: \quad \log_a M > \log_a N \iff M < N \quad \text{(inequality flips!)}$$

> When base > 1: log is **increasing** → inequality stays same.
> When 0 < base < 1: log is **decreasing** → inequality **reverses**.

---

#### Example 29:

> Solve: $\log_2 x > 3$

Base = 2 > 1 (increasing function).

$$x > 2^3 = 8$$

Also need $x > 0$ (for log to exist).

**Solution: x > 8**

---

#### Example 30:

> Solve: $\log_{0.5} x > 2$

Base = 0.5 < 1 (decreasing function) → inequality reverses!

$$x < (0.5)^2 = 0.25$$

Also need $x > 0$.

**Solution: 0 < x < 0.25**

---

#### Example 31:

> Solve: $\log_3(x-1) > \log_3(5-x)$

Base = 3 > 1 (no flip):

$$x-1 > 5-x \Rightarrow 2x > 6 \Rightarrow x > 3$$

Domain: $x-1 > 0$ and $5-x > 0$ → $1 < x < 5$.

Combined: **3 < x < 5**

---

## 10. CHARACTERISTIC AND MANTISSA

---

### Definition:

When a logarithm is expressed in the form **integer + decimal**, the two parts are:

| Part | Name | Range |
|------|------|-------|
| Integer part | **Characteristic** | Can be negative or positive |
| Decimal part | **Mantissa** | Always between 0 and 1 (inclusive) |

$$\log N = \underbrace{\text{characteristic}}_{\text{integer part}} + \underbrace{\text{mantissa}}_{\text{decimal part}}$$

---

### Finding Characteristic:

For $N > 1$:
$$\text{Characteristic} = (\text{number of digits in } N) - 1$$

For $0 < N < 1$:
$$\text{Characteristic} = -(\text{number of zeros after decimal point including the one before first significant digit})$$

More precisely: if N = 0.00xyz... → characteristic = −(leading zeros + 1)

---

### Examples:

| N | log N (approx) | Characteristic | Mantissa |
|---|---------------|---------------|---------|
| 2345 | 3.3701 | 3 | 0.3701 |
| 23.45 | 1.3701 | 1 | 0.3701 |
| 2.345 | 0.3701 | 0 | 0.3701 |
| 0.2345 | −0.6299 = −1 + 0.3701 | −1 (written as $\bar{1}$) | 0.3701 |
| 0.02345 | −1.6299 = −2 + 0.3701 | −2 (written as $\bar{2}$) | 0.3701 |

> Note: Numbers with **same significant digits** have the **same mantissa** but different characteristics.

---

### Number of Digits / Zeros:

**Digits in N** (for integer N) = Characteristic + 1 = ⌊log N⌋ + 1

**Zeros after decimal** (for N < 1) = |Characteristic| - 1

---

#### Example 32:

> How many digits does $5^{20}$ have? (Given log 5 = 0.699)

$$\log(5^{20}) = 20 \times 0.699 = 13.98$$

Characteristic = 13 → **Number of digits = 13 + 1 = 14**

---

#### Example 33:

> How many zeros after decimal in $(0.5)^{25}$? (log 5 = 0.699, log 2 = 0.301)

$$\log(0.5)^{25} = 25 \times \log(0.5) = 25 \times (-0.301) = -7.525$$

Write as: $-7.525 = -8 + 0.475 = \bar{8}.475$

Characteristic = -8 → **Zeros after decimal = 8 - 1 = 7**

## 11. APPLICATIONS OF LOGARITHMS

---

### Application 1: Finding Number of Digits

Number of digits in $a^n$ = $\lfloor n \log a \rfloor + 1$

---

### Application 2: Exponential Equations

> Solve $3^x = 17$

$$x = \log_3 17 = \frac{\log 17}{\log 3} = \frac{1.2304}{0.4771} \approx 2.579$$

---

### Application 3: Compound Interest

> At what rate does money double in 10 years?

$$2 = \left(1 + \frac{r}{100}\right)^{10}$$

$$\log 2 = 10 \log\left(1 + \frac{r}{100}\right)$$

$$\log\left(1 + \frac{r}{100}\right) = \frac{0.3010}{10} = 0.0301$$

$$1 + \frac{r}{100} = 10^{0.0301} \approx 1.0718$$

$$r \approx 7.18\%$$

---

### Application 4: Half-Life (Exponential Decay)

> Element decays to half in T years. After n years, fraction remaining = $(1/2)^{n/T}$.

Using logs: $\log\left(\frac{N}{N_0}\right) = \frac{n}{T} \times \log(1/2) = -\frac{n}{T} \times 0.3010$

## 12. QUICK TRICKS & SHORTCUTS

---

### 🚀 TRICK 1: Fundamental Conversion Shortcut

Every log problem can be converted to/from exponential form.
If stuck: convert, then solve.

$\log_a N = x \iff a^x = N$

---

### 🚀 TRICK 2: Chain Rule of Logarithms

$$\log_a b \cdot \log_b c \cdot \log_c d = \log_a d$$

All intermediate bases cancel! Telescoping effect.

---

### 🚀 TRICK 3: Reciprocal Rule

$$\log_a b = \frac{1}{\log_b a}$$

If you need $\log_a b$, compute $\log_b a$ and take reciprocal.

---

### 🚀 TRICK 4: Number of Digits Formula

$$\text{Digits in } n^k = \lfloor k \log n \rfloor + 1$$

Apply characteristic concept.

---

### 🚀 TRICK 5: log 5 in Terms of log 2

$$\log_{10} 5 = 1 - \log_{10} 2$$

Since $\log_{10} 10 = 1$ and $10 = 2 \times 5$ → $\log 2 + \log 5 = 1$.

This avoids needing log 5 separately!

---

### 🚀 TRICK 6: Simplify by Converting to Same Base

When comparing or combining logs, convert all to log base 10 (or ln).

---

### 🚀 TRICK 7: Quadratic in Log — Let t = log x

Any equation of form $a(\log x)^2 + b(\log x) + c = 0$:

Let $t = \log x$, solve $at^2 + bt + c = 0$ for t, then find x.

---

### 🚀 TRICK 8: Log Inequality Direction Depends on Base

- Base > 1 → log is increasing → inequality SAME direction
- 0 < Base < 1 → log is decreasing → inequality REVERSES

---

### 🚀 TRICK 9: x + 1/x Type in Log Problems

If $\log_a b + \log_b a = k$, let $\log_a b = t$, then $t + 1/t = k$.
$t^2 - kt + 1 = 0$. Find t, then find b.

---

### 🚀 TRICK 10: log of 1, log of base — Instant Values

$$\log_a 1 = 0, \quad \log_a a = 1, \quad \log_a a^n = n$$

These three should be INSTANT — no calculation needed.

---

### 🚀 TRICK 11: Product to Sum (Solve Complex)

$$\log_a(b \times c \times d) = \log_a b + \log_a c + \log_a d$$

Break complex products into sums.

---

### 🚀 TRICK 12: Logarithm of Power (Instant Simplification)

$$\log_a(a^n) = n \quad \text{(instant!)}$$

$$2^{\log_2 5} = 5 \quad \text{(exponential-log cancel)}$$

$$10^{\log_{10} 7} = 7 \quad \text{(instant!)}$$

---

### 🚀 TRICK 13: Change of Base for Same Numerator

$$\frac{1}{\log_a N} = \log_N a$$

Use this to simplify $\sum \frac{1}{\log_k N}$ type problems.

---

### 🚀 TRICK 14: log m = log n → m = n (Same Base)

When you have $\log f(x) = \log g(x)$ (same base), just set $f(x) = g(x)$.

---

### 🚀 TRICK 15: Sum of Logs → Log of Product

$$\log_N 2 + \log_N 3 + \log_N 4 + \ldots + \log_N n = \log_N(n!)$$

Use for finding $\sum \frac{1}{\log_k N}$ type expressions.

## 13. COMMON MISTAKE ALERTS

---

### ⚠️ MISTAKE 1: log(a + b) ≠ log a + log b

**Wrong:** $\log(3+4) = \log 3 + \log 4$
**Right:** $\log(3+4) = \log 7$. Log of sum has NO simple law!

$\log(a+b)$ cannot be split. Only $\log(a \times b) = \log a + \log b$.

---

### ⚠️ MISTAKE 2: (log a)² ≠ 2 log a

**Wrong:** $(\log 5)^2 = 2\log 5$
**Right:** $(\log 5)^2 = (\log 5) \times (\log 5)$ (square of the number log 5)

Power rule says: $\log(5^2) = 2\log 5$, NOT $(\log 5)^2 = 2\log 5$.

---

### ⚠️ MISTAKE 3: log(a/b) ≠ log a / log b

**Wrong:** $\log(6/2) = \frac{\log 6}{\log 2}$
**Right:** $\log(6/2) = \log 6 - \log 2 = \log 3$

Quotient rule: $\log(a/b) = \log a - \log b$. The change of base formula gives $\log_2 6 = \frac{\log 6}{\log 2}$, which is different!

---

### ⚠️ MISTAKE 4: Negative Argument in Log

**Wrong:** $\log(-5) = \log 5$ (just ignoring the sign)
**Right:** $\log(-5)$ is **undefined** (in real numbers). Log is only defined for positive arguments!

When solving equations, always reject solutions that make any argument ≤ 0.

---

### ⚠️ MISTAKE 5: Base = 1 in Logarithm

$\log_1 N$ is **undefined** for ALL N (since $1^x = 1 \neq N$ for $N \neq 1$).

Similarly, $\log_0 N$ and $\log_{-1} N$ are undefined.

**Valid base:** $a > 0$ and $a \neq 1$.

---

### ⚠️ MISTAKE 6: Forgetting to Check Solutions in Log Equations

After solving, ALWAYS verify:
- All arguments are positive
- Base conditions are met

Extraneous solutions (from squaring or cross-multiplying) must be rejected.

---

### ⚠️ MISTAKE 7: Log Base Inequality Direction

For $\log_{0.3} x > \log_{0.3} y$: since base 0.3 < 1, the function is DECREASING, so **x < y** (not x > y).

---

### ⚠️ MISTAKE 8: Characteristic of Numbers < 1

$\log(0.05) = \log\frac{5}{100} = \log 5 - 2 \approx 0.699 - 2 = -1.301$

This is written as $\bar{2}.699$ (characteristic = -2, mantissa = 0.699).

**Wrong:** Characteristic = -1 (miscounting zeros).
**Right:** 0.05 has 1 leading zero after decimal → characteristic = -2.

---

### ⚠️ MISTAKE 9: Using log Laws Across Different Bases

$$\log_2 3 + \log_3 5 \neq \log_2 5$$

You can only combine logs with the **SAME BASE**.

---

### ⚠️ MISTAKE 10: log ab ÷ log a ≠ log b

$$\frac{\log(ab)}{\log a} \neq \log b$$

The correct version: $\frac{\log(ab)}{\log a} = \frac{\log a + \log b}{\log a} = 1 + \frac{\log b}{\log a} = 1 + \log_a b$

### Problem Type → Approach:

| Problem | Approach |
|---------|---------|
| log_a(x) = k | Convert: x = aᵏ |
| log f(x) = log g(x) | Set f(x) = g(x), check domain |
| (log x)² + b·log x + c = 0 | Let t = log x, solve quadratic |
| Sum log_a·log_b... | Chain rule (telescoping) |
| 1/log_k N (sum) | Convert to log_N k, then sum |
| Digits in aⁿ | ⌊n·log a⌋ + 1 |
| GP → AP (log) | If a,b,c in GP, log a, log b, log c in AP |
| Proof: a/b + b/a type | Use AM-GM with log identity |
| Nested log | Evaluate from innermost outward |
| Different base same problem | Use change of base, find common base |

## 🏆 TOP EXAM TIPS

1. **Memorise log 2, 3, 7** — From these three, you can derive log of any number 1-10.
2. **log(a+b) has NO simple law** — The most tempting but completely wrong step.
3. **Always check domain** — After solving log equations, verify all arguments > 0.
4. **Chain rule is your best friend** — log_a b × log_b c × log_c d = log_a d (all middle terms cancel).
5. **log of base = 1, log of 1 = 0** — These should be instant. No calculation needed.
6. **Quadratic in log** — Let t = log x and solve the quadratic. Very common exam question.
7. **Digits formula** — ⌊n × log(base)⌋ + 1. Apply characteristic concept.
8. **Inequality direction** — Base > 1 → same direction; 0 < base < 1 → reverse direction.
9. **Reciprocal rule** — log_a b = 1/log_b a. Quick way to switch base and argument.
10. **Convert all to same base** — When combining logs of different bases, change all to base 10 (or log) first.
