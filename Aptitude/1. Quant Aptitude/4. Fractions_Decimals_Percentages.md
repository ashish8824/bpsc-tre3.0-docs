## 1. THE BIG PICTURE — HOW THEY CONNECT

> These three topics are **three ways of expressing the same thing**. Master the conversions first.

```
╔══════════════════════════════════════════════════════════════╗
║                                                              ║
║         FRACTION  ←————————————→  DECIMAL                   ║
║            ↑                          ↑                     ║
║            |                          |                     ║
║            ↓                          ↓                     ║
║                    PERCENTAGE                               ║
║                                                              ║
║   1/4  =  0.25  =  25%   (ALL represent the same value)     ║
║   3/5  =  0.6   =  60%                                      ║
║   1/3  =  0.333 =  33.33%                                   ║
║                                                              ║
╚══════════════════════════════════════════════════════════════╝
```

### The Three Conversion Rules

```
Fraction → Decimal   : Divide numerator by denominator
                        3/4 → 3÷4 = 0.75

Decimal → Fraction   : Write over 10/100/1000... then simplify
                        0.75 → 75/100 = 3/4

Fraction → Percent   : Multiply by 100
                        3/4 × 100 = 75%

Percent → Fraction   : Divide by 100 and simplify
                        75% → 75/100 = 3/4

Decimal → Percent    : Multiply by 100 (shift decimal 2 right)
                        0.75 → 75%

Percent → Decimal    : Divide by 100 (shift decimal 2 left)
                        75% → 0.75
```

## 2. FRACTIONS — COMPLETE COVERAGE

### 2.1 Types of Fractions

#### Proper Fraction
> Numerator **<** Denominator → Value is **less than 1**

```
Examples: 1/3,  3/7,  5/8,  11/15
```

#### Improper Fraction
> Numerator **≥** Denominator → Value is **≥ 1**

```
Examples: 7/4,  9/5,  15/7,  22/3
```

#### Mixed Number (Mixed Fraction)
> A whole number + a proper fraction

```
Examples: 1¾,  2⅔,  5⅛

Converting Mixed → Improper:
  2¾ = (2×4 + 3)/4 = 11/4
  Formula: a(b/c) = (a×c + b)/c

Converting Improper → Mixed:
  17/5 = 3 remainder 2 = 3(2/5)
  Formula: divide, quotient is whole, remainder is numerator
```

#### Like Fractions
> Same denominator

```
1/7,  3/7,  5/7,  6/7  → all have denominator 7
Easy to add/subtract directly
```

#### Unlike Fractions
> Different denominators

```
1/3,  2/5,  3/7  → different denominators
Need to find LCM before adding/subtracting
```

#### Unit Fraction
> Numerator = 1

```
1/2,  1/3,  1/7,  1/11
Key property: Among unit fractions, LARGER denominator = SMALLER value
  1/2 > 1/3 > 1/5 > 1/7
```

#### Equivalent Fractions
> Different fractions representing the **same value**

```
1/2 = 2/4 = 3/6 = 4/8 = 50/100
3/4 = 6/8 = 9/12 = 75/100

Created by: Multiply or divide both parts by the SAME number
```

#### Complex Fraction
> Fraction within a fraction

```
(3/4) / (5/8)  or  (1 + 1/2) / (2 + 1/3)
```

#### Decimal Fraction
> Denominator is a power of 10

```
3/10,  47/100,  125/1000
These convert directly to decimals: 0.3,  0.47,  0.125
```

### 2.2 Equivalent Fractions

```
To create equivalent fractions:
  Multiply top AND bottom by the same number.

  2/3 × 4/4 = 8/12  ← equivalent to 2/3 ✓
  2/3 × 7/7 = 14/21 ← equivalent to 2/3 ✓

To check if two fractions are equivalent:
  Cross multiply — if products are equal, fractions are equivalent.

  Is 3/4 equivalent to 9/12?
  3 × 12 = 36   and   4 × 9 = 36   → YES ✓

  Is 2/5 equivalent to 6/16?
  2 × 16 = 32   and   5 × 6 = 30   → NO ✗
```

### 2.3 Simplifying Fractions (Reducing to Lowest Terms)

> Divide both numerator and denominator by their **HCF**.

```
Simplify 36/48:
  HCF(36, 48) = 12
  36/48 = (36÷12)/(48÷12) = 3/4 ✓

Simplify 126/210:
  HCF(126, 210) = 42
  126/210 = 3/5 ✓

Shortcut — Keep dividing by common small primes:
  126/210 ÷2→ 63/105 ÷3→ 21/35 ÷7→ 3/5 ✓
```

### 2.4 Comparing Fractions

#### Method 1: Same Denominator (Direct Compare)
```
3/7  vs  5/7  → same denominator → 3 < 5 → 3/7 < 5/7
```

#### Method 2: Cross Multiplication ⭐ (Fastest)
```
Compare 5/8 and 7/11:
  5 × 11 = 55   vs   7 × 8 = 56
  55 < 56  →  5/8 < 7/11 ✓

Rule: a/b vs c/d → compare ad vs bc
If ad > bc → a/b > c/d
```

#### Method 3: Convert to Decimals
```
Compare 3/7 and 4/9:
  3/7 ≈ 0.4286
  4/9 ≈ 0.4444
  So 4/9 > 3/7 ✓
```

#### Method 4: Comparing Unit Fractions
```
For unit fractions (1/a vs 1/b):
  Larger denominator = Smaller fraction
  1/5 < 1/3 < 1/2  (because 5 > 3 > 2)
```

#### Arranging Multiple Fractions (Ascending/Descending)

```
Arrange in ascending order: 3/4, 5/6, 7/12, 2/3

  Method: LCM of denominators (4,6,12,3) = 12
  Convert all to /12:
  3/4 = 9/12,  5/6 = 10/12,  7/12 = 7/12,  2/3 = 8/12

  Ascending: 7/12, 8/12, 9/12, 10/12
           = 7/12, 2/3, 3/4, 5/6  ✓
```

### 2.5 Operations on Fractions

#### Addition of Fractions

```
SAME denominators → add numerators directly:
  3/8 + 5/8 = 8/8 = 1 ✓

DIFFERENT denominators → LCM method:
  3/4 + 5/6
  LCM(4,6) = 12
  = 9/12 + 10/12
  = 19/12 = 1 7/12 ✓

Quick Formula (2 fractions):
  a/b + c/d = (ad + bc) / bd   [then simplify]
  3/4 + 5/6 = (18 + 20)/24 = 38/24 = 19/12 ✓
```

#### Subtraction of Fractions

```
SAME denominators:
  7/9 − 4/9 = 3/9 = 1/3 ✓

DIFFERENT denominators:
  5/6 − 3/8
  LCM(6,8) = 24
  = 20/24 − 9/24
  = 11/24 ✓

Mixed numbers subtraction (borrowing):
  3 1/4 − 1 3/4
  = 13/4 − 7/4
  = 6/4 = 3/2 = 1½ ✓
```

#### Multiplication of Fractions

```
Multiply straight across (then simplify):
  3/4 × 8/9
  = 24/36 = 2/3 ✓

BETTER — Cancel first (cross-cancel), then multiply:
  3/4 × 8/9
  ↓ cancel 3 with 9 → 1/3 and cancel 8 with 4 → 2/1
  = 1/1 × 2/3 = 2/3 ✓  (much faster!)

Mixed × fraction:
  2½ × 3/5
  = 5/2 × 3/5
  = 3/2 = 1½ ✓
```

#### Division of Fractions (KFC Rule)

```
KFC = Keep → Flip → Change (operation to ×)

  a/b ÷ c/d = a/b × d/c = ad/bc

Examples:
  3/4 ÷ 9/16 = 3/4 × 16/9 = 48/36 = 4/3 ✓
  
  5/6 ÷ 5    = 5/6 × 1/5 = 1/6 ✓
  
  3 ÷ 3/4    = 3/1 × 4/3 = 4 ✓
  
  2⅔ ÷ 1⅓
  = 8/3 ÷ 4/3
  = 8/3 × 3/4
  = 2 ✓
```

#### Order of Operations with Fractions

```
Evaluate: 3/4 + 2/3 × 1/2 − 1/6

  Step 1 (×): 2/3 × 1/2 = 1/3
  Expression: 3/4 + 1/3 − 1/6

  Step 2 (+ and −): LCM(4,3,6) = 12
  = 9/12 + 4/12 − 2/12
  = 11/12 ✓
```

### 2.6 Complex & Compound Fractions

```
Simplify: (2/3 + 1/4) / (5/6 − 1/3)

  Numerator:   2/3 + 1/4 = 8/12 + 3/12 = 11/12
  Denominator: 5/6 − 1/3 = 5/6 − 2/6 = 3/6 = 1/2

  Result: (11/12) ÷ (1/2) = 11/12 × 2 = 11/6 ✓
```

```
Continued fraction:
  1 + 1/(2 + 1/(1 + 1/2))

  Innermost: 1 + 1/2 = 3/2
  Next:      1 + 1/(3/2) = 1 + 2/3 = 5/3
  Next:      2 + 1/(5/3) = 2 + 3/5 = 13/5
  Outermost: 1 + 1/(13/5) = 1 + 5/13 = 18/13 ✓
```

## 3. DECIMALS — COMPLETE COVERAGE

### 3.1 Place Value System

```
Number: 4 7 3 . 8 9 6
        ↑ ↑ ↑   ↑ ↑ ↑
        H T U . t h th
        u e n   e u o
        n n i   n n u
        d s t   t d s
        r   s   h r a
        e       s e n
        d       → → d
                        t  h  t
                        e  u  h
                        n  n  o
                        t  d  u
                        h  r  s
                        s  e  a
                           d  n
                              d
                              t
                              h
                              s

Position:  100 10  1 . 1/10 1/100 1/1000

4 is in hundreds place   → 400
7 is in tens place        → 70
3 is in units place       → 3
8 is in tenths place      → 8/10  = 0.8
9 is in hundredths place  → 9/100 = 0.09
6 is in thousandths place → 6/1000= 0.006
```

### 3.2 Types of Decimals

#### Terminating Decimal
> The decimal **ends** (finite digits after decimal point).

```
0.25,   1.375,   3.5,   0.125

These occur when the denominator (in lowest terms) has
ONLY factors of 2 and 5.

3/8    → 8 = 2³         → terminates → 0.375 ✓
7/25   → 25 = 5²        → terminates → 0.28 ✓
1/6    → 6 = 2×3        → does NOT terminate (has 3)
```

#### Non-Terminating, Non-Recurring
> Goes on forever with NO pattern.

```
π = 3.14159265358979...  (irrational)
√2 = 1.41421356...       (irrational)
```

#### Non-Terminating, Recurring (Repeating)
> Goes on forever WITH a repeating pattern.

```
1/3 = 0.3333... = 0.3̄
1/7 = 0.142857142857... = 0.1̄4̄2̄8̄5̄7̄
2/9 = 0.2222... = 0.2̄
5/6 = 0.8333... = 0.83̄

Notation: dot above or bar above the repeating part
  0.3̄  means 0.3333...
  0.1̄2̄ means 0.121212...
```

### 3.3 Operations on Decimals

#### Addition & Subtraction
```
RULE: Line up the decimal points! (Pad with zeros if needed)

  24.75
+  3.8       →   3.80
+  0.045     →   0.045
─────────
  28.595 ✓

  12.3 − 4.785:
  12.300
−  4.785
─────────
   7.515 ✓
```

#### Multiplication
```
RULE: Ignore decimals, multiply normally, then count total
      decimal places in BOTH numbers and place decimal there.

  1.25 × 0.4
  = 125 × 4 = 500
  Decimal places: 2 + 1 = 3
  Answer: 0.500 = 0.5 ✓

  0.03 × 0.07
  = 3 × 7 = 21
  Decimal places: 2 + 2 = 4
  Answer: 0.0021 ✓

  2.5 × 2.5
  = 25 × 25 = 625
  Decimal places: 1 + 1 = 2
  Answer: 6.25 ✓
```

#### Division
```
RULE: Multiply both by 10^n to make divisor a whole number.

  1.44 ÷ 0.12
  = 144 ÷ 12  (multiply both by 100)
  = 12 ✓

  0.0036 ÷ 0.06
  = 0.36 ÷ 6  (multiply both by 100... or just move decimals)
  = 0.06 ✓

  Shortcut:
  0.0036/0.06 = 36/600 = 6/100 = 0.06 ✓
```

### 3.4 Recurring Decimals — Converting to Fractions

> This is a HIGH-FREQUENCY exam topic. Master all three cases.

#### Case 1: Fully Recurring (all digits repeat)

```
Formula: Fraction = (Repeating block) / (same number of 9s)

  0.3̄  = 0.333... = 3/9 = 1/3 ✓
  0.7̄  = 7/9 ✓
  0.2̄7̄ = 27/99 = 3/11 ✓
  0.1̄4̄2̄8̄5̄7̄ = 142857/999999 = 1/7 ✓
  0.6̄3̄ = 63/99 = 7/11 ✓
```

#### Case 2: Partially Recurring (some digits don't repeat)

```
Formula: Fraction = (Whole block − Non-repeating block) / (9s for repeating, 0s for non-repeating)

  0.13̄ = 0.1333...
  = (13 − 1) / 90
  = 12/90 = 2/15 ✓

  0.41̄6̄ = 0.41666...
  = (416 − 41) / 900
  = 375/900 = 5/12 ✓

  Pattern of denominator:
  - Count repeating digits → that many 9s
  - Count non-repeating digits (after decimal) → that many 0s
  - Denominator = 9s followed by 0s
```

#### Case 3: Converting back to decimal

```
2/11:
  11 × 0 = 0 rem 2
  11 × 1 = 11 → 20÷11=1 rem 9
  90÷11=8 rem 2  ← same remainder as start → repeats!
  2/11 = 0.1̄8̄ = 0.181818... ✓

Shortcut for /9, /99, /999:
  1/9 = 0.1̄,  2/9 = 0.2̄,  ..., 8/9 = 0.8̄
  1/99 = 0.0̄1̄, 12/99 = 0.1̄2̄, 35/99 = 0.3̄5̄
  1/999 = 0.0̄0̄1̄, 142/999 = 0.1̄4̄2̄
```

#### Operations on Recurring Decimals

```
0.3̄ + 0.6̄ = 1/3 + 2/3 = 1 ✓  (NOT 0.9̄, though 0.9̄ = 1 too!)

0.3̄ × 0.6̄ = 1/3 × 2/3 = 2/9 = 0.2̄ ✓

Tip: ALWAYS convert recurring decimals to fractions
     before doing any arithmetic with them.
```

## 4. PERCENTAGES — COMPLETE COVERAGE

### 4.1 What is Percentage?

> **Percent** means "per hundred" (from Latin *per centum*).
> It is a way of expressing a number as a **fraction of 100**.

```
45% = 45 per hundred = 45/100 = 0.45

Visualize: If you have 100 apples and 45 are red → 45% are red.
```

### 4.2 Percentage Conversions

#### Fraction → Percentage
```
Multiply the fraction by 100:
  3/4 × 100 = 75%
  7/8 × 100 = 87.5%
  5/3 × 100 = 166.67%
```

#### Percentage → Fraction
```
Divide by 100 and simplify:
  35% = 35/100 = 7/20
  12.5% = 12.5/100 = 125/1000 = 1/8
  33.33% = 1/3 (recognize this!)
```

#### Decimal → Percentage
```
Multiply by 100 (shift decimal 2 places RIGHT):
  0.75  → 75%
  0.025 → 2.5%
  1.4   → 140%
  0.003 → 0.3%
```

#### Percentage → Decimal
```
Divide by 100 (shift decimal 2 places LEFT):
  48%   → 0.48
  7%    → 0.07
  125%  → 1.25
  0.5%  → 0.005
```

### 4.3 Types of Percentage Problems

#### Type 1: Find Percentage OF a Number (x% of N)

```
Formula: x% of N = (x/100) × N

Examples:
  25% of 80  = 25/100 × 80 = 20 ✓
  35% of 240 = 35/100 × 240 = 84 ✓
  8% of 1250 = 8/100 × 1250 = 100 ✓

TRICK: Use fractions you've memorized:
  25% of 80 = 1/4 × 80 = 20 ✓  (instant!)
  12.5% of 96 = 1/8 × 96 = 12 ✓
```

#### Type 2: What Percent IS x of N?

```
Formula: (x/N) × 100 = ?%

Examples:
  What % is 15 of 60?
  = (15/60) × 100 = 25% ✓

  What % is 42 of 35?
  = (42/35) × 100 = 120% ✓   (can exceed 100%!)
```

#### Type 3: Find the Whole When Part and Percent Are Known

```
Formula: Whole (N) = (Given value / Given %) × 100

Examples:
  35% of a number is 70. Find the number.
  N = (70/35) × 100 = 200 ✓

  20% of N = 45 → N = 45 × 100/20 = 225 ✓
```

#### Type 4: Percentage More/Less Than

```
"A is 20% more than B"   → A = B × (1 + 20/100) = 1.2B
"A is 20% less than B"   → A = B × (1 − 20/100) = 0.8B

"A is what % more than B?" → [(A−B)/B] × 100
"A is what % less than B?" → [(B−A)/B] × 100

Examples:
  60 is what % more than 50?
  = [(60−50)/50] × 100 = [10/50] × 100 = 20% ✓

  40 is what % less than 80?
  = [(80−40)/80] × 100 = [40/80] × 100 = 50% ✓
```

### 4.4 Percentage Change

```
╔══════════════════════════════════════════════════════════╗
║              PERCENTAGE CHANGE FORMULA                   ║
║                                                          ║
║  % Change = [(New − Old) / Old] × 100                   ║
║                                                          ║
║  % Increase = [(New − Old) / Old] × 100  [when New>Old] ║
║  % Decrease = [(Old − New) / Old] × 100  [when Old>New] ║
╚══════════════════════════════════════════════════════════╝
```

```
Examples:

Price rises from ₹80 to ₹100:
  % increase = [(100−80)/80] × 100 = [20/80] × 100 = 25% ✓

Price falls from ₹500 to ₹400:
  % decrease = [(500−400)/500] × 100 = [100/500] × 100 = 20% ✓

Marks go from 60 to 45:
  % change = [(45−60)/60] × 100 = [−15/60] × 100 = −25% (decrease of 25%) ✓
```

#### Reverse Percentage (Finding Original Value)

```
After a 20% increase, price is ₹120. Original price?
  Multiplier for 20% increase = 1.2
  Original = 120 / 1.2 = ₹100 ✓
  
  OR: If 120% of original = 120 → original = 120 × 100/120 = 100 ✓

After a 25% decrease, price is ₹300. Original price?
  75% of original = 300
  Original = 300 × 100/75 = ₹400 ✓
```

### 4.5 Successive Percentage Change

> When two or more percentage changes are applied **one after another**.

```
╔══════════════════════════════════════════════════════════╗
║         SUCCESSIVE % CHANGE FORMULA                      ║
║                                                          ║
║  Net % change = a + b + ab/100                          ║
║                                                          ║
║  where a = first % change (+ for increase, − for dec)   ║
║        b = second % change                              ║
╚══════════════════════════════════════════════════════════╝
```

```
Example 1: Price increases 20% then decreases 20%.
  a = +20, b = −20
  Net = 20 + (−20) + (20×−20)/100
      = 0 − 400/100
      = 0 − 4 = −4%
  So overall: 4% DECREASE (NOT zero!) ✓

Example 2: Salary increased 10% then 15%.
  Net = 10 + 15 + (10×15)/100
      = 25 + 1.5 = 26.5% increase ✓

Example 3: Population increased 20% then 25% then 10%.
  First apply 20% and 25%:
  Net₁₂ = 20 + 25 + (20×25)/100 = 45 + 5 = 50%
  Then apply 50% and 10%:
  Net = 50 + 10 + (50×10)/100 = 60 + 5 = 65% increase ✓
```

#### Multiplier Method (Faster for 3+ changes)

```
Overall multiplier = Product of all individual multipliers

+20% → × 1.20
−20% → × 0.80
+15% → × 1.15
−5%  → × 0.95

Example: 20% increase then 20% decrease:
  = 1.2 × 0.8 = 0.96 → 4% decrease ✓

Example: 10%, 15%, 10% successive increases on ₹1000:
  = 1000 × 1.10 × 1.15 × 1.10
  = 1000 × 1.3915
  = ₹1391.50 ✓
```

### 4.6 Percentage & Fractions Interplay

#### Relationship Trick — If A is x% of B, then B is (?%) of A

```
If A = x% of B
Then B = (100/x)% of A

Example:
  If A is 25% of B → B is 400% of A
  (because 100/25 = 4 = 400%)

  If A is 20% of B → B is 500% of A ✓
```

#### Percentage Points vs Percentage Change

```
Percentage POINT change = simple arithmetic difference
Percentage CHANGE = relative to original value

If interest rate goes from 6% to 8%:
  Percentage point increase = 8 − 6 = 2 percentage points
  Percentage change = [(8−6)/6] × 100 = 33.33%

⚠️ These are VERY different! Examiners test this.
```

## 6. WORD PROBLEM PATTERNS — ALL TYPES

---

### Pattern 1: Basic "Of" Problems

```
Q: A shopkeeper sold 2/5 of his stock of 450 items. How many sold?

  2/5 of 450 = 2/5 × 450 = 180 items ✓
```

### Pattern 2: Fraction of Remaining

```
Q: Ram spends 1/3 of his salary on rent and 2/5 of the remainder
   on food. What fraction of salary is left?

  Remaining after rent = 1 − 1/3 = 2/3
  Spent on food = 2/5 × 2/3 = 4/15
  Left = 2/3 − 4/15 = 10/15 − 4/15 = 6/15 = 2/5 ✓
```

### Pattern 3: Percentage Application (Tax, Discount)

```
Q: A TV costs ₹15,000. A 12% discount is given, then 5% GST is added.
   Find the final price.

  After discount: 15000 × (1 − 0.12) = 15000 × 0.88 = ₹13,200
  After GST:      13200 × (1 + 0.05) = 13200 × 1.05 = ₹13,860 ✓
```

### Pattern 4: Profit and Loss Percentages

```
Q: CP = ₹200, SP = ₹250. Find profit %.

  Profit = 50
  Profit% = (50/200) × 100 = 25% ✓

Q: If profit% = 20% and SP = ₹360, find CP.

  SP = CP × 1.20
  CP = 360/1.20 = ₹300 ✓
```

### Pattern 5: Fraction/Percent Word Problem

```
Q: A student scored 72 marks out of 80. What % did he score?

  % = (72/80) × 100 = 90% ✓

Q: A number is increased by 20% and then decreased by 20%.
   What is the net % change?

  Net = 20 + (−20) + (20×−20)/100 = −4% (decrease of 4%) ✓
```

### Pattern 6: Comparison Problems

```
Q: A is 40% of B. B is 50% of C. A is what % of C?

  A = 0.4B and B = 0.5C
  A = 0.4 × 0.5C = 0.2C
  A is 20% of C ✓
```

### Pattern 7: Fraction of a Fraction

```
Q: 2/3 of a number exceeds 3/4 of the same number by 6.
   Find the number.

  Let the number = x
  2x/3 − 3x/4 = 6
  (8x − 9x)/12 = 6
  −x/12 = 6
  x = −72

  So: 3/4 exceeds 2/3 by 6 → let 3x/4 − 2x/3 = 6
  (9x − 8x)/12 = 6 → x = 72 ✓
```

### Pattern 8: Population/Value Change Over Years

```
Q: Population of a town is 10,000. It increases by 10% per year.
   What is the population after 3 years?

  P = 10000 × (1.1)³ = 10000 × 1.331 = 13,310 ✓
```

### Pattern 9: Mixture Percentage Problems

```
Q: In a 100L mixture, milk:water = 3:2.
   How much water must be added to make water 50% of the mixture?

  Milk = 60L, Water = 40L
  Let water added = x
  40 + x = 50% of (100 + x)
  40 + x = 50 + 0.5x
  0.5x = 10
  x = 20L ✓
```

### Pattern 10: Percentage Error / Approximation

```
Q: A person measures a wire as 2.5m but actual is 2.4m.
   Find the % error.

  Error = 2.5 − 2.4 = 0.1
  % Error = (0.1/2.4) × 100 = 4.17% ✓
```

### Pattern 11: Fraction-Based Seat/Share Problems

```
Q: A property is shared: 1/3 to son, 1/4 to daughter,
   1/6 to wife. Rest to charity. Property = ₹2,40,000.

  To son: 1/3 × 240000 = 80,000
  To daughter: 1/4 × 240000 = 60,000
  To wife: 1/6 × 240000 = 40,000
  Fraction given = 1/3+1/4+1/6 = 4/12+3/12+2/12 = 9/12 = 3/4
  To charity: 1/4 × 240000 = ₹60,000 ✓
```

### Pattern 12: Increasing/Decreasing Parts Sequentially

```
Q: A tank is 3/4 full. After removing 2/3 of the water, what
   fraction of the tank remains?

  Removed: 2/3 of 3/4 = 2/4 = 1/2
  Remaining: 3/4 − 1/2 = 3/4 − 2/4 = 1/4 ✓
```

## 7. SHORT TRICKS & SPEED METHODS

---

### ⚡ Trick 1: The % Triangle (Instant Formula Selection)

```
          VALUE
           /\
          /  \
         /    \
        /______\
      RATE    BASE
     (%)       (of)

  To find VALUE: Rate × Base / 100
  To find RATE:  Value × 100 / Base
  To find BASE:  Value × 100 / Rate
```

---

### ⚡ Trick 2: x% of y = y% of x (Commutativity of %)

```
This is a HUGE time saver!

  4% of 75 = 75% of 4 = 3  ✓  (75% of 4 is easier!)
  8% of 25 = 25% of 8 = 2  ✓
  16% of 50 = 50% of 16 = 8 ✓

  Whenever one number is a "nice" percentage of the other,
  swap and compute instantly!
```

---

### ⚡ Trick 3: Recurring Decimal Shortcuts

```
Fraction with 9s in denominator:
  n/9   = 0.n̄         (n is single digit)
  n/99  = 0.n̄n̄        (n is 2 digits, pad with 0 if needed)
  n/999 = 0.n̄n̄n̄       (n is 3 digits)

Examples:
  5/9   = 0.5̄ = 0.555...
  07/99 = 0.0̄7̄ = 0.070707...
  142/999 = 0.1̄4̄2̄ = 0.142142142...
  1/7   = 142857/999999 = 0.1̄4̄2̄8̄5̄7̄
```

---

### ⚡ Trick 4: Fraction Comparison — "Bigger Top, Closer to 1" Rule

```
For fractions with:
  Same numerator → bigger denominator = SMALLER fraction
    3/7 < 3/5 < 3/4

  Same denominator → bigger numerator = BIGGER fraction
    4/9 < 6/9 < 8/9

  Both numerator and denominator differ → cross multiply
```

---

### ⚡ Trick 5: Fast % Calculation — Break into Parts

```
Find 37% of 400:
  = 30% + 7%
  = 120 + 28 = 148 ✓

Find 17.5% of 640:
  = 10% + 5% + 2.5%
  = 64 + 32 + 16 = 112 ✓

Find 87.5% of 480:
  = 480 − 12.5% of 480
  = 480 − 60 = 420 ✓
  (87.5% = 7/8, so 480 × 7/8 = 420 ✓)
```

---

### ⚡ Trick 6: Successive % — The −(ab/100) Shortcut

```
When SAME % is first increased then decreased (or vice versa):
  Net loss = (x²/100)%

  20% increase then 20% decrease:
  Net = −(20²/100)% = −400/100 = −4% (loss of 4%) ✓

  10% increase then 10% decrease:
  Net = −(10²/100)% = −1% (loss of 1%) ✓

  15% increase then 15% decrease:
  Net = −(225/100)% = −2.25% (loss of 2.25%) ✓
```

---

### ⚡ Trick 7: Fraction Addition — Diagonal Cross Method

```
a/b + c/d = (ad + bc) / bd  then simplify

  3/5 + 4/7 = (21 + 20)/35 = 41/35 ✓

This is faster than finding LCM for simple 2-fraction additions.
```

```
0.125  → 125/1000 → ÷125 → 1/8 ✓
0.375  → 375/1000 → ÷125 → 3/8 ✓
0.625  → 625/1000 → ÷125 → 5/8 ✓
```

### ⚡ Trick 9: "What % is A more/less than B" — Common Confusion Resolver

```
IMPORTANT: The BASE changes depending on the question!

"A is what % more than B?" → Base is B → [(A−B)/B]×100
"B is what % less than A?" → Base is A → [(A−B)/A]×100

Example:
  A = 60, B = 40
  A is what % more than B?  = (20/40)×100 = 50% ✓
  B is what % less than A?  = (20/60)×100 = 33.33% ✓

  Same difference, DIFFERENT percentages! Never confuse.
```

### ⚡ Trick 11: Simplify Fractions — "Vedic Division" approach

```
To simplify quickly, find the GCD by successive subtraction
or recognize common divisors:

  Are both even? ÷2
  Digit sum divisible by 3? ÷3
  End in 0 or 5? ÷5
  Keep going until no common factor.

  180/252: both even → 90/126 → both even → 45/63 → both÷9 → 5/7 ✓
```

### ⚡ Trick 12: The "Of" Chain — Work Right to Left

```
1/2 of 2/3 of 3/4 of 240

Work right to left:
  3/4 of 240 = 180
  2/3 of 180 = 120
  1/2 of 120 = 60 ✓

OR multiply all fractions × base:
  = 1/2 × 2/3 × 3/4 × 240
  = (1×2×3×240)/(2×3×4) = 1440/24 = 60 ✓
```

### ⚡ Trick 13: Multiplier Approach for Price Problems

```
Instead of computing step by step, combine multipliers:

  After 10% increase and 5% decrease:
  Multiplier = 1.10 × 0.95 = 1.045 → 4.5% net increase ✓

  After 20% discount and 8% GST:
  Net multiplier = 0.80 × 1.08 = 0.864 → 13.6% net decrease ✓
```

```
To compare 0.7̄ and 0.7:
  0.7̄ = 7/9 ≈ 0.777...
  0.7  = 7/10 = 0.700

  0.7̄ > 0.7 ✓

To compare 0.3̄ and 0.33:
  0.3̄ = 1/3 = 0.3333...
  0.33 = 33/100 = 0.3300
  0.3̄ > 0.33 ✓
```

### ⚡ Trick 15: Percentage in Ratio Form

```
If A:B = 3:5, then:
  A/(A+B) = 3/8 = 37.5%
  B/(A+B) = 5/8 = 62.5%

If exam says "A is 3/8 of total" → A:Total = 3:8 → A:B = 3:5
```

## 8. COMMON MISTAKES TO AVOID

| ❌ Mistake | ✅ Correct Approach | Example |
|-----------|---------------------|---------|
| Adding fractions: a/b + c/d = (a+c)/(b+d) | Use LCM or cross method | 1/2+1/3 = 5/6 NOT 2/5 |
| 33% ≈ 1/3 used as exact | 33.33% = 1/3 exactly | Use 1/3 = 33.33%, not 33% |
| % more vs % of confusion | "A is 20% more than B" ≠ "A is 20% of B" | 120 is 20% more than 100; 20 is 20% of 100 |
| Increase then decrease = net 0 | Always net loss = x²/100% | +20% then −20% = −4%, not 0% |
| Same base for % comparisons | Base CHANGES with question | "A more than B" uses B as base |
| 0.3̄ = 0.3 | 0.3̄ = 0.333... ≠ 0.3 | 0.3̄ = 1/3, 0.3 = 3/10 |
| Cancelling wrong parts | Only cancel FACTORS, not addends | (x+2)/(x+3) ≠ 2/3 |
| Dividing fractions: a/b ÷ c = a/(bc) but forget to flip | Flip the divisor | 3/4 ÷ 3/8 = 3/4 × 8/3 = 2 |
| (a+b)% of c = a% + b% of c | Correct if applied to same base | 37% = 30%+7% (only if same base) |
| Percentage point = percentage | They are different | 6% to 8% = 2pp but 33.33% change |
