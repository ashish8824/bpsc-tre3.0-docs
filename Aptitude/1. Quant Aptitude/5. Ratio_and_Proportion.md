## 1. RATIO вҖ” FOUNDATION & CORE CONCEPTS

### 1.1 What is a Ratio?

> A **ratio** is a comparison of two or more quantities of the **same kind** and the **same unit** by division.

```
в•”в•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•—
в•‘                                                              в•‘
в•‘   Ratio of a to b  =  a : b  =  a/b                        в•‘
в•‘                                                              в•‘
в•‘   вҖў a is called the ANTECEDENT (first term)                 в•‘
в•‘   вҖў b is called the CONSEQUENT (second term)                в•‘
в•‘   вҖў b вү  0 (denominator can never be zero)                   в•‘
в•‘                                                              в•‘
в•ҡв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•қ
```

```
Examples:
  If a bag has 3 red balls and 5 blue balls:
  Ratio of red to blue = 3:5
  Ratio of blue to red = 5:3
  Ratio of red to total = 3:8
  Ratio of blue to total = 5:8

  вҡ пёҸ ORDER MATTERS: 3:5 вү  5:3
```

#### Key Rules About Ratios

```
Rule 1: Ratio has NO UNITS вҖ” units must cancel.
  5 km to 10 km = 5:10 = 1:2  вң“
  5 km to 500 m вҶ’ convert first: 5000m to 500m = 10:1  вң“
  5 kg to 10 km вҶ’ CANNOT form ratio (different kinds)  вң—

Rule 2: Ratio is dimensionless (a pure number).

Rule 3: If you multiply OR divide BOTH terms by the same
        non-zero number, the ratio remains unchanged.
  3:4 = 6:8 = 9:12 = 15:20 = 3k:4k  (all equivalent)

Rule 4: A ratio must always be expressed in its LOWEST TERMS
        (simplest form) unless otherwise stated.
```

### 1.2 Types of Ratios

#### Simple Ratio
```
Comparison of two quantities.
  5:8,   3:7,   15:20 = 3:4
```

#### Compound Ratio
```
Product of two or more ratios.
  (a:b) compounded with (c:d) = ac : bd

  Compound ratio of 2:3 and 4:5 = (2Г—4):(3Г—5) = 8:15  вң“
```

#### Duplicate Ratio
```
Square of a ratio.
  Duplicate of a:b = aВІ:bВІ

  Duplicate of 3:4 = 9:16  вң“
  Duplicate of 5:7 = 25:49  вң“
```

#### Sub-Duplicate Ratio
```
Square root of a ratio.
  Sub-duplicate of a:b = вҲҡa:вҲҡb

  Sub-duplicate of 9:25 = 3:5  вң“
  Sub-duplicate of 4:49 = 2:7  вң“
```

#### Triplicate Ratio
```
Cube of a ratio.
  Triplicate of a:b = aВі:bВі

  Triplicate of 2:3 = 8:27  вң“
```

#### Sub-Triplicate Ratio
```
Cube root of a ratio.
  Sub-triplicate of a:b = вҲӣa : вҲӣb

  Sub-triplicate of 8:27 = 2:3  вң“
  Sub-triplicate of 64:125 = 4:5  вң“
```

#### Reciprocal / Inverse Ratio
```
Swap antecedent and consequent.
  Inverse of a:b = b:a

  Inverse of 3:7 = 7:3  вң“
```

#### Ratio of Equality, Inequality, Greater/Lesser Inequality

```
  a:b is ratio of equality if a = b  (e.g., 5:5)
  a:b is ratio of greater inequality if a > b  (e.g., 7:3)
  a:b is ratio of lesser inequality if a < b  (e.g., 3:7)
```

### 1.3 Properties of Ratios

#### Property 1: Multiplying/Dividing Changes Nothing
```
  a:b = ka:kb  for any k вү  0

  4:6 = 2:3 = 8:12 = 20:30  (all the same ratio)
```

#### Property 2: Adding Same Number Changes the Ratio
```
  (a+k):(b+k) вү  a:b  in general

  3:5 вҶ’ add 2 вҶ’ 5:7 вү  3:5  вң“

  EXCEPTION: If a = b (ratio of equality), adding k gives:
  (a+k):(a+k) = 1:1, same ratio of equality.
```

#### Property 3: Adding to a Ratio of Inequality
```
  If a > b (ratio of greater inequality):
    Adding k to both вҶ’ (a+k):(b+k) < a:b  (ratio DECREASES toward 1)

  If a < b (ratio of lesser inequality):
    Adding k to both вҶ’ (a+k):(b+k) > a:b  (ratio INCREASES toward 1)

  Example: 7:3 (greater inequality)
    Add 2: 9:5 вҶ’ 9/5=1.8 < 7/3=2.33  вң“ (decreased)

  Example: 3:7 (lesser inequality)
    Add 2: 5:9 вҶ’ 5/9=0.555 > 3/7=0.429  вң“ (increased toward 1)
```

#### Property 4: Ratio Between More Than Two Quantities
```
  a:b:c means:
    a/b is the ratio of first to second
    b/c is the ratio of second to third
    a/c is the ratio of first to third

  In 2:3:5:
    a=2, b=3, c=5
    a+b+c = 10 (total parts)
    a's fraction = 2/10 = 1/5
    b's fraction = 3/10
    c's fraction = 5/10 = 1/2
```

### 1.4 Simplifying Ratios

```
Divide all terms by their HCF.

  24:36   вҶ’ HCF=12 вҶ’ 2:3  вң“
  15:25   вҶ’ HCF=5  вҶ’ 3:5  вң“
  42:70:56 вҶ’ HCF=14 вҶ’ 3:5:4  вң“

Simplifying ratios with fractions:
  3/4 : 5/6 вҶ’ Multiply both by LCM(4,6)=12
            вҶ’ 9:10  вң“

  1/2 : 1/3 : 1/4 вҶ’ Multiply by LCM(2,3,4)=12
               вҶ’ 6:4:3  вң“

Simplifying ratios with decimals:
  0.3 : 0.45 вҶ’ Multiply by 100 вҶ’ 30:45 вҶ’ Г·15 вҶ’ 2:3  вң“
  1.5 : 2.5  вҶ’ Multiply by 10  вҶ’ 15:25 вҶ’ Г·5  вҶ’ 3:5  вң“
```

### 1.5 Comparing Ratios

#### Method: Convert to Same Denominator (or Cross Multiply)

```
Compare 3:4 and 5:7

  3:4 = 3/4 = 0.75
  5:7 = 5/7 вүҲ 0.714

  3:4 > 5:7  вң“

Cross multiply: 3Г—7=21 vs 5Г—4=20 вҶ’ 21>20 вҶ’ 3/4 > 5/7  вң“
```

#### Arranging Multiple Ratios

```
Arrange in ascending order: 3:5, 4:7, 2:3, 5:8

  3/5=0.600, 4/7вүҲ0.571, 2/3вүҲ0.667, 5/8=0.625

  Ascending: 4:7, 3:5, 5:8, 2:3  вң“
```

### 1.6 Combining Ratios (Finding a:b:c from a:b and b:c)

> This is one of the most tested skills. Master it.

#### Method: Make the Common Term Equal

```
Given: a:b = 3:4  and  b:c = 5:6
Find:  a:b:c = ?

Step 1: Identify common term (b)
Step 2: Make b equal in both ratios
        In a:b=3:4, b=4
        In b:c=5:6, b=5
        LCM(4,5) = 20

Step 3: Scale both ratios so b=20
        a:b = 3:4 вҶ’ Г—5 вҶ’ 15:20
        b:c = 5:6 вҶ’ Г—4 вҶ’ 20:24

Step 4: Combine
        a:b:c = 15:20:24  вң“
```

```
Example 2: a:b = 2:3, b:c = 4:5, c:d = 6:7
Find a:b:c:d

  a:b = 2:3 вҶ’ b=3
  b:c = 4:5 вҶ’ b=4, LCM(3,4)=12
  Scale: a:b = 8:12 and b:c = 12:15
  Now a:b:c = 8:12:15

  c:d = 6:7 вҶ’ c=6
  In a:b:c, c=15, LCM(15,6)=30
  Scale: a:b:c = 16:24:30 and c:d = 30:35
  a:b:c:d = 16:24:30:35  вң“
```

```
SHORTCUT (when values are simple):
  a:b = 3:4 and b:c = 5:6
  a:b:c = (3Г—5):(4Г—5):(4Г—6) = 15:20:24  вң“

  Formula: a:b:c = (aГ—bвӮӮ):(bвӮҒГ—bвӮӮ):(bвӮҒГ—c)
  where bвӮҒ = b in first ratio, bвӮӮ = b in second ratio
```

## 2. PROPORTION вҖ” FOUNDATION & CORE CONCEPTS

### 2.1 What is Proportion?

> **Proportion** is a statement that two ratios are **equal**.

```
в•”в•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•—
в•‘                                                              в•‘
в•‘   a:b :: c:d   means   a/b = c/d                            в•‘
в•‘                                                              в•‘
в•‘   Read as: "a is to b as c is to d"                         в•‘
в•‘                                                              в•‘
в•‘   вҖў a and d are called EXTREMES                              в•‘
в•‘   вҖў b and c are called MEANS                                 в•‘
в•‘                                                              в•‘
в•‘   GOLDEN RULE: Product of MEANS = Product of EXTREMES       в•‘
в•‘                    b Г— c = a Г— d                            в•‘
в•ҡв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•қ
```

```
Example: 3:4 :: 6:8
  Means: 4 Г— 6 = 24
  Extremes: 3 Г— 8 = 24  вң“  (equal вҶ’ it IS a proportion)

Example: 2:5 :: 4:10
  Means: 5 Г— 4 = 20
  Extremes: 2 Г— 10 = 20  вң“
```

### 2.2 Types of Proportion

#### Continued Proportion
```
a, b, c are in continued proportion if:
  a:b = b:c  (equivalently: bВІ = ac)

  b is called the MEAN PROPORTIONAL of a and c.
  b = вҲҡ(ac)

Examples:
  4, 6, 9 вҶ’ 4:6 = 2:3 and 6:9 = 2:3 вҶ’ YES, continued proportion вң“
  bВІ = 4Г—9 = 36 вҶ’ b = 6  вң“

  Find mean proportional of 9 and 25:
  b = вҲҡ(9Г—25) = вҲҡ225 = 15  вң“
  (9:15 = 3:5 and 15:25 = 3:5  вң“)
```

#### Third Proportion
```
If a:b :: b:c, then c is the THIRD PROPORTION to a and b.
  c = bВІ/a

  Third proportion to 4 and 6:
  c = 6ВІ/4 = 36/4 = 9  вң“
```

#### Fourth Proportion
```
If a:b :: c:d, then d is the FOURTH PROPORTION to a, b, c.
  d = (b Г— c) / a

  Fourth proportion to 3, 4, 15:
  d = (4 Г— 15) / 3 = 20  вң“
  (3:4 :: 15:20  вң“)
```

### 2.4 Direct Proportion

> When one quantity **increases**, the other also **increases** (or both decrease together) by the **same ratio**.

```
y вҲқ x  means  y = kx  (k is constant of proportionality)

  yвӮҒ/xвӮҒ = yвӮӮ/xвӮӮ = k  (constant)

Examples:
  More workers вҶ’ more work done  (direct)
  More speed вҶ’ more distance (in same time)  (direct)
  More items вҶ’ more cost  (direct)
```

```
Solving direct proportion problems:

  If 5 pens cost вӮ№35, how much do 8 pens cost?

  Method 1 (Unitary): 1 pen = 35/5 = вӮ№7 вҶ’ 8 pens = 8Г—7 = вӮ№56 вң“

  Method 2 (Ratio): 5:8 = 35:x вҶ’ x = (35Г—8)/5 = вӮ№56 вң“

  Method 3 (Table):
    Pens:  5    8
    Cost: 35    x
    More pens вҶ’ more cost вҶ’ multiply ratio same way
    x = 35 Г— (8/5) = вӮ№56 вң“
```

### 2.5 Inverse Proportion

> When one quantity **increases**, the other **decreases** in the **same ratio**.

```
y вҲқ 1/x  means  y = k/x  OR  xy = k  (constant)

  xвӮҒyвӮҒ = xвӮӮyвӮӮ = k  (product is constant)

Examples:
  More workers вҶ’ less time to complete work  (inverse)
  More speed вҶ’ less time for same distance  (inverse)
  More pipes вҶ’ less time to fill tank  (inverse)
```

```
Solving inverse proportion problems:

  If 6 workers complete a job in 12 days, how many days will
  10 workers take?

  Method 1: xвӮҒyвӮҒ = xвӮӮyвӮӮ
    6 Г— 12 = 10 Г— d вҶ’ d = 72/10 = 7.2 days вң“

  Method 2 (Ratio вҖ” flip for inverse):
    Workers: 6    10
    Days:   12     x
    More workers вҶ’ less days вҶ’ flip ratio
    x = 12 Г— (6/10) = 7.2 days вң“
```

#### How to Identify Direct vs Inverse Proportion

```
Ask: "If I increase the first quantity, does the second:
  INCREASE вҶ’ DIRECT proportion
  DECREASE вҶ’ INVERSE proportion"

  Workers & Days?     More workers вҶ’ fewer days вҶ’ INVERSE  вң“
  Workers & Work?     More workers вҶ’ more work done вҶ’ DIRECT  вң“
  Speed & Distance?   More speed вҶ’ more distance (same time) вҶ’ DIRECT вң“
  Speed & Time?       More speed вҶ’ less time (same distance) вҶ’ INVERSE вң“
  Pipes & Fill Time?  More pipes вҶ’ less time вҶ’ INVERSE  вң“
  Items & Total Cost? More items вҶ’ more cost вҶ’ DIRECT  вң“
```

## 3. COMPOUND RATIO & PROPORTION

### 3.1 Compound Ratio

```
Compound ratio of (a:b), (c:d), (e:f) = ace : bdf

Example: Compound ratio of (2:3), (4:5), (3:7)
  = (2Г—4Г—3):(3Г—5Г—7)
  = 24:105
  = 8:35  вң“
```

### 3.2 Compound Proportion (Chain Rule)

> Used when more than two quantities are in proportion.

```
Problem template:
  AвӮҒ workers working BвӮҒ hours/day do work CвӮҒ in DвӮҒ days.
  AвӮӮ workers working BвӮӮ hours/day do work CвӮӮ in how many days?

  Chain Rule / Unitary Method:

  DвӮӮ = DвӮҒ Г— (AвӮҒ/AвӮӮ) Г— (BвӮҒ/BвӮӮ) Г— (CвӮӮ/CвӮҒ)
              вҶ‘ inverse   вҶ‘ inverse  вҶ‘ direct
  (More workers = less days вҶ’ inverse)
  (More hours/day = less days вҶ’ inverse)
  (More work = more days вҶ’ direct)
```

```
Example:
  12 men working 8 hours/day can do a job in 10 days.
  How many days will 16 men working 6 hours/day take for
  the same job?

  DвӮӮ = 10 Г— (12/16) Г— (8/6)
     = 10 Г— (3/4) Г— (4/3)
     = 10 Г— 1
     = 10 days вң“
```

```
Example (harder):
```

## 4. VARIATIONS

### 4.1 Direct Variation
```
y вҲқ x  вҶ’  y = kx

If (xвӮҒ, yвӮҒ) and (xвӮӮ, yвӮӮ) are two observations:
  yвӮҒ/xвӮҒ = yвӮӮ/xвӮӮ
```

### 4.2 Inverse Variation
```
y вҲқ 1/x  вҶ’  y = k/x  вҶ’  xy = constant

  xвӮҒyвӮҒ = xвӮӮyвӮӮ
```

### 4.3 Joint Variation
```
z вҲқ x Г— y  вҶ’  z = kxy

Example: z varies jointly with x and y.
  z=12 when x=2, y=3 вҶ’ k = 12/(2Г—3) = 2
  So z = 2xy
  When x=4, y=5: z = 2Г—4Г—5 = 40  вң“
```

### 4.4 Combined (Partial) Variation
```
z вҲқ x/y  вҶ’  z = kx/y

Example: z varies directly with x and inversely with y.
  z=6 when x=4, y=2 вҶ’ 6 = kГ—4/2 вҶ’ k=3
  z = 3x/y
  When x=5, y=3: z = 3Г—5/3 = 5  вң“
```

## 5. PARTNERSHIPS

> Partnership problems are ENTIRELY based on ratios. Master ratio вҶ’ master partnerships.

### 5.1 Simple Partnership (Same Time Period)

```
When partners invest for the SAME duration:
  Profit ratio = Ratio of investments

Example:
  A invests вӮ№3000, B invests вӮ№5000, C invests вӮ№7000.
  They share profit in ratio 3:5:7.
  Total profit = вӮ№15,000.
  A's share = (3/15) Г— 15000 = вӮ№3000
  B's share = (5/15) Г— 15000 = вӮ№5000
  C's share = (7/15) Г— 15000 = вӮ№7000  вң“
```

### 5.2 Compound Partnership (Different Time Periods)

```
When partners invest for DIFFERENT durations:
  Profit ratio = Investment Г— Time  (for each partner)

Example:
  A invests вӮ№5000 for 6 months
  B invests вӮ№4000 for 8 months
  C invests вӮ№3000 for 12 months

  A's share ratio: 5000 Г— 6 = 30,000
  B's share ratio: 4000 Г— 8 = 32,000
  C's share ratio: 3000 Г— 12 = 36,000

  Ratio = 30:32:36 = 15:16:18

  Total profit = вӮ№9800 (example)
  A: (15/49) Г— 9800 = вӮ№3000
  B: (16/49) Г— 9800 = вӮ№3200
  C: (18/49) Г— 9800 = вӮ№3600  вң“
```

### 5.3 Working Partner vs Sleeping Partner

```
Working partner: invests capital AND manages business вҶ’ gets extra salary/commission
Sleeping partner: invests capital only

Example:
  A and B invest вӮ№3000 and вӮ№5000.
  A is working partner and gets 10% of profit as salary.
  Total profit = вӮ№800.

  A's salary = 10% of 800 = вӮ№80
  Remaining profit = 720
  Split in 3:5: A gets 3/8Г—720=270, B gets 5/8Г—720=450

  A's total = 80 + 270 = вӮ№350
  B's total = вӮ№450  вң“
```

## 6. MIXTURES & ALLIGATION

### 6.1 Alligation Rule (The Cross Method)

> **Alligation** finds the ratio in which two ingredients at different prices/concentrations must be mixed to get a desired mean value.

```
в•”в•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•—
в•‘                  ALLIGATION DIAGRAM                      в•‘
в•‘                                                          в•‘
в•‘      Cheaper (C)          Dearer (D)                    в•‘
в•‘                                                          в•‘
в•‘           C        Mean (M)       D                     в•‘
в•‘            \                    /                        в•‘
в•‘             \                  /                         в•‘
в•‘              (D вҲ’ M) : (M вҲ’ C)                          в•‘
в•‘                                                          в•‘
в•‘   Ratio of cheaper to dearer = (DвҲ’M) : (MвҲ’C)           в•‘
в•‘                                                          в•‘
в•ҡв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•қ
```

```
Example 1: In what ratio should tea at вӮ№40/kg be mixed with
tea at вӮ№60/kg to get a blend at вӮ№50/kg?

  C=40, D=60, M=50
  Ratio = (DвҲ’M):(MвҲ’C) = (60вҲ’50):(50вҲ’40) = 10:10 = 1:1  вң“
```

```
Example 2: In what ratio should water (free = вӮ№0) be mixed
with milk at вӮ№24/litre to get a mixture at вӮ№20/litre?

  C=0 (water), D=24 (milk), M=20
  Ratio = (24вҲ’20):(20вҲ’0) = 4:20 = 1:5

  Water:Milk = 1:5  вң“
```

### 6.2 Alligation for Mixtures of Concentrations

```
Example: 30% alcohol solution and 60% alcohol solution
         are mixed to get 50% alcohol solution. Find ratio.

  C=30, D=60, M=50
  Ratio = (60вҲ’50):(50вҲ’30) = 10:20 = 1:2

  Mix 1 part of 30% with 2 parts of 60%  вң“

  Verify: (1Г—30 + 2Г—60)/3 = (30+120)/3 = 150/3 = 50% вң“
```

### 6.3 Repeated Dilution / Replacement Formula

```
When a vessel contains x litres of liquid and y litres are
replaced by another liquid n times:

  Amount of original liquid remaining = x Г— (1 вҲ’ y/x)вҒҝ

Example: A 50L vessel has pure milk. Each time, 10L is
         removed and replaced with water. After 3 replacements,
         how much milk remains?

  Milk remaining = 50 Г— (1 вҲ’ 10/50)Ві
                 = 50 Г— (4/5)Ві
                 = 50 Г— 64/125
                 = 25.6 litres  вң“
```

### 6.4 Mixing Two Mixtures

```
Example: Solution A has milk:water = 3:2 (60% milk)
         Solution B has milk:water = 4:1 (80% milk)
         They are mixed in ratio 2:3. Find milk% in mixture.

  Milk from A: 60% Г— 2 parts = 120%В·parts
  Milk from B: 80% Г— 3 parts = 240%В·parts
  Total milk: 360 in 5 total parts
  Milk % = 360/5 = 72%  вң“
```

## 11. COMMON MISTAKES TO AVOID

| вқҢ Mistake | вң… Correct Approach | Example |
|-----------|---------------------|---------|
| Ratio 3:5 same as 5:3 | ORDER matters: 3:5 вү  5:3 | Ratio of boys:girls вү  girls:boys |
| a:b = (a+k):(b+k) after adding k | Adding k CHANGES ratio | 3:5 вҶ’ add 2 вҶ’ 5:7 вү  3:5 |
| Applying HCF to 3-way ratio wrong | Find HCF of ALL three terms | HCF of 6:9:15 = 3 вҶ’ 2:3:5 |
| Cross multiply wrong in proportion | ad = bc (extremes Г— extremes = means Г— means) | 2:3::x:6 вҶ’ 2Г—6=3x вҶ’ x=4 |
| Using sum of ratios as denominator wrong | Denominator = SUM of ALL ratio parts | 3:5 вҶ’ denominator is 8, not 5 |
| Ignoring units before forming ratio | Always convert to SAME unit first | 2kg:500g = 2000g:500g = 4:1 |
| Direct/Inverse confusion | More workers вҶ’ LESS days вҶ’ INVERSE | Never assume direct always |
| Alligation: subtracting wrong way | (DearerвҲ’Mean):(MeanвҲ’Cheaper) | C=20, D=50, M=30: ratio=20:10=2:1 |
