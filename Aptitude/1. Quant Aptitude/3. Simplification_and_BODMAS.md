## 1. WHAT IS SIMPLIFICATION?

> **Simplification** means reducing a complex mathematical expression to its **simplest/single numerical value** by performing operations in the correct order.

### Why it matters in exams:
- Every quantitative aptitude exam has 5вҖ“15 simplification questions.
- They test: order of operations, fraction/decimal skills, and speed.
- A single wrong step = wrong answer. Order is everything.

### What you will simplify:
```
Examples of what gets simplified:

  3 + 4 Г— 2 вҲ’ 1            = ?   (basic BODMAS)
  (18 Г· 3) Г— 4 + 2ВІ        = ?   (brackets + powers)
  2/3 + 5/6 вҲ’ 1/4          = ?   (fractions)
  вҲҡ144 + вҲӣ27 Г— 4ВІ          = ?   (roots + powers)
  [3{2(4+1) вҲ’ 3} + 2] Г— 5  = ?   (nested brackets)
```

## 2. BODMAS вҖ” THE GOLDEN RULE

> **BODMAS** is the universally accepted order in which operations must be performed.

```
в•”в•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•—
в•‘                  B O D M A S                          в•‘
в•‘                                                       в•‘
в•‘  B  вҶ’  Brackets          (solve innermost first)      в•‘
в•‘  O  вҶ’  Orders/Of         (powers, roots, 'of')        в•‘
в•‘  D  вҶ’  Division          (Г·)                          в•‘
в•‘  M  вҶ’  Multiplication    (Г—)                          в•‘
в•‘  A  вҶ’  Addition          (+)                          в•‘
в•‘  S  вҶ’  Subtraction       (вҲ’)                          в•‘
в•ҡв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•қ
```

> вҡ пёҸ **CRITICAL RULE:** D and M have **equal priority** вҖ” do left to right.
> вҡ пёҸ **CRITICAL RULE:** A and S have **equal priority** вҖ” do left to right.

### 2.1 BODMAS vs PEMDAS vs BEDMAS

These are all the same rule with different names used in different countries:

| Name | Countries | Full Form |
|------|-----------|-----------|
| **BODMAS** | India, UK, Pakistan | Brackets, Orders, Division, Multiplication, Addition, Subtraction |
| **PEMDAS** | USA | Parentheses, Exponents, Multiplication, Division, Addition, Subtraction |
| **BEDMAS** | Canada | Brackets, Exponents, Division, Multiplication, Addition, Subtraction |

> рҹҮ®рҹҮі For Indian exams (SSC, IBPS, CAT, RRB): always use **BODMAS**.

### 2.2 Step-by-Step Application

```
Evaluate: 18 вҲ’ 3 Г— (4 + 2) Г· 9 + 5ВІ

Step 1 вҶ’ Brackets:     18 вҲ’ 3 Г— (6) Г· 9 + 5ВІ
Step 2 вҶ’ Orders:       18 вҲ’ 3 Г— 6 Г· 9 + 25
Step 3 вҶ’ Division:     18 вҲ’ 3 Г— (6Г·9) + 25     вҶҗ left to right with Г—
         Actually D and M left to right:
         3 Г— 6 = 18, then 18 Г· 9 = 2
         So:            18 вҲ’ 2 + 25
Step 4 вҶ’ Addition:     18 + 25 вҲ’ 2 = 41
Step 5 вҶ’ Subtraction:  41 вҲ’ 2 = 41

Wait вҖ” let's redo carefully:
  After step 2: 18 вҲ’ 3 Г— 6 Г· 9 + 25
  D and M left to right: 3 Г— 6 = 18, then 18 Г· 9 = 2
  Expression: 18 вҲ’ 2 + 25
  A and S left to right: 18 вҲ’ 2 = 16, then 16 + 25 = 41

  ANSWER = 41 вң“
```

### 2.3 The Left-to-Right Rule Explained

```
WRONG:  8 вҲ’ 3 + 2  =  8 вҲ’ (3+2)  =  8 вҲ’ 5  =  3  вң—
RIGHT:  8 вҲ’ 3 + 2  =  (8вҲ’3) + 2  =  5 + 2  =  7  вң“

```

## 3. BRACKETS вҖ” TYPES & ORDER

### 3.1 The Four Types of Brackets

| Symbol | Name | Also Called | Priority |
|--------|------|-------------|----------|
| `( )` | Round brackets / Parentheses | Small brackets | Innermost вҶ’ solve first |
| `{ }` | Curly brackets / Braces | Middle brackets | Second |
| `[ ]` | Square brackets | Big brackets | Third |
| `вҖ”`  | Vinculum / Bar | Line above | Solved BEFORE all brackets |

### 3.2 Order of Solving Brackets

```
Correct order (inside out):

  Vinculum вҶ’ ( ) вҶ’ { } вҶ’ [ ]
  
  INNERMOST bracket is ALWAYS solved first.
```

### 3.3 Vinculum (Bar) вҖ” The Hidden First Step

> A **vinculum** is a horizontal bar drawn over an expression. Solve it **before any bracket**.

```
Example: 2 Г— [3 + {4 вҲ’ вҖҫ(8вҲ’5)}]
                              вҶ‘ vinculum: 8вҲ’5 = 3

Step 1 Vinculum: = 2 Г— [3 + {4 вҲ’ 3}]
Step 2 ( ): None here
Step 3 { }: = 2 Г— [3 + {1}]
           = 2 Г— [3 + 1]
Step 4 [ ]: = 2 Г— [4]
           = 2 Г— 4
           = 8 вң“
```

### 3.4 Full Nested Bracket Example

```
Evaluate: 5 Г— [3 + {8 вҲ’ (4 + 2) + 1} вҲ’ 2]

Step 1 вҲ’ ( ):  (4+2) = 6
  вҶ’ 5 Г— [3 + {8 вҲ’ 6 + 1} вҲ’ 2]

Step 2 вҲ’ { }:  8 вҲ’ 6 + 1 = 3   (left to right: 8вҲ’6=2, 2+1=3)
  вҶ’ 5 Г— [3 + 3 вҲ’ 2]

Step 3 вҲ’ [ ]:  3 + 3 вҲ’ 2 = 4
  вҶ’ 5 Г— 4

Step 4:  = 20 вң“
```

### 3.5 Negative Sign Before a Bracket

> When a **negative (вҲ’) sign** is placed before a bracket:
> - **Removing the bracket reverses the sign of every term inside.**

```
вҲ’ (a + b)  =  вҲ’a вҲ’ b
вҲ’ (a вҲ’ b)  =  вҲ’a + b
вҲ’ (вҲ’a + b) =  +a вҲ’ b
вҲ’ (вҲ’a вҲ’ b) =  +a + b
```

```
Example: 15 вҲ’ (8 вҲ’ 3 + 2) = ?
WRONG:  15 вҲ’ 8 вҲ’ 3 + 2 = 6   вң—  (didn't reverse all signs)
RIGHT:  15 вҲ’ 8 + 3 вҲ’ 2 = 8   вң“
```

### 3.6 Positive Sign Before a Bracket

> When a **positive (+) sign** is before a bracket:
> - Removing it keeps all signs the same.

```
+ (a + b) = a + b
+ (a вҲ’ b) = a вҲ’ b
```

## 4. EACH OPERATION IN DETAIL

### 4.1 "Of" вҖ” What Does It Mean?

> **"Of" = Multiplication**

```
1/2 of 60  =  1/2 Г— 60  =  30
25% of 80  =  25/100 Г— 80  =  20
3/4 of 1/2 =  3/4 Г— 1/2  =  3/8
```

> рҹ”‘ In BODMAS, **"Of" comes under Orders (O)** вҖ” so it is done before Г· and Г—.
> In practice: treat "of" as multiplication but with high priority (like power/root level).

### 4.2 Division

```
Rules:
  a Г· b = a/b
  a Г· b Г· c = (aГ·b) Г· c = a/(bГ—c)    вҶҗ always left to right
  
  NEVER do:  a Г· (b Г· c) = a Г· b/c  вң—  (unless brackets say so)
```

```
Example:  48 Г· 6 Г· 2
  Correct: (48Г·6) Г· 2 = 8 Г· 2 = 4 вң“
  Wrong:   48 Г· (6Г·2) = 48 Г· 3 = 16 вң—
```

### 4.3 Multiplication

```
Rules:
  a Г— b Г— c  =  any order  (commutative & associative)
  a Г— (вҲ’b)   =  вҲ’ab
  (вҲ’a) Г— (вҲ’b) = +ab
  0 Г— anything = 0
```

### 4.4 Addition and Subtraction

```
Key rule: Always go LEFT to RIGHT when only + and вҲ’ remain.

  20 вҲ’ 5 + 3 вҲ’ 7 + 2
  = (20вҲ’5) + 3 вҲ’ 7 + 2
  = 15 + 3 вҲ’ 7 + 2
  = 18 вҲ’ 7 + 2
  = 11 + 2
  = 13 вң“
```

## 5. FRACTIONS IN SIMPLIFICATION

### 5.1 Types of Fractions

| Type | Definition | Example |
|------|-----------|---------|
| Proper | Numerator < Denominator | 3/7 |
| Improper | Numerator > Denominator | 9/4 |
| Mixed | Whole number + Fraction | 2Вј |
| Like fractions | Same denominator | 1/5, 3/5 |
| Unlike fractions | Different denominators | 1/3, 2/5 |
| Unit fractions | Numerator = 1 | 1/4, 1/7 |

### 5.2 Operations on Fractions

#### Addition & Subtraction вҖ” Make Denominators Equal

```
a/b + c/d = (ad + bc) / bd

Example: 3/4 + 5/6
  LCM(4,6) = 12
  = 9/12 + 10/12 = 19/12 вң“

Example: 7/8 вҲ’ 3/5
  LCM(8,5) = 40
  = 35/40 вҲ’ 24/40 = 11/40 вң“
```

#### Multiplication вҖ” Straight Across

```
a/b Г— c/d = ac/bd

Example: 3/4 Г— 8/9 = 24/36 = 2/3 вң“
Shortcut: Cancel first вҶ’ 3/4 Г— 8/9 = (3Г—8)/(4Г—9) = 1Г—2/(1Г—3) = 2/3 вң“
```

#### Division вҖ” Flip & Multiply (KFC: Keep, Flip, Change)

```
a/b Г· c/d = a/b Г— d/c = ad/bc

Example: 5/6 Г· 10/3 = 5/6 Г— 3/10 = 15/60 = 1/4 вң“
```

### 5.3 Mixed Number Operations

```
Convert mixed to improper before operating:
  2Вҫ = (2Г—4 + 3)/4 = 11/4
  3в…“ = (3Г—3 + 1)/3 = 10/3

Example: 2Вҫ + 1в…”
  = 11/4 + 5/3
  = 33/12 + 20/12
  = 53/12
  = 4 5/12 вң“
```

### 5.4 Complex / Compound Fractions

```
Simplify: (2/3) / (4/9)
  = 2/3 Г· 4/9
  = 2/3 Г— 9/4
  = 18/12
  = 3/2 вң“

Simplify: [1 + 1/(1 + 1/2)]
  Inner: 1 + 1/2 = 3/2
  Then:  1/(3/2) = 2/3
  Then:  1 + 2/3 = 5/3 вң“
```

### 5.5 Continued Fractions (Advanced)

```
Simplify: 1 / (1 + 1/(1 + 1/(1+1)))

  Innermost: 1 + 1 = 2
  Next:      1 + 1/2 = 3/2
  Next:      1 + 1/(3/2) = 1 + 2/3 = 5/3
  Result:    1/(5/3) = 3/5 вң“
```
