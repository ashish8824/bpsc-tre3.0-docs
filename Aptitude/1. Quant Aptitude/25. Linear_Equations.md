## 1. THE BIG PICTURE — WHAT IS A LINEAR EQUATION?

---

### 🔑 Definition:

> A **Linear Equation** is a polynomial equation where the **highest power of the variable is 1** (degree = 1). The graph of a linear equation is always a **straight line**.

$$\boxed{ax + b = 0 \quad \text{where } a \neq 0}$$

---

### Why "Linear"?

- "Linear" = **Line** (Latin: *linearis*)
- The graph of every linear equation is a **straight line**
- Variables appear only to the **first power** — no x², no √x, no 1/x

---

### Examples vs Non-Examples:

| Equation | Linear? | Reason |
|----------|---------|--------|
| 3x + 5 = 0 | ✅ Yes | Degree = 1 |
| 2x + 3y = 7 | ✅ Yes | Two variables, both degree 1 |
| x/3 − 4 = 2 | ✅ Yes | Equivalent to a degree-1 equation |
| x² + 2x = 0 | ❌ No | Degree = 2 (quadratic) |
| √x + 3 = 5 | ❌ No | Variable under square root |
| 1/x + 2 = 0 | ❌ No | Variable in denominator (degree = −1) |
| 0·x + 5 = 0 | ❌ No | a = 0, becomes 5 = 0 (contradiction, not linear) |

---

### The Big Picture — Concept Map:

```
              LINEAR EQUATION
                     │
        ┌────────────┼────────────┐
        │            │            │
   ONE VARIABLE  TWO VARIABLES  THREE VARIABLES
    ax + b = 0   ax + by = c   ax + by + cz = d
        │            │
   One solution  Methods:
                 ├── Substitution
                 ├── Elimination
                 ├── Cross-Multiplication
                 └── Graphical
```

## 2. STANDARD FORMS & KEY TERMINOLOGY

---

### Standard Forms:

| Type | Form | Example |
|------|------|---------|
| One variable | ax + b = 0 | 3x − 9 = 0 |
| Two variables | ax + by + c = 0 | 2x + 3y − 6 = 0 |
| Two variables (slope-intercept) | y = mx + c | y = 2x + 5 |
| Two variables (point-slope) | y − y₁ = m(x − x₁) | y − 3 = 2(x − 1) |
| Three variables | ax + by + cz = d | x + 2y − z = 4 |

---

### Key Terminology:

| Symbol | Name | Meaning |
|--------|------|---------|
| a, b, c | Coefficients | Numbers multiplying variables |
| x, y, z | Variables / Unknowns | Values to be found |
| Solution | Root | Value(s) that satisfy the equation |
| m | Slope / Gradient | Steepness of the line |
| c | y-intercept | Where line crosses y-axis |
| Consistent | — | System has at least one solution |
| Inconsistent | — | System has no solution |
| Dependent | — | System has infinitely many solutions |

---

### What is a "Solution"?

A **solution** is a value (or set of values) that, when substituted into the equation, makes the equation **true** (LHS = RHS).

**Example:** For 2x + 3 = 7:
- Try x = 2: LHS = 2(2) + 3 = 7 = RHS ✅ → **x = 2 is the solution**
- Try x = 1: LHS = 2(1) + 3 = 5 ≠ 7 ❌ → x = 1 is NOT a solution

## 3. LINEAR EQUATIONS IN ONE VARIABLE

---

### Standard Form:

$$ax + b = 0 \quad (a \neq 0) \quad \Rightarrow \quad x = -\frac{b}{a}$$

---

### Golden Rules for Solving:

> **The fundamental principle:** Whatever you do to one side of the equation, you MUST do to the other side.

| Operation | Rule |
|-----------|------|
| Add same number | LHS + k = RHS + k |
| Subtract same number | LHS − k = RHS − k |
| Multiply by same non-zero number | LHS × k = RHS × k |
| Divide by same non-zero number | LHS ÷ k = RHS ÷ k |
| Move term across = sign | Sign changes (+ becomes −, and vice versa) |

---

### Step-by-Step Solving Strategy:

```
Step 1: Clear fractions — multiply both sides by LCM of denominators
Step 2: Expand brackets — use distributive law a(b+c) = ab + ac
Step 3: Collect variable terms on LEFT, constant terms on RIGHT
Step 4: Simplify both sides
Step 5: Divide by the coefficient of the variable
Step 6: VERIFY — substitute back into the original equation
```

---

### Example 1: Basic Linear Equation

> Solve: 3x − 7 = 14

$$3x = 14 + 7 = 21$$
$$x = \frac{21}{3} = \boxed{7}$$

**Verify:** 3(7) − 7 = 21 − 7 = 14 ✅

---

### Example 2: Variables on Both Sides

> Solve: 5x + 3 = 3x + 11

$$5x - 3x = 11 - 3$$
$$2x = 8$$
$$x = \boxed{4}$$

**Verify:** 5(4) + 3 = 23 and 3(4) + 11 = 23 ✅

---

### Example 3: With Brackets

> Solve: 3(2x − 1) − 2(x + 4) = 5

$$6x - 3 - 2x - 8 = 5$$
$$4x - 11 = 5$$
$$4x = 16$$
$$x = \boxed{4}$$

---

### Example 4: With Fractions

> Solve: $\frac{x+1}{3} + \frac{x-2}{4} = 5$

LCM of 3 and 4 = 12. Multiply throughout by 12:

$$4(x+1) + 3(x-2) = 60$$
$$4x + 4 + 3x - 6 = 60$$
$$7x - 2 = 60$$
$$7x = 62$$
$$x = \boxed{\frac{62}{7}}$$

---

### Example 5: Cross-Multiplication for Fractional Equations

> Solve: $\frac{2x-1}{3} = \frac{x+2}{5}$

Cross-multiply:

$$5(2x-1) = 3(x+2)$$
$$10x - 5 = 3x + 6$$
$$7x = 11$$
$$x = \boxed{\frac{11}{7}}$$

---

### Example 6: Complex Fractions

> Solve: $\frac{x}{x-1} + \frac{x-1}{x} = \frac{13}{6}$ (x ≠ 0, x ≠ 1)

Let $t = x + \frac{1}{x}$... Actually, multiply by 6x(x−1):

$$6x^2 + 6(x-1)^2 = 13x(x-1)$$
$$6x^2 + 6x^2 - 12x + 6 = 13x^2 - 13x$$
$$12x^2 - 12x + 6 = 13x^2 - 13x$$
$$x^2 - x - 6 = 0$$
$$(x-3)(x+2) = 0$$
$$\boxed{x = 3 \text{ or } x = -2}$$

> ⚠️ Note: This became quadratic after simplification — always be alert!

---

### Special Cases:

| Result After Solving | Meaning | Example |
|---------------------|---------|---------|
| x = constant | Unique solution | x = 5 |
| 0 = 0 (identity) | Infinitely many solutions | 2x + 4 = 2(x + 2) |
| 0 = non-zero (contradiction) | No solution | x + 1 = x + 5 |

---

#### Example 7: No Solution

> Solve: 2(x + 3) = 2x + 7

$$2x + 6 = 2x + 7$$
$$6 = 7 \quad \text{(Contradiction!)}$$

$$\boxed{\text{No Solution}}$$

---

#### Example 8: Infinite Solutions

> Solve: 3(x − 2) = 3x − 6

$$3x - 6 = 3x - 6$$
$$0 = 0 \quad \text{(Always True)}$$

$$\boxed{\text{Infinitely Many Solutions (All real x)}}$$

## 4. LINEAR EQUATIONS IN TWO VARIABLES

---

### Standard Form:

$$\boxed{ax + by + c = 0 \quad (a, b \text{ not both zero})}$$

---

### Key Insight:

- A single equation in **two variables** does NOT have a unique solution
- It has **infinitely many solutions** — the solution set forms a **straight line**
- A point (x₀, y₀) is a solution if and only if **ax₀ + by₀ + c = 0**

---

### Example 9: Finding Solutions of a Two-Variable Equation

> Find three solutions of 2x + y = 6.

| x | y = 6 − 2x | Solution |
|---|------------|----------|
| 0 | 6 | (0, 6) |
| 1 | 4 | (1, 4) |
| 3 | 0 | (3, 0) |

All three points lie on the same straight line!

---

### Pair of Linear Equations (System of 2 equations, 2 variables):

$$\boxed{a_1x + b_1y + c_1 = 0}$$
$$\boxed{a_2x + b_2y + c_2 = 0}$$

This system is solved to find a **unique point (x, y)** where both lines intersect (if consistent).

## 5. METHODS OF SOLVING A PAIR OF LINEAR EQUATIONS

---

### METHOD 1: Substitution Method

**Steps:**

```
Step 1: From one equation, express one variable in terms of the other
Step 2: Substitute into the second equation → single variable equation
Step 3: Solve the resulting equation
Step 4: Back-substitute to find the other variable
Step 5: Verify in BOTH original equations
```

---

#### Example 10: Substitution

> Solve: x + y = 7 and x − y = 3

From equation 1: **x = 7 − y**

Substitute into equation 2:

$$(7 - y) - y = 3$$
$$7 - 2y = 3$$
$$2y = 4 \Rightarrow y = 2$$

Back-substitute: x = 7 − 2 = **5**

$$\boxed{x = 5, \quad y = 2}$$

**Verify:** 5 + 2 = 7 ✅ and 5 − 2 = 3 ✅

---

#### Example 11: Substitution — Slightly Harder

> Solve: 3x + 2y = 16 and x − y = 1

From equation 2: **x = 1 + y**

Substitute into equation 1:

$$3(1 + y) + 2y = 16$$
$$3 + 3y + 2y = 16$$
$$5y = 13 \Rightarrow y = \frac{13}{5}$$

$$x = 1 + \frac{13}{5} = \frac{18}{5}$$

$$\boxed{x = \frac{18}{5}, \quad y = \frac{13}{5}}$$

---

### METHOD 2: Elimination Method

**Steps:**

```
Step 1: Multiply equations to make the coefficient of one variable EQUAL
Step 2: Add or subtract the equations to eliminate that variable
Step 3: Solve for the remaining variable
Step 4: Substitute back to find the eliminated variable
Step 5: Verify in both original equations
```

---

#### Example 12: Elimination — Add

> Solve: 2x + 3y = 12 and 2x − y = 4

Subtract Eq 2 from Eq 1 to eliminate x:

$$(2x + 3y) - (2x - y) = 12 - 4$$
$$4y = 8 \Rightarrow y = 2$$

Substitute into Eq 2: 2x − 2 = 4 → **x = 3**

$$\boxed{x = 3, \quad y = 2}$$

---

#### Example 13: Elimination — Multiply First

> Solve: 3x + 4y = 10 and 2x − 3y = 1

Multiply Eq 1 by 3 and Eq 2 by 4 (to equate y coefficients):

$$9x + 12y = 30 \quad \cdots (1')$$
$$8x - 12y = 4 \quad \cdots (2')$$

Add both: $17x = 34 \Rightarrow x = 2$

Substitute into Eq 1: 6 + 4y = 10 → 4y = 4 → **y = 1**

$$\boxed{x = 2, \quad y = 1}$$

---

#### Example 14: Elimination — Classic Exam Style

> Solve: 5x − 4y = 22 and 3x + 7y = 11

Multiply Eq 1 by 7 and Eq 2 by 4 (to equate y coefficients):

$$35x - 28y = 154 \quad \cdots (1')$$
$$12x + 28y = 44 \quad \cdots (2')$$

Add: $47x = 198 \Rightarrow x = \frac{198}{47}$

> ⚠️ If fractions appear, re-check your multiplication — or try eliminating x instead.

Multiply Eq 1 by 3 and Eq 2 by 5 (to equate x):

$$15x - 12y = 66 \quad \cdots (1'')$$
$$15x + 35y = 55 \quad \cdots (2'')$$

Subtract: $-47y = 11 \Rightarrow y = \frac{-11}{47}$

Substitute into Eq 1: $5x - 4\left(\frac{-11}{47}\right) = 22$

$$5x + \frac{44}{47} = 22 \Rightarrow 5x = \frac{1034-44}{47} = \frac{990}{47} \Rightarrow x = \frac{198}{47}$$

$$\boxed{x = \frac{198}{47}, \quad y = \frac{-11}{47}}$$

---

### METHOD 3: Cross-Multiplication Method

For the system:

$$a_1x + b_1y + c_1 = 0$$
$$a_2x + b_2y + c_2 = 0$$

The solution is:

$$\boxed{\frac{x}{b_1c_2 - b_2c_1} = \frac{y}{c_1a_2 - c_2a_1} = \frac{1}{a_1b_2 - a_2b_1}}$$

So:

$$x = \frac{b_1c_2 - b_2c_1}{a_1b_2 - a_2b_1}, \quad y = \frac{c_1a_2 - c_2a_1}{a_1b_2 - a_2b_1}$$

---

### 🔑 Cross-Multiplication Memory Device:

Write the coefficients in a matrix and cross-multiply in a cycle:

```
       b₁  c₁  a₁  b₁
       b₂  c₂  a₂  b₂

x ↔ b-c cross    y ↔ c-a cross    1 ↔ a-b cross
```

The **denominator** D = a₁b₂ − a₂b₁ is the same for both x and y.

> ⚠️ If D = 0, the system is NOT consistent-unique (lines are parallel or coincident).

---

#### Example 15: Cross-Multiplication

> Solve: 2x + 3y − 5 = 0 and 4x − y − 3 = 0

Here: a₁=2, b₁=3, c₁=−5 and a₂=4, b₂=−1, c₂=−3

$$x = \frac{(3)(−3) − (−1)(−5)}{(2)(−1) − (4)(3)} = \frac{-9 - 5}{-2 - 12} = \frac{-14}{-14} = 1$$

$$y = \frac{(−5)(4) − (−3)(2)}{(2)(−1) − (4)(3)} = \frac{-20 + 6}{-14} = \frac{-14}{-14} = 1$$

$$\boxed{x = 1, \quad y = 1}$$

---

### METHOD 4: Graphical Method

**Steps:**

```
Step 1: Find at least 2 points for each equation (use x=0 and y=0)
Step 2: Plot both lines on the same graph
Step 3: The point of intersection = the solution
```

| Lines Relationship | System | Solution |
|-------------------|--------|----------|
| Intersecting at one point | Consistent | Unique solution |
| Parallel (never meet) | Inconsistent | No solution |
| Coincident (same line) | Consistent, Dependent | Infinite solutions |

---

#### Example 16: Graphical Method

> Solve graphically: x + y = 6 and x − y = 2

For x + y = 6:

| x | 0 | 6 |
|---|---|---|
| y | 6 | 0 |

For x − y = 2:

| x | 0 | 2 |
|---|---|---|
| y | −2 | 0 |

The two lines intersect at **(4, 2)**.

$$\boxed{x = 4, \quad y = 2}$$

---

### Choosing the Right Method:

| Situation | Best Method |
|-----------|------------|
| One equation is easy to isolate one variable | Substitution |
| Coefficients can be made equal easily | Elimination |
| Need a formula / general solution | Cross-Multiplication |
| Visual understanding needed | Graphical |
| Both equations messy | Elimination (most reliable) |

## 6. CONSISTENCY & INCONSISTENCY OF LINEAR SYSTEMS

---

### The Three Cases — Summary Table:

For the system $a_1x + b_1y + c_1 = 0$ and $a_2x + b_2y + c_2 = 0$:

| Condition | Type | Geometry | Solutions |
|-----------|------|----------|-----------|
| $\dfrac{a_1}{a_2} \neq \dfrac{b_1}{b_2}$ | Consistent, Independent | Lines intersect | **Unique solution** |
| $\dfrac{a_1}{a_2} = \dfrac{b_1}{b_2} = \dfrac{c_1}{c_2}$ | Consistent, Dependent | Coincident lines | **Infinitely many** |
| $\dfrac{a_1}{a_2} = \dfrac{b_1}{b_2} \neq \dfrac{c_1}{c_2}$ | Inconsistent | Parallel lines | **No solution** |

---

### Visual Picture:

```
CASE 1: Unique Solution          CASE 2: No Solution          CASE 3: Infinite Solutions
  (Lines Intersect)              (Lines Parallel)             (Lines Coincident)

     /  /                           /  /                         /
    / ×/    ← intersection         / ≠ /   ← never meet         /  ← same line
   /  /                           /  /                         /
```

---

### Examples:

#### Example 17: Check Consistency

> Check: 2x + 3y = 5 and 4x + 6y = 10

$$\frac{a_1}{a_2} = \frac{2}{4} = \frac{1}{2}, \quad \frac{b_1}{b_2} = \frac{3}{6} = \frac{1}{2}, \quad \frac{c_1}{c_2} = \frac{5}{10} = \frac{1}{2}$$

All ratios equal → **Consistent, Dependent — Infinitely many solutions** (same line!)

---

#### Example 18: Check Consistency

> Check: x + 2y = 3 and 3x + 6y = 5

$$\frac{1}{3} = \frac{2}{6} = \frac{1}{3}, \quad \frac{c_1}{c_2} = \frac{3}{5} \neq \frac{1}{3}$$

First two ratios equal, but third is different → **Inconsistent — No solution** (parallel lines!)

---

#### Example 19: Find k for Infinite Solutions

> For what value of k does $2x + 3y = 7$ and $6x + 9y = k$ have infinitely many solutions?

For infinite solutions: $\frac{2}{6} = \frac{3}{9} = \frac{7}{k}$

$$\frac{1}{3} = \frac{7}{k} \Rightarrow k = 21$$

$$\boxed{k = 21}$$

---

#### Example 20: Find k for No Solution

> For what value of k does $3x + ky = 7$ and $6x + 10y = 14$ have no solution?

For no solution: $\frac{3}{6} = \frac{k}{10} \neq \frac{7}{14}$

$$\frac{1}{2} = \frac{k}{10} \Rightarrow k = 5$$

Check: $\frac{7}{14} = \frac{1}{2}$ — but this equals $\frac{1}{2}$!

So k = 5 gives ALL ratios equal → infinite solutions, not no solution.

Try differently: For NO solution, need $\frac{a_1}{a_2} = \frac{b_1}{b_2} \neq \frac{c_1}{c_2}$

$$\frac{3}{6} = \frac{k}{10} \Rightarrow k = 5, \quad \text{but } \frac{7}{14} = \frac{1}{2} = \frac{1}{2}$$

Here all three ratios are equal → system is inconsistent only if $\frac{c_1}{c_2} \neq \frac{1}{2}$.

> ⚠️ Always check all THREE ratios carefully!

## 7. LINEAR EQUATIONS IN THREE VARIABLES

---

### System Form:

$$a_1x + b_1y + c_1z = d_1$$
$$a_2x + b_2y + c_2z = d_2$$
$$a_3x + b_3y + c_3z = d_3$$

---

### Solving Strategy — Elimination in Stages:

```
Step 1: Use two pairs of equations to eliminate z → get two equations in x and y
Step 2: Solve the resulting 2×2 system for x and y
Step 3: Substitute x and y into any original equation to find z
Step 4: Verify in ALL THREE original equations
```

---

#### Example 21: Three Variables

> Solve: x + y + z = 6, 2x + y − z = 3, x − y + 2z = 7

**Eliminate z:**

Add Eq 1 and Eq 2:
$$3x + 2y = 9 \quad \cdots (A)$$

From Eq 1 × 2 and Eq 3, eliminate z:
$$2x + 2y + 2z = 12 \quad \cdots (1')$$
$$x - y + 2z = 7 \quad \cdots (3)$$
Subtract: $x + 3y = 5 \quad \cdots (B)$

**Solve (A) and (B):**

From (B): x = 5 − 3y. Substitute into (A):
$$3(5 - 3y) + 2y = 9$$
$$15 - 9y + 2y = 9$$
$$-7y = -6 \Rightarrow y = \frac{6}{7}$$

$$x = 5 - 3 \cdot \frac{6}{7} = 5 - \frac{18}{7} = \frac{17}{7}$$

From Eq 1: $z = 6 - x - y = 6 - \frac{17}{7} - \frac{6}{7} = 6 - \frac{23}{7} = \frac{19}{7}$

$$\boxed{x = \frac{17}{7}, \quad y = \frac{6}{7}, \quad z = \frac{19}{7}}$$

---

#### Example 22: Simpler Three-Variable

> Solve: x + y + z = 9, x + y = 7, y + z = 7

From Eq 2: x + y = 7, and Eq 1: z = 9 − 7 = **2**

From Eq 3: y + 2 = 7 → **y = 5**

From Eq 2: x + 5 = 7 → **x = 2**

$$\boxed{x = 2, \quad y = 5, \quad z = 2}$$

## 8. WORD PROBLEMS — MODELLING WITH LINEAR EQUATIONS

---

### 🔑 The 5-Step Framework:

```
Step 1: IDENTIFY — Read carefully, list what is unknown
Step 2: DEFINE — Assign variables (let x = ..., let y = ...)
Step 3: FORMULATE — Translate conditions into equations
Step 4: SOLVE — Use appropriate method
Step 5: INTERPRET — Convert mathematical answer back to context. VERIFY.
```

---

### Type 1: Number Problems

---

#### Example 23: Sum and Difference

> The sum of two numbers is 50 and their difference is 16. Find the numbers.

Let numbers = x and y (x > y).

$$x + y = 50 \quad \cdots (1)$$
$$x - y = 16 \quad \cdots (2)$$

Add: $2x = 66 \Rightarrow x = 33$. Then $y = 50 - 33 = 17$.

$$\boxed{\text{Numbers are 33 and 17}}$$

---

#### Example 24: Digit Problem

> A two-digit number is 7 times the sum of its digits. If 27 is subtracted, the digits are reversed. Find the number.

Let tens digit = x, units digit = y.

Number = 10x + y.

**Condition 1:** 10x + y = 7(x + y) → 10x + y = 7x + 7y → **3x = 6y → x = 2y**

**Condition 2:** (10x + y) − 27 = 10y + x → 9x − 9y = 27 → **x − y = 3**

Substituting x = 2y: 2y − y = 3 → **y = 3, x = 6**

$$\boxed{\text{Number} = 63}$$

---

### Type 2: Age Problems

---

#### Example 25: Age Problem

> Father is 3 times as old as his son. 10 years later, father will be twice as old. Find their current ages.

Let son's age = x, father's age = y.

$$y = 3x \quad \cdots (1)$$
$$y + 10 = 2(x + 10) \quad \cdots (2)$$

From (1) and (2): $3x + 10 = 2x + 20 \Rightarrow x = 10, \quad y = 30$

$$\boxed{\text{Son: 10 years, Father: 30 years}}$$

---

#### Example 26: Age Problem — Past and Future

> 5 years ago, A was twice as old as B. In 5 years, A will be 1.5 times as old as B. Find current ages.

Let current ages: A = x, B = y.

**5 years ago:** x − 5 = 2(y − 5) → x − 5 = 2y − 10 → **x − 2y = −5** ... (1)

**5 years later:** x + 5 = 1.5(y + 5) → x + 5 = 1.5y + 7.5 → **x − 1.5y = 2.5** ... (2)

Subtract (1) from (2): **0.5y = 7.5 → y = 15**

From (1): x = 2(15) − 5 = **25**

$$\boxed{A = 25 \text{ years}, \quad B = 15 \text{ years}}$$

---

### Type 3: Distance, Speed, Time Problems

**Key Formula:**

$$\boxed{Distance = Speed \times Time \quad \Rightarrow \quad d = st}$$

| To find | Formula |
|---------|---------|
| Distance | d = st |
| Speed | s = d/t |
| Time | t = d/s |

---

#### Example 27: Upstream-Downstream

> A boat covers 30 km upstream and 44 km downstream in 10 hours. It covers 40 km upstream and 55 km downstream in 13 hours. Find the speed of the boat and the stream.

Let boat speed in still water = x km/h, stream speed = y km/h.

Upstream speed = x − y, Downstream speed = x + y.

$$\frac{30}{x-y} + \frac{44}{x+y} = 10 \quad \cdots (1)$$
$$\frac{40}{x-y} + \frac{55}{x+y} = 13 \quad \cdots (2)$$

Let $u = \frac{1}{x-y}$ and $v = \frac{1}{x+y}$:

$$30u + 44v = 10 \quad \cdots (A)$$
$$40u + 55v = 13 \quad \cdots (B)$$

Multiply (A) by 4 and (B) by 3:
$$120u + 176v = 40$$
$$120u + 165v = 39$$

Subtract: $11v = 1 \Rightarrow v = \frac{1}{11}$ → x + y = 11

From (A): $30u + 4 = 10 \Rightarrow u = \frac{1}{5}$ → x − y = 5

Solving: **x = 8 km/h, y = 3 km/h**

$$\boxed{\text{Boat speed} = 8 \text{ km/h}, \quad \text{Stream speed} = 3 \text{ km/h}}$$

---

### Type 4: Work Problems

**Key Formula:**

$$\boxed{\text{Work done} = \text{Rate} \times \text{Time} \quad \text{(Rate = 1/days to complete alone)}}$$

---

#### Example 28: Work Problem

> A and B together can do a job in 12 days. A alone can do it in 20 days. In how many days can B alone do it?

Rate of A = 1/20 per day, Rate of (A+B) = 1/12 per day.

Rate of B = 1/12 − 1/20 = (5 − 3)/60 = **2/60 = 1/30** per day.

$$\boxed{\text{B alone: 30 days}}$$

---

### Type 5: Mixture and Alligation Problems

---

#### Example 29: Mixture Problem

> A shopkeeper mixes two types of rice: one at ₹40/kg and another at ₹60/kg to make a 50 kg mixture worth ₹48/kg. How much of each is used?

Let x kg of ₹40 rice and y kg of ₹60 rice.

$$x + y = 50 \quad \cdots (1)$$
$$40x + 60y = 48 \times 50 = 2400 \quad \cdots (2)$$

From (1): x = 50 − y. Substitute into (2):

$$40(50 - y) + 60y = 2400$$
$$2000 + 20y = 2400$$
$$y = 20, \quad x = 30$$

$$\boxed{30 \text{ kg of } ₹40 \text{ rice and } 20 \text{ kg of } ₹60 \text{ rice}}$$

### Type 6: Money / Interest Problems

---

#### Example 30: Simple Interest

> A sum of ₹5000 is invested partly at 6% and partly at 9% per annum. If the total interest in one year is ₹390, find the amount invested at each rate.

Let amount at 6% = x, at 9% = y.

$$x + y = 5000 \quad \cdots (1)$$
$$0.06x + 0.09y = 390 \quad \cdots (2)$$

Multiply (2) by 100: $6x + 9y = 39000$ → Divide by 3: $2x + 3y = 13000$ ... (2')

From (1): x = 5000 − y. Substitute into (2'):

$$2(5000 - y) + 3y = 13000$$
$$10000 + y = 13000$$
$$y = 3000, \quad x = 2000$$

$$\boxed{₹2000 \text{ at 6\%} \text{ and } ₹3000 \text{ at 9\%}}$$

## 9. LINEAR INEQUALITIES

---

### Basics:

A **linear inequality** is like a linear equation but uses <, >, ≤, or ≥ instead of =.

$$ax + b < 0, \quad ax + b > 0, \quad ax + b \leq 0, \quad ax + b \geq 0$$

---

### Rules for Inequalities:

| Operation | Effect on Inequality |
|-----------|---------------------|
| Add or subtract any number | Direction UNCHANGED |
| Multiply/divide by **positive** number | Direction UNCHANGED |
| **Multiply/divide by NEGATIVE number** | ⚠️ Direction **REVERSES** |
| Replace x with −x | ⚠️ Direction **REVERSES** |

---

### Key Rule to Remember:

> 🚨 **Multiplying or dividing by a NEGATIVE number FLIPS the inequality sign!**

$$5 > 3 \quad \Rightarrow \quad -5 < -3 \quad \text{(sign flipped!)}$$

---

#### Example 31: Simple Inequality

> Solve: 3x − 7 < 5

$$3x < 12 \Rightarrow x < 4$$

$$\boxed{x \in (-\infty, 4)}$$

---

#### Example 32: Inequality with Negative Division

> Solve: −2x + 5 ≥ 11

$$-2x \geq 6$$
$$x \leq -3 \quad \text{(sign flips when dividing by −2!)}$$

$$\boxed{x \in (-\infty, -3]}$$

---

#### Example 33: Compound Inequality (Double-sided)

> Solve: −3 < 2x + 1 ≤ 9

Subtract 1 from all parts:
$$-4 < 2x \leq 8$$

Divide by 2:
$$-2 < x \leq 4$$

$$\boxed{x \in (-2, 4]}$$

---

#### Example 34: Inequality with Fractions

> Solve: $\frac{3x-2}{5} \geq \frac{x+3}{3}$

Cross-multiply (denominators are positive, so sign stays same):

$$3(3x-2) \geq 5(x+3)$$
$$9x - 6 \geq 5x + 15$$
$$4x \geq 21$$
$$x \geq \frac{21}{4}$$

$$\boxed{x \in \left[\frac{21}{4}, \infty\right)}$$

---

### Number Line Notation:

| Interval | Notation | Graph |
|----------|----------|-------|
| x > a | (a, ∞) | ○─────→ (open dot at a) |
| x ≥ a | [a, ∞) | ●─────→ (closed dot at a) |
| x < a | (−∞, a) | ←─────○ (open dot at a) |
| x ≤ a | (−∞, a] | ←─────● (closed dot at a) |
| a < x < b | (a, b) | ○───○ |
| a ≤ x ≤ b | [a, b] | ●───● |

## 10. GRAPHICAL INTERPRETATION

---

### Line from Equation — Quick Method:

To draw $ax + by = c$:

1. **x-intercept:** Set y = 0 → x = c/a → point **(c/a, 0)**
2. **y-intercept:** Set x = 0 → y = c/b → point **(0, c/b)**
3. Join the two points with a straight line

---

### Slope (Gradient):

$$\boxed{m = \frac{y_2 - y_1}{x_2 - x_1} = \frac{\text{rise}}{\text{run}}}$$

| Slope | Line |
|-------|------|
| m > 0 | Rising (left to right) |
| m < 0 | Falling (left to right) |
| m = 0 | Horizontal |
| m = ∞ (undefined) | Vertical |

---

### Slope-Intercept Form:

$$y = mx + c$$

- **m** = slope
- **c** = y-intercept (value of y when x = 0)

---

#### Example 35: Convert to Slope-Intercept

> Write 3x − 2y + 6 = 0 in slope-intercept form.

$$-2y = -3x - 6$$
$$y = \frac{3}{2}x + 3$$

Slope = 3/2, y-intercept = 3.

---

### Two Lines: Relationship by Slope:

| Condition | Lines |
|-----------|-------|
| m₁ = m₂ and same intercept | Coincident (same line) |
| m₁ = m₂ and different intercept | Parallel (no solution) |
| m₁ ≠ m₂ | Intersecting (unique solution) |
| m₁ × m₂ = −1 | Perpendicular |

## 11. QUICK TRICKS & SHORTCUTS

---

### 🚀 TRICK 1: Instant Solution for Simple Equations

For ax = b: $x = b/a$ directly. Don't overthink simple equations.

---

### 🚀 TRICK 2: Elimination — Make Coefficients Equal Fast

To eliminate y from $a_1x + b_1y = c_1$ and $a_2x + b_2y = c_2$:

Multiply Eq 1 by $b_2$ and Eq 2 by $b_1$, then subtract.

$$x = \frac{b_2c_1 - b_1c_2}{a_1b_2 - a_2b_1}$$

---

### 🚀 TRICK 3: Cross-Multiplication Formula

$$\frac{x}{b_1c_2 - b_2c_1} = \frac{y}{c_1a_2 - c_2a_1} = \frac{1}{a_1b_2 - a_2b_1}$$

Memorize the denominator: **D = a₁b₂ − a₂b₁** (called the determinant).

---

### 🚀 TRICK 4: Consistency Check in 3 Seconds

Write ratios: $\frac{a_1}{a_2}$, $\frac{b_1}{b_2}$, $\frac{c_1}{c_2}$.

- All three equal → **Infinitely many solutions**
- First two equal, third different → **No solution**
- First two not equal → **Unique solution**

---

### 🚀 TRICK 5: Digit Problems Shortcut

For a two-digit number with tens digit = x and units digit = y:
- Number = **10x + y**
- Reversed number = **10y + x**
- Sum of digits = **x + y**
- Difference of reversed and original = **9(y − x)** (always a multiple of 9!)

---

### 🚀 TRICK 6: Upstream-Downstream Shortcut

Let boat speed = b, stream speed = s.

- Downstream speed = b + s
- Upstream speed = b − s
- $b = \frac{(\text{downstream speed}) + (\text{upstream speed})}{2}$
- $s = \frac{(\text{downstream speed}) - (\text{upstream speed})}{2}$

---

### 🚀 TRICK 7: Age Problems — Substitute at the End

Always define variables as **current ages**. Write all conditions in terms of current age (add/subtract years), then solve.

---

### 🚀 TRICK 8: Reciprocal Substitution for Ugly Equations

If you see $\frac{1}{x-a}$ and $\frac{1}{y-b}$ repeating, substitute:

$$u = \frac{1}{x-a}, \quad v = \frac{1}{y-b}$$

Solve linear system in u and v, then find x and y.

---

### 🚀 TRICK 9: k for Unique / No / Infinite Solutions

- **Unique:** $a_1b_2 \neq a_2b_1$ (D ≠ 0)
- **Infinite:** $\frac{a_1}{a_2} = \frac{b_1}{b_2} = \frac{c_1}{c_2}$
- **None:** $\frac{a_1}{a_2} = \frac{b_1}{b_2} \neq \frac{c_1}{c_2}$

---

### 🚀 TRICK 10: Negative Coefficient Inequality — Always Flip

If the coefficient of x is negative, when you isolate x by dividing, the inequality FLIPS.

$$-3x > 6 \Rightarrow x < -2 \quad \text{(NOT } x > -2\text{!)}$$

---

### 🚀 TRICK 11: LCM Method for Fractions

Multiply the ENTIRE equation by the LCM of all denominators first. This immediately removes all fractions and simplifies solving.

---

### 🚀 TRICK 12: Check Your Answer — 30-Second Verify

Always substitute back. If LHS ≠ RHS, you made an error somewhere. This catches arithmetic mistakes before they cost marks.

---

### 🚀 TRICK 13: Slope from Standard Form

From $ax + by + c = 0$: slope = $-\frac{a}{b}$.

---

### 🚀 TRICK 14: Parallel Line Slope

Two lines are parallel if and only if their slopes are equal but y-intercepts are different.

If one line is $ax + by + c = 0$, a parallel line has the form $ax + by + k = 0$ (same a and b, different c).

---

### 🚀 TRICK 15: Adding and Subtracting Equations

If you need the **sum** of variables: Add the two equations directly (sometimes the answer pops out without further solving!).

Example: x + y = 5 and 2x + 3y = 12. To find x + 2y, subtract the first from the second: x + 2y = 7. Done instantly!

---

### 🚀 TRICK 16: Symmetric Equations Shortcut

If the system is symmetric (i.e., swapping x and y gives the same equations), then **x = y** is always one solution.

Example: x + 2y = 9 and 2x + y = 9 → Adding: 3(x+y) = 18 → x+y = 6. Subtracting: x−y = 0 → x = y = 3.

---

### 🚀 TRICK 17: Word Problem — "More Than / Less Than" Translation

| English | Math |
|---------|------|
| "x is 5 more than y" | x = y + 5 |
| "x is 5 less than y" | x = y − 5 |
| "x exceeds y by 5" | x − y = 5 |
| "x is twice y" | x = 2y |
| "x is k% of y" | x = (k/100)y |

---

### 🚀 TRICK 18: Verify Consistency Without Solving

To determine if the system has a unique/no/infinite solutions — just compute the ratios. **You don't need to fully solve the system!**

---

### 🚀 TRICK 19: Substitution When One Coefficient is 1

If any coefficient of x or y is 1 or −1 in any equation, use **substitution** — it avoids fractions.

---

### 🚀 TRICK 20: Compound Inequality — Do All Steps Simultaneously

For $a < cx + d < b$: subtract d from all three parts, then divide by c (flip if c < 0). Never split into two separate inequalities — it wastes time.

---

## 12. COMMON MISTAKE ALERTS

---

### ⚠️ MISTAKE 1: Forgetting to Flip the Inequality Sign

**Wrong:** $-2x > 8 \Rightarrow x > -4$
**Right:** $-2x > 8 \Rightarrow x < -4$ (divide by negative → sign FLIPS!)

---

### ⚠️ MISTAKE 2: Transposing Without Changing Sign

**Wrong:** 3x + 5 = 2x + 8 → 3x − 2x = 8 + 5 (wrong! the 5 stays positive)
**Right:** 3x + 5 = 2x + 8 → 3x − 2x = 8 − 5 = 3

When you move a term across =, it changes sign: +5 becomes −5.

---

### ⚠️ MISTAKE 3: Multiplying Only Part of the Equation

**Wrong:** $\frac{x}{3} + \frac{x}{4} = 5$ → Multiply by 12: $4x + 3x = 5$ (forgot to multiply the RHS!)
**Right:** $4x + 3x = 60$

Multiply **EVERY term** (including both sides!) by the LCM.

---

### ⚠️ MISTAKE 4: Sign Error When Expanding Brackets

**Wrong:** 3(x − 2) = 3x − 2
**Right:** 3(x − 2) = 3x − **6** (distribute 3 to BOTH terms inside)

---

### ⚠️ MISTAKE 5: Cross-Multiplication Setup Error

For $\frac{a}{b} = \frac{c}{d}$:
**Wrong:** ad = bc (sometimes people swap)
**Right:** **ad = bc** — This IS correct, but be careful: $\frac{2x-1}{3} = \frac{x+2}{5}$ means 5(2x−1) = 3(x+2), NOT 3(2x−1) = 5(x+2).

---

### ⚠️ MISTAKE 6: Setting Up Word Problems — Wrong Variable Definition

Always clearly state: "Let x = ..." Be specific. Writing "Let x = the number" is too vague when there are two numbers. Write "Let x = the larger number, y = the smaller number."

---

### ⚠️ MISTAKE 7: Confusion Between "Sum" and "Difference"

- Sum = x **+** y (both positive)
- Difference = x **−** y (be careful which is subtracted from which!)

"The difference between A and B is 5" means A − B = 5 **if A > B**. Context matters!

---

### ⚠️ MISTAKE 8: Age Problems — Mixing Past/Present/Future

Only one of these is correct at a time. Always define **current** ages as variables, then:
- k years ago → subtract k
- k years later → add k

---

### ⚠️ MISTAKE 9: Reciprocal vs. Negative

- Reciprocal of x = **1/x**
- Negative of x = **−x**

"A number's reciprocal added to itself is 5/2" means $x + \frac{1}{x} = \frac{5}{2}$, **NOT** x + (−x) = 5/2.

---

### ⚠️ MISTAKE 10: Not Checking Extraneous Solutions

When equations involve fractions with variables in denominators, solutions that make the denominator zero are **invalid**. Always check:

If you get x = 3 but the original equation had (x − 3) in the denominator → **reject x = 3**.

---

### ⚠️ MISTAKE 11: Confusion of Consistent vs. Inconsistent

- **Consistent** = has solution(s) [includes both unique and infinite cases]
- **Inconsistent** = has NO solution (parallel lines)

Students often confuse "consistent" with "unique solution only."

---

### ⚠️ MISTAKE 12: Ratio Check — Getting c Ratio Wrong

For $a_1x + b_1y + c_1 = 0$: the ratio for c is $\frac{c_1}{c_2}$, **not** $\frac{-c_1}{-c_2}$ which is the same thing.

But if equations are written as $a_1x + b_1y = d_1$ (not = 0), then $c_1 = -d_1$. Always rewrite in standard form ax + by + c = 0 first before comparing ratios!

### Problem Type → Best Approach:

| Problem Type | Approach |
|-------------|----------|
| One variable, simple | Direct solving (move terms) |
| One variable, fractions | Clear fractions with LCM first |
| Two variables, one has coefficient 1 | Substitution |
| Two variables, coefficients easy to match | Elimination |
| Two variables, need general formula | Cross-multiplication |
| Check uniqueness/consistency only | Compare ratios (no solving!) |
| Word problem | 5-step: Identify → Define → Formulate → Solve → Verify |
| Reciprocal terms (1/x, 1/y) | Substitute u = 1/x, v = 1/y |
| Inequality with negative coefficient | Always check sign flip |
| Compound inequality | Do all operations on all three parts together |

## 🏆 TOP EXAM TIPS

1. **Verify every answer** — Substitute back into the original equation(s). This is worth full marks and catches silly errors.

2. **Clear fractions FIRST** — Before doing anything else with a fractional equation, multiply through by the LCM. Makes everything cleaner.

3. **Sign flip in inequalities** — Dividing or multiplying by a negative number reverses the inequality. This is the #1 inequality mistake.

4. **Ratio method for consistency** — Don't solve the system just to check consistency. Compute the three ratios: a₁/a₂, b₁/b₂, c₁/c₂.

5. **Two-digit number = 10x + y** — Always use this, not "xy". Never confuse the "two-digit number" with the product x × y.

6. **"Let" clearly defines variables** — Write "Let x = current age of A" not just "Let x = age". Full credit in exams requires this clarity.

7. **Elimination is the most reliable method** — When in doubt about which method to use, go with elimination.

8. **Reciprocal substitution is a power move** — When you see 1/(ax+b) appearing repeatedly in both equations, substitute u = 1/(ax+b). It instantly converts the system to a simple linear one.

9. **Word problems: write equations first** — Resist the urge to start computing. Read twice, write ALL conditions as equations, THEN solve.

10. **Parallel lines → NO solution; Same line → INFINITE solutions** — Always pair the geometric picture with the algebraic condition.
