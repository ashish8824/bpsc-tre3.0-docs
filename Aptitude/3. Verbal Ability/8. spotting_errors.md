## 1. WHAT IS SPOTTING ERRORS?

**Spotting Errors** is a question type where a sentence is divided into **three or four labelled parts (A, B, C, D)**. One part contains a grammatical error. You must identify WHICH part has the error.

---

### Standard Question Format:

```
"Find the part of the sentence that has an error. If there is no error, choose (D)."

She / goes to the market / every days / No error
 (A)        (B)               (C)           (D)

Answer: (C) — "every days" should be "every day" (every + singular noun)
```

---

### What Makes Spotting Errors Different From Other Grammar Topics:

```
In Sentence Correction → you fix the error
In Spotting Errors    → you FIND the error (just identify the part)

You don't need to write the correction — you only identify the LOCATION.
This requires you to scan each part quickly and systematically.
```

---

### What Is Being Tested:

```
SPOTTING ERRORS tests:
  ├── Knowledge of grammar rules
  ├── Ability to scan sentences quickly
  ├── Precision (error is in one specific part)
  └── Awareness of common exam-targeted error types
```

## 3. HOW TO READ A SPOTTING ERRORS QUESTION

---

### The Correct Reading Strategy:

```
STEP 1 — Read the ENTIRE sentence first (5 seconds)
          Get the overall meaning and context.

STEP 2 — Identify what TYPE of sentence it is
          Is it a statement? A question? Does it have conditionals/comparisons?

STEP 3 — Apply your scanning checklist (see Section 20)
          Check each part against the most common error types.

STEP 4 — Identify which PART contains the error
          Remember: only ONE part has the error.

STEP 5 — Verify
          Read the corrected version mentally. Does it make sense?
          If yes → mark that option. If not → recheck.
```

---

### Critical Rule — The Error Is ALWAYS in One Specific Part:

```
The sentence is divided into parts A, B, C (sometimes D).
ONLY ONE part has the grammatical error.

Even if multiple things seem slightly "off," choose the part
with the CLEAR, DEFINITE grammatical rule violation.

The other parts are designed to sound slightly odd to confuse you,
but they are grammatically correct.
```

## 4. ERROR TYPE 1 — SUBJECT-VERB AGREEMENT

> The most frequently tested error in all competitive exams. The verb must agree with its subject in number (singular/plural).

---

### Rule 1 — Ignore the Interrupting Phrase

The subject is BEFORE the interrupting phrase. Cross it out mentally.

```
WRONG: The quality of these goods [are] not satisfactory.
                    ↑ interrupting phrase: "of these goods"
The real subject = "quality" (singular) → needs "is"
RIGHT: The quality of these goods [is] not satisfactory.

WRONG: The manager, as well as his employees, [were] invited.
                        ↑ interrupting: "as well as his employees"
Real subject = "manager" (singular) → needs "was"
RIGHT: The manager, as well as his employees, [was] invited.

INTERRUPTING PHRASES TO IGNORE:
along with / together with / as well as / in addition to /
accompanied by / including / rather than / except / besides
```

---

### Rule 2 — Either...Or / Neither...Nor → Proximity Rule

The verb agrees with the **subject NEAREST to it**.

```
WRONG: Neither the students nor the teacher [are] present.
Nearest subject to verb = "teacher" (singular) → needs "is"
RIGHT: Neither the students nor the teacher [is] present.

WRONG: Either the manager or the employees [is] responsible.
Nearest subject = "employees" (plural) → needs "are"
RIGHT: Either the manager or the employees [are] responsible.
```

---

### Rule 3 — Collective Nouns → Singular Verb

```
WRONG: The committee [have] decided to postpone the meeting.
RIGHT: The committee [has] decided to postpone the meeting.

Collective nouns = committee / jury / team / board / class /
family / government / army / council / audience / crowd / staff
→ ALL take SINGULAR verbs in standard exam English
```

---

### Rule 4 — Indefinite Pronouns → Singular

```
ALWAYS SINGULAR:
each / every / either / neither / one / anyone / someone /
everyone / no one / nobody / anybody / somebody / everybody /
nothing / anything / something / everything

WRONG: Each of the players [were] given a medal.
RIGHT: Each of the players [was] given a medal.

WRONG: Everyone [are] welcome to attend the ceremony.
RIGHT: Everyone [is] welcome to attend the ceremony.
```

---

### Rule 5 — "The Number Of" vs "A Number Of"

```
"The number of" + noun → SINGULAR verb
"A number of" + noun   → PLURAL verb

WRONG: The number of accidents [have] increased this year.
RIGHT: The number of accidents [has] increased this year.

WRONG: A number of students [was] absent from class.
RIGHT: A number of students [were] absent from class.
```

---

### Rule 6 — Titles, Amounts, Distances → Singular

```
WRONG: "Romeo and Juliet" [are] a famous play.
RIGHT: "Romeo and Juliet" [is] a famous play.

WRONG: Ten kilometres [are] a long distance to walk.
RIGHT: Ten kilometres [is] a long distance to walk.

WRONG: Five thousand rupees [are] a large sum for him.
RIGHT: Five thousand rupees [is] a large sum for him.
```

---

### Rule 7 — "Every" + Multiple Nouns → Singular

```
WRONG: Every boy and every girl [were] given a prize.
RIGHT: Every boy and every girl [was] given a prize.
(Every + noun + and + every + noun → singular verb always)
```

---

### Rule 8 — Arithmetic Operations → Singular

```
WRONG: Two and two [are] four.
RIGHT: Two and two [is] four.

WRONG: Six divided by two [are] three.
RIGHT: Six divided by two [is] three.
```

---

### Rule 9 — "There is / There are" — Verb Agrees with Subject After It

```
WRONG: There [is] many students in the hall.
RIGHT: There [are] many students in the hall.
("students" = plural → "are")

WRONG: There [are] a cat on the roof.
RIGHT: There [is] a cat on the roof.
("cat" = singular → "is")
```

---

### Rule 10 — News, Information, Furniture → Always Singular

```
WRONG: The news [are] shocking.
RIGHT: The news [is] shocking.

WRONG: The furniture [were] damaged in the fire.
RIGHT: The furniture [was] damaged in the fire.

UNCOUNTABLE NOUNS ALWAYS SINGULAR:
news / advice / information / furniture / luggage /
equipment / knowledge / progress / scenery / traffic
```

## 5. ERROR TYPE 2 — TENSE ERRORS

---

### Rule 1 — Signal Words Determine Tense

```
SIGNAL WORD                    CORRECT TENSE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
yesterday / last / ago / in [year]   → Simple Past
just / already / yet / recently      → Present Perfect
since / for (duration, present)      → Present Perfect / Pres. Perf. Cont.
when [past + past]                   → Past Perfect + Simple Past
by the time [past]                   → Past Perfect
by [future time]                     → Future Perfect
always / usually / every             → Simple Present
now / at this moment                 → Present Continuous
Scientific / universal facts         → ALWAYS Simple Present
No sooner...THAN / Hardly...WHEN    → Past Perfect + Simple Past
```

---

### Rule 2 — Since → Present Perfect (Never Simple Past)

```
WRONG: She [lived] here since 2010.
RIGHT: She [has been living] here since 2010.

WRONG: I [know] him since childhood.
RIGHT: I [have known] him since childhood.

ANY sentence with "since" + ongoing state → MUST use has/have + V3
or has/have + been + V-ing
```

---

### Rule 3 — Past Perfect for Earlier of Two Past Actions

```
WRONG: When I reached the station, the train [left].
RIGHT: When I reached the station, the train [had left].

Pattern: The action that happened FIRST uses Past Perfect (had + V3)
         The action that happened SECOND uses Simple Past

WRONG: She told me that she [finishes] the work yesterday.
RIGHT: She told me that she [had finished] the work yesterday.
```

---

### Rule 4 — "No Sooner...Than" and "Hardly...When"

```
WRONG: No sooner did he arrive [when] the lights went out.
RIGHT: No sooner [had] he arrived [than] the lights went out.

WRONG: Hardly did she sit [than] the phone rang.
RIGHT: Hardly [had] she sat [when] the phone rang.

Structure: No sooner + HAD + subject + V3 + THAN + simple past
           Hardly/Scarcely + HAD + subject + V3 + WHEN + simple past
```

---

### Rule 5 — Universal/Scientific Truths → Always Simple Present

```
WRONG: The teacher said that the earth [revolved] around the sun.
RIGHT: The teacher said that the earth [revolves] around the sun.

WRONG: Newton proved that gravity [attracted] all objects.
RIGHT: Newton proved that gravity [attracts] all objects.

Universal truths DO NOT change tense in reported speech.
```

---

### Rule 6 — Stative Verbs → Never Use Continuous Form

```
Stative verbs CANNOT be in -ing form:
know / believe / understand / want / need / love / hate /
like / prefer / see / hear / smell / taste / seem / appear /
belong / own / contain / have (possession)

WRONG: I [am knowing] the answer.
RIGHT: I [know] the answer.

WRONG: She [is having] a car.
RIGHT: She [has] a car.

EXCEPTION: "Having" is allowed when meaning "experiencing":
"She is having a good time." ✓ (experiencing, not possessing)
```

---

### Rule 7 — Future Tense After Time Conjunctions → Use Present

```
After: when / until / as soon as / before / after / unless / once / by the time
→ Use PRESENT TENSE (not future) for the time clause

WRONG: I will call you when I [will reach] home.
RIGHT: I will call you when I [reach] home.

WRONG: Wait here until he [will come].
RIGHT: Wait here until he [comes].
```

## 6. ERROR TYPE 3 — ARTICLE ERRORS (A / AN / THE)

---

### Rule 1 — A vs AN → Based on SOUND (Not Spelling)

```
AN before VOWEL SOUND: an hour / an honest / an MBA / an heir / an NGO
A before CONSONANT SOUND: a university / a European / a one-way / a uniform

WRONG: She is [an] university student.
RIGHT: She is [a] university student.
("university" starts with "yoo" sound = consonant sound → A)

WRONG: He is [a] honest man.
RIGHT: He is [an] honest man.
("honest" starts with "on" sound = vowel sound → AN)
```

---

### Rule 2 — THE Before Superlatives and Ordinals

```
WRONG: She is [a] best student in the class.
RIGHT: She is [the] best student in the class.
(superlative "best" → always THE)

WRONG: He was [a] first person to arrive.
RIGHT: He was [the] first person to arrive.
(ordinal "first" → always THE)
```

---

### Rule 3 — THE Before Unique Things and Second Mention

```
WRONG: [A] sun rises in the east.
RIGHT: [The] sun rises in the east. (unique = only one sun)

WRONG: I saw a dog. [A] dog was barking.
RIGHT: I saw a dog. [The] dog was barking. (second mention → THE)
```

---

### Rule 4 — THE Before River / Mountain Range / Ocean / Newspaper Names

```
WRONG: She crossed [a] Ganga on her way to the temple.
RIGHT: She crossed [the] Ganga on her way to the temple.

THE before: rivers (the Ganga), mountain ranges (the Himalayas),
oceans (the Pacific), newspapers (the Times of India),
island groups (the Andamans), historical docs (the Constitution)
```

---

### Rule 5 — NO Article Before Abstract Nouns (General)

```
WRONG: [The] honesty is the best policy.
RIGHT: [No article] Honesty is the best policy.

WRONG: [The] love is a beautiful emotion.
RIGHT: [No article] Love is a beautiful emotion.

Abstract nouns in general sense → NO article:
honesty / love / peace / freedom / knowledge / courage
```

---

### Rule 6 — NO Article Before Names of Subjects, Languages, Sports

```
WRONG: She is studying [the] mathematics.
RIGHT: She is studying [no article] mathematics.

WRONG: He plays [the] cricket.
RIGHT: He plays [no article] cricket.

WRONG: She speaks [the] Hindi fluently.
RIGHT: She speaks [no article] Hindi fluently.
```

---

### Rule 7 — "A" or "An" Before Profession (One of Many)

```
WRONG: He is [the] doctor.
RIGHT: He is [a] doctor.
("the doctor" implies a specific one; "a doctor" = one of many)
```

## 7. ERROR TYPE 4 — PREPOSITION ERRORS

---

### Most Tested Preposition Rules:

```
WRONG PREPOSITION          CORRECT PREPOSITION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Married WITH               → Married TO
Different THAN             → Different FROM
Superior/Inferior THAN     → Superior/Inferior TO
Junior/Senior THAN         → Junior/Senior TO
Suffer WITH                → Suffer FROM
Reason OF                  → Reason FOR
Congratulate FOR           → Congratulate ON
Comply TO                  → Comply WITH
Compare TO (when examining differences) → Compare WITH
Blind OF                   → Blind TO (opportunity) / Blind IN one eye
Deaf OF                    → Deaf TO (appeal) / Deaf IN one ear
Good IN                    → Good AT
Interested ABOUT           → Interested IN
Angry ON                   → Angry WITH (person) / Angry AT (thing)
Afraid FROM                → Afraid OF
Ashamed FROM               → Ashamed OF
Accused WITH               → Accused OF
Addicted IN                → Addicted TO
Die BY illness             → Die OF illness
Die BY accident            → Die IN an accident
```

---

### Time Prepositions:

```
AT: specific time → at 5 PM / at noon / at night / at Christmas
ON: days, dates → on Monday / on 15th August / on my birthday
IN: months, years, parts of day → in January / in 2024 / in the morning
BY: deadline → by Friday / by 5 PM
SINCE: point in time → since 2010 / since Monday
FOR: duration → for three years / for two days

WRONG: She has lived here [since] three years.
RIGHT: She has lived here [for] three years.
("three years" = duration → FOR, not SINCE)

WRONG: I met him [in] Monday.
RIGHT: I met him [on] Monday.
(day of week → ON)
```

---

### Fixed Preposition Phrases (Most Tested):

```
Abide BY (rules)              Listen TO (music)
Apply FOR (a job)             Look AT / AFTER / INTO / FOR
Approve OF (a plan)           Object TO (a proposal)
Believe IN (God)              Operate ON (a patient)
Blame FOR (something)         Participate IN (an event)
Capable OF (doing)            Pay FOR (something)
Care FOR / ABOUT              Pleased WITH (result)
Complain ABOUT/TO             Prefer TO (alternative)
Comply WITH (rules)           Proud OF (achievement)
Congratulate ON (success)     Rely ON (someone)
Deal WITH (problem)           Responsible FOR (task)
Depend ON (circumstances)     Result IN (outcome)
Differ FROM (another)         Search FOR (something)
Eligible FOR (a post)         Sensitive TO (criticism)
Envious OF (someone)          Similar TO (something)
Fond OF (something)           Suffer FROM (illness)
Good AT (a subject)           Sure OF/ABOUT (a fact)
```

## 8. ERROR TYPE 5 — PRONOUN ERRORS

---

### Rule 1 — Subject vs Object Case

```
SUBJECT pronouns (before verb): I, he, she, we, they, who
OBJECT pronouns (after preposition/verb): me, him, her, us, them, whom

WRONG: Between you and [I], the plan is flawed.
RIGHT: Between you and [me]. (after preposition "between" → object)

WRONG: She gave the prize to [he and I].
RIGHT: She gave the prize to [him and me]. (after "to" → object)

TRICK: Remove the other person and test alone:
"Between I" sounds wrong → "Between me" ✓
"Give it to I" sounds wrong → "Give it to me" ✓
```

---

### Rule 2 — Pronoun-Antecedent Agreement

```
WRONG: Every student must submit [their] project.
RIGHT: Every student must submit [his or her] project.
("every student" = singular → his or her, not their)

WRONG: The company lost [their] profits.
RIGHT: The company lost [its] profits.
(company = singular → its, not their)

WRONG: Each of the boys must carry [their] bag.
RIGHT: Each of the boys must carry [his] bag.
(each = singular → his)
```

---

### Rule 3 — Who vs Whom

```
WHO = subject (replace test: if "he/she" fits → WHO)
WHOM = object (replace test: if "him/her" fits → WHOM)

WRONG: To [who] did you speak at the meeting?
RIGHT: To [whom] did you speak? (you spoke to him → WHOM)

WRONG: [Whom] is the leader of the group?
RIGHT: [Who] is the leader? (he is the leader → WHO)
```

---

### Rule 4 — Reflexive Pronoun Errors

```
Reflexive (myself/yourself/himself/herself/itself/ourselves/themselves):
Used ONLY when subject = object (the person acts on themselves)

WRONG: Please contact [myself] for further information.
RIGHT: Please contact [me]. (different people → no reflexive)

WRONG: He hurt [him] while climbing.
RIGHT: He hurt [himself]. (same person → reflexive needed)

WRONG: [Myself] and my team completed the project.
RIGHT: [I] and my team completed the project. (subject position → I, not myself)
```

---

### Rule 5 — Relative Pronoun Errors

```
WHO/WHOM → for PEOPLE
WHICH → for THINGS/ANIMALS
THAT → for PEOPLE or THINGS (restrictive clauses)
WHAT → "the thing that" (never used after a noun)

WRONG: The man [which] I met was helpful.
RIGHT: The man [who] I met was helpful.

WRONG: This is [what] the book I need.
RIGHT: This is the book [that] I need.
OR: This is [what] I need. (no noun before "what")
```

## 9. ERROR TYPE 6 — ADJECTIVE & ADVERB ERRORS

---

### Rule 1 — Adjective vs Adverb Usage

```
ADJECTIVE → modifies a NOUN (placed before noun or after linking verb)
ADVERB → modifies a VERB, ADJECTIVE, or another ADVERB

WRONG: She sings [beautiful].
RIGHT: She sings [beautifully]. (modifies verb "sings" → adverb)

WRONG: He drives [careful].
RIGHT: He drives [carefully]. (modifies verb "drives" → adverb)

WRONG: She looks at me [angry].
RIGHT: She looks at me [angrily]. (modifies verb "looks" → adverb)

EXCEPTION — Linking verbs take ADJECTIVES:
She looks [beautiful]. ✓ (looks = linking verb → adjective)
The food tastes [good]. ✓ (tastes = linking verb → adjective)
He seems [tired]. ✓ (seems = linking verb → adjective)
```

---

### Rule 2 — "Hardly," "Scarcely," "Barely" → Already Negative

```
These words are ALREADY NEGATIVE. Adding "not" = double negative (WRONG).

WRONG: I can [hardly not] believe it.
RIGHT: I can [hardly] believe it.

WRONG: She [barely didn't] make it in time.
RIGHT: She [barely] made it in time.
```

---

### Rule 3 — Comparative vs Superlative

```
TWO items → Comparative (-er / more)
THREE+ items → Superlative (-est / most)

WRONG: She is [the tallest] of the two sisters.
RIGHT: She is [the taller] of the two sisters.

WRONG: He is [more intelligent] of all the students.
RIGHT: He is [the most intelligent] of all the students.
```

---

### Rule 4 — Double Comparative (Wrong)

```
WRONG: She is [more smarter] than him.
RIGHT: She is [smarter] than him.

WRONG: This is [the most tallest] building.
RIGHT: This is [the tallest] building.

Never use "more" + "-er" or "most" + "-est" together.
```

---

### Rule 5 — Flat Adverbs (No -ly Needed)

```
Some words function as both adjective AND adverb without changing:
fast / hard / late / early / high / deep / long / near / far / straight

WRONG: He ran [fastly] to catch the bus.
RIGHT: He ran [fast] to catch the bus.

WRONG: She worked [hardly] all day.
RIGHT: She worked [hard] all day.
(Note: "hardly" = barely/scarcely — completely different meaning!)
```

---

## 10. ERROR TYPE 7 — COMPARISON ERRORS

---

### Rule 1 — Compare Like with Like

```
You must compare the SAME TYPE of thing.

WRONG: The population of China is more than [India].
(Comparing "population" with "India" — illogical)
RIGHT: The population of China is more than [that of India].

WRONG: His handwriting is better than [his brother].
(Comparing "handwriting" with "his brother" — illogical)
RIGHT: His handwriting is better than [his brother's].
```

---

### Rule 2 — "Any" vs "Any Other" in Comparisons

```
When comparing a member WITH its own group → use "any other"

WRONG: Mumbai is larger than [any city] in Maharashtra.
(Mumbai IS a city in Maharashtra — comparing with itself = illogical)
RIGHT: Mumbai is larger than [any other city] in Maharashtra.

WRONG: She is smarter than [any student] in her class.
RIGHT: She is smarter than [any other student] in her class.
```

---

### Rule 3 — "As...As" Comparison

```
WRONG: She is as [tall or taller] than him.
RIGHT: She is [as tall as or taller than] him.

WRONG: He is not as [smarter] as her.
RIGHT: He is not as [smart] as her.
(No comparative form within "as...as")

WRONG: [More] she studies, [more] she learns.
RIGHT: [The more] she studies, [the more] she learns.
(Double "the" structure for proportional comparisons)
```

---

### Rule 4 — Superior / Inferior / Senior / Junior → Always "TO"

```
WRONG: This product is superior [than] that one.
RIGHT: This product is superior [to] that one.

WRONG: He is junior [than] her by two years.
RIGHT: He is junior [to] her by two years.

Words that ALWAYS take "to" (never "than"):
superior / inferior / senior / junior / prior / anterior / posterior
```

---

## 11. ERROR TYPE 8 — CONJUNCTION ERRORS

---

### Rule 1 — Correlative Conjunction Pairs (Must Stay Together)

```
BOTH...AND (not "both...as well as")
EITHER...OR (not "either...nor")
NEITHER...NOR (not "neither...or")
NOT ONLY...BUT ALSO (not "not only...but")
BOTH...AS WELL AS → WRONG combination
NO SOONER...THAN (not "no sooner...when")
HARDLY/SCARCELY...WHEN (not "hardly...than")
WHETHER...OR (not "whether...or not...or")
AS...AS (not "as...so" in positive comparisons)

WRONG: Neither she [or] he was present.
RIGHT: Neither she [nor] he was present.

WRONG: Not only she sings [but] she also dances.
RIGHT: Not only does she sing [but also] dances. (parallel)
```

---

### Rule 2 — "Unless" Never Takes "Not"

```
"Unless" = "if not" — already contains the negative.

WRONG: Unless you [don't] hurry, you will miss the train.
RIGHT: Unless you [hurry], you will miss the train.

WRONG: Unless he [doesn't] study, he will fail.
RIGHT: Unless he [studies], he will fail.

Any time "unless...not" appears → it's an ERROR. Remove "not."
```

---

### Rule 3 — "Although / Though" vs "But" vs "Despite"

```
"Although / Though" + SUBJECT + VERB (full clause)
"Despite / In spite of" + NOUN / PRONOUN / V-ING (NOT full clause)

WRONG: [Despite] she was tired, she continued working.
RIGHT: [Although] she was tired, she continued working.

WRONG: [Although] being tired, she continued.
RIGHT: [Despite] being tired, she continued.

WRONG: She is honest, [but however] she makes mistakes.
RIGHT: She is honest, [but] she makes mistakes.
(Never use "but" and "however" together — choose one)
```

---

### Rule 4 — "Because" Never Used with "The Reason Is"

```
WRONG: The reason for his failure is [because] he was lazy.
RIGHT: The reason for his failure is [that] he was lazy.
(OR: The reason for his failure is his laziness.)

"The reason is because" = redundant. Always use "that."
```

---

### Rule 5 — "Lest...Should" (Never "Lest...Might/Would")

```
"Lest" means "for fear that" → always followed by "should"

WRONG: He ran fast lest he [might] miss the bus.
RIGHT: He ran fast lest he [should] miss the bus.

WRONG: Study hard lest you [would] fail.
RIGHT: Study hard lest you [should] fail.
```

---

## 12. ERROR TYPE 9 — MODIFIER ERRORS

---

### Dangling Modifier (Most Tested Modifier Error)

```
A dangling modifier has no logical subject in the sentence.
The action in the modifier must be performed by the SUBJECT of the main clause.

WRONG: Running to catch the bus, [my bag fell].
(Who is running? My bag cannot run.)
RIGHT: Running to catch the bus, [I dropped my bag].

WRONG: Having studied all night, [the exam seemed easy to him].
(Who studied? The exam didn't study.)
RIGHT: Having studied all night, [he found the exam easy].

WRONG: Walking along the road, [the scenery was beautiful].
(Scenery doesn't walk.)
RIGHT: Walking along the road, [she found the scenery beautiful].
```

---

### Misplaced Modifier

```
WRONG: She almost drove her children to school every day.
(almost modifies "drove" → she almost drove = didn't drive? Odd.)
RIGHT: She drove her children to school almost every day.
(almost modifies "every day" → the intended meaning)

WRONG: I only eat vegetables on Mondays.
(only modifies "eat" → sounds like she does nothing but eat)
RIGHT: I eat only vegetables on Mondays.
(only modifies "vegetables" → the intended meaning)
```

---

## 13. ERROR TYPE 10 — PARALLELISM ERRORS

---

### Rule 1 — Items in a List Must Have the Same Grammatical Form

```
WRONG: She likes reading, to swim, and [dances].
RIGHT: She likes reading, swimming, and [dancing]. (all V-ing)

WRONG: The job requires skill, patience, and [being dedicated].
RIGHT: The job requires skill, patience, and [dedication]. (all nouns)

WRONG: He not only sings [but is also playing] guitar.
RIGHT: He not only [sings] but also [plays] guitar. (both simple present)
```

---

### Rule 2 — Correlative Conjunctions → Parallel Structures

```
BOTH...AND → same form after both elements
EITHER...OR → same form after both elements

WRONG: She is both [intelligent] and [has creativity].
RIGHT: She is both [intelligent] and [creative]. (both adjectives)

WRONG: Either [come with us] or [you can stay here].
RIGHT: Either [come with us] or [stay here]. (both imperative verbs)
```

---

### Rule 3 — Comparison Must Be Parallel

```
WRONG: Swimming is more tiring than [to run].
RIGHT: Swimming is more tiring than [running]. (both V-ing)

WRONG: To err is human; [forgiving] is divine.
RIGHT: To err is human; [to forgive] is divine. (both "to + V")
```

---

## 14. ERROR TYPE 11 — NOUN ERRORS (COUNTABLE / UNCOUNTABLE)

---

### Rule 1 — Uncountable Nouns Have No Plural Form

```
NO PLURAL for: advice / information / furniture / luggage /
               equipment / news / knowledge / progress /
               scenery / machinery / traffic / work (abstract)

WRONG: She gave me some useful [advices].
RIGHT: She gave me some useful [advice]. (no plural)

WRONG: The [furnitures] were damaged.
RIGHT: The [furniture] was damaged. (no plural + singular verb)

WRONG: I received many [informations] about the job.
RIGHT: I received [much information] / [a lot of information].
```

---

### Rule 2 — Fewer vs Less

```
FEWER → countable nouns (fewer students, fewer books)
LESS → uncountable nouns (less water, less information)

WRONG: There are [less] students in the class today.
RIGHT: There are [fewer] students in the class today.

WRONG: We have [fewer] time to complete this task.
RIGHT: We have [less] time to complete this task. (time = uncountable)
```

---

### Rule 3 — Many vs Much / Few vs Little

```
MANY / FEW → countable nouns
MUCH / LITTLE → uncountable nouns

WRONG: There is [many] water in the tank.
RIGHT: There is [much] water in the tank.

WRONG: She has [little] friends.
RIGHT: She has [few] friends.
```

---

### Rule 4 — "Number" vs "Amount"

```
NUMBER → countable (a large number of students)
AMOUNT → uncountable (a large amount of water)

WRONG: A large [amount] of people attended the rally.
RIGHT: A large [number] of people attended the rally.
(people = countable → number)
```

---

## 15. ERROR TYPE 12 — CONDITIONAL SENTENCE ERRORS

---

### The Four Types and Their Formulas:

```
TYPE 0 (Always true):
If + Simple Present → Simple Present
If you heat ice, it melts.

TYPE 1 (Real future):
If + Simple Present → will + V1
WRONG: If she [will study], she will pass.
RIGHT: If she [studies], she will pass.
(NEVER use "will" in the if-clause of Type 1)

TYPE 2 (Hypothetical present):
If + Simple Past / WERE → would + V1
WRONG: If I [was] you, I would apologise.
RIGHT: If I [were] you, I would apologise.
(Use "were" for ALL persons in Type 2)

WRONG: If she worked harder, she [will] succeed.
RIGHT: If she worked harder, she [would] succeed.

TYPE 3 (Unreal past):
If + Past Perfect → would have + V3
WRONG: If he [would have tried], he would have succeeded.
RIGHT: If he [had tried], he would have succeeded.
(NEVER use "would have" in the if-clause of Type 3)

WRONG: If she had studied, she would [passed].
RIGHT: If she had studied, she would [have passed].
```

---

## 16. ERROR TYPE 13 — REPORTED SPEECH ERRORS

---

### Rule 1 — Tense Shifts in Reported Speech

```
DIRECT → REPORTED
Simple Present → Simple Past
Present Continuous → Past Continuous
Present Perfect → Past Perfect
Simple Past → Past Perfect
Will → Would
Can → Could
May → Might
Must/Have to → Had to

WRONG: She said that she [is] tired.
RIGHT: She said that she [was] tired.

WRONG: He told me that he [will] call me later.
RIGHT: He told me that he [would] call me later.
```

---

### Rule 2 — No Inversion in Reported Questions

```
WRONG: She asked where [did I live].
RIGHT: She asked where [I lived]. (no auxiliary "did" in reported questions)

WRONG: He asked what [was my name].
RIGHT: He asked what [my name was]. (subject before verb in reported speech)
```

---

### Rule 3 — Universal Truths Stay in Simple Present

```
WRONG: The teacher said that the sun [rose] in the east.
RIGHT: The teacher said that the sun [rises] in the east.
(Universal truth → always Simple Present even in reported speech)
```

---

### Rule 4 — "That" in Reported Statements

```
For reporting statements: said that / told + object + that

WRONG: She [told that] she was hungry.
RIGHT: She [said that] she was hungry.
(OR: She [told me] she was hungry — "told" needs an object)

"Tell" needs an object: told me / told him / told her
"Say" doesn't need an object: said / said that
```

---

## 17. ERROR TYPE 14 — DOUBLE NEGATIVES & REDUNDANCY

---

### Rule 1 — Double Negative = Wrong

```
WRONG: I [don't know nothing] about it.
RIGHT: I [don't know anything] about it.
OR: I [know nothing] about it.

WRONG: She [never did nothing] wrong.
RIGHT: She [never did anything] wrong.
OR: She [never did] anything wrong.

Negative words: no / not / never / neither / nor / nobody /
                nothing / nowhere / hardly / barely / scarcely
→ These already make the sentence negative. Adding another = error.
```

---

### Rule 2 — Redundancy (Saying the Same Thing Twice)

```
WRONG: They [returned back] home after the trip.
RIGHT: They [returned] home. ("back" is redundant with "returned")

WRONG: She [completely finished] the work.
RIGHT: She [finished] the work. ("finished" is already complete)

WRONG: The [final conclusion] was drawn.
RIGHT: The [conclusion] was drawn. ("final" is redundant with "conclusion")

WRONG: He is [advancing forward] rapidly.
RIGHT: He is [advancing] rapidly. ("forward" is redundant)

COMMON REDUNDANT PHRASES:
past history / advance forward / free gift / end result /
unexpected surprise / safe haven / close proximity / future plans /
repeat again / revert back / ATM machine / PIN number
```

---

### Rule 3 — "Hardly," "Barely," "Scarcely" = Already Negative

```
WRONG: She [can't hardly] speak.
RIGHT: She [can hardly] speak.

WRONG: He [barely didn't] finish in time.
RIGHT: He [barely] finished in time.
```

---

## 18. ERROR TYPE 15 — WORD CHOICE & CONFUSING WORDS

---

### Most Tested Word Confusion in Spotting Errors:

```
WORD PAIR            CORRECT USAGE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
affect / effect      affect (verb) = to influence
                     effect (noun) = the result
                     WRONG: The rain [effected] his mood.
                     RIGHT: The rain [affected] his mood.

lie / lay            lie (intransitive) = to recline → lay, lain
                     lay (transitive) = to place → laid, laid
                     WRONG: He was [laying] on the bed.
                     RIGHT: He was [lying] on the bed.

rise / raise         rise (intransitive) = to go up → rose, risen
                     raise (transitive) = to lift → raised, raised
                     WRONG: She [raised] from her seat.
                     RIGHT: She [rose] from her seat.

eminent / imminent   eminent = distinguished/famous
                     imminent = about to happen
                     WRONG: The danger is [eminent].
                     RIGHT: The danger is [imminent].

flaunt / flout       flaunt = to show off
                     flout = to openly disobey
                     WRONG: He [flaunted] the rules.
                     RIGHT: He [flouted] the rules.

amount / number      amount = uncountable (amount of water)
                     number = countable (number of students)
                     WRONG: A large [amount] of people came.
                     RIGHT: A large [number] of people came.

beside / besides     beside = next to (physical position)
                     besides = in addition to / apart from
                     WRONG: [Beside] her excellent grades, she won awards.
                     RIGHT: [Besides] her excellent grades, she won awards.

Principal / Principle principal = main / head of school
                     principle = a rule or belief
                     WRONG: The principal [principal] of honesty guides her.
                     RIGHT: The [principle] of honesty guides her.

complement /         complement = completes / goes well with
compliment           compliment = an expression of praise
                     WRONG: The wine [compliments] the food.
                     RIGHT: The wine [complements] the food.

infer / imply        infer = to conclude (reader/listener)
                     imply = to suggest (speaker/writer)
                     WRONG: He [inferred] that we should leave.
                     RIGHT: He [implied] that we should leave.
                     (He was the speaker → implies, not infers)

disinterested /      disinterested = impartial
uninterested         uninterested = not interested/bored
                     WRONG: A good judge must be [uninterested].
                     RIGHT: A good judge must be [disinterested].
```

## 19. MASTER TRICKS & SHORTCUTS

---

### 🚀 TRICK 1 — The Subject-Finder (Cross-Out Method)

```
Cross out everything between commas or after "along with / as well as / together with."
What remains BEFORE the verb = the real subject.

"The manager, [as well as his entire team], were present."
Cross out: [as well as his entire team]
Remaining: "The manager _____ present."
Subject = "manager" (singular) → "was" not "were" → ERROR in verb part.
```

---

### 🚀 TRICK 2 — The Pronoun Test (Remove One)

```
When you see "X and I/me" or "X and he/him":
Remove the other person and test alone.

"Between you and I" → "Between I" → WRONG → "Between me" ✓
"Give it to she and I" → "Give it to I" → WRONG → "Give it to her and me" ✓
"He and me went" → "Me went" → WRONG → "He and I went" ✓
```

---

### 🚀 TRICK 3 — The "Since" Alert

```
ANY sentence containing "SINCE" + a duration or point in time:
→ Must use Present Perfect or Present Perfect Continuous
→ NEVER Simple Past

"She lived here since 2010." → WRONG → "has lived" ✓
"I know him since childhood." → WRONG → "have known" ✓

Spot "since" → immediately check: is the verb in present perfect? If not → ERROR.
```

---

### 🚀 TRICK 4 — The "Will in If-Clause" Flag

```
ANY Type 1 conditional with "will" in the if-clause = WRONG

"If she will come, I will tell her." → "will come" is the ERROR.
Correct: "If she comes, I will tell her."

Spot "if" + "will" in the same clause → instant error identification.
```

---

### 🚀 TRICK 5 — The Collective Noun Check

```
ANY time you see: committee / jury / team / board / class / family / government:
→ The verb MUST be singular

"The team have won the match." → WRONG → "has won" ✓
"The jury were divided in opinion." → WRONG → "was divided" ✓
```

---

### 🚀 TRICK 6 — The "Unless...Not" Trap

```
AUTOMATICALLY wrong: "Unless...not" combination in any sentence.
"Unless you don't hurry..." → ERROR: remove "don't" → "Unless you hurry..."

Spot "unless" → check if "not" follows anywhere in the clause → if yes → ERROR.
```

---

### 🚀 TRICK 7 — The Article Sound Test

```
When checking A/AN errors:
Say the word after A/AN aloud (in your head).
Vowel sound → AN | Consonant sound → A

"She is a honest person." → "honest" = "on-est" sound → AN → ERROR.
"He is an university student." → "university" = "yoo" sound → A → ERROR.
```

---

### 🚀 TRICK 8 — The Preposition Pair Flash Cards

```
Memorise these pairs as triggers:
Married → TO (never "with") → any "married with" = ERROR
Suffer → FROM (never "with") → any "suffer with illness" = ERROR
Superior/Inferior → TO (never "than") → "superior than" = ERROR
Reason → FOR (never "of") → "reason of failure" = ERROR
Congratulate → ON (never "for") → "congratulate for" = ERROR
Different → FROM (not "than") → "different than" = ERROR
```

---

### 🚀 TRICK 9 — The Stative Verb Alarm

```
ANY time these verbs appear in -ING form → likely ERROR:
know / believe / want / need / understand / prefer / love / hate /
like / see / hear / smell / taste / seem / appear / own / contain / belong

"I am knowing the answer." → ERROR → "I know the answer."
"She is having a car." → ERROR → "She has a car."
(Exception: "is having fun/time" = experiencing → correct)
```

---

### 🚀 TRICK 10 — The "More...More" Double "The" Rule

```
WRONG: "More she studies, more she learns."
RIGHT: "The more she studies, the more she learns."
→ Proportional comparisons ALWAYS need double "The"

Spot "more X, more Y" pattern without "The" → instant ERROR identification.
```

---

### 🚀 TRICK 11 — The Redundancy Quick-Scan

```
Watch for these instant redundancy errors:
returned back / advance forward / end result / past history /
free gift / unexpected surprise / close proximity / future plans /
repeat again / revert back / final conclusion

Spot any of these in a sentence part → that part has the ERROR.
```

---

### 🚀 TRICK 12 — The Parallelism Balance Test

```
When you see a list or "not only...but also / both...and / either...or":
Check: Are all items in the SAME grammatical form?

"She is known for her honesty, her dedication, and [being punctual]."
Items: noun phrase, noun phrase, V-ing phrase → MISMATCH → ERROR
Correction: "her punctuality"
```

---

### 🚀 TRICK 13 — The "The Reason Is Because" Flag

```
"The reason is because..." = ALWAYS WRONG
Correct: "The reason is that..."

Any sentence with "the reason is/was because" → instant ERROR in the "because" part.
```

---

### 🚀 TRICK 14 — Lay vs Lie Quick Check

```
LIE (no object): I lie down. / He lay on the bed yesterday. (past = lay)
LAY (needs object): I lay the book on the table.

"He was [laying] on the bed." → ERROR → "lying" ✓
(lying = present participle of "lie" = to recline)
"She [lay] the pen on the desk." → CORRECT (past tense of "lay" = to place)
```

---

### 🚀 TRICK 15 — The "No Error" Discipline

```
Don't force an error when there isn't one.
If you've checked all parts and nothing is clearly wrong → choose "No Error."

Common trap: The sentence sounds slightly awkward but is grammatically correct.
RULE: Only mark an error if you can clearly state the RULE being violated.
If you can't name the rule → it may not be an error.
```

---

## 20. THE 60-SECOND SCANNING FRAMEWORK

> Use this systematic checklist on EVERY spotting errors question. Apply it in order.

---

```
READ THE FULL SENTENCE (5 seconds)
         ↓
APPLY THE CHECKLIST IN ORDER:

PRIORITY 1 — HIGHEST FREQUENCY (Check first, takes 10 seconds)
  ☐ Subject-Verb Agreement
      → Find real subject (cross out interrupting phrases)
      → Is the verb singular/plural correctly?
      → Check: collective nouns, either/or, neither/nor, each/every

PRIORITY 2 — VERY COMMON (Check second, takes 10 seconds)
  ☐ Tense
      → Is there a signal word? (since, yesterday, just, when, by the time)
      → Does the verb match the signal word?
      → Are stative verbs used in -ing form? (know/want/like in -ing = error)

PRIORITY 3 — COMMON (Check third, takes 10 seconds)
  ☐ Articles
      → A/AN: check the sound of the following word
      → THE: before superlatives, ordinals, rivers, unique things?
      → No article: before abstract nouns, languages, sports?

  ☐ Prepositions
      → Check fixed pairs: married TO, suffer FROM, superior TO, reason FOR
      → Check time prepositions: since/for, at/on/in

PRIORITY 4 — MEDIUM (Check fourth, takes 10 seconds)
  ☐ Pronouns
      → Subject/object case (between you and me vs between you and I)
      → WHO vs WHOM (subject vs object)
      → Pronoun-antecedent agreement (every student → his/her not their)

  ☐ Conjunctions
      → Correct pair: neither...nor / not only...but also / no sooner...than
      → Unless + not = error / lest + should (not might/would)

PRIORITY 5 — CHECK IF STILL UNSURE (takes 5 seconds)
  ☐ Comparison: same type? "any other" when comparing within group?
  ☐ Parallelism: same grammatical form in lists?
  ☐ Double negative / redundancy: unless...not / returned back?
  ☐ Conditional: will in if-clause? were (not was) in Type 2?
         ↓
ANSWER (5 seconds)
  → If error found → mark the part
  → If no error found → mark "No Error"
```

## 21. COMMON MISTAKE ALERTS

---

### ⚠️ MISTAKE 1 — Marking Error Based on "Sounds Wrong"

```
WRONG approach: "Part B sounds odd to me, so it must be wrong."
RIGHT approach: Identify the SPECIFIC RULE being violated.

If you can't name the rule → don't mark it as an error.
Many things sound unusual but are grammatically correct.
```

---

### ⚠️ MISTAKE 2 — Ignoring the Subject Finder Step

```
The most common error: choosing the verb part as the error
without first finding the REAL subject.

"The list of requirements were not met."
→ Students mark the verb part (were/not met)
→ Real subject = "list" (singular) → "was" not "were"
→ ERROR IS IN THE VERB: must be "was"

Always find the real subject BEFORE checking the verb.
```

---

### ⚠️ MISTAKE 3 — Confusing "Since" with "For"

```
SINCE = point in time (since 2010, since Monday)
FOR = duration (for three years, for two days)

"She has worked here since three years." → ERROR: should be "for"
"She has worked here for Monday." → ERROR: should be "since"

Match the WORD after with the correct preposition.
```

---

### ⚠️ MISTAKE 4 — Choosing "No Error" Too Quickly

```
WRONG: Checking only one part and saying "looks fine = No Error."
RIGHT: Check ALL parts systematically before choosing No Error.

Spotting Errors questions are specifically designed to have subtle errors.
If you don't find it immediately, apply all checklist items before selecting D.
```

---

### ⚠️ MISTAKE 5 — Missing Stative Verb Errors

```
"I am knowing the answer to the question."
→ Students often miss this because the sentence sounds "almost right."
→ KNOW is a stative verb → NEVER in -ing form
→ ERROR: "am knowing" → "know"

Always check: are any stative verbs (know/want/like/have/seem) in -ing form?
```

---

### ⚠️ MISTAKE 6 — Missing "Lest...Should" Rule

```
"She spoke quietly lest she might disturb the others."
→ After "lest" → must use "should" not "might/would/could"
→ ERROR: "might" → "should"

This rule is specifically tested in SSC and UPSC exams.
```

---

### ⚠️ MISTAKE 7 — Not Recognising "The Reason Is Because" Error

```
"The reason for his failure is because he was careless."
→ ERROR: "because" → "that"
→ "The reason is because" is ALWAYS wrong

This appears frequently. The error is always in the "because" part.
```

---

### ⚠️ MISTAKE 8 — Missing Redundancy Errors

```
"He returned back home after the trip."
→ "returned" already means going back → "back" is redundant → ERROR

Redundancy errors are subtle. Scan for:
returned back / advance forward / end result / past history /
unexpected surprise / free gift / close proximity / future plans
```

## 23. ANSWERS & EXPLANATIONS

---

### SET A — SUBJECT-VERB AGREEMENT

**A1. (C) are → is**
Real subject = "quality" (singular). "Of the mangoes" = interrupting prepositional phrase (cross it out). Singular subject → singular verb → "is not good."

**A2. (C) was → are**
"Neither...nor" → verb agrees with NEAREST subject. Nearest = "teachers" (plural) → "are" not "was."

**A3. (A) have → has**
"The committee" = collective noun → always singular verb → "has decided."

**A4. (B) was → were**
"A number of" + plural noun → plural verb. "A number of students" = several students (plural) → "were absent." (Compare: "The number of students was high.")

**A5. (C) were → was**
"News" = uncountable noun → always singular verb → "was deeply disturbing."

**A6. (B) were → was**
"Each of the participants" — "each" is always singular → "was awarded."

**A7. (C) have → has**
"The number of" + noun → singular verb → "has increased." (Compare: "A number of accidents HAVE occurred.")

**A8. (B) are → is**
"Five hundred rupees" refers to a single sum/amount → singular verb → "is a significant amount."

**A9. (C) is → are**
"Either...or" → verb agrees with NEAREST subject. Nearest = "senior managers" (plural) → "are responsible."

**A10. (C) were → was**
"Every boy and girl" — "every + noun + and + every/noun" → always singular verb → "was required."

---

### SET B — TENSES

**A11. (C) since → for**
"Since" is used for a POINT in time (since Monday, since 2010). "The past five years" = a DURATION → needs "for" not "since." → "for the past five years."

**A12. (C) collapsed → had collapsed**
Two past actions: fire brigade arrived (later) and building collapsed (earlier). The earlier action = Past Perfect → "had already collapsed."

**A13. (A) did → had / (B) when → than**
"No sooner" structure: No sooner + HAD + subject + V3 + THAN (not "when"). "No sooner did" → should be "No sooner had he entered...than." The error is in the structure — "when" should be "than." Mark (B) "when" as the error.

**A14. (C) revolved → revolves**
The teacher said that the earth revolves (not "revolved") around the sun. Universal/scientific truth → always Simple Present, even in reported speech.

**A15. (A) am knowing → know**
"Know" is a stative verb → NEVER used in continuous (-ing) form → "I know the answer."

**A16. (C) will reach → reaches**
After time conjunctions (as soon as, when, until, before, after) → use Present tense, NOT future. "As soon as she reaches" not "will reach."

**A17. (B) than → when**
"Hardly...when" is the correct pair. "Hardly...than" is wrong. → "Hardly had she sat down WHEN the telephone began to ring."

**A18. (B) finished → had finished**
He finished BEFORE the deadline (two past actions). Earlier action = Past Perfect → "he had finished the assignment before the deadline."

**A19. (C) started → had started**
"By the time we reached" = a past moment. The programme started BEFORE that → Past Perfect → "had already started."

**A20. (C) since → for**
"Two hours" = a DURATION → use "for" not "since." → "for two hours." ("since" is used for a point in time like "since two o'clock.")

---

### SET C — ARTICLES & PREPOSITIONS

**A21. (B) an university → a university**
"University" begins with the consonant sound "yoo" (not a vowel sound) → use "A" not "AN." → "a university student."

**A22. (B) with → from**
"Suffer FROM" is the fixed preposition. "Suffer with" is incorrect. → "suffering FROM a severe fever."

**A23. (A) Himalaya → Himalayas / The Himalayas**
Mountain RANGES take "The" and are usually plural. → "The Himalayas is the longest mountain range." (Error is the missing "The" and wrong form — "Himalaya" should be "The Himalayas.")

**A24. (B) than → from**
"Different FROM" is standard formal English. "Different than" is incorrect in formal usage. → "different FROM the one she wore."

**A25. (C) because → that**
"The reason is because" = always wrong (redundant). → "The reason for his dismissal was THAT he was always late."

**A26. (B) a → an**
"Honest" begins with a silent H → sounds like "on-est" (vowel sound) → use "AN" not "A." → "an honest woman."

**A27. (B) for → on**
"Congratulate ON" is the fixed preposition. → "congratulated her ON winning the national championship."

**A28. (D) No Error**
"Honesty is the best policy" — "honesty" (abstract noun in general sense) needs no article. "the best" correctly uses THE before superlative. The sentence is grammatically correct.

**A29. (C) in → to**
"Admitted TO the hospital" is correct (going to a place = TO). "In the hospital" is used when already there. But "admitted" requires TO. → "admitted TO the hospital immediately."

**A30. (B) with → to**
"Married TO" is the correct preposition. "Married with" is always wrong. → "married TO a successful businessman."

---

### SET D — PRONOUNS & MODIFIERS

**A31. (B) I → me**
"To" is a preposition → requires object pronoun. "To both my colleague and I" → "I" should be "me." → "to both my colleague and ME." (Test: "to I" sounds wrong → "to me" ✓)

**A32. (B) their → his or her**
"Every student" = singular → pronoun must be singular → "his or her" not "their." → "must submit HIS OR HER project."

**A33. (B) the bag was dropped → she dropped the bag**
Dangling modifier: "Running to catch the bus" — who is running? Not the bag. The subject of the main clause must be the runner. → "Running to catch the bus, SHE dropped the bag."

**A34. (A) which → who**
"Which" refers to things/animals. "Who" refers to people. "The man" = a person → "who." → "The man WHO I met."

**A35. (B) the manager was informed → the team members informed the manager**
Dangling modifier: "Having finished the report" — who finished it? The team members, but the main clause says "the manager was informed." The subject of the main clause must be the team members. → "Having finished the report, THE TEAM MEMBERS informed the manager."

**A36. (A) I → me**
"Between" = preposition → requires object pronoun. "Between you and I" → "I" should be "me." → "Between you and ME." (Test: "Between I" sounds wrong → "Between me" ✓)

**A37. (A) almost → (move to after "every")**
"Almost drove" means she didn't drive (she "almost" did = she didn't). The meaning should be "nearly every day." → "She drove her children to school ALMOST EVERY DAY." The position of "almost" is the error (misplaced modifier).

**A38. (B) myself → me**
"Myself" (reflexive) is used only when subject = object. Here, "please contact" is directed to another person → object pronoun "me" is needed. → "Please contact ME if you have queries."

---

### SET E — COMPARISONS & CONJUNCTIONS

**A39. (B) any city → any other city**
Mumbai IS a city in Maharashtra. Comparing it to "any city in Maharashtra" includes Mumbai itself → illogical. → "larger than ANY OTHER city in Maharashtra."

**A40. (C) India → that of India**
Comparing "population of China" with "India" = comparing unlike things. Must compare population with population. → "larger than THAT OF India."

**A41. (B) most tallest → tallest**
Double superlative: "most" + "-est" = error. Use only one form. → "the TALLEST player on the team."

**A42. (B) than → to**
"Senior" takes the preposition "TO" not "than." → "He is senior TO me by three years." (Same for junior/superior/inferior/prior → always TO.)

**A43. (B) or → nor**
"Neither...NOR" is the correct correlative pair. "Neither...or" is always wrong. → "Neither she NOR he was present."

**A44. (B) don't submit → submit**
"Unless" = "if not" → already contains the negative. Adding "don't" creates a double negative. → "Unless you SUBMIT the form." Remove "don't."

**A45. (B) more → the more**
Proportional comparisons need double "THE": "The more...the more." → "The more she studies, THE MORE she understands."

**A46. (B) might → should**
After "lest" → must use "should" (not might/would/could). → "lest she SHOULD miss the last train."

**A47. (B) but → (remove "but")**
"Although" and "but" cannot both be used in the same sentence (they both show contrast). Remove one. → "Although she worked hard, she did not get the promotion." (Remove "but")

**A48. (D) No Error**
"No sooner HAD he left...THAN it began to rain." — Structure is correct: Past Perfect + "than" + Simple Past. This is grammatically correct.

---

### SET F — NOUNS & WORD CHOICE

**A49. (B) advices → advice**
"Advice" is UNCOUNTABLE — it has NO plural form. → "some useful ADVICE." Never "advices."

**A50. (B) less → fewer**
"Opportunities" is a COUNTABLE noun → use "fewer" not "less." → "FEWER opportunities for fresh graduates."

**A51. (A) returned back → returned**
"Returned" already means going back. "Back" is redundant. → "He RETURNED to his hometown." (Remove "back.")

**A52. (C) affect → effect**
"Effect" (noun) = the result. "Affect" (verb) = to influence. Here a noun is needed after "disastrous" → "EFFECT on the crops."

**A53. (A) laying → lying**
"Lay" (to place something, needs object) vs "Lie" (to recline, no object). "He was lying on the sofa" = reclining (no object → "lie"). Past continuous: was + lying. "Laying" = placing something (wrong here). → "He was LYING on the sofa."

**A54. (A) amount → number**
"People" is COUNTABLE → use "number" not "amount." → "A large NUMBER of people attended."

**A55. (A) flaunted → flouted**
"Flaunt" = to show off proudly. "Flout" = to openly disobey rules. Traffic rules are disobeyed, not shown off. → "He FLOUTED the traffic rules."

**A56. (A) Beside → Besides**
"Beside" = next to (physical position). "Besides" = in addition to / apart from. The meaning requires "in addition to her excellent academic record." → "BESIDES her excellent academic record."

---

### SET G — MIXED ADVANCED

**A57. (A) would have studied → had studied**
Type 3 Conditional: If + Past Perfect (had + V3) → would have + V3. NEVER use "would have" in the if-clause. → "If he HAD STUDIED harder, he would have secured a better rank."

**A58. (B) where did I live → where I lived**
In reported/indirect questions: NO inversion (no auxiliary "did"). → "She asked me WHERE I LIVED." (Statement word order, past tense.)

**A59. (C) are → is**
"The manager" = subject. "Together with all his team members" = interrupting phrase (cross it out). Singular subject "manager" → singular verb → "IS attending."

**A60. (C) to paint → painting**
Parallelism error: "reading novels, swimming, and [painting]" — all must be in V-ing form. "to paint" breaks the parallel structure. → "reading novels, swimming, and PAINTING landscapes."

**A61. (B) nothing → anything**
Double negative: "don't" + "nothing" = two negatives = error. → "I don't know ANYTHING about the incident." (OR: "I know nothing about the incident.")

**A62. (A) was → were**
Type 2 Conditional (hypothetical): "If I WERE the Prime Minister" — use "were" for ALL persons (not "was"). → "If I WERE the Prime Minister, I would reform..."

**A63. (D) No Error**
"She is not only hardworking but also highly creative" — "not only...but also" is the correct pair. Both elements are adjectives (hardworking, creative) = parallel. The sentence is grammatically correct.

**A64. (B) will meet → would meet**
In reported speech, "will" changes to "would." → "He told me that he WOULD MEET me at the railway station."

**A65. (C) were → was**
"Furniture" = uncountable noun → always singular verb → "WAS damaged during the flooding."

**A66. (C) most brilliantest → most brilliant OR brightest**
Double superlative: "most" + "-est" is wrong. "Brilliant" forms superlative as "most brilliant" (not "brilliantest"). → "the MOST BRILLIANT scientists."

**A67. (C) till now yet → till now / yet**
Both "till now" and "yet" mean the same thing in this context = redundancy. Use one: either "He has not submitted the report YET" OR "He has not submitted the report TILL NOW." Using both = error.

**A68. (A/B) comprising of → comprising**
"Comprise" means "consist of" — it does NOT take "of." "Comprising of" is always wrong. → "The committee COMPRISING five members." (Not "comprising OF.")

## 🏆 TOP 15 EXAM TIPS

1. **Find the real subject first** — Cross out every interrupting phrase before checking verb agreement. The most common trap is a plural noun near the verb that isn't the subject.

2. **"Since" = point in time → Present Perfect is mandatory** — Any sentence with "since" + simple past = automatic error. This appears in almost every exam.

3. **Collective nouns are singular in exams** — Committee, jury, team, board, family, class, government → always take singular verbs in all competitive exams.

4. **"Will" in the if-clause of a conditional = instant error** — Type 1 conditional never has "will" in the if-clause. Spot it in 2 seconds.

5. **Either/Or and Neither/Nor → proximity rule** — Verb agrees with the subject NEAREST to it (immediately before the verb). Not the first subject.

6. **A/AN is about sound, not spelling** — "An university" is wrong because "university" starts with a consonant sound "yoo." Always say the word aloud mentally.

7. **"Unless + not" is always wrong** — "Unless" already contains the negative. Seeing "unless...don't/won't/not" = automatic error, always.

8. **Stative verbs in -ing form = error** — Know, believe, want, understand, prefer, love, hate, like, seem, appear, own, contain → Never in -ing form. Spot this quickly.

9. **"Although + but" = two contrasts in one = error** — Never use both in the same sentence. Remove one.

10. **Lest + should (never might/would/could)** — After "lest," only "should" is correct. This appears regularly in SSC and UPSC exams.

11. **Universal truths stay in Simple Present** — Even in reported speech, scientific/universal facts don't change tense. "The teacher said that the sun rises" = correct.

12. **Modifier error test: ask "Who is doing this action?"** — If the answer doesn't match the main clause subject, it's a dangling modifier. "Running to catch the bus, the bag fell" → bags don't run = error.

13. **"The reason is because" = always wrong** — "The reason IS THAT" is always the correct form. Seeing "reason is because" = instant error in the "because" part.

14. **Fewer vs Less: countable vs uncountable** — Students, books, opportunities = countable → FEWER. Water, time, information = uncountable → LESS. "Less opportunities" = always wrong.

15. **Name the rule before marking the error** — If you can't clearly state WHICH grammatical rule is being violated, don't mark it as an error. "It sounds odd" is not enough. This prevents false positives and helps you choose "No Error" with confidence.
