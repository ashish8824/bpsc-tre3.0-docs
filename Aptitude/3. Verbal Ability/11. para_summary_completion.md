## 1. WHAT IS PARA SUMMARY & PARA COMPLETION?

---

### Para Summary

**Para Summary** questions give you a paragraph (150–300 words) and ask you to choose which of four given options **best summarises** the main idea of the passage.

```
GIVEN: A full paragraph discussing a topic.
TASK:  Choose the ONE sentence that best captures the COMPLETE meaning
       of the paragraph — not too broad, not too narrow, not distorted.

Key principle: A summary is NOT a paraphrase of one sentence.
               It captures the ENTIRE passage in ONE precise statement.
```

---

### Para Completion

**Para Completion** questions give you a paragraph with one sentence **missing** (usually at the beginning, middle, or end). You must choose which of the given options **best fits** the missing position.

```
GIVEN: A paragraph with one blank sentence (marked as ___ or [BLANK]).
TASK:  Choose the ONE sentence from four options that makes the paragraph
       most logically coherent and complete.

Key principle: The correct sentence must CONNECT perfectly with
               what comes BEFORE and AFTER the blank.
```

---

### Why These Are Different Skills:

```
PARA SUMMARY  = Reading comprehension + abstraction
               → You identify the CORE IDEA from the whole
               → You reject details, examples, and peripheral ideas

PARA COMPLETION = Logical flow + structural awareness
               → You identify what's MISSING from the flow
               → You match tone, tense, direction, and logic
```

## 3. TYPES OF QUESTIONS ASKED

---

### Para Summary — Question Types:

**Type 1 — Standard Summary MCQ**
```
"Which of the following best summarises the above passage?"
→ 4 options, choose 1
```

**Type 2 — Best Captures the Main Idea**
```
"Which of the following best captures the main idea of the passage?"
→ Same as above but specifically about the central argument
```

**Type 3 — Identify the Correct Summary (Odd One Out)**
```
"Three of the following are incorrect summaries. Which is the correct one?"
→ Requires knowing common traps (too broad, too narrow, distorted)
```

---

### Para Completion — Question Types:

**Type 1 — Opening Sentence Missing**
```
"[BLANK]. The remaining three sentences then discuss this idea..."
→ Find the sentence that INTRODUCES the topic
```

**Type 2 — Middle Sentence Missing**
```
"The paragraph begins with X. [BLANK]. The paragraph then continues with Z."
→ Find the sentence that CONNECTS X to Z logically
```

**Type 3 — Closing Sentence Missing**
```
"The paragraph discusses X and Y. [BLANK]."
→ Find the sentence that CONCLUDES or SUMMARISES the paragraph
```

**Type 4 — Sentence Insertion (CAT/GMAT Style)**
```
"The following sentence needs to be inserted into the paragraph at the right position.
Where does it fit best?"
→ Find the position in the paragraph where the given sentence fits
```

## 4. PARA SUMMARY — CORE CONCEPTS

---

### 4.1 What Makes a Good Summary

A good summary has FOUR essential qualities:

```
QUALITY 1 — COMPLETENESS
The summary must capture what the ENTIRE passage is about.
Not just one paragraph, not just the opening line.

QUALITY 2 — ACCURACY
The summary must not add information not in the passage.
It must not distort or misrepresent what the passage says.

QUALITY 3 — APPROPRIATE SCOPE
Not too broad (so general it could describe anything)
Not too narrow (so specific it describes only one detail)

QUALITY 4 — TONE MATCH
The summary's tone should match the passage's tone.
A critical passage → summary should reflect criticism
An objective passage → summary should be neutral
```

---

### 4.2 Structure of a Paragraph

Understanding paragraph structure helps you identify where the main idea lives:

```
MOST COMMON STRUCTURES:

1. DEDUCTIVE STRUCTURE (Topic sentence first):
   Main Idea → Supporting Details → Examples → Conclusion
   WHERE IS MAIN IDEA? → First sentence (usually)
   EXAMPLE: "Technology is transforming education. [3 supporting examples follow]"

2. INDUCTIVE STRUCTURE (Conclusion at the end):
   Examples/Details → Analysis → Main Idea/Conclusion
   WHERE IS MAIN IDEA? → Last sentence (usually)
   EXAMPLE: "[3 problems described] → Therefore, this crisis requires immediate action."

3. QUESTION-ANSWER STRUCTURE:
   Question/Problem posed → Discussion → Answer/Resolution
   WHERE IS MAIN IDEA? → The answer/resolution (usually last or near last)

4. COMPARE-CONTRAST STRUCTURE:
   Position A → Position B → Comparison → Judgement
   WHERE IS MAIN IDEA? → The judgement/conclusion

5. PROBLEM-SOLUTION STRUCTURE:
   Problem described → Causes → Solution proposed
   WHERE IS MAIN IDEA? → Either the problem OR the solution (or both together)
```

---

### 4.3 How to Identify the Main Idea

**The Three-Pass Method:**

```
PASS 1 — Read the FIRST sentence and LAST sentence of the passage.
          These two sentences usually contain the main idea.
          The first introduces the topic; the last often states the conclusion.

PASS 2 — Ask: "What is this passage primarily ABOUT?"
          Imagine explaining it in ONE sentence to a friend.
          That sentence = the main idea.

PASS 3 — Test against all sentences in the passage.
          Does every sentence connect to your identified main idea?
          If yes → you have it. If no → revisit.
```

---

**The "What → So What" Method:**

```
WHAT: What is the passage describing or discussing?
SO WHAT: Why does it matter? What is the author's point?

"What" gives you the topic.
"So what" gives you the main idea.

EXAMPLE:
Passage about how social media algorithms create information bubbles.

WHAT: Social media algorithms and information bubbles.
SO WHAT: These algorithms narrow our perspectives and increase polarisation.

MAIN IDEA: Social media algorithms, by curating content based on
           existing preferences, narrow users' perspectives and
           contribute to social polarisation.
```

---

### 4.4 What to Include vs What to Exclude

```
INCLUDE IN A SUMMARY:
✓ The central claim or argument
✓ The main subject and what is said about it
✓ The key conclusion or implication

DO NOT INCLUDE:
✗ Specific statistics or numbers (unless the number IS the point)
✗ Names of specific people or places (unless central to the idea)
✗ Individual examples (the general category is what matters)
✗ Metaphors or figurative language (capture the literal meaning)
✗ Quotations or direct speech
✗ Background information or historical context (unless it IS the main point)
✗ Information from outside the passage (your general knowledge)
```

---

**Example Application:**

```
PASSAGE:
"The Amazon rainforest, often called 'the lungs of the Earth,' produces
approximately 20% of the world's oxygen. Home to over 10% of all known
species, it represents Earth's most biodiverse ecosystem. Yet, despite
its irreplaceable importance, nearly 17% of the Amazon has been destroyed
in the past 50 years due to agricultural expansion and illegal logging.
Scientists warn that once deforestation crosses a tipping point, the
forest's own water cycle will collapse, turning it from a carbon sink
into a carbon source. This threshold, some estimate, is only two decades away."

WHAT: Amazon deforestation
SO WHAT: It has crossed dangerous levels and is approaching an irreversible point.

CORRECT SUMMARY: The Amazon rainforest, despite its critical ecological
importance, is being destroyed at an alarming rate and may soon reach
an irreversible tipping point.

WRONG — Too narrow: "The Amazon produces 20% of the world's oxygen."
(Only one detail, not the main point.)

WRONG — Too broad: "Deforestation is a global environmental problem."
(Could describe any forest; doesn't capture the Amazon's specific tipping point.)

WRONG — Distorted: "The Amazon has already been destroyed and is now a
carbon source." (Passage says this MIGHT happen, not that it HAS happened.)

WRONG — Added info: "Amazon deforestation, like climate change, is
caused by human greed." ('Human greed' not stated; 'like climate change' added.)
```

---

### 4.5 The Umbrella Test

> The correct summary option must cover the ENTIRE passage like an umbrella — nothing outside it, nothing inside uncovered.

```
UMBRELLA TEST:
Take each option and ask:
1. Does this option cover EVERYTHING in the passage? (No details left outside)
2. Does this option contain ONLY what's in the passage? (Nothing added from outside)
3. Is every sentence in the passage somehow connected to this option?

If an option fails ANY of these → eliminate it.

VISUAL:
         [CORRECT SUMMARY]
        ←——— umbrella ———→
        ↓                 ↓
[detail1] [detail2] [detail3] [detail4]
All details sit UNDER the umbrella.
The umbrella doesn't extend BEYOND what the passage contains.
```

## 5. PARA COMPLETION — CORE CONCEPTS

---

### 5.1 Types of Blanks in Para Completion

---

#### Opening Sentence Blank:

```
Characteristics of a correct OPENING sentence:
✓ Introduces the topic for the first time
✓ Uses indefinite reference (a/an, general nouns)
✓ Does NOT refer back to something unstated ("this / that / such")
✓ Does NOT begin with a contrast word (however / but / although)
✓ Does NOT begin with a pronoun without an antecedent (he/she/it/they/this)
✓ Is broad enough to accommodate everything that follows

The opening sentence sets the SUBJECT and DIRECTION for the whole paragraph.
```

---

#### Middle Sentence Blank:

```
Characteristics of a correct MIDDLE sentence:
✓ Connects logically to what PRECEDES it (backward context)
✓ Connects logically to what FOLLOWS it (forward context)
✓ Maintains the same topic and direction
✓ Uses appropriate transition words (moreover / however / therefore / etc.)
✓ Pronouns refer correctly to nouns in previous sentences
✓ Tense matches the surrounding sentences
✓ Neither introduces a new tangential topic nor repeats what was already said
```

---

#### Closing Sentence Blank:

```
Characteristics of a correct CLOSING sentence:
✓ Concludes or wraps up the discussion
✓ Uses conclusion signals (therefore / thus / hence / in conclusion / ultimately)
✓ Does NOT introduce a new idea that needs further development
✓ Does NOT end with a question that leaves the paragraph open-ended
   (unless the paragraph's purpose is to pose a question)
✓ Is broad enough to summarise the whole paragraph
✓ May provide a recommendation, call to action, or final judgement
```

---

### 5.2 The Logical Flow Principle

> **A paragraph is a chain of connected ideas. Each sentence must LOGICALLY FOLLOW from the previous one AND LEAD TO the next.**

```
CHAIN OF LOGIC:

Sentence 1 → [logical link] → Sentence 2 → [logical link] → Sentence 3 → ...

The missing sentence must:
→ RECEIVE the logical output of the sentence before it
→ PASS a logical input to the sentence after it

If you insert the wrong sentence:
→ There will be a LOGICAL GAP between sentences (ideas don't connect)
→ There will be a PRONOUN without an antecedent (he/she/it refers to nothing)
→ There will be a TENSE JUMP without signal words
→ There will be a TOPIC SHIFT that is unexplained
```

---

### 5.3 Paragraph Structures to Recognise

---

#### Structure 1 — Problem → Analysis → Solution

```
[Problem introduced] → [Analysis of causes] → [Solution proposed]

If PROBLEM is missing: The solution + analysis mention problems → opening sentence = problem
If ANALYSIS is missing: The gap is between problem and solution → middle = analysis
If SOLUTION is missing: Problem + analysis build up to an answer → closing = solution
```

---

#### Structure 2 — Claim → Evidence → Conclusion

```
[Claim made] → [Evidence/Examples given] → [Conclusion drawn]

If CLAIM is missing: Evidence + conclusion present → opening = the general claim
If EVIDENCE is missing: Claim + conclusion present → middle = supporting evidence
If CONCLUSION is missing: Claim + evidence present → closing = the logical conclusion
```

---

#### Structure 3 — General → Specific → Back to General

```
[General statement] → [Specific example/data] → [Return to general significance]

If GENERAL is missing: Specific detail + return to general → opening = broad introduction
If SPECIFIC is missing: General + return to general → middle = specific example/data
If RETURN is missing: General + specific → closing = returning to broader implications
```

---

#### Structure 4 — Background → Contrast → Current Situation

```
[Historical background] → [Change/Contrast] → [Present situation/implications]

Signal words: "however," "but," "in contrast," "recently," "today"
If BACKGROUND is missing: Change + present → opening = the historical context
If CHANGE is missing: Background + present → middle = the transition/contrast sentence
If PRESENT is missing: Background + change → closing = current situation or implication
```

---

### 5.4 Forward and Backward Context

> **The most critical skill in Para Completion is reading BOTH directions from the blank.**

---

```
BACKWARD CONTEXT (sentences BEFORE the blank):
→ What topic has been introduced?
→ What pronouns are being used?
→ What is the last idea that needs to continue?
→ What direction (positive/negative) has been established?

FORWARD CONTEXT (sentences AFTER the blank):
→ What words or phrases pick up from the blank? (pronouns? repeated nouns?)
→ What idea is being developed next?
→ Is there a transition word that signals what came before?
→ Does the forward sentence assume something that only the blank can provide?

THE BRIDGE FUNCTION:
The missing sentence must ACT AS A BRIDGE between backward and forward context.
It receives the output of backward context and supplies the input for forward context.
```

---

**Example of Forward Context Revealing the Answer:**

```
PARAGRAPH:
"The Industrial Revolution transformed manufacturing but also created
unprecedented social problems. [BLANK]. Women and children worked in
appalling conditions for minimal wages, while factory owners accumulated
vast wealth."

BACKWARD CONTEXT: Industrial Revolution → social problems
FORWARD CONTEXT: "Women and children worked in appalling conditions"
               → The forward sentence describes specific workers' exploitation.
               → This means the blank must INTRODUCE the concept of worker exploitation
                 as a general statement.

CORRECT BLANK: "The most visible of these problems was the widespread exploitation of workers."
→ "These problems" refers back to "social problems" (backward)
→ "Exploitation of workers" is then illustrated by women and children (forward)
```

---

### 5.5 Transition Words as Guides

```
If the FIRST WORD of the sentence AFTER the blank is:

"However / But / Yet / Nevertheless"
→ The sentence BEFORE the blank (the blank itself or the sentence before)
  must present the CONTRASTING idea.
→ The blank likely presents the POSITIVE side that "however" then contrasts.

"Therefore / Thus / Hence / Consequently"
→ The blank must be the CAUSE or PREMISE from which the consequence follows.
→ The blank presents the reason; what follows is the result.

"Moreover / Furthermore / Additionally"
→ The blank adds ANOTHER POINT in the same direction.
→ Both the blank and what follows must be supporting the same argument.

"For example / For instance / Such as"
→ The blank makes a GENERAL CLAIM that the example illustrates.
→ The blank = the general statement that needs exemplification.

"This / These / Such"
→ The blank contains the noun that "this/these/such" refers to.
→ Whatever "this" points to in the forward sentence MUST be present in the blank.
```

## 6. STEP-BY-STEP STRATEGY

---

### 6.1 Para Summary Strategy

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 1 — READ THE FULL PASSAGE (60 seconds)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Read the entire passage without looking at options.
Focus on: What is this about? What is the author's main point?
Identify: Topic + Main Claim + Tone

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 2 — FORM YOUR OWN SUMMARY (30 seconds)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Before looking at options, mentally form your own one-sentence summary.
Use the WHAT + SO WHAT method.
This prevents the options from misleading you.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 3 — APPLY THE ELIMINATION FILTERS (60 seconds)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
For each option, ask:

FILTER 1 — Too Narrow?
"Does this option focus on only ONE detail or example?"
If YES → eliminate (it's a supporting detail, not the main idea)

FILTER 2 — Too Broad?
"Could this option describe almost ANY passage on this topic?"
If YES → eliminate (it lacks the specific point the passage makes)

FILTER 3 — Distorted?
"Does this option change what the passage actually says?"
"Does it say something is certain when the passage says 'might'?"
"Does it assign blame or praise not in the passage?"
If YES → eliminate (it's a misrepresentation)

FILTER 4 — Added Information?
"Does this option include facts, ideas, or claims not in the passage?"
If YES → eliminate (summaries cannot introduce new information)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 4 — APPLY THE UMBRELLA TEST (30 seconds)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
For the surviving option(s), test:
"Can ALL the passage's key ideas fit under this summary?"
"Does this summary cover NOTHING outside the passage?"
The option that passes both = your answer.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 5 — VERIFY (15 seconds)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Re-read your chosen summary option alongside the passage.
Does it capture the passage's essence without adding or distorting?
If yes → confirm your answer.
```

### 6.2 Para Completion Strategy

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 1 — IDENTIFY THE BLANK POSITION (10 seconds)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Is the blank at the BEGINNING, MIDDLE, or END?
This immediately tells you the function of the missing sentence.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 2 — READ BACKWARD AND FORWARD CONTEXT (45 seconds)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
BACKWARD: Read all sentences BEFORE the blank.
Note: Last topic introduced | Last pronoun used | Direction of argument

FORWARD: Read all sentences AFTER the blank.
Note: First word of next sentence | Any pronouns that need antecedents |
      Any transition words that reveal what came before

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 3 — FORM A PREDICTION (20 seconds)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
What TYPE of sentence is needed?
(Introducing? Bridging? Concluding? Contrasting? Adding evidence?)
What CONTENT should it have?
(What idea connects backward to forward?)
```

## 7. MASTER TRICKS & SHORTCUTS

---

### 🚀 TRICK 1 — The WHAT + SO WHAT Method (Para Summary)

```
WHAT = The topic of the passage (factual content)
SO WHAT = The author's point/claim about that topic (significance)

The correct summary = WHAT + SO WHAT combined.

PASSAGE: [Discusses how social media algorithms work and then argues
         that they create polarisation in society.]

WHAT: How social media algorithms work.
SO WHAT: They create social polarisation.

CORRECT SUMMARY: Social media algorithms, designed to maximise engagement
by showing content aligned with users' existing views, are a key driver
of social polarisation.

Any option that only describes WHAT (algorithms) = too narrow.
Any option that only says SO WHAT (polarisation) = too narrow.
The option that combines both = correct.
```

---

### 🚀 TRICK 2 — The First and Last Sentence Anchor (Para Summary)

```
The FIRST and LAST sentences of a passage usually together contain the main idea.

If the passage is DEDUCTIVE:
→ First sentence = main claim | Last sentence = restatement/conclusion
→ A good summary combines these two sentences' ideas.

If the passage is INDUCTIVE:
→ First sentence = context/example | Last sentence = the actual conclusion
→ A good summary emphasises the LAST sentence's idea.

APPLICATION:
Before reading all options, read FIRST sentence + LAST sentence.
Form a mental summary from these.
Now look at options and find the one closest to this.
```

---

### 🚀 TRICK 3 — The "Can't Be Too Specific" Rule (Para Summary)

```
If a summary option mentions:
→ A specific statistic (17% / $3.4 billion / 40 countries)
→ A specific name (Einstein / Harvard / 1985)
→ A specific example used in the passage (but not the main point)

→ ELIMINATE IT. It's too specific for a summary.

Exception: If the number IS the main point (e.g., "90% of a species
is extinct" when the passage is about species extinction).

QUICK TEST: If you could change the number/name without changing
the essential truth of the summary → the number shouldn't be in the summary.
```

---

### 🚀 TRICK 4 — The Extreme Language Trap (Para Summary)

```
Be wary of summary options that use EXTREME or ABSOLUTE language
when the passage uses MODERATE or QUALIFIED language.

PASSAGE: "Researchers suggest that social media may contribute to
increased levels of depression in some users."

WRONG SUMMARY: "Social media causes depression in users."
(Passage: "may contribute" → "suggest" | Wrong summary: "causes" definitive)

WRONG SUMMARY: "Social media is the primary cause of the depression epidemic."
("Primary cause" + "epidemic" both added and exaggerated)

The correct summary matches the CERTAINTY LEVEL of the passage.
Suggestions/possibilities → should → summary should also use "may/might/suggest"
Definite claims → is/are → summary can use definitive language
```

---

### 🚀 TRICK 5 — The Pronoun-Antecedent Check (Para Completion)

```
When a sentence begins with a PRONOUN (he/she/it/they/this/these/such):
→ The pronoun's antecedent MUST appear in the sentence immediately before it.

This works in both directions:
1. The sentence BEFORE the blank ends with a noun → the blank begins with that pronoun.
2. The sentence AFTER the blank begins with a pronoun → the blank ends with that noun.

EXAMPLE (Para Completion):
"[BLANK]. It has since been adopted by governments in over 40 countries."

"It" = singular pronoun → must refer to a singular noun/policy/concept in the blank.
The blank must INTRODUCE what "it" refers to.
Options: (A) "A new environmental framework was proposed in 2015." ✓
         ("it" = "the framework")
         (B) "Several countries began discussing climate action." ✗
         ("it" can't refer to "several countries" = plural)
```

---

### 🚀 TRICK 6 — The Transition Word Forward Clue (Para Completion)

```
The FIRST WORD of the sentence immediately AFTER the blank is the strongest clue.

"However" at start of next sentence:
→ The blank presents the OPPOSITE idea.
→ The next sentence will contradict/qualify what the blank says.

"Therefore" at start of next sentence:
→ The blank is the PREMISE from which the conclusion follows.
→ The blank contains a fact/argument that leads logically to the result.

"For example" at start of next sentence:
→ The blank makes a GENERAL CLAIM.
→ The example in the next sentence illustrates the blank.

"Moreover" at start of next sentence:
→ The blank makes ONE POINT in a direction.
→ The next sentence adds ANOTHER POINT in the same direction.

"This" or "These" at start of next sentence:
→ The blank introduces what "this/these" refers to.
→ Find options that end with the noun "this" points back to.
```

---

### 🚀 TRICK 7 — The Opening Sentence Cannot Start With... (Para Completion)

```
For OPENING SENTENCE blanks, instantly eliminate any option that:
✗ Begins with: "However / But / Yet / Nevertheless / Although"
  (These contrast words imply something was already said)

✗ Begins with: "He / She / It / They / This / These / Such / The"
  (These reference something already introduced — can't be opening)
  Exception: "The" with a universally known noun (The sun / The world)

✗ Begins with: "Therefore / Thus / Hence / Consequently / As a result"
  (These signal conclusions — nothing has been argued yet)

✗ Begins with: "Moreover / Furthermore / Additionally / Also"
  (These add to something already said — nothing said yet)

ONLY THESE CAN OPEN A PARAGRAPH:
✓ Begins with: A/An + new noun (introduces topic for first time)
✓ Begins with: A proper noun as topic (Newton's theory... / The Amazon...)
✓ Begins with: A general truth statement ("Education is...", "Technology has...")
✓ Begins with: A year/time marker that sets context ("In the 18th century...")
```

---

### 🚀 TRICK 8 — The Closing Sentence Must NOT... (Para Completion)

```
For CLOSING SENTENCE blanks, instantly eliminate any option that:
✗ Introduces a brand-new idea that requires further development
✗ Ends with a question that opens a new inquiry
✗ Uses "Firstly / To begin with / Initially"
  (These signal beginnings, not endings)
✗ Says something that contradicts the paragraph's conclusion

THE CLOSING SENTENCE SHOULD:
✓ Resolve/wrap up what was discussed
✓ Use conclusion signals (therefore / hence / thus / ultimately / in conclusion)
✓ Make a broader point that connects back to the opening
✓ Present a recommendation, warning, or final judgement
✓ Provide the "SO WHAT" of everything that came before
```

---

### 🚀 TRICK 9 — The Tense Consistency Check (Para Completion)

```
The missing sentence MUST match the dominant tense of the paragraph.

If the paragraph is in Simple Present:
→ The blank should also be in Simple Present (unless a specific time signal overrides)

If the paragraph is in Simple Past:
→ The blank should also be in Simple Past

COMMON TRAP: The correct-sounding option is in the wrong tense.
This eliminates options that are thematically perfect but grammatically inconsistent.

QUICK CHECK: Scan the verbs in 2 sentences before and after the blank.
All past tense? → Blank must also be past tense.
Mixed? → Look for the tense that matches the part the blank connects to most closely.
```

---

### 🚀 TRICK 10 — The Scope Ladder (Para Summary)

```
Imagine a SCOPE LADDER with these levels:

TOO SPECIFIC (details)
— specific number or statistic
— specific name or example
— one event or occurrence

JUST RIGHT (main idea)
— captures the central argument
— covers the whole passage
— not too broad, not too narrow

TOO BROAD (general)
— could describe any passage on this topic
— loses the specific point the passage makes
— sounds like a textbook definition

Options at the TOP of the ladder = eliminate (too broad)
Options at the BOTTOM = eliminate (too narrow)
The option at the JUST RIGHT level = correct
```

---

### 🚀 TRICK 11 — The "Outside Information" Alarm (Para Summary)

```
During reading, note anything in summary options that is NOT in the passage.

Even if the outside information is TRUE in the real world:
→ If it's not in the passage → the summary that includes it is WRONG.

PASSAGE: "The Renaissance brought a renewed interest in human potential
and shifted art from religious themes to secular ones."

WRONG SUMMARY: "The Renaissance, which began in Italy in the 14th century,
shifted artistic focus from religious to secular themes."
(14th century + Italy — not mentioned in the passage)

RIGHT SUMMARY: "The Renaissance represented a shift in art away from
religious themes, reflecting a new emphasis on human potential."
(Everything in this summary is in the passage)
```

---

### 🚀 TRICK 12 — The Logic Gap Detector (Para Completion)

```
After inserting an option, read [before] + [option] + [after] quickly.
Ask: "Is there a LOGICAL GAP anywhere?"

TYPES OF LOGICAL GAPS:
1. TOPIC JUMP: The before discusses A, the option discusses B, the after goes back to A.
   → The option creates a tangent. Eliminate.

2. PRONOUN WITHOUT ANTECEDENT: The option says "He did X" but no male has been introduced.
   → The option is misplaced. Eliminate.

3. RESULT WITHOUT CAUSE: The after-sentence says "As a result, prices fell."
   But the option doesn't describe anything that would cause prices to fall.
   → The option is wrong. The option must contain the cause.

4. CLAIM WITHOUT SUPPORT: The option makes a strong claim but the surrounding
   sentences don't support or elaborate on it.
   → Wrong position or wrong option.

5. CONTRADICTION: The option says something that directly contradicts what the
   passage says before or after.
   → Wrong option.
```

---

### 🚀 TRICK 13 — The Signposting Word Match (Para Completion)

```
LOOK FOR SIGNPOSTING WORDS in the sentences around the blank:

"On the one hand..." → The blank likely contains "On the other hand..."
"First..." → The blank might contain "Second..." or "However..."
"The problem is..." → The blank might contain "The solution is..."
"Traditionally..." → The blank might contain "However, today..." or "But recently..."
"Critics argue..." → The blank might contain "Proponents counter that..."

These signposting pairs PREDICT what the missing sentence contains.
```

---

### 🚀 TRICK 14 — The Author's Argument Test (Para Summary)

```
Ask: What is the AUTHOR ARGUING?

A passage rarely just presents neutral facts — it has a position or implication.

Is the author:
→ FOR something? (advocating, recommending)
→ AGAINST something? (warning, criticising)
→ EXPLAINING a phenomenon? (neutral, analytical)
→ COMPARING two things? (balanced, weighing)

The summary must reflect the AUTHOR'S ARGUMENT, not just the topic.

WRONG: Passage argues technology is harmful → Summary says "Technology is neutral."
RIGHT: Passage argues technology is harmful → Summary says "Technology poses significant risks."
```

---

### 🚀 TRICK 15 — The 2-Second Elimination for Para Summary Options

```
QUICK SCAN ORDER for each option:

1. Does it start with a specific name/number? → Probably too narrow → suspect
2. Does it contain ANY information not in the passage? → Eliminate
3. Does it miss the author's main point (only covers context)? → Eliminate
4. Does it use extreme language (always/never/all/only) when passage is moderate? → Eliminate
5. Is it vague enough to describe ANY passage on this topic? → Too broad → Eliminate

Usually 3 of 4 options fail at least one of these tests.
The surviving option = answer.
Apply 2-second scan → skip deep analysis for obviously wrong options.
```

## 8. COMMON TRAP OPTIONS — HOW TO ELIMINATE

---

### Trap Type 1 — Too Narrow (Most Common in Para Summary)

```
DESCRIPTION: Focuses on ONE detail, example, or supporting point rather than the WHOLE.

IDENTIFICATION:
→ Mentions a specific statistic, name, date, or example from the passage
→ Describes only what one paragraph or sentence says
→ Could NOT be the main idea because the passage continues after this point

EXAMPLE TRAP:
Passage: [About global warming's effects on biodiversity, including ocean
acidification, habitat loss, and species extinction.]
Trap option: "Ocean acidification is destroying coral reef ecosystems."
(True and in the passage, but only ONE of several effects — not the whole.)
```

---

### Trap Type 2 — Too Broad (Common in Para Summary)

```
DESCRIPTION: So general that it doesn't capture the passage's SPECIFIC POINT.

IDENTIFICATION:
→ Could describe hundreds of different passages
→ Lacks the specific argument or claim that the passage makes
→ Reads like a textbook definition or general statement

EXAMPLE TRAP:
Passage: [About how income inequality specifically in urban areas is
driving crime rates higher.]
Trap option: "Income inequality is a major social problem."
(True, but doesn't capture 'urban areas' + 'driving crime rates' — too broad.)
```

---

### Trap Type 3 — Distorted (Trickiest Trap)

```
DESCRIPTION: Contains most of the right elements but CHANGES something — a conclusion,
a relationship, a certainty level, or who is responsible.

IDENTIFICATION:
→ The passage says "may" → the option says "does"
→ The passage says "A causes B" → the option says "B causes A"
→ The passage describes two causes → the option says only one cause
→ The passage expresses a concern → the option calls it a catastrophe
→ The passage credits X → the option credits Y

EXAMPLE TRAP:
Passage: "Studies suggest a correlation between social media use and
depression, though a causal link has not been established."
Trap option: "Social media use causes depression according to research."
(Passage: correlation only, not causation | Trap: says it causes — DISTORTED)
```

---

### Trap Type 4 — Added Information (Para Summary)

```
DESCRIPTION: Includes correct and relevant ideas BUT also adds information
NOT present in the passage.

IDENTIFICATION:
→ Mentions something you know is true but the passage didn't say
→ Adds context, causes, or effects not discussed in the passage
→ Makes connections between ideas that the passage didn't make

EXAMPLE TRAP:
Passage: [About how automation is replacing jobs in manufacturing.]
Trap option: "Automation, driven by AI advances, is displacing manufacturing
workers and creating social inequality."
('Social inequality' and 'AI advances' = not mentioned in the passage — ADDED INFO)
```

---

### Trap Type 5 — Half-True (Para Completion)

```
DESCRIPTION: An option that is thematically correct but wrong for the POSITION.
It could fit somewhere in the passage, just not where the blank is.

IDENTIFICATION:
→ Insert the option; something before OR after doesn't connect
→ The option has a pronoun with no antecedent in the immediate context
→ The option's transition word signals the wrong relationship

EXAMPLE TRAP (Opening blank):
Trap option: "However, these benefits come with significant risks."
("However" can't open a paragraph — what is it contrasting with?)
Even though this sentence fits the topic, it CAN'T be the opening.
```

## 10. COMMON MISTAKE ALERTS

---

### ⚠️ MISTAKE 1 — Choosing a Summary That Is the First Sentence

```
WRONG: Just restating the first sentence of the passage as the summary.
RIGHT: The first sentence introduces the topic but is not always the main idea.
       The main idea often emerges from combining the first and last sentences.

Ask: Does the summary capture EVERYTHING in the passage,
or only what the opening line says?
```

---

### ⚠️ MISTAKE 2 — Choosing an Option Because It Sounds Familiar

```
WRONG: Picking an option because it uses language from the passage and "sounds like" the passage.
RIGHT: Verify that the option accurately represents the COMPLETE and CENTRAL idea.

Trap options deliberately use vocabulary from the passage to seem plausible.
The test is not "does this sound like the passage?" but
"does this capture what the passage is ABOUT?"
```

---

### ⚠️ MISTAKE 3 — Ignoring the Certainty Level

```
WRONG: Choosing a summary that says "X causes Y" when the passage says "X may contribute to Y."
RIGHT: The summary must match the DEGREE of certainty the passage expresses.

Signal words: "suggests / may / could / might / appears to / tends to" = uncertain
Signal words: "proves / demonstrates / shows / is / does" = certain
These must match between passage and summary.
```

---

### ⚠️ MISTAKE 4 — Not Using Forward Context in Para Completion

```
WRONG: Reading only the sentences BEFORE the blank and choosing accordingly.
RIGHT: Reading BOTH the sentences before AND the sentences after the blank.

The sentences after the blank often contain the most revealing clues
(pronouns that need antecedents, transition words that signal what came before).
```

---

### ⚠️ MISTAKE 5 — Inserting an Option Without Reading Before + After

```
WRONG: Evaluating options in isolation without reading them in context.
RIGHT: Always read [sentence before blank] + [chosen option] + [sentence after blank]
       as a continuous paragraph segment.

An option that reads well in isolation may create a logical gap in context.
```

---

### ⚠️ MISTAKE 6 — Choosing by Topic Familiarity Instead of Logic

```
WRONG: Choosing a para completion option because you know it to be true about the topic.
RIGHT: Choose based on LOGICAL FLOW within the paragraph, not on outside knowledge.

An option may be factually correct in the real world but logically wrong for this paragraph.
The paragraph determines what fits, not your knowledge of the subject.
```

---

### ⚠️ MISTAKE 7 — Missing the Author's Stance

```
WRONG: Summarising a passage as neutral when the author clearly has a position.
RIGHT: Identify whether the author is FOR, AGAINST, CONCERNED ABOUT, or EXPLAINING something.

The summary must reflect the author's stance.
"Economic growth leads to environmental damage" (neutral)
vs "The author argues that economic growth inevitably leads to environmental damage" (author's claim)
In a summary question, the latter (capturing the author's position) is more accurate.
```

---

### ⚠️ MISTAKE 8 — Confusing Para Completion with Para Jumbles

```
WRONG: Treating Para Completion as rearranging all sentences.
RIGHT: In Para Completion, most sentences are FIXED. Only ONE is missing.
       You're inserting one sentence into a paragraph that is otherwise complete.

The paragraph already has its logical flow established.
Your job is to find the ONE sentence that makes the flow perfect.
```

**Why B is wrong:** Generational transmission — a consequence of the poverty-education link, not a bridge to the specific pathways described in the forward sentence.

## 🏆 TOP 15 EXAM TIPS

1. **Form your own summary BEFORE reading options** — This is the #1 habit of top scorers. Read the passage, form a mental summary using WHAT + SO WHAT, then read the options. This prevents the options from misleading you with plausible-sounding but incorrect choices.

2. **The correct summary covers EVERYTHING, adds NOTHING** — Any option that covers only part of the passage (too narrow) or adds information not in the passage (added info) is automatically wrong. These two filters eliminate most wrong options.

3. **First sentence + last sentence = the main idea** — For most passages, reading only the first and last sentences gives you 80% of the information you need to identify the correct summary. Practise this as a speed technique.

4. **The "SO WHAT" is usually in the last paragraph or last few sentences** — Authors typically state the significance of their argument at the end. If the summary doesn't capture the conclusion/implication, it's incomplete.

5. **Extreme language is always a trap** — If the passage uses "may suggest," the correct summary will not say "proves." Options that use "all," "always," "never," "completely," or "only" when the passage is moderate are almost always wrong.

6. **Distortion traps change relationships, not just words** — A summary can use the passage's exact vocabulary but distort the relationship (A causes B → B causes A) or the certainty level (might → does). Read each option critically, not just for vocabulary match.

7. **Para Completion: blank position determines function** — Opening = introduce topic | Middle = bridge or pivot | Closing = conclude or summarise. Identify position FIRST because it immediately tells you what type of sentence you're looking for.

8. **The first word of the sentence AFTER the blank is your strongest clue** — "However" = blank says the opposite. "Therefore" = blank is the cause. "For example" = blank makes a general claim. "This" = blank contains the noun "this" refers to. These signal what must precede them.

9. **No opening sentence can begin with a contrast, conclusion, or addition signal** — "However / Therefore / Moreover" at the start of an option ELIMINATES it from being the opening sentence. Apply this instantly to save time.

10. **Read [before] + [option] + [after] as a unit** — Never evaluate a completion option in isolation. Always read the full three-sentence sequence. A logical gap anywhere = wrong option.

11. **Pronouns are your connective tissue** — If the sentence after the blank starts with "It / They / This / These," your blank must end with the noun these pronouns refer to. If the blank starts with "He / She," there must be a person introduced before the blank.

12. **The closing sentence must not introduce new ideas** — Any option that starts a new topic or asks a question that opens a new inquiry cannot close the paragraph. The closing sentence must RECEIVE from what came before, not LAUNCH something new.

13. **Author's stance matters for summary** — A critical passage needs a critical summary. A cautious passage needs a qualified summary. Matching the author's tone and certainty level is as important as matching the content.

14. **The WHAT is never enough — you need the SO WHAT** — "The passage is about X" only gives you the topic. The correct summary includes WHY X matters / what the author argues about X / what conclusion about X the passage reaches.

15. **For Para Completion, wrong options are "thematically correct but positionally wrong"** — Options are usually about the right topic but don't fit the specific position. The test is not "is this true about the topic?" but "does this bridge backward to forward perfectly?" Always test the bridge function.

Both ultimately test whether you understand how ideas CONNECT and how arguments are STRUCTURED.
