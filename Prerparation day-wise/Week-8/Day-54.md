## 🔷 BLOCK 1: E-Commerce — All Models and Facts

### What is E-Commerce?
```
E-COMMERCE = Electronic Commerce
           = Conducting business transactions ELECTRONICALLY over the INTERNET
           = Buying and selling goods/services online

Key characteristics:
  → Available 24×7 (no physical store hours)
  → Global reach (not limited to local area)
  → Lower operational costs (no physical store needed)
  → No physical inspection of goods (limitation)
  → Security concerns (limitation)
  → Digital payments: credit/debit cards, UPI, net banking
```

### The Four Major E-Commerce Models:
```
B2B — Business to Business:
  = Transactions BETWEEN businesses
  Examples: Alibaba, IndiaMART, TradeIndia
  Products: Raw materials, wholesale goods, industrial equipment
  ← "Business buys from Business"

B2C — Business to Consumer: ← MOST TESTED MODEL
  = Business sells DIRECTLY to end consumers
  Examples: Amazon, Flipkart, Snapdeal ← TRE 3.0 PYQ listed these exactly
            Myntra, Nykaa, BigBasket
  ← "Business sells to You (Consumer)"
  ⚠️ PYQ TRAP: "Which of these is a B2C e-commerce platform?"
               Answer includes: Amazon, Flipkart, Snapdeal

C2C — Consumer to Consumer:
  = Transactions BETWEEN consumers (peer-to-peer)
  Examples: OLX, eBay, Quikr, Facebook Marketplace
  Platform: Acts as intermediary; does not own goods
  ← "You sell to Another Consumer"

B2G — Business to Government:
  = Business sells to GOVERNMENT bodies
  Examples: GeM (Government e-Marketplace), CPPP (Central Public Procurement Portal)
  Context: Government procurement of goods/services from companies

G2C — Government to Consumer (also called G2B):
  = Government provides services to citizens/businesses online
  Examples: e-filing of taxes, passport portals, DigiLocker
```

### E-Commerce Advantages vs Disadvantages:
```
ADVANTAGES:
  ✅ 24×7 availability — no closing hours
  ✅ Global market reach — sell to anyone, anywhere
  ✅ Lower costs — no physical store rent/staff
  ✅ Easier price comparison for consumers
  ✅ Personalised recommendations (AI-based)
  ✅ Faster transactions

DISADVANTAGES:
  ❌ No physical inspection before buying
  ❌ Security/fraud risks (data theft, payment fraud)
  ❌ Delivery delays (unlike instant in-store purchase)
  ❌ No personal relationship with seller
  ❌ Returns process can be complex
  ❌ Digital divide — not accessible to all
```

### Payment Gateway in E-Commerce:
```
PAYMENT GATEWAY = Software that processes payment transactions
                = Authorises and facilitates online payments
Examples: Razorpay, PayU, CCAvenue, Stripe
Process: Customer → Payment gateway → Bank → Confirmation → Merchant
```

### 🚨 PYQ TRAP #1:
> B2C examples = **Amazon, Flipkart, Snapdeal** — these three appeared verbatim in TRE 3.0.
> B2B = **Alibaba, IndiaMART** (businesses buying wholesale from businesses).
> C2C = **OLX, eBay** (used goods between individuals).
> **GeM = B2G** (businesses selling to Government of India).

## 🔷 BLOCK 2: Multimedia — Complete Facts

### What is Multimedia?
```
MULTIMEDIA = Combination of TWO OR MORE media types in a single platform:

The 5 elements of multimedia:
  1. TEXT      → written words, fonts, formatting
  2. AUDIO     → sound, music, voice
  3. VIDEO     → moving images, films
  4. ANIMATION → computer-generated moving graphics
  5. GRAPHICS  → static images, illustrations, photographs

⚠️ PYQ TRAP: "Multimedia = combination of text, audio, video, animation and graphics"
              All five must be mentioned — any subset is incomplete
              Key: multimedia is INTEGRATION of these, not just storing them separately
```

### Multimedia Authoring Tools:
```
AUTHORING TOOL = Software used to CREATE multimedia content
               = Allows integration of text, audio, video, animation into one project

EXAMPLES OF AUTHORING TOOLS (appeared in TRE 3.0):
  → Adobe Director (formerly Macromedia Director)
  → Macromedia Flash / Adobe Animate
  → Authorware (Macromedia/Adobe)
  → ToolBook (Asymetrix/SumTotal)
  → HyperCard (older Apple tool)

⚠️ PYQ TRAP: PowerPoint = PRESENTATION TOOL, NOT an authoring tool
             PowerPoint creates slideshows; it cannot integrate full media programming
             This distinction appeared in TRE 3.0: "Which is NOT a multimedia authoring tool?"
             Answer: PowerPoint (it's a presentation tool)
```

### Graphics Package:
```
GRAPHICS PACKAGE = Set of LIBRARIES providing 2D GRAPHICS FUNCTIONS programmatically ← PYQ definition
                 = Allows programmers to call functions for drawing shapes, lines, images
Examples: OpenGL, DirectX, SDL, SFML
NOT examples: MS Paint (it's a bitmap editor, not a programming library)

⚠️ EXACT PYQ DEFINITION: "Graphics package = set of libraries providing 2D graphics functions
                          that can be called by applications"
```

## 🔷 BLOCK 3: Computer Generations — The Complete Table

### Overview:
```
5 GENERATIONS of computers, defined by the primary TECHNOLOGY used
Each generation brought: faster speed, smaller size, lower cost, higher capability
```

### Generation-by-Generation Detail:

```
════════════════════════════════════════════════════════════════════════════
GENERATION 1 — VACUUM TUBES (1940s–1950s)
════════════════════════════════════════════════════════════════════════════
Technology:   VACUUM TUBES (thermionic valves) ← the defining component
Size:         Enormous — filled entire rooms
Speed:        Very slow (milliseconds)
Memory:       Magnetic drums
Language:     MACHINE LANGUAGE (binary — 0s and 1s only) ← PYQ
              Also ASSEMBLY LANGUAGE in later 1st gen machines
Programming:  Extremely tedious; single mistake = hours of debugging
Input/Output: Punched cards, paper tape

Famous computers:
  → ENIAC (Electronic Numerical Integrator and Computer, 1945) ← PYQ
    First general-purpose electronic computer
    Built by Mauchly and Eckert (University of Pennsylvania)
    Weighed 30 tons; used 18,000 vacuum tubes
  → UNIVAC I (1951) — first commercial computer
  → IBM 701

Characteristics:
  → Very unreliable (vacuum tubes burned out frequently)
  → Generated enormous heat
  → Consumed huge amounts of electricity
  → Only government and large corporations could afford them
════════════════════════════════════════════════════════════════════════════

GENERATION 2 — TRANSISTORS (1950s–1960s)
════════════════════════════════════════════════════════════════════════════
Technology:   TRANSISTORS ← replaced vacuum tubes
              Transistor: invented at Bell Labs (1947) by Shockley, Bardeen, Brattain
Size:         Much smaller than 1st gen; room-sized to desk-sized
Speed:        Microseconds
Memory:       Magnetic core memory
Language:     ASSEMBLY LANGUAGE primarily ← PYQ
              HIGH-LEVEL languages began: FORTRAN, COBOL, ALGOL
Input/Output: Punched cards → magnetic tape

Famous computers:
  → IBM 1401
  → IBM 7094 ← PYQ (appeared in TRE 2.0)
  → CDC 1604
  → UNIVAC 1107

Characteristics:
  → More reliable than 1st gen
  → Less heat generation
  → Lower cost; more organisations could afford
  → Batch processing operating systems emerged
════════════════════════════════════════════════════════════════════════════

GENERATION 3 — INTEGRATED CIRCUITS (1960s–1970s)
════════════════════════════════════════════════════════════════════════════
Technology:   INTEGRATED CIRCUITS (ICs) ← multiple transistors on one chip
              IC invented by Jack Kilby (Texas Instruments, 1958) ← PYQ
Size:         Much smaller; mini-computers possible
Speed:        Nanoseconds
Memory:       Magnetic core → semiconductor memory
Language:     HIGH-LEVEL LANGUAGES: FORTRAN, COBOL, BASIC, Pascal
Operating System: MULTIPROGRAMMING ← key OS feature of 3rd gen ← PYQ
                  Multiple programs running on same machine
              Time-sharing systems emerged

Famous computers:
  → IBM System/360 (most successful 3rd gen)
  → PDP-8 (mini-computer)
  → CDC 6600

Characteristics:
  → Mini-computers possible (smaller organisations could afford)
  → Multiple users could share one computer (time-sharing)
  → Software industry began growing
  → First operating systems with multi-programming
════════════════════════════════════════════════════════════════════════════

GENERATION 4 — MICROPROCESSORS / VLSI (1970s–present)
════════════════════════════════════════════════════════════════════════════
Technology:   MICROPROCESSORS — entire CPU on a single chip
              VLSI (Very Large Scale Integration) ← PYQ term
              ULSI (Ultra Large Scale Integration) in later 4th gen
Size:         Personal computers (PCs) possible; laptops; tablets
Speed:        Picoseconds
Memory:       RAM, ROM, hard drives, SSDs
Language:     HIGH-LEVEL LANGUAGES: C, C++, Java, Python ← PYQ (C++ is 4th gen)
Operating System: GUI-based OS (Windows, macOS, Linux)
                  Multi-tasking, Multi-user

Famous computers/developments:
  → Intel 4004 (first microprocessor, 1971) ← PYQ
  → Apple II (1977), IBM PC (1981)
  → Personal Computer revolution

Characteristics:
  → Personal computers for everyone
  → Graphical User Interface (GUI)
  → Internet emerged
  → Networks — Local Area Networks (LAN) widely used
  → C++ and Java are 4th generation languages ← PYQ
════════════════════════════════════════════════════════════════════════════

GENERATION 5 — ARTIFICIAL INTELLIGENCE (present–future)
════════════════════════════════════════════════════════════════════════════
Technology:   AI + PARALLEL PROCESSING + NATURAL LANGUAGE PROCESSING ← PYQ
              Ultra-large scale integration (ULSI)
Size:         Laptops, smartphones, wearables
Speed:        Extremely fast
Language:     NATURAL LANGUAGE (humans give instructions in English/Hindi/etc.)
              OBJECT-ORIENTED PROGRAMMING ← PYQ (5th gen = OOP) ← TRE 3.0 PYQ
Key features:
  → ARTIFICIAL INTELLIGENCE ← #1 defining feature ← PYQ
  → NATURAL LANGUAGE PROCESSING (NLP) — computers understand human language
  → Machine Learning and Neural Networks
  → Expert systems (make decisions like human experts)
  → Voice recognition, face recognition

Famous systems:
  → IBM Deep Blue, Watson

## 🔷 SECTION 1: PHYSICS — ALL PYQ-TESTED FACTS

### 1.1 Bernoulli's Principle
```
BERNOULLI'S PRINCIPLE:
  "As the speed of a fluid increases, its pressure DECREASES"
  → Fast-moving fluid = LOW pressure
  → Slow-moving fluid = HIGH pressure

REAL-WORLD APPLICATIONS: ← PYQ asks "which device works on Bernoulli's principle?"
  → SPRAY BOTTLE / ATOMISER ← most tested application
  → PERFUME SPRAY
  → Aircraft wings (lift is created — fast air above, slow below = lower pressure above)
  → CARBURETOR in engines
  → Venturi meter (measuring flow rate)

⚠️ PYQ TRAP: "A spray bottle/atomiser works on which principle?"
             Answer: BERNOULLI'S PRINCIPLE
```

### 1.2 Stefan's Law (Blackbody Radiation)
```
STEFAN'S LAW (Stefan-Boltzmann Law):
  E ∝ T⁴
  = Energy (power) radiated per unit area ∝ FOURTH POWER of absolute temperature

CALCULATION PYQ:
  "If temperature is reduced to T/3, what happens to energy?"
  New E = (T/3)⁴ × old_coefficient
        = T⁴/81 × coefficient
        = E/81

⚠️ PYQ EXACT QUESTION: Temperature reduced to T/3 → energy becomes E/81
  ← This calculation appeared in TRE 2.0!

  Formula to remember: E ∝ T⁴
  If T → T/3: E → E/3⁴ = E/81
  If T → T/2: E → E/2⁴ = E/16
  If T → 2T:  E → 16E
```

### 1.3 Speed of Electron in Hydrogen Atom
```
Speed of electron in ground state (n=1) of hydrogen atom:
  = c/137 (where c = speed of light)
  = approximately 2.19 × 10⁶ m/s
  ← This EXACT fact appeared in TRE paper

The number 137 = Fine structure constant (inverse)
(You don't need to know why — just remember c/137 for hydrogen ground state)
```

### 1.4 Isotropic Light Source — Candela to Lumen
```
ISOTROPIC light source:
  = Light source that radiates equally in ALL directions

Relationship: 1 candela (cd) = 4π lumens (lm) for isotropic source
  Because: A sphere has solid angle = 4π steradians
  So: Luminous flux = Intensity × 4π = 1 × 4π ≈ 12.57 lumens

⚠️ PYQ: "An isotropic light source of 1 candela emits how many lumens?"
         Answer: 4π lumens ≈ 12.57 lumens
```

### 1.5 Light and Optics
```
CONVEX LENS IN WATER:
  → A convex lens placed in water:
    Still acts as a CONVEX LENS (converging) ← PYQ
    But its FOCAL LENGTH INCREASES (less converging power) ← PYQ
  Reason: Refractive index difference between glass and water < glass and air
          Less bending of light → longer focal length

FREQUENCY OF LIGHT CHANGING MEDIUM:
  When light moves from one medium to another:
  → FREQUENCY = UNCHANGED ← PYQ key fact
  → Wavelength changes
  → Speed changes
  → Frequency is the one property that does NOT change

FOG AND LIGHT SCATTERING:
  → Fog makes light OPAQUE because light is SCATTERED by water droplets
  → Scattering makes it difficult to see through fog
  → Blue light scatters MORE than red light (Rayleigh scattering)
    → That's why sky is blue (more blue scattered) and sunsets are red
```

### 1.6 Electricity
```
ELECTRICAL POWER:
  P = I²R = V²/R = VI ← all three forms are equivalent ← PYQ
  Where: P = power (watts), I = current (amperes), V = voltage (volts), R = resistance (ohms)

RESISTANCE IN PARALLEL:
  If n equal resistors of resistance R are connected in parallel:
  Combined resistance = R/n
  
  For two unequal resistors R₁ and R₂ in parallel:
  1/R_total = 1/R₁ + 1/R₂
  R_total = (R₁ × R₂)/(R₁ + R₂)

⚠️ PYQ TRAP: Parallel combination → TOTAL resistance DECREASES (less than smallest individual)
             Series combination   → TOTAL resistance INCREASES (sum of all)
```

### 1.7 Electromagnetic Waves and Sound
```
ELECTROMAGNETIC SPECTRUM (low frequency → high frequency):
  Radio → Microwave → Infrared → Visible → UV → X-ray → Gamma

Speed of light = 3 × 10⁸ m/s (in vacuum)
All EM waves travel at the same speed in vacuum

SOUND:
  → Sound is a LONGITUDINAL wave (compressions and rarefactions)
  → Cannot travel through VACUUM (needs a medium)
  → Speed of sound in air = ~343 m/s at 20°C
  → Speed of sound: solid > liquid > gas
  → Ultrasound = frequency > 20,000 Hz
```

### 2.1 Photoperiodism
```
PHOTOPERIODISM:
  = Response of plants to the RELATIVE LENGTH OF DAY AND NIGHT (photoperiod)
  = Plants flower based on how many hours of light/darkness they receive

DISCOVERED BY: GARNER AND ALLARD ← PYQ key fact
               (W.W. Garner and H.A. Allard, American scientists, 1920)

THREE TYPES OF PLANTS:
  Short-day plants  → flower when day length is SHORT (night > critical length)
                     → Examples: Chrysanthemum, tobacco, poinsettia
  Long-day plants   → flower when day length is LONG (day > critical length)
                     → Examples: Wheat, oat, radish
  Day-neutral plants → not affected by photoperiod
                     → Examples: Tomato, sunflower, cucumber

⚠️ PYQ: "Photoperiodism was discovered by ___?"
         Answer: GARNER AND ALLARD
```

### 2.3 Diaphragm During Respiration
```
NORMAL INHALATION (breathing in):
  Diaphragm → CONTRACTS and FLATTENS ← PYQ
              (moves downward; increases chest cavity volume)
  Lungs expand; air rushes in

NORMAL EXHALATION (breathing out):
  Diaphragm → RELAXES and returns to domed shape
  Lungs compress; air pushed out

⚠️ PYQ: "During normal respiration, what happens to the diaphragm?"
         Answer: It FLATTENS (contracts downward)
         The question is specifically about INHALATION
```

### 2.4 Immunity and Lymphocytes
```
IMMUNITY:
  = Body's ability to resist disease-causing microorganisms

TYPES:
  Innate immunity    → non-specific; present from birth; first line of defense

LYMPHOCYTES = MOST IMPORTANT in immunity ← PYQ

  B-lymphocytes → produce ANTIBODIES (humoral immunity)

### 2.5 Ginger — Underground Stem
```
GINGER (Zingiber officinale):
  = UNDERGROUND STEM (rhizome) ← PYQ
  
Why it is a STEM (not a root):
  → Has NODES and INTERNODES ← key distinguishing feature ← PYQ
  → Nodes = points where buds/leaves arise
  → Roots do NOT have nodes and internodes
  → Ginger has scale leaves at nodes (visible as rings)
  → Grows horizontally underground

Other underground stems (not ginger, but related):
  → Potato = underground stem (tuber) — has "eyes" (buds at nodes)
  → Onion = underground stem (bulb) — has nodes with scale leaves
  → Colocasia (arbi) = corm

⚠️ PYQ: "Ginger is an example of ___?"
         Answer: Underground stem (rhizome) — because it has NODES and INTERNODES
         NOT a root — roots don't have nodes
```

### 2.6 Asexual Reproduction by Budding
```
BUDDING = Type of asexual reproduction where a new organism grows from a BUD on the parent

ORGANISM THAT REPRODUCES BY BUDDING: YEAST ← PYQ
  → Yeast (a fungus) reproduces asexually by budding
  → A small bud grows on the parent yeast cell
  → Bud gets nucleus from parent, grows, and eventually separates

Other organisms that reproduce by budding:
  → Hydra (aquatic animal)
  → Sponges

⚠️ PYQ: "Which organism reproduces by budding?"
         Answer: YEAST (among organisms listed in options)
```

### 2.8 Antacid — Indigestion Medicine
```
ANTACID:
  = Medicine that NEUTRALISES excess stomach acid ← PYQ
  = Used to treat INDIGESTION (acidity, heartburn, acid reflux)
  
How it works:
  Stomach produces HCl (hydrochloric acid)
  Excess acid = indigestion, burning sensation
  Antacid = alkaline substance that neutralises the excess acid
  
Common antacids: Milk of Magnesia (Mg(OH)₂), Baking soda (NaHCO₃), Gelusil, ENO

⚠️ PYQ: "Antacid is a medicine for ___?"
         Answer: INDIGESTION (acidity)
         NOT: Infection (that's antibiotics), NOT: Fever (that's antipyretics)
```

### 2.9 Vas Deferens
