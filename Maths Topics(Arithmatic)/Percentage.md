# PART 1: THE FOUNDATION — WHAT IS PERCENTAGE?

## 1.1 Basic Concept

**Percent** means **"per hundred"** — literally "out of 100".

```
x% = x/100

So:   25% = 25/100 = 1/4
      50% = 50/100 = 1/2
      75% = 75/100 = 3/4
     100% = 100/100 = 1 (the whole thing)
     150% = 150/100 = 3/2 (more than the whole)
```

**Think of it this way:**
If you have ₹100 and spend ₹30, you spent **30%** of your money.
If marks are 45 out of 60, percentage = (45/60) × 100 = **75%**

---

## 1.2 The 3 Core Formulas — Memorize These First

### Formula 1: Finding Percentage
```
Percentage = (Part / Whole) × 100
```
**Example:** 36 out of 80 = (36/80) × 100 = 45%

---

### Formula 2: Finding the Part (Value)
```
Part = (Percentage × Whole) / 100
```
**Example:** 35% of 200 = (35 × 200) / 100 = 70

---

### Formula 3: Finding the Whole (Original)
```
Whole = (Part × 100) / Percentage
```
**Example:** 40 is 25% of what number?
Whole = (40 × 100) / 25 = **160**

# PART 3: PERCENTAGE INCREASE & DECREASE

## 3.1 Percentage Increase

```
% Increase = [(New Value - Original Value) / Original Value] × 100
```

**Example 1:** Salary increased from ₹8,000 to ₹10,000.
```
% Increase = [(10,000 - 8,000) / 8,000] × 100
           = [2,000 / 8,000] × 100
           = 0.25 × 100
           = 25%
```

**Example 2:** Population grew from 40,000 to 46,000.
```
% Increase = [(46,000 - 40,000) / 40,000] × 100
           = [6,000 / 40,000] × 100
           = 15%
```

---

## 3.2 Percentage Decrease

```
% Decrease = [(Original Value - New Value) / Original Value] × 100
```

**Example:** Price reduced from ₹500 to ₹400.
```
% Decrease = [(500 - 400) / 500] × 100
           = [100 / 500] × 100
           = 20%
```

---

## 3.3 ⚠️ CRITICAL TRAP: Base Always Matters!

> The percentage is ALWAYS calculated on the **ORIGINAL (base) value**, NOT the new value.

**Wrong thinking:** Price increased from 100 to 120. Some students think "decrease from 120 to 100 is also 20%." **WRONG!**

```
Increase from 100 to 120:
% increase = (20/100) × 100 = 20%   ← Base is 100 ✅

Decrease from 120 to 100:
% decrease = (20/120) × 100 = 16.67%  ← Base is 120 ✅
```

**They are DIFFERENT because the base changed!**

# PART 4: THE MULTIPLIER METHOD — Fastest Technique

## 4.1 What is the Multiplier?

Instead of calculating step by step, use a **single multiplier**:

```
% Increase by x%  →  Multiply by (1 + x/100)
% Decrease by x%  →  Multiply by (1 - x/100)
```

### Examples:

| Operation | Multiplier | Example |
|-----------|-----------|---------|
| Increase by 10% | × 1.10 | 200 → 200 × 1.10 = 220 |
| Increase by 25% | × 1.25 | 200 → 200 × 1.25 = 250 |
| Decrease by 20% | × 0.80 | 200 → 200 × 0.80 = 160 |
| Decrease by 15% | × 0.85 | 200 → 200 × 0.85 = 170 |
| Increase by 100% | × 2.00 | 200 → 200 × 2 = 400 |

---

## 4.2 Successive Percentage Changes

When two percentage changes happen one after another:

**WRONG method:** Add the percentages
**CORRECT method:** Multiply the multipliers

### Formula for Net Effect of Two Changes (a% then b%):
```
Net % Change = a + b + (a × b)/100
```

**Example:** Price increased by 20%, then decreased by 20%.
```
Net change = 20 + (-20) + (20 × -20)/100
           = 0 + (-400/100)
           = 0 - 4
           = -4%  (4% DECREASE overall!)
```

**Why?** Because 20% is calculated on different bases each time.

**Multiplier method (even faster):**
```
1.20 × 0.80 = 0.96 → 4% decrease ✅
```

---

## 4.3 Shortcut for Equal but Opposite Changes

> **If a value increases by x% and then decreases by x%, the net result is always a DECREASE of (x²/100)%**

| Change | Net Result |
|--------|-----------|
| +10% then -10% | -1% (decrease) |
| +20% then -20% | -4% (decrease) |
| +25% then -25% | -6.25% (decrease) |
| +50% then -50% | -25% (decrease) |

**Memory trick:** Equal + and - never cancel out — there's always a net LOSS.

# PART 5: "X IS WHAT % OF Y" TYPE QUESTIONS

## Standard Format:
```
X is what % of Y?
Answer = (X/Y) × 100
```

**Example 1:** 15 is what % of 75?
```
(15/75) × 100 = (1/5) × 100 = 20%
```

**Example 2:** 60 is what % of 40?
```
(60/40) × 100 = 1.5 × 100 = 150%
```
*(This is more than 100% — perfectly valid when X > Y!)*

## Reverse Format:
```
X% of Y = Z, find Y
Y = (Z × 100) / X
```

**Example:** 15% of a number is 45. Find the number.
```
Number = (45 × 100) / 15 = 300
```

# PART 6: PERCENTAGE POINT vs PERCENTAGE CHANGE

> This is a HIGH-LEVEL concept that separates toppers from average students.

**Percentage Point** = Simple arithmetic difference between two percentages
**Percentage Change** = Change calculated on the base percentage

**Example:**
- Pass rate in 2022 = 40%
- Pass rate in 2023 = 50%

```
Percentage Point increase = 50 - 40 = 10 percentage points

Percentage Change = [(50 - 40) / 40] × 100 = 25%
```

They are DIFFERENT numbers (10 points ≠ 25%)!

# PART 7: POPULATION & EXAM-TYPE PROBLEMS

## 7.1 Population Problems

**Formula:** If population is P and increases by r% per year:
```
Population after n years = P × (1 + r/100)ⁿ
```

**Example:** A city has population 10,000. It grows at 10% per year. Population after 2 years?
```
= 10,000 × (1.10)²
= 10,000 × 1.21
= 12,100
```

---

## 7.2 Marks/Exam Problems

**Type 1:** Passing marks problem
```
A student needs X% to pass. He gets Y marks and fails by Z marks.
Passing marks = Y + Z
Total marks = (Y + Z) × 100 / X
```

**Example:** A student gets 40 marks and fails by 10. Passing is 25%. Find total marks.
```
Passing marks = 40 + 10 = 50
Total marks = (50 × 100) / 25 = 200
```

**Type 2:** Two students marks comparison
```
A gets X%, B gets Y%. A gets Z more marks than B.
Total marks = Z × 100 / (X - Y)
```

# PART 8: INCOME & EXPENDITURE PROBLEMS

## The Classic "Income-Expenditure" Template

> **Key Formula:**
> If income increases by a% and expenditure increases by b%, then:
> ```
> % change in savings = [(a × I - b × E) / S] × 100
> where S = I - E (savings = income - expenditure)
> ```

**Example (Classic BPSC type):**
Income = ₹5000, Expenditure = ₹4000, Savings = ₹1000
Income increases by 20%, Expenditure increases by 25%.

```
New Income = 5000 × 1.20 = 6000
New Expenditure = 4000 × 1.25 = 5000
New Savings = 6000 - 5000 = 1000
% change in savings = [(1000 - 1000) / 1000] × 100 = 0%
```
Savings unchanged!

# PART 9: 10 POWERFUL SHORTCUT TRICKS

## ⚡ TRICK 1: The "ab/100" Shortcut for Successive %

If a thing changes by a% and then b%:
```
Net % = a + b + ab/100
```
Remember: Use negative for decrease.

---

## ⚡ TRICK 2: Finding 10% Instantly

To find 10% of any number → just move the decimal one place left.
```
10% of 450 = 45.0
10% of 3700 = 370
10% of 25 = 2.5
```
Then build other percentages from this:
- 5% = half of 10%
- 20% = double of 10%
- 15% = 10% + 5%
- 30% = 3 × 10%

---

## ⚡ TRICK 3: The x% of y = y% of x Rule

```
x% of y = y% of x
```

**Example:** Find 16% of 25.
→ Instead, find 25% of 16 = 16/4 = **4** ✅ (Much easier!)

**Example:** Find 8% of 75.
→ Instead, find 75% of 8 = 6 ✅

---

## ⚡ TRICK 4: Percentage by Parts

Break awkward percentages into easy parts:
```
35% = 30% + 5%
65% = 50% + 15%
48% = 50% - 2%
```

**Example:** 35% of 240:
```
30% of 240 = 72
5% of 240 = 12
35% of 240 = 72 + 12 = 84 ✅
```

---

## ⚡ TRICK 5: If A is x% more than B, then B is less than A by?

```
If A is x% more than B:
B is less than A by = [x / (100 + x)] × 100 %
```

**Example:** A's salary is 25% more than B's. By what % is B's salary less than A's?
```
= [25 / (100 + 25)] × 100
= [25/125] × 100
= 20%
```

---

## ⚡ TRICK 6: If A is x% less than B, then B is more than A by?

```
If A is x% less than B:
B is more than A by = [x / (100 - x)] × 100 %
```

**Example:** A earns 20% less than B. By what % does B earn more than A?
```
= [20 / (100 - 20)] × 100
= [20/80] × 100
= 25%
```

---

## ⚡ TRICK 7: Quick Calculation Using Fractions

| Need to find | Use fraction | Example |
|--------------|-------------|---------|
| 12.5% | 1/8 | 12.5% of 160 = 160/8 = 20 |
| 16.67% | 1/6 | 16.67% of 300 = 300/6 = 50 |
| 37.5% | 3/8 | 37.5% of 80 = (3×80)/8 = 30 |
| 62.5% | 5/8 | 62.5% of 80 = (5×80)/8 = 50 |
| 87.5% | 7/8 | 87.5% of 80 = (7×80)/8 = 70 |

---

## ⚡ TRICK 8: Percentage to Find Original Before Increase/Decrease

```
If after x% increase, value = V, then original = V × 100/(100+x)
If after x% decrease, value = V, then original = V × 100/(100-x)
```

**Example:** After 20% increase, price is ₹600. Original price?
```
Original = 600 × 100/120 = 600 × 5/6 = ₹500
```

---

## ⚡ TRICK 9: The "Equal Fractions" Trick

To check if two fractions are equal percent of different numbers:
```
a is p% of b → b is (100/p)% of a
```

**Example:** 30 is 60% of 50. Then 50 is what % of 30?
```
= (100/60) × 100 = 166.67%
```

---

## ⚡ TRICK 10: Percentage in Mixtures/Alloys

If a container has A litres of acid and B litres of water:
```
% of acid = A/(A+B) × 100
% of water = B/(A+B) × 100
```

**Example:** 30 litres acid + 20 litres water. % of acid?
```
= 30/50 × 100 = 60%
```

# PART 10: SOLVED EXAMPLES — ALL QUESTION TYPES

## Type 1: Basic Percentage
**Q:** A class has 40 students. 30% are girls. How many boys?
```
Girls = 30% of 40 = 12
Boys = 40 - 12 = 28
```

## Type 2: Finding Original Number
**Q:** 120% of a number is 96. Find the number.
```
Number = (96 × 100) / 120 = 80
```

## Type 3: Comparison
**Q:** Which is greater: 5/8 or 62%?
```
5/8 = 62.5%
62.5% > 62% → 5/8 is greater
```

## Type 4: Two Successive Changes
**Q:** A number is first increased by 10% then decreased by 10%. Net change?
```
Net = 10 + (-10) + (10 × -10)/100 = 0 - 1 = -1%
→ 1% decrease
```

## Type 5: Reverse Calculation
**Q:** A shopkeeper marks up price by 25% and then gives 20% discount. What is overall % change?
```
Multiplier = 1.25 × 0.80 = 1.00
→ 0% change (no profit, no loss)
```

## Type 6: Comparison Between Two Values
**Q:** Ravi earns ₹15,000 and spends 60% of it. Priya earns ₹18,000 and spends ₹9,000. Who saves more?
```
Ravi saves: 40% of 15,000 = 6,000
Priya saves: 18,000 - 9,000 = 9,000
Priya saves more.
```

## Type 7: Percentage of Percentage
**Q:** 30% of 40% of 500 = ?
```
= 30/100 × 40/100 × 500
= 0.30 × 0.40 × 500
= 0.12 × 500
= 60
```

## Type 8: Classic Exam/Marks Problem
**Q:** A student scores 75% in an exam. If he scored 45 more marks, he would get 90%. Total marks?
```
Let total = T
75% of T + 45 = 90% of T
45 = 15% of T
T = 45 × 100/15 = 300
```

# PART 11: COMMON MISTAKES TO AVOID

| ❌ Wrong Approach | ✅ Correct Approach |
|-------------------|---------------------|
| 20% increase + 20% decrease = 0% change | Net = -4% (always a loss) |
| 10% of 200 = 20, and 10% of 300 = 30, so 10% of 500 = 50 | This one is actually correct ✅ |
| If A is 20% more than B, then B is 20% less than A | B is 16.67% less than A |
| Base changes in successive % | Always apply each % to the NEW value |
| 150% of a number means 1.5 times | ✅ This is correct |

# QUICK FORMULA REFERENCE CARD

```
┌─────────────────────────────────────────────────────────┐
│              PERCENTAGE FORMULA CARD                    │
├─────────────────────────────────────────────────────────┤
│ % = (Part/Whole) × 100                                  │
│ Part = (% × Whole) / 100                                │
│ Whole = (Part × 100) / %                                │
├─────────────────────────────────────────────────────────┤
│ % increase = [(New-Old)/Old] × 100                      │
│ % decrease = [(Old-New)/Old] × 100                      │
├─────────────────────────────────────────────────────────┤
│ Successive changes a% and b%:                           │
│ Net = a + b + ab/100                                    │
├─────────────────────────────────────────────────────────┤
│ x% more than y → y is [x/(100+x) × 100]% less than x   │
│ x% less than y → y is [x/(100-x) × 100]% more than x   │
├─────────────────────────────────────────────────────────┤
│ After r% increase, original = New × 100/(100+r)         │
│ After r% decrease, original = New × 100/(100-r)         │
├─────────────────────────────────────────────────────────┤
│ KEY: x% of y = y% of x  (Use this to simplify!)        │
└─────────────────────────────────────────────────────────┘
```
