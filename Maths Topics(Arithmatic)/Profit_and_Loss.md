## 1. 🔑 Key Terms & Definitions {#key-terms}

| Term | Meaning | Also Called |
|------|---------|-------------|
| **Cost Price (CP)** | Price at which an item is bought | Purchase Price |
| **Selling Price (SP)** | Price at which an item is sold | Sale Price |
| **Profit** | When SP > CP → Gain | Gain |
| **Loss** | When SP < CP → Loss | — |
| **Marked Price (MP)** | Price printed on the item / tag | List Price, MRP |
| **Discount** | Reduction given on Marked Price | Rebate |
| **Net Price** | Price after discount | SP after discount |
| **Overhead** | Extra costs (transport, repair, etc.) | Additional cost |

> 💡 **Golden Rule:**
> - If SP > CP → **PROFIT**
> - If SP < CP → **LOSS**
> - If SP = CP → **No Profit, No Loss**

## 2. 📐 Core Formulas {#core-formulas}

### 2.1 Profit & Loss

```
Profit       = SP − CP
Loss         = CP − SP

Profit %     = (Profit / CP) × 100
Loss %       = (Loss / CP) × 100
```

> ⚠️ **Always remember:** Profit% and Loss% are **ALWAYS calculated on CP** (unless stated otherwise).

### 2.2 Finding SP from CP

```
SP = CP × (100 + Profit%) / 100      ← when profit is given
SP = CP × (100 − Loss%) / 100        ← when loss is given
```

### 2.3 Finding CP from SP

```
CP = SP × 100 / (100 + Profit%)      ← when profit is given
CP = SP × 100 / (100 − Loss%)        ← when loss is given
```

### 2.4 Discount

```
Discount     = MP − SP
Discount %   = (Discount / MP) × 100
SP           = MP × (100 − Discount%) / 100
```

## 3. 📗 Basic Concepts with Examples {#basic-concepts}

---

### ✅ Concept 1: Simple Profit Calculation

**Example 1:**
A shopkeeper buys a book for ₹200 and sells it for ₹250. Find Profit and Profit%.

```
Profit    = SP − CP = 250 − 200 = ₹50
Profit%   = (50 / 200) × 100 = 25%
```
✔ Answer: Profit = ₹50, Profit% = **25%**

---

### ✅ Concept 2: Simple Loss Calculation

**Example 2:**
A person buys a phone for ₹8,000 and sells it for ₹6,500. Find Loss and Loss%.

```
Loss    = CP − SP = 8000 − 6500 = ₹1500
Loss%   = (1500 / 8000) × 100 = 18.75%
```
✔ Answer: Loss = ₹1500, Loss% = **18.75%**

---

### ✅ Concept 3: Finding SP when Profit% is given

**Example 3:**
CP = ₹500, Profit = 20%. Find SP.

```
SP = CP × (100 + 20) / 100
SP = 500 × 120 / 100 = ₹600
```
✔ Answer: SP = **₹600**

---

### ✅ Concept 4: Finding SP when Loss% is given

**Example 4:**
CP = ₹1,200, Loss = 15%. Find SP.

```
SP = CP × (100 − 15) / 100
SP = 1200 × 85 / 100 = ₹1020
```
✔ Answer: SP = **₹1020**

---

### ✅ Concept 5: Finding CP when SP and Profit% are given

**Example 5:**
SP = ₹660, Profit = 10%. Find CP.

```
CP = SP × 100 / (100 + Profit%)
CP = 660 × 100 / 110 = ₹600
```
✔ Answer: CP = **₹600**

---

### ✅ Concept 6: Finding CP when SP and Loss% are given

**Example 6:**
SP = ₹850, Loss = 15%. Find CP.

```
CP = SP × 100 / (100 − Loss%)
CP = 850 × 100 / 85 = ₹1000
```
✔ Answer: CP = **₹1000**

## 4. 📘 Intermediate Concepts {#intermediate-concepts}

---

### ✅ Concept 7: Discount & Marked Price

**Example 7:**
MP of a shirt = ₹800. A discount of 25% is given. Find SP and Discount amount.

```
Discount     = 25% of 800 = ₹200
SP           = 800 − 200 = ₹600

OR using formula:
SP = MP × (100 − Discount%) / 100
SP = 800 × 75 / 100 = ₹600
```
✔ Answer: SP = **₹600**, Discount = **₹200**

---

### ✅ Concept 8: Profit% on SP (Tricky — asked in exams!)

> Some problems ask profit as % of SP instead of CP.

**Example 8:**
A man sells an item at 20% profit on SP. If SP = ₹500, find actual profit.

```
Profit = 20% of SP = 20% of 500 = ₹100
CP = SP − Profit = 500 − 100 = ₹400

Actual Profit% on CP = (100/400) × 100 = 25%
```
✔ Answer: Actual Profit on CP = **25%** (though profit on SP was 20%)

> 💡 **Key insight:** Profit% on SP is ALWAYS less than Profit% on CP.

---

### ✅ Concept 9: Overhead Costs (Added to CP)

**Example 9:**
A merchant buys a TV for ₹10,000. He pays ₹500 for transport and ₹300 for repairs. He sells it for ₹12,000. Find Profit%.

```
Total CP = 10000 + 500 + 300 = ₹10,800
Profit   = 12000 − 10800 = ₹1200
Profit%  = (1200 / 10800) × 100 = 11.11%
```
✔ Answer: Profit% = **11.11%**

---

### ✅ Concept 10: Selling Multiple Items

**Example 10:**
A person sells 2 items at ₹1,200 each. On the first item he gains 20%, on the second he loses 20%. Find overall profit or loss.

```
CP of item 1 = 1200 × 100 / 120 = ₹1000
CP of item 2 = 1200 × 100 / 80  = ₹1500

Total CP = 1000 + 1500 = ₹2500
Total SP = 1200 + 1200 = ₹2400

Loss = 2500 − 2400 = ₹100
Loss% = (100 / 2500) × 100 = 4%
```
✔ Answer: **Loss of 4%** (There is ALWAYS a loss in this type!)

> 🔥 **Magic Trick for this type:**
> When same SP, same %gain and %loss → **Always a LOSS**
> Loss% = (Common%)² / 100
> Loss% = 20² / 100 = 400/100 = **4%** ✅

---

### ✅ Concept 11: Dishonest Shopkeeper (False Weight)

**Example 11:**
A shopkeeper cheats by using a weight of 900g instead of 1kg. He also sells at 10% profit on CP. Find actual profit%.

```
Let CP of 1000g = ₹1000 (assume)
He sells 900g but charges for 1000g.
So his effective CP for 900g = 900

He sells at 10% profit:
SP = 900 × 110/100 = ₹990? 

Wait — let's use the standard approach:

Profit% = [(True weight − False weight) / False weight × 100] + Profit on CP%
         + (Profit% × (True − False)/False)

Easier formula:
Actual Profit% = [(100 + Profit%) × True Wt / False Wt] − 100

= [(110) × 1000 / 900] − 100
= 1100/9 − 100
= 122.22 − 100
= 22.22%
```
✔ Answer: Actual Profit = **22.22%**

---

### ✅ Concept 12: Profit/Loss on Cost with Given Ratio

**Example 12:**
If CP : SP = 4 : 5, find Profit%.

```
CP = 4, SP = 5
Profit = 5 − 4 = 1
Profit% = (1/4) × 100 = 25%
```
✔ Answer: **25% profit**

## 5. 📕 Advanced Concepts {#advanced-concepts}

---

### ✅ Concept 13: Successive Discounts

**Example 13:**
Two successive discounts of 20% and 10% are given on MP = ₹1,000. Find SP.

```
After 1st discount: SP = 1000 × 80/100 = ₹800
After 2nd discount: SP = 800 × 90/100 = ₹720

OR use formula:
Net Discount = d1 + d2 − (d1 × d2)/100
             = 20 + 10 − (20×10)/100
             = 30 − 2 = 28%

SP = 1000 × (100−28)/100 = 1000 × 72/100 = ₹720
```
✔ Answer: SP = **₹720** | Net Discount = **28%** (not 30%!)

> 💡 Successive discounts are NEVER simply added. Use the formula!

---

### ✅ Concept 14: MP and CP Relationship with Profit and Discount

**Example 14:**
A shopkeeper marks goods 40% above CP and gives a 20% discount. Find Profit%.

```
Let CP = 100
MP = 140
Discount = 20% of 140 = 28
SP = 140 − 28 = 112

Profit = SP − CP = 112 − 100 = 12
Profit% = 12%
```
✔ Answer: **12% Profit**

> 🔥 **Formula shortcut:**
> Profit% = (1 + Mark-up%/100)(1 − Discount%/100) × 100 − 100
> = (1.4)(0.8) × 100 − 100 = 112 − 100 = **12%** ✅

---

### ✅ Concept 15: Find Mark-up % to earn Profit after Discount

**Example 15:**
A trader wants to earn 25% profit after giving 20% discount. By what % should he mark above CP?

```
Let CP = 100
SP = 125 (25% profit)
Discount = 20%, so SP = MP × 80/100

125 = MP × 80/100
MP  = 125 × 100/80 = 156.25

Mark-up % = (156.25 − 100)/100 × 100 = 56.25%
```
✔ Answer: Mark up by **56.25%**

> 🔥 **General Formula:**
> Mark-up% = [(100 + Profit%) / (100 − Discount%) − 1] × 100
> = [125/80 − 1] × 100 = [0.5625] × 100 = **56.25%** ✅

---

### ✅ Concept 16: When Articles are Bought and Sold with Different Counts

**Example 16:**
A person buys 10 oranges for ₹10 and sells 8 oranges for ₹10. Find Profit%.

```
CP of 1 orange = 10/10 = ₹1
SP of 1 orange = 10/8 = ₹1.25

Profit% = (1.25 − 1)/1 × 100 = 25%
```
✔ Answer: **25% Profit**

---

### ✅ Concept 17: Selling at Loss to Cover Previous Profit

**Example 17:**
A man sold an article for ₹800 gaining 25%. He then repurchased the same article and sold it at a loss of 20%. Find overall P/L.

```
Step 1: CP1 = 800 × 100/125 = ₹640
         Profit1 = 800 − 640 = ₹160

Step 2: CP2 = 800 (he bought again at 800)
         SP2 = 800 × 80/100 = ₹640
         Loss2 = 800 − 640 = ₹160

Overall: Net = Profit1 − Loss2 = 160 − 160 = 0
```
✔ Answer: **No Profit, No Loss**

---

### ✅ Concept 18: Profit% and Loss% in terms of SP

**Formula when % is given on SP (not CP):**

```
If Profit is x% of SP:
   Profit% on CP = 100x / (100 − x)

If Loss is x% of SP:
   Loss% on CP = 100x / (100 + x)
```

**Example 18:**
Profit is 25% of SP. What is profit% on CP?

```
Profit% on CP = 100 × 25 / (100 − 25) = 2500/75 = 33.33%
```
✔ Answer: **33.33%** profit on CP

---

### ✅ Concept 19: Equal Profit and Loss % on Different Items

**Example 19:**
A dealer sells two items, one at 30% profit and other at 30% loss. If CP of both items is same (₹500 each), find total P/L.

```
CP1 = 500, SP1 = 500 × 130/100 = ₹650
CP2 = 500, SP2 = 500 × 70/100  = ₹350

Total CP = 1000
Total SP = 1000

Result: NO PROFIT, NO LOSS
```
✔ When CP is same and %profit = %loss → Always **No Profit No Loss**

But when **SP is same** → Always a **Loss** (Concept 10 above)

---

### ✅ Concept 20: CP of Multiple Units

**Example 20:**
A person buys 12 pens for ₹120 and sells 10 pens for ₹120. Find Profit%.

```
CP per pen = 120/12 = ₹10
SP per pen = 120/10 = ₹12

Profit% = (12 − 10)/10 × 100 = 20%
```
✔ Answer: **20% Profit**

## 6. ⚡ Short Tricks & Mental Math {#short-tricks}

---

### 🔥 Trick 1: Same SP, Same %age → Loss Formula
> Two items sold at same SP, one at x% profit, other at x% loss.
```
Loss% = x² / 100
```
Example: x = 20% → Loss% = 400/100 = **4%**

---

### 🔥 Trick 2: Quick Profit% from CP:SP ratio
> If CP : SP = a : b
```
Profit% = (b − a)/a × 100
Loss%   = (a − b)/a × 100
```
Example: CP:SP = 5:6 → Profit% = 1/5 × 100 = **20%**

---

### 🔥 Trick 3: Successive Discount Net Formula
> Two discounts d1% and d2%:
```
Net Discount% = d1 + d2 − (d1 × d2)/100
```
Example: 30% then 20% → 30 + 20 − 6 = **44%** net discount

---

### 🔥 Trick 4: Mark-up Formula
> To earn P% profit after D% discount:
```
Mark-up% = [(100 + P) / (100 − D) − 1] × 100
```

---

### 🔥 Trick 5: Profit on SP Conversion to Profit on CP
> If profit = x% on SP:
```
Profit% on CP = 100x / (100 − x)
```
Example: 20% on SP → 100×20/80 = **25%** on CP

---

### 🔥 Trick 6: CP from SP (one-line)
> Profit given: CP = SP × 100 / (100 + P%)
> Loss given:   CP = SP × 100 / (100 − L%)

---

### 🔥 Trick 7: False Weight Profit
> Shopkeeper cheats by selling x grams as 1000g, and also claims P% profit on CP:
```
Actual Profit% = [(100 + P%) × 1000 / x] − 100
```
Example: sells 800g as 1000g, claims 10% profit:
= [110 × 1000/800] − 100 = 137.5 − 100 = **37.5%**

---

### 🔥 Trick 8: Buying and Selling Different Quantities
> If person buys m items for ₹x and sells n items for ₹x:
```
Profit/Loss% = [(m − n)/n] × 100
Profit if m > n, Loss if m < n
```
Example: Buys 12 for ₹10, sells 8 for ₹10
= [(12 − 8)/8] × 100 = **50% Profit**

---

### 🔥 Trick 9: SP when Profit% on MP is given
> If profit x% on MP:
```
Profit on CP = [MP(100 + x%) / CP] − 100
```
(just substitute MP and CP values)

---

### 🔥 Trick 10: Multiplier Method (Fastest!)
| % Change | Multiplier |
|----------|-----------|
| 10% gain | × 1.10 |
| 20% gain | × 1.20 |
| 25% gain | × 1.25 |
| 10% loss | × 0.90 |
| 20% loss | × 0.80 |
| 15% discount | × 0.85 |

> Example: CP = 400, 25% gain
> SP = 400 × 1.25 = **₹500** (no formula needed!)

## 7. ⚠️ Common Exam Traps {#exam-traps}

| Trap | What looks right | What IS right |
|------|-----------------|---------------|
| Profit% base | Some say "profit on SP" | **Always CP unless stated** |
| Same SP rule | Thinks profit and loss cancel | **Always results in LOSS** |
| Successive discounts | Adds 20+30 = 50% | **Net = 44%, not 50%** |
| Overhead costs | Forgets transport/repair | **Add all to CP first** |
| CP:SP ratio | Confusion on which is which | **Profit% = (SP−CP)/CP × 100** |
| False weight | Only counts weight cheat | **May also have stated profit—add both effects** |

## 10. 📋 Quick Formula Cheat Sheet {#cheat-sheet}

```
┌─────────────────────────────────────────────────────────────────┐
│                     PROFIT & LOSS CHEAT SHEET                   │
├─────────────────────────────────────────────────────────────────┤
│  Profit = SP − CP                                               │
│  Loss   = CP − SP                                               │
│                                                                 │
│  Profit% = (Profit/CP) × 100  [ALWAYS on CP]                   │
│  Loss%   = (Loss/CP)   × 100  [ALWAYS on CP]                   │
│                                                                 │
│  SP = CP × (100 ± %)  / 100                                     │
│  CP = SP × 100 / (100 ± %)                                      │
│                                                                 │
│  Discount = MP − SP                                             │
│  Discount% = (Discount/MP) × 100  [ALWAYS on MP]               │
│  SP = MP × (100 − D%) / 100                                     │
│                                                                 │
│  SAME SP, SAME %   →  Loss% = x²/100                           │
│  Successive D = d1+d2 − (d1×d2)/100                            │
│  False Weight Profit = [(100+P%) × 1000/False] − 100           │
│  Markup % = [(100+P)/(100−D) − 1] × 100                        │
│  Profit on SP → CP: = 100x/(100−x)                             │
└─────────────────────────────────────────────────────────────────┘
```

## 🎯 Final Tips for Exam Day

1. **Always identify CP first** — every % is calculated on CP (unless MP/SP is explicitly stated).
2. **Write out what's given** before jumping to formulas.
3. **Assume CP = 100** for percentage problems — it simplifies mental math dramatically.
4. **Watch for overhead costs** — always add them to CP.
5. **Same SP + same % = always loss** — don't second-guess this.
6. **Successive discounts ≠ simple addition** — always use the formula.
7. **Profit% on SP < Profit% on CP** — always.
8. **In ratio problems** (CP:SP = a:b), profit% = (b−a)/a × 100.
